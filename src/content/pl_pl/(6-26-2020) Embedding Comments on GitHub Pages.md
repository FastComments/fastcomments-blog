# [postlink]Osadzanie komentarzy na stronach GitHub[/postlink]

{{#unless isPost}}
Chcesz dodać komentarze do statycznej strony, takiej jak GitHub Pages?
{{/unless}}

{{#isPost}}
#### Dla programistów
Zainstalowanie FastComments na stronie internetowej wymaga tylko wstawienia małego fragmentu kodu. Oznacza to, że działa na stronach statycznych bez żadnych problemów - w tym na bezpłatnym hostingu stron GitHub.

#### Demo

Ustawiliśmy przykładową stronę GitHub <a href="https://fastcomments.github.io/fastcomments-demo.github.io/" target="_blank">tutaj</a>.

<a href="https://github.com/fastcomments/fastcomments-demo.github.io/blob/master/index.html#L29" target="_blank">Tutaj</a> jest link do miejsca, gdzie dodaliśmy fragment kodu dla naszego
konta testowego.

#### Konfiguracja

Będziesz musiał dodać swoją domenę GitHub jako ważną domenę w swoim koncie.
