### <i class="circle">!</i> Эта статья содержит техническую терминологию

### Что нового

FastComments теперь имеет компоненты React Native, которые соответствуют функционалу нашего виджета VanillaJS.

Мы решили назвать библиотеку `fastcomments-react-native`. Вы можете найти репозиторий GitHub с исходным кодом <a href="https://github.com/FastComments/fastcomments-react-native" target="_blank">здесь</a>.

В репозитории также содержится пример использования виджета.

Он доступен на NPM, который вы можете найти <a href="https://www.npmjs.com/package/fastcomments-react-native" target="_blank">здесь</a>.

### Как это работает

Новый компонент работает так, что React-компонент является оберткой вокруг существующего виджета FastComments.
 
Это означает, что если мы добавим функции в наши существующие компоненты или исправим ошибки, вы все равно сможете сразу получить от этого пользу!

### Время разработки

Виджеты React Native можно добавить в ваше приложение всего за несколько минут с минимальным временем разработки.

### SSO

Все библиотеки React Native поддерживают SSO прямо из коробки, и процесс интеграции SSO такой же, как и для компонентов Vanilla JS. Существующие примеры кода на стороне сервера тоже применимы.

### Настройка стиля?

Компоненты React Native можно стилизовать с помощью CSS через интерфейс настройки виджетов, так же, как и веб-виджеты.
