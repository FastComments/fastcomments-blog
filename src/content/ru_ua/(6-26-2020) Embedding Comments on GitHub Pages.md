{{#unless isPost}}
Хотите добавить комментарии на статичный сайт, такой как GitHub Pages?
{{/unless}}

#### Для разработчиков
Установка FastComments на веб-странице требует только вставки небольшого фрагмента кода. Это означает, что он работает на статичных страницах без лишних хлопот - включая бесплатный хостинг GitHub Pages.

#### Демонстрация

Мы подготовили пример GitHub Page <a href="https://fastcomments.github.io/fastcomments-demo.github.io/" target="_blank">здесь</a>.

<a href="https://github.com/fastcomments/fastcomments-demo.github.io/blob/master/index.html#L29" target="_blank">Вот</a> ссылка на место, где мы добавили фрагмент кода для нашего
тестового аккаунта.

#### Настройка

Вам нужно будет добавить ваш домен GitHub как действительный домен в вашем аккаунте.
