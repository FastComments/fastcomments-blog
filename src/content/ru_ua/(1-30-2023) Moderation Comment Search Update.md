# [postlink]Обновление поиска комментариев для модерации[/postlink]

{{#unless isPost}}
FastComments теперь позволяет вам искать по страницам и пользователям на странице модерации комментариев.
{{/unless}}

{{#isPost}}

### Что нового

Ранее FastComments позволял вам искать по фразам или точным соответствиям строк в странице модерации комментариев.

Теперь вы можете искать комментарии следующими способами:

- Неясный поиск слов: **кошки любят**
- Точное совпадение фразы: **"Я люблю кошек."**
- **_Новинка!_** По заголовку страницы: **page:"Заголовок страницы"**
  - Поддерживает автозаполнение.
  - Показывает количество комментариев для каждой страницы в списке предложений автозаполнения.
- **_Новинка!_** По URL страницы: **page:"https://example.com/some-page"**
  - Поддерживает автозаполнение.
  - Показывает количество комментариев для каждой страницы в списке предложений автозаполнения.
- **_Новинка!_** По пользователю: **user:"Боб"**
  - Поддерживает автозаполнение.

Вы можете делиться результатами поиска с другими модераторами или администраторами, поделившись URL страницы с страницы модерации. Значение поля поиска будет включено в URL вашего браузера после нажатия на "Перейти".
