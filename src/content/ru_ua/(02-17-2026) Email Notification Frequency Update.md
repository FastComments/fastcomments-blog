# [postlink]Обновление Частоты Уведомлений по Электронной Почте[/postlink]

{{#unless isPost}}
Пользователи теперь могут контролировать, как часто они получают уведомления по электронной почте о новых комментариях и ответах: каждую минуту, почасовой дайджест или ежедневный дайджест, с отдельными настройками для уведомлений об ответах и администраторов, а также индивидуальными настройками для каждой подписки.
{{/unless}}

{{#isPost}}

### Что нового

Мы добавили настройки **Частота Уведомлений об Ответах** и **Частота Уведомлений для Администраторов**, которые контролируют, как часто вы получаете уведомления по электронной почте о ответах и новых комментариях соответственно. Три варианта для каждой из них:

- **Каждую минуту** - получать уведомление по электронной почте, как только поступят новые комментарии (проверяется каждую минуту).
- **Почасовой дайджест** - получать сводку новых комментариев раз в час.
- **Ежедневный дайджест** - получать сводку новых комментариев раз в день.

Частота Уведомлений об Ответах доступна всем пользователям и по умолчанию установлена на **Каждую минуту**. Частота Уведомлений для Администраторов доступна администраторам сайта и по умолчанию установлена на **Почасовой дайджест**. Обратите внимание, что уведомления @mention всегда отправляются немедленно, независимо от этих настроек.

Вы также можете изменить частоту для каждой подписки в таблице Подписок, чтобы иметь более точный контроль над отдельными страницами.

### Как Настроить

1. Перейдите в ваши [Настройки Уведомлений](https://fastcomments.com/auth/my-account/edit-notifications).
2. Используйте выпадающие списки **Частота Уведомлений об Ответах** и **Частота Уведомлений для Администраторов**, чтобы установить предпочитаемую частоту.
3. Опционально измените частоту для отдельных подписок в таблице Подписок.
4. Нажмите **Сохранить Изменения**.

### Поддержка API

Поле `notificationFrequency` в объекте пользователя контролирует частоту уведомлений об ответах, а поле `adminNotificationFrequency` контролирует частоту уведомлений для администраторов. Периодические изменения подписки могут быть установлены через поле `notificationFrequency` подписки. См. [документацию API](https://docs.fastcomments.com/guide-api.html) для подробностей.
