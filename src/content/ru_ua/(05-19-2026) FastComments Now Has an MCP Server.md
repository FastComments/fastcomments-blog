# [postlink]FastComments Теперь имеет сервер MCP[/postlink]

{{#unless isPost}}
Теперь вы можете использовать Claude Code, Claude Desktop, Cursor или любой другой клиент протокола Model Context Protocol с FastComments и вызывать наш API напрямую.
{{/unless}}

{{#isPost}}

### Что нового

FastComments теперь предлагает размещенный [Model Context Protocol](https://modelcontextprotocol.io/) (MCP) сервер. MCP — это открытый стандарт, который используют AI-ассистенты для вызова серверных инструментов. Укажите своему клиенту один URL, и он сможет перечислить ваши комментарии, модерировать их, управлять пользователями SSO, настраивать виджеты и делать всё, что позволяет [API](https://docs.fastcomments.com/guide-api.html).

### Где это находится

С вашего дашборда FastComments откройте **Интеграция, затем MCP Сервер**, или посетите его напрямую по адресу 
[fastcomments.com/auth/my-account/mcp-setup](https://fastcomments.com/auth/my-account/mcp-setup).
Выберите, какой API ключ должен использовать ваш клиент, и страница сгенерирует предзаполненный URL и фрагменты кода для Claude Code и конфигурационных файлов Claude Desktop / Cursor.

### Claude Code

Одна команда для регистрации:

    claude mcp add --transport http fastcomments 'https://fastcomments.com/mcp?tenantId=YOUR_TENANT_ID&API_KEY=YOUR_API_KEY'

Затем `/mcp` внутри сессии отображает соединение и перечисляет каждый инструмент, который предоставляет сервер. Задавайте вопросы, такие как "покажи мне последние 20 помеченных комментариев на /blog/launch", "заблокируй этого комментатора на всех моих сайтах" или "какова наша ставка предотвращения спама за последнюю неделю". Claude с маршрутизирует правильный вызов инструмента.

### Claude Desktop и Cursor

Добавьте этот блок в конфигурацию MCP вашего клиента (`claude_desktop_config.json` для Claude Desktop, `mcp.json` для Cursor):

```json
{
  "mcpServers": {
    "fastcomments": {
      "type": "http",
      "url": "https://fastcomments.com/mcp?tenantId=YOUR_TENANT_ID&API_KEY=YOUR_API_KEY"
    }
  }
}
```

Перезапустите клиента, и сервер появится рядом с другими настроенными MCP серверами.

### Безопасность

API ключ встроен в URL, поэтому рассматривайте сам URL как секрет. Не вставляйте его в публичные чаты, снимки экрана или коммиты. Если ключ будет раскрыт, измените его на странице [API Keys](https://fastcomments.com/auth/my-account/api-secret) и заново скопируйте фрагмент с страницы настройки. Чтобы ограничить действия агента, создайте отдельный API ключ для каждого ассистента и привяжите его к нужному домену. Выпадающее меню на странице настройки позволяет переключать, какой ключ используют фрагменты кода всего в один клик.

### Почему мы это сделали

Люди всё чаще используют агенционные инструменты для повседневных задач взаимодействия с SaaS продуктами. Все, кто подключает сессию Claude или Cursor для работы с очередью модерации, или настраивает индивидуального агента для подсветки тем, достойных закрепления, теперь могут полностью пропустить слой обертки.

### В заключение

Откройте [Интеграция, затем MCP Сервер](https://fastcomments.com/auth/my-account/mcp-setup) в вашем дашборде, скопируйте фрагмент для вашего любимого MCP клиента, и вы на связи. Полная документация доступна в [руководстве по LLM Kit](https://docs.fastcomments.com/guide-llm-kit.html).

Как всегда, дайте нам знать ниже, если у вас возникнут вопросы.

Удачи!
