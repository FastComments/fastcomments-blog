#### Para Desenvolvedores
Instalar o FastComments em uma página da web requer apenas inserir um pequeno trecho de código. Isso significa que funciona em páginas estáticas sem complicações - incluindo a hospedagem gratuita do GitHub Pages.

#### Uma Demonstração

<a href="https://github.com/fastcomments/fastcomments-demo.github.io/blob/master/index.html#L29" target="_blank">Aqui</a> está o link para onde adicionamos o trecho de código para nossa conta de teste.
