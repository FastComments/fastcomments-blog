# [postlink]Como Desativar Uploads de Imagens Nos Seus Comentários[/postlink]

Com o FastComments, você pode desativar uploads de imagens na seção de comentários do seu site, tanto para todo o site quanto apenas para uma página específica.

{{#isPost}}
Para fazer isso, vá para <a href="https://fastcomments.com/auth/my-account/customize-widget" target="_blank">Personalizar</a> no painel e adicione uma regra de personalização.

Habilitar a configuração fica assim:

<img 
    src="images/fc-disable-image-uploads.png"
    alt="Desativar Uploads de Imagens"
    title="Desativar Uploads de Imagens"
    class='lozad' />

... e clique em salvar.
Nesta página, você deve definir seu domínio se ele não estiver preenchido automaticamente. Deixe o ID da URL em branco para aplicar a regra a todo o seu site.

Esta página tem uploads de imagens desativados como exemplo.

{{/isPost}}
