# [postlink]Personalize sua Blacklist de Comentários com o FastComments[/postlink]

Para alguns, a blacklist de palavras padrão é muito restritiva ou muito aberta.

{{#isPost}}

A blacklist de palavras padrão agora pode ser editada ou completamente alterada. Basta ir para Moderar Comentários -> Editar Configurações de Moderação -> Editar Blacklist.

<img 
    src="images/fc-customize-blacklist.png"
    alt="Personalizar Blacklist"
    title="Personalizar Blacklist"
    class='lozad' />

As palavras nesta lista devem ser separadas por vírgulas - mas você pode usar espaços ou quebras de linha se isso facilitar a leitura.

Quaisquer alterações podem levar no máximo cinco minutos para serem atualizadas.

{{/isPost}}
