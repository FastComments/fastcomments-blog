# [postlink]Aprovações Agora Podem Ser Feitas Uma Vez Por Usuário[/postlink]

{{#isPost}}

### O Que Há de Novo

Anteriormente, se a aprovação manual estava habilitada, cada comentário teria que ser moderado. Agora, se ***Apenas Aprovar Automaticamente Para Usuários Aprovados*** está habilitado, o comentário de um usuário não requer aprovação se ele já foi aprovado anteriormente.

Isso serve como uma espécie de porta de entrada para uma comunidade, o que pode ser desejável para comunidades que querem ter aprovações, mas não querem aprovar cada comentário individualmente.

### Como Ativar

Você pode habilitar este recurso através das [Configurações de Moderação de Comentários](https://fastcomments.com/auth/my-account/moderate-comments/settings), acessível via
`Moderate Comments -> Settings`.

Sob `Aprovação de Comentários`, selecione `Aprovar Comentários Automaticamente` e, em seguida, `Apenas Aprovar Automaticamente Para Usuários Aprovados`.

### Mantendo a Funcionalidade Existente

Se você aprova manualmente todos os comentários hoje, e deseja continuar a fazê-lo, simplesmente não faça nada. Não ativaremos esse recurso automaticamente ou por padrão.

### Em Conclusão

Sabemos que este recurso foi aguardado por algum tempo por alguns. Como todos os grandes lançamentos, estamos felizes por termos podido gastar tempo para otimizar, testar e liberar corretamente este recurso.

Saudações!
