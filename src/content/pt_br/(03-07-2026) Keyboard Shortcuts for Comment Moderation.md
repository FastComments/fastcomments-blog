# [postlink]Atalhos de Teclado para Moderação de Comentários[/postlink]

{{#unless isPost}}
Usuários avançados agora podem moderar comentários inteiramente do teclado com um conjunto completo de atalhos para navegação, ações e mais.
{{/unless}}

{{#isPost}}

### Novidades

A página de Moderação de Comentários agora suporta atalhos de teclado abrangentes. Navegue entre os comentários, aprove, exclua, bane,
marque como spam, altere contexto, veja registros e muito mais - tudo sem tocar no mouse.

Isso é muito importante para equipes que moderam um alto volume de comentários. Cada ação que anteriormente exigia mover o mouse e clicar em um botão
está agora a apenas uma tecla de distância.

### Exemplo de Fluxo

Suponha que você tenha uma fila de comentários para revisar. Pressione **j** para focar no primeiro comentário. Pressione **c** para expandir seu
contexto e ver a thread ao redor. Parece bom, então pressione **c** novamente para colapsar o contexto, depois **a** para aprová-lo.
Pressione **j** para mover para o próximo comentário e repita. Se algo parecer errado, pressione **s** para marcá-lo como spam ou **d** para
excluí-lo. Você pode percorrer uma página inteira de comentários sem nunca precisar alcançar o mouse.

### Navegação

Pressione **j** para mover o foco para o próximo comentário e **k** para voltar ao anterior. O comentário focado é destacado
com um contorno azul para distingui-lo dos comentários selecionados (checados).

Pressione **x** ou **Enter** para alternar a seleção no comentário focado. Mantenha **Shift** pressionado e pressione **j** ou **k** para selecionar
o comentário atual e mover o foco em um movimento - útil para selecionar rapidamente uma sequência de comentários.

### Ações

Com um comentário focado, os atalhos de uma tecla acionam as mesmas ações que os botões da barra de ferramentas:

- **a** - Alternar aprovar / desaprovar
- **s** - Alternar spam / não spam
- **d** - Excluir
- **u** - Restaurar
- **r** - Alternar revisado / não revisado
- **e** - Editar
- **b** - Banir / gerenciar banimento
- **p** - Alternar fixar / desfixar
- **f** - Desmarcar
- **i** - Ver todos do IP
- **t** - Definir fator de confiança
- **m** - Gerenciar crachás
- **l** - Ver registros
- **c** - Alternar contexto

### Atalhos Globais

Alguns atalhos funcionam independentemente de um comentário estar focado:

- **Ctrl+a** - Selecionar todos os comentários
- **Ctrl+z** - Desfazer a última ação
- **/** - Focar na entrada de pesquisa
- **[** e **]** - Navegar para a página anterior ou próxima

### Ajuda e Guias

Pressione **?** a qualquer momento para abrir um diálogo de ajuda listando todos os atalhos disponíveis.

Pressione **g** para alternar o modo "guias", que sobrepõe pequenos crachás nos botões de ação do comentário focado mostrando
a tecla de atalho correspondente. Isso é útil enquanto aprende os atalhos.

Todos os atalhos são suprimidos ao digitar na caixa de pesquisa ou em qualquer outro campo de entrada, e quando um diálogo está aberto.
Pressione **Escape** para desfocar uma entrada focada e retornar ao modo de atalho.

### Documentação

Para uma referência completa, veja o <a href="https://docs.fastcomments.com/guide-moderation.html" target="_blank">Guia de Moderação</a>.

### Em Conclusão

Esperamos que os atalhos de teclado tornem a moderação de alto volume mais rápida e confortável. Os atalhos seguem convenções
familiares para usuários de ferramentas como Gmail e GitHub, então devem parecer naturais desde o início.
