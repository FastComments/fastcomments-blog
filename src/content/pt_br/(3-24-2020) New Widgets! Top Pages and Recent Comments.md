# [postlink]Novos Widgets! Páginas Principais e Comentários Recentes[/postlink]

Hoje estamos introduzindo dois novos tipos de widgets, totalizando três widgets suportados: Contagem de Comentários, Comentários Recentes e Páginas Principais.

{{#isPost}}

Se você estiver logado, pode ver a lista <a href="https://fastcomments.com/auth/my-account/widgets" target="_blank">aqui</a>; caso contrário, vá para Minha Conta -> Obter Código do Widget -> Outros Widgets.

Tenha em mente que todos os nossos widgets têm um estilo muito minimalista - eles são projetados para serem rápidos e fáceis de estilizar de acordo com suas necessidades.

O widget de Comentários Recentes mostrará os dez comentários aprovados mais recentes, ordenados por data.

O widget de Páginas Principais mostrará as 30 páginas principais ordenadas pela contagem de comentários.

Os valores para esses widgets são pré-determinados antes de você carregar a página, **e** armazenados em cache para garantir a melhor experiência para seus usuários. Para documentação sobre os widgets, vá para a lista de widgets e selecione um. A partir daí, você verá o código que pode adicionar ao seu site, assim como qualquer configuração que o widget aceite.

Se você achar que esses widgets não fornecem as informações que você precisa, ou se gostaria de ajuda para estilizá-los, sinta-se à vontade para <a href="https://fastcomments.com/auth/my-account/help" target="_blank">entrar em contato</a>.

{{/isPost}}
