{{#isPost}}
Por padrão, o FastComments requer um e-mail para comentar se você não estiver logado. Usamos esse e-mail para criar automaticamente uma conta para o usuário e permitir que eles verifiquem seu comentário. 
Para permitir comentários anônimos, você só precisa criar uma regra de personalização. Defina o domínio e, opcionalmente, a página, onde você quer permitir comentários anônimos.

Isso significa que você pode controlar, página por página, onde você permite comentários anônimos.
