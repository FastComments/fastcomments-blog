# [postlink]Avatares Padrão do FastComments Usando Gravatar[/postlink]

### O que é um Avatar?

Um avatar é o ícone de perfil associado a cada conta e exibido ao lado de cada comentário.

### O que é Gravatar?

<a href="https://en.gravatar.com/" target="_blank">Gravatar</a> se apresenta como "Um Avatar Reconhecido Globalmente".
É um serviço onde você pode definir um avatar para um determinado e-mail. Quaisquer produtos que você se inscrever usando esse e-mail podem então
usar esse avatar. Isso remove uma etapa para as pessoas que se inscrevem no FastComments, uma vez que seu avatar é configurado automaticamente.

### Então, qual é a mudança?

O FastComments agora irá puxar avatares padrão do Gravatar. Você ou seus usuários não precisam fazer nada para obter esse recurso.

Se seus usuários existentes já definiram seu próprio avatar - sem preocupações - não vamos sobrescrever seus avatares personalizados.
Além disso, os usuários ainda podem sobrescrever o avatar do Gravatar no FastComments.
