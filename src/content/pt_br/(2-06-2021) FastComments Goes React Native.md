### <i class="circle">!</i> Este Artigo Contém Jargão Técnico

### O que Há de Novo

O FastComments agora possui componentes React Native que estão em paridade de recursos com nosso widget VanillaJS.

Decidimos nomear a biblioteca como `fastcomments-react-native`. Você pode encontrar o repositório no GitHub com o código-fonte <a href="https://github.com/FastComments/fastcomments-react-native" target="_blank">aqui</a>.

O repositório também contém exemplos de uso do widget.

Está no NPM, que você pode encontrar <a href="https://www.npmjs.com/package/fastcomments-react-native" target="_blank">aqui</a>.

### Como Funciona

A forma como este novo componente funciona é que o componente React é um envoltório em torno do widget FastComments existente.

Isso significa que se adicionarmos recursos aos nossos componentes existentes ou corrigirmos bugs, você ainda se beneficiará imediatamente!

### Tempo de Desenvolvimento

Os widgets React Native podem ser adicionados à sua aplicação em apenas alguns minutos, com pouco tempo de desenvolvimento necessário.

### SSO

Todas as bibliotecas React Native suportam SSO diretamente, e o processo de integração de SSO é o mesmo que os componentes Vanilla JS. Os exemplos de código de backend existentes também se aplicam.

### Estilização Personalizada?

Os componentes React Native podem ser estilizados usando CSS através da interface de personalização de widget, da mesma forma que os widgets baseados na web.

### Alternativas

Você poderia construir sua própria interface em torno 

### Em Conclusão

Ao lançar a biblioteca `fastcomments-react-native`, e futuras bibliotecas, esperamos facilitar para os desenvolvedores a adoção do FastComments enquanto utilizam metodologias de desenvolvimento modernas.
