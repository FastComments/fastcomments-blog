# [postlink]Πώς να Απενεργοποιήσετε τις Μεταφορτώσεις Εικόνων στα Σχόλιά σας[/postlink]

Με το FastComments μπορείτε να απενεργοποιήσετε τις μεταφορτώσεις εικόνων στην ενότητα σχολίων της ιστοσελίδας σας, είτε για ολόκληρη την ιστοσελίδα είτε μόνο για μια συγκεκριμένη σελίδα.

{{#isPost}}
Για να το κάνετε αυτό, πηγαίνετε στο <a href="https://fastcomments.com/auth/my-account/customize-widget" target="_blank">Προσαρμογή</a> στον πίνακα ελέγχου και προσθέστε έναν κανόνα προσαρμογής.

Η ενεργοποίηση της ρύθμισης φαίνεται έτσι:

<img 
    src="images/fc-disable-image-uploads.png"
    alt="Απενεργοποίηση Μεταφορτώσεων Εικόνων"
    title="Απενεργοποίηση Μεταφορτώσεων Εικόνων"
    class='lozad' />

... και πατήστε αποθήκευση.
Σε αυτή τη σελίδα πρέπει να ορίσετε το τομέα σας αν δεν είναι ήδη προεπιλεγμένος. Αφήστε το URL ID κενό για να εφαρμοστεί ο κανόνας σε ολόκληρη την ιστοσελίδα σας.

Αυτή η σελίδα έχει απενεργοποιημένες τις μεταφορτώσεις εικόνων ως παράδειγμα.

{{/isPost}}
