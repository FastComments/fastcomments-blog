# [postlink]FastComments Moodle Plugin[/postlink]

{{#unless isPost}}
Η FastComments διαθέτει πλέον ένα ειδικό plugin για το Moodle με SSO, θεματικές συζητήσεις και συνεργατικό chat για τα μαθήματά σας.
{{/unless}}

{{#isPost}}

### Τι Νέο

Είμαστε ενθουσιασμένοι να ανακοινώσουμε το επίσημο plugin FastComments για το Moodle. Αν χρησιμοποιείτε το Moodle και θέλετε να προσθέσετε ζωντανές,
θεματικές συζητήσεις ή συνεργατικό chat στα μαθήματά σας, αυτό το plugin το καθιστά απλό. Αντικαθιστά την παλιά χειροκίνητη προσέγγιση PHP
με ένα κατάλληλο plugin Moodle που εγκαθίσταται σε λίγα λεπτά και ενσωματώνεται με τους υπάρχοντες λογαριασμούς χρηστών Moodle. Το plugin λειτουργεί
με το Moodle 4.1 και μετά, και ο πηγαίος κώδικας είναι διαθέσιμος στο [GitHub](https://github.com/FastComments/fastcomments-moodle).

### Ξεκινώντας

Για να εγκαταστήσετε, κατεβάστε το ZIP από το [GitHub repository](https://github.com/FastComments/fastcomments-moodle) και εξαγάγετέ το
στο `<moodle-root>/local/fastcomments`. Στη συνέχεια, συνδεθείτε ως διαχειριστής του ιστότοπου και επισκεφθείτε **Διαχείριση Ιστότοπου > Ειδοποιήσεις**.
Το Moodle θα ανιχνεύσει το νέο plugin και θα εκτελέσει την εγκατάσταση αυτόματα.

Μόλις εγκατασταθεί, μεταβείτε στο **Διαχείριση Ιστότοπου > Πρόσθετα > Τοπικά πρόσθετα > FastComments** για να το ρυθμίσετε. Θα χρειαστείτε το
**Tenant ID** σας (που θα βρείτε στον πίνακα ελέγχου FastComments) και το **API Secret** σας αν θέλετε να χρησιμοποιήσετε Secure SSO, το οποίο συνιστούμε.

Σημειώστε ότι η χειροκίνητη λήψη από το GitHub είναι προσωρινή ενώ περιμένουμε την έγκριση του plugin στον κατάλογο προσθέτων του Moodle.
Μόλις εγκριθεί, θα μπορείτε να το εγκαταστήσετε απευθείας από τον εγκαταστάτη προσθέτων του Moodle.

### Στυλ Σχολίων

Το plugin υποστηρίζει τρία στυλ σχολίων, ώστε να μπορείτε να επιλέξετε αυτό που λειτουργεί καλύτερα για τους μαθητές σας και τη δομή του μαθήματος.

**Σχόλια**: Το mode αυτό τοποθετεί ένα πλήρες widget σχολίων κάτω από το περιεχόμενο της σελίδας. Οι μαθητές λαμβάνουν θεματικές απαντήσεις, @αναφορές,
ψηφοφορία, επεξεργαστή πλούσιου κειμένου και την καμπάνα ειδοποιήσεων για να εγγραφούν σε νέα σχόλια σε μια σελίδα.

**Collab Chat**: Το mode αυτό προσθέτει μια μπάρα στην κορυφή της σελίδας που προτρέπει τους χρήστες να επιλέξουν κείμενο και να ξεκινήσουν μια συζήτηση. Το
υπογραμμισμένο κείμενο συνδέεται με το περιεχόμενο, ώστε η συνομιλία να παραμένει συνδεδεμένη με ακριβώς αυτό που συζητείται. Επίσης
δείχνει τους online χρήστες και τον αριθμό των ενεργών συζητήσεων. Αυτό το mode δεν εμφανίζει widget στο κάτω μέρος.

### Αυτόματη SSO

Το plugin υποστηρίζει τρία modes SSO. **Secure SSO** είναι η συνιστώμενη επιλογή. Επικυρώνει την ταυτότητα του χρήστη στον server με
HMAC-SHA256 χρησιμοποιώντας το API Secret σας, έτσι ώστε τα διαπιστευτήρια να μην εκτίθενται ποτέ στον πελάτη. Με το Secure SSO, οι διαχειριστές Moodle είναι
αυτόματα συγχρονισμένοι ως συντονιστές FastComments, πράγμα που σημαίνει ότι οι διαχειριστές του ιστότοπου μπορούν να moderat σχολιασμό χωρίς καμία
επιπλέον ρύθμιση. Οι avatar χρηστών, τα ονόματα και οι διευθύνσεις email περνούν όλα με ασφάλεια.

**Απλό SSO**: περνάει τα δεδομένα του χρήστη (όνομα, email, avatar) από την πλευρά του πελάτη χωρίς υπογραφή. Είναι γρήγορο στην εγκατάσταση αλλά λιγότερο ασφαλές
από την προσέγγιση HMAC. Τέλος, **Καμία** απενεργοποιεί εντελώς το SSO, έτσι ώστε οι χρήστες να σχολιάζουν ανώνυμα.

### Προτιμήσεις Ειδοποιήσεων Χρηστών

Οι μαθητές μπορούν να διαχειρίζονται τις ρυθμίσεις ειδοποιήσεων FastComments απευθείας από το προφίλ τους στο Moodle. Στην ενότητα **FastComments**
μπορούν να ενεργοποιήσουν ή να απενεργοποιήσουν ειδοποιήσεις απαντήσεων (να λαμβάνουν email όταν κάποιος απαντά στο σχόλιό τους) και ειδοποιήσεις συνδρομής
(να λαμβάνουν emails για νήματα που έχουν εγγραφεί). Αυτό διατηρεί τα πάντα σε ένα μέρος και δίνει στους μαθητές έλεγχο πάνω στο πόσο email θα λαμβάνουν.

### Στο Συμπέρασμα

Το plugin Moodle είναι διαθέσιμο τώρα. Για τον πλήρη οδηγό εγκατάστασης, δείτε τον
[Οδηγό Ενσωμάτωσης Moodle](https://docs.fastcomments.com/guide-installation-moodle.html), και ο πηγαίος κώδικας είναι διαθέσιμος στο
[GitHub](https://github.com/FastComments/fastcomments-moodle). Ενημερώστε μας παρακάτω αν έχετε οποιεσδήποτε παρατηρήσεις!

Cheers!
