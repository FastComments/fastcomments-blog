# [postlink]Σχόλια Για Το Jamstack[/postlink]

{{#unless isPost}}
Ό,τι είναι παλιό είναι καινούριο ξανά. Η προ-παραγωγή και η εξυπηρέτηση ιστοσελίδων από ένα CDN ονομάζεται τώρα "JamStack" - αλλά δεν χρειάζεται να είναι εντελώς στατικές!
{{/unless}}

{{#isPost}}

### <i class="circle">!</i> Αυτό το Άρθρο Περιέχει Τεχνική Ορολογία

<a href="https://jamstack.org/" target="_blank">Το Jamstack</a> είναι ένας τρόπος για να γράψεις μια ιστοσελίδα που δεν εξαρτάται από έναν παραδοσιακό web server. Αντίθετα, παρέχονται από servers που ονομάζουμε "CDN" για την καλύτερη εμπειρία χρήστη (φορτώνουν πολύ γρήγορα). Η απόδοση είναι σημαντική προτεραιότητα για τις ιστοσελίδες που είναι γραμμένες με το Jamstack - όπως και το FastComments.

#### Γιατί Το FastComments Λειτουργεί Με Το Jamstack

Το FastComments απαιτεί μόνο ένα μικρό (< 11kb προς το παρόν) απόσπασμα κώδικα στο κάτω μέρος της στατικής σελίδας στην οποία θέλεις να προσθέσεις σχόλια. Έτσι, η εγκατάσταση του FastComments σε μια τυπική ιστοσελίδα Jamstack διαρκεί μόνο μερικά δευτερόλεπτα, όπως σε οποιαδήποτε στατική ιστοσελίδα.

#### Πιθανά Ζητήματα

Κατανοούμε ότι οι ιστοσελίδες Jamstack δεν αρχίζουν πάντα ως κάτι στατικό. Μπορεί να είναι φτιαγμένες με Preact, React ή Vue για παράδειγμα. Γραμμένο σε vanilla JS, δημιουργήσαμε το widget του FastComments για μέγιστη διαλειτουργικότητα ώστε να λειτουργεί με όλα τα κύρια frameworks. Ωστόσο, αν συναντήσεις προβλήματα με το FastComments και το framework που προτιμάς τότε
<a href="https://fastcomments.com/auth/my-account/help" target="_blank">ενημέρωσέ μας</a>.

#### Σε Συμπέρασμα

Ελπίζουμε ότι θα διαπιστώσεις ότι το FastComments επιτρέπει στους χρήστες σου να αλληλεπιδρούν με την ιστοσελίδα σου, να παρέχουν ανατροφοδότηση και να αφήνουν σχόλια με τρόπους που κανονικά δεν θα μπορούσαν σε μια στατική ιστοσελίδα. Για παράδειγμα, αυτή η σελίδα είναι προ-παραγωγμένη όπως μια παραδοσιακή ιστοσελίδα Jamstack, και μπορείς να δεις το FastComments να λειτουργεί ευτυχισμένα στο κάτω μέρος.

Στην υγειά σας!
