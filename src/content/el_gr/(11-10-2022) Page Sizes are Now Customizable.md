# [postlink]Οι Διαστάσεις Σελίδας Είναι Τώρα Προσαρμόσιμες[/postlink]

{{#unless isPost}}
Το FastComments είχε μια σταθερή διάσταση σελίδας 30 σχολίων από την εκκίνηση. Αυτό είναι τώρα προσαρμόσιμο!
{{/unless}}

{{#isPost}}

### Τι Είναι Νέο

Οι νήματα σχολίων σας μπορούν τώρα να έχουν ποικιλία μεγεθών σελιδοποίησης, και αυτό μπορεί να προσαρμοστεί παγκοσμίως σε όλο τον λογαριασμό σας, ή σε επίπεδο ιστότοπου, ή ακόμα και σε επίπεδο σελίδας. Αυτό μπορεί να είναι επιθυμητό εάν θέλετε ορισμένες σελίδες να έχουν λιγότερα ή περισσότερα σχόλια εμφανιζόμενα πριν ο χρήστης χρειαστεί να αλλάξει σελίδα.

Αυτό επηρεάζει επίσης την ατελείωτη κύλιση, και οποιαδήποτε πλαίσια ή ενοποιήσεις που εμφανίζουν σχόλια από το FastComments.

Οι διαστάσεις σελίδας μπορεί να κυμαίνονται από `10` έως `200`. Η προεπιλογή παραμένει στα `30`.

### Πώς Να Το Αποκτήσετε

Απλά κατευθυνθείτε στην αξιόπιστη [Δ διεπαφή Προσαρμογής Widget](https://fastcomments.com/auth/my-account/customize-widget) και ορίστε την διάσταση σελίδας σας.

Σημειώστε ότι το FastComments προ-υπολογίζει τα σχόλια που εμφανίζονται σε κάθε σελίδα, επομένως η αλλαγή αυτή της παραμέτρου θα ενεργοποιήσει το σύστημά μας να ξαναυπολογίσει όλα τα νήματα σχολίων σας. Αυτό δεν θα προκαλέσει καθυστέρηση, αλλά μπορεί να χρειαστούν μερικά λεπτά για να εμφανιστεί η αλλαγή.

### Τεκμηρίωση

Ενημερωμένη τεκμηρίωση σχετικά με τις διαστάσεις σελίδας μπορείτε να βρείτε στο [docs.fastcomments.com](https://docs.fastcomments.com/guide-customizations-and-configuration.html#page-size).

### Σημαντικές Αλλαγές

Ως μέρος αυτής της αλλαγής, δύο μεταφράσεις που καθορίστηκαν στο widget σχολίων έχουν μετονομαστεί. Το `PREV_30` και το `NEXT_30` είναι τώρα `PREV` και `NEXT`, αντίστοιχα, και περιέχουν μια μεταβλητή `[count]`.

Εάν έχετε καθορίσει μεταφράσεις για τα κουμπιά σελιδοποίησης μέσω της διεπαφής Προσαρμογής Widget, τότε έχουμε ήδη μεταφέρει αυτές για εσάς. Εάν περνάτε τις μεταφράσεις μέσω κάποιου είδους ενοποίησης μέσω κώδικα, θα πρέπει να ενημερώσετε αυτές:

- `PREV_30` -> `PREV`
- `NEXT_30` -> `NEXT`

### Συμπερασματικά

Όπως και σε όλες τις κύριες εκδόσεις, χαιρόμαστε που μπορέσαμε να αφιερώσουμε χρόνο για να βελτιστοποιήσουμε, να δοκιμάσουμε και να κυκλοφορήσουμε σωστά αυτή τη δυνατότητα. Ενημερώστε μας παρακάτω αν ανακαλύψετε τυχόν προβλήματα.

Στην υγειά σας!
