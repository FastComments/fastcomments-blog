# [postlink]Η FastComments Λανσάρει AI Agents[/postlink]

{{#unless isPost}}
Μπορείτε τώρα να δημιουργήσετε AI Agents που διαβάζουν κάθε νέο σχόλιο, απαντούν με τη φωνή σας, κλιμακώνουν τις δύσκολες περιπτώσεις και απονέμουν διακριτικά στους καλύτερους συνεισφέροντες. Λειτουργούν σύμφωνα με το πρόγραμμα σας, εντός του προϋπολογισμού σας, και κάνουν μόνο ό,τι τους επιτρέπετε.

### Μια Σημείωση προς την Κοινότητα

Θέλω απλώς να ξεκαθαρίσω ότι δεν δημιουργούμε κάτι που επιτρέπει στους bots να προσποιούνται ότι είναι πραγματικοί χρήστες. Αυτό δεν είναι κάτι που πιστεύω ότι είναι καλό για το διαδίκτυο, τις online κοινότητες ή την κατεύθυνση του προϊόντος μας. Ο στόχος με τους AI Agents είναι να ενδυναμώσει τις ομάδες μέτρησης ή τους ιδιοκτήτες κοινοτήτων που δεν μπορούν να αντέξουν οικονομικά αφιερωμένους μετρητές.

Όλα τα σχόλια που αφήνονται από το AI πρέπει να έχουν την ετικέτα "Bot". Έτσι αποστέλλεται το προϊόν και οποιαδήποτε ιστοσελίδα αφαιρεί κάπως την ετικέτα Bot από τα σχόλια των bots θα έχει αυτή τη δυνατότητα απενεργοποιημένη. Επιπλέον, χρησιμοποιούμε μόνο παρόχους LLM που δεν εκπαιδεύονται με τα δεδομένα μας. Αυτή τη στιγμή χρησιμοποιούμε το DeepInfra, το οποίο παρέχει μόνο προσβάσεις για μοντέλα ανοιχτού κώδικα.

Τώρα που το ξεκαθαρίσαμε :)

### Τι Νέο Υπάρχει

Η FastComments παρέχει πλέον αυτοματοποιημένα agents που μπορούν να εκτελούν εργασίες μέτρησης, να καρφιτσώνουν σχόλια, να συνοψίζουν συζητήσεις και πολλά άλλα.

Μπορείτε να τους βρείτε κάτω από [Customize, then AI Agents](https://fastcomments.com/auth/my-account/ai-agents).

### Trigger και Εργαλεία

Κάθε agent ρυθμίζεται με μια λίστα triggers και μια λίστα επιτρεπόμενων ενεργειών. Τα triggers περιλαμβάνουν νέα σχόλια, επεξεργασίες, αφαιρέσεις, όρια ψήφων, όρια σημαίας, ενέργειες μετρητών, πρωτότυπους σχολιαστές και άλλα. Αν δεν θέλετε ένας agent να απαγορεύει χρήστες, μπορείτε να εξαιρέσετε το εργαλείο απαγόρευσης.

Για παράδειγμα, εάν θέλετε μόνο έναν "συνοψιστή συζητήσεων" που απαντά μόνο μετά από έναν συγκεκριμένο αριθμό σχολίων και καρφιτσώνει αυτό το σχόλιο, μπορείτε να ρυθμίσετε τα δικαιώματα του agent έτσι ώστε να έχει πρόσβαση μόνο σε αυτές τις ενέργειες.

Μπορείτε επίσης να περιορίσετε ένα agent σε συγκεκριμένα μοτίβα URL όπως `/news/*` ή `/forums/*` και σε συγκεκριμένες τοποθεσίες, ώστε ένας agent κοινότητας γαλλικής γλώσσας να μην αρχίσει να απαντά στις αγγλικές σας σελίδες.

### Dry Run, Εγκρίσεις και Προϋπολογισμοί

Κάθε agent ξεκινά σε λειτουργία dry-run από προεπιλογή. Στη λειτουργία dry-run, ο agent διαβάζει το πλαίσιο και γράφει τι θα έκανε, αλλά δεν εκτελεί πράγματι τίποτα. Μπορείτε να παρακολουθήσετε μερικά πραγματικά triggers, να δείτε τις προτεινόμενες ενέργειες και τη λογική του agent και να αποφασίσετε αν τον εμπιστεύεστε πριν τον ενεργοποιήσετε.

Για ευαίσθητα εργαλεία, μπορείτε να απαιτήσετε ανθρώπινη έγκριση. Ο agent ουράζει την ενέργεια, ένας διαχειριστής εξετάζει την προτεινόμενη ενέργεια μαζί με τη λογική και την εμπιστοσύνη του agent και εγκρίνει ή απορρίπτει. Οι απαγορεύσεις απαιτούν πάντα έγκριση στην ΕΕ σύμφωνα με το Άρθρο 17 DSA, αλλά μπορείτε να απαιτήσετε έγκριση για οποιοδήποτε εργαλείο, οπουδήποτε.

Κάθε agent έχει ένα ημερήσιο και ένα μηνιαίο όριο προϋπολογισμού. Όταν το όριο χτυπηθεί, ο agent σταματά να λειτουργεί μέχρι να ολοκληρωθεί η περίοδος. Σας στέλνουμε email στα προεπιλεγμένα όρια του 80% και 100% (μπορείτε επίσης να επιλέξετε να λάβετε ειδοποίηση στο 50%) ώστε να μην υπάρχουν εκπλήξεις με τιμολόγια.

### Μνήμη

Οι agents έχουν μνήμη που διαβάζει και γράφει για τον εαυτό τους. Μπορούν να αποθηκεύσουν σύντομες σημειώσεις για έναν χρήστη ή ένα μοτίβο, όπως "προειδοποιήθηκε για εκτός θέματος τρεις φορές" ή "σπαμ-σύστημα που δημοσιεύει συνδέσμους στο τομέα X", και να διαβάσουν αυτές τις σημειώσεις στην επόμενη ενεργοποίηση. Η μνήμη μοιράζεται μεταξύ των agents που είναι στο τέντωμα σας, έτσι οι σημειώσεις ενός agent καλωσορίσματος μπορούν να ενημερώσουν τις αποφάσεις ενός agent μέτρησης αργότερα. Η μνήμη που σχετίζεται με έναν χρήστη καθαρίζεται αν διαγράψει τον λογαριασμό του.

### Επαναπαραγκιακή Δοκιμή

Πριν απελευθερώσετε έναν agent σε ζωντανή κυκλοφορία, μπορείτε να τον εκτελέσετε ενάντια σε ιστορικά σχόλια. Το εργαλείο επαναπαρασκευής περνά μέσα από πραγματικά παρελθόντα σχόλια σε dry-run, σας δείχνει τι θα είχε κάνει ο agent και κατηγοριοποιεί κάθε προτεινόμενη ενέργεια σε σχέση με το τι πραγματικά συνέβη. Αυτός είναι ο πιο γρήγορος τρόπος να ρυθμίσετε ένα σύστημα προτροπής χωρίς να περιμένετε ζωντανά γεγονότα.

### Μοντέλα, Ιδιωτικότητα και Κόστος

Οι agents λειτουργούν σε open-weight μοντέλα που φιλοξενούνται από παρόχους υποδομής που δεν εκπαιδεύονται με δεδομένα πελατών. Σήμερα μπορείτε να επιλέξετε μεταξύ ενός ταχύτερου μοντέλου (gpt-oss-120B-turbo) και ενός πιο έξυπνου, πιο αργού (GLM 5.1), και θα συνεχίσουμε να προσθέτουμε επιλογές καθώς εξελίσσεται το τοπίο. Το κόστος μετριέται ανά token και μετατρέπεται σε δολάρια στη σελίδα χρήσης σας, μαζί με τις ημερήσιες και μηνιαίες τάσεις.

Ως μέρος αυτής της αλλαγής, μειώνουμε τα κόστη tokens από $80 1M Tokens σε $20 ανά 1M συνδυασμένων tokens εισόδου/εξόδου. Οι περισσότεροι από τους πελάτες μας ξοδεύουν πολύ λίγα σε tokens για ανίχνευση spam, καθώς το αρχικό σύστημα ανίχνευσης spam με βάση LLM μόνο ταξινόμησε στο LLM αν το spam ξεπερνούσε τον ναϊβ-μπέις ταξινομητή, γι' αυτό και το αρχικό υψηλό κόστος. Μειώνουμε το κόστος για να ενθαρρύνουμε τη χρήση της νέας λειτουργικότητας agents.

Σε κάποια στιγμή όταν τα κόστη μνήμης επανέλθουν και τα μοντέλα όπως το GLM 5.1 γίνουν πιο λογικά για λειτουργία σε προσωπικό εξοπλισμό, θα μεταβούμε από τη χρήση εξωτερικού παρόχου και θα λειτουργούμε τον εξοπλισμό εμείς οι ίδιοι.

### Συμπέρασμα

Σκεφτήκαμε πολύ για το πώς να το χτίσουμε αυτό χωρίς να κάνει τους agents να φαίνονται ότι έχουν την εξουσία. Η λειτουργία dry-run, οι εγκρίσεις, τα περιορισμένα εργαλεία, τα όρια προϋπολογισμού και η επαναπαραγωγή στην ιστορία υπάρχουν όλα για να σας βοηθήσουν να παραμείνετε υπό έλεγχο. Ελπίζουμε ότι οι AI Agents απελευθερώνουν τους μετρητές σας για τις συζητήσεις που χρειάζονται πραγματικά έναν άνθρωπο.

Ευχαριστούμε τους πελάτες μας που μας δίνουν συνεχή ανατροφοδότηση ώστε να μπορούμε να σκεφτόμαστε ιδέες όπως αυτή και επίσης μας δίνουν τον χρόνο να τις λανσάρουμε με επιτυχία. Όπως πάντα, ενημερώστε μας παρακάτω αν ανακαλύψετε οποιαδήποτε προβλήματα.

Υγειά!
