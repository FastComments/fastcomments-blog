# [postlink]Όλα τα Widgets FastComments Είναι Τώρα Live[/postlink]

{{#unless isPost}}
Κάθε ενσωματωμένο widget FastComments (μετρήσεις σχολίων, πρόσφατα σχόλια, κορυφαίες σελίδες, πρόσφατες συζητήσεις, συνοψίσεις αξιολογήσεων, πλωτές αντιδράσεις) τώρα ενημερώνεται σε πραγματικό χρόνο.
{{/unless}}

{{#isPost}}

### Τι Νέο

Το κύριο widget σχολίων είναι live από την πρώτη μέρα. Ωστόσο, τα μικρότερα ενσωματωμένα widgets δεν ήταν. Οι μετρήσεις σχολίων, τα widget "πρόσφατα σχόλια" ή "κορυφαίες σελίδες", κ.λπ., θα έδειχναν έναν αριθμό που μπορεί να ήταν έως 1 λεπτό παμπάλαιος.

Τώρα κάθε widget που αποστέλλει το FastComments εγγράφεται σε ζωντανές ενημερώσεις και ανανεώνεται αμέσως :) Αυτό περιλαμβάνει και τις συνοψίσεις αξιολογήσεων!

### Ποια Widgets

Όλα. Συγκεκριμένα:

- `FastCommentsCommentCount` - η μετρήση σχολίων ανά σελίδα
- `FastCommentsCommentCountBulk` - η μαζική έκδοση που ενημερώνει πολλές μετρήσεις σε μια σελίδα λίστας/αρχείου
- `FastCommentsRecentComments` και `FastCommentsRecentCommentsV2`
- `FastCommentsTopPages` και `FastCommentsTopPagesV2`
- `FastCommentsRecentDiscussionsV2`
- `FastCommentsReviewsSummaryWidget`
- `FastCommentsFlyoverTrigger` (η πλωτή ετικέτα μετρήσεων σχολίων)
- `FastCommentsEmbedPageLikesFloating` (οι πλωτές αντιδράσεις + μετρήσεις σχολίων)

### Τι Πρέπει Να Κάνετε

Τίποτα. Αν έχετε ήδη κάποιο από αυτά ενσωματωμένο, φρεσκάρετε τη σελίδα μία φορά. Την επόμενη φορά που θα δημοσιευτεί, επεξεργαστεί ή διαγραφεί ένα σχόλιο, ή κάποιος αντιδράσει σε μια σελίδα, το widget θα ενημερωθεί.

Η παλιά σημαία ρύθμισης `isLive: true` στα widgets μετρήσεων σχολίων είναι τώρα περιττή - τα widgets είναι πάντα live.

Αν φορτώσετε το widget JS από το CDN μας, έχετε την νέα έκδοση, συμπεριλαμβανομένου αν χρησιμοποιείτε μία από τις βιβλιοθήκες wrapper μας (React, Vue, κ.λπ.).

### Οι Αντιδράσεις Είναι Live Και Αυτές

Το πλωτό widget αντιδράσεων επίσης ανταποκρίνεται σε γεγονότα αντιδράσεων και αρέσκειας τώρα. Πατήστε την καρδιά σε μια σελίδα και άλλες ανοιχτές καρτέλες βλέπουν τη μετρήση να αλλάζει.

### Πώς Λειτουργεί

Κάθε widget ανοίγει μια μοναδική σύνδεση WebSocket. Τα widgets που εμφανίζουν δεδομένα για μία συγκεκριμένη σελίδα (`comment-count` (συμπεριλαμβανομένων των μαζικών), `reviews-summary`, `embed-page-likes-floating`) εγγράφονται στη ροή γεγονότων αυτής της σελίδας. Τα widgets που εμφανίζουν δεδομένα που εκτείνονται σε έναν ενοικιαστή (`recent-comments`, `top-pages`, `recent-discussions`) εγγράφονται σε μια λεπτή ροή παλμού ανά ενοικιαστή, που στέλνει σήμα όποτε κάτι αλλάζει σε αυτόν τον ενοικιαστή.

Το αποτέλεσμα είναι ότι ένα ανενεργό widget κοστίζει ουσιαστικά τίποτα και ένα ενεργό widget δείχνει έναν φρέσκο αριθμό μέσα σε ένα ή δύο δευτερόλεπτα από την υποκείμενη αλλαγή.

### Συνοχή Cache

Οι server-side cache που υποστηρίζουν αυτά τα widgets χρησιμοποιούσαν να λήγουν σε TTL 60 δευτερολέπτων. Τώρα αποτυγχάνουν τη στιγμή που καταφτάνει ένα σχετικό γεγονός σχολίου ή αντίδρασης, έτσι η πρώτη αίτηση μετά από μια αλλαγή επιστρέφει φρέσκα δεδομένα, και όχι την αποθηκευμένη έκδοση.

### Συμπέρασμα

Είμαστε χαρούμενοι που μπορέσαμε να αφιερώσουμε χρόνο για να δοκιμάσουμε και να βελτιστοποιήσουμε αυτήν την αλλαγή ώστε οι πελάτες μας να την απολαμβάνουν στην ίδια τιμή όπως πάντα. Νομίζουμε ότι οι ζωντανές συνοψίσεις αξιολογήσεων είναι μια ιδιαίτερα cool διαφοροποίηση για εμάς.

Ενημερώστε μας παρακάτω αν παρατηρήσετε κάτι εκτός.

Cheers!
