# [postlink]Κλείσιμο Μιας Συζήτησης με το FastComments[/postlink]

Έχετε μια συζήτηση που θέλετε να κλείσετε; Η επιλογή είναι εδώ!

{{#isPost}}

Το FastComments είναι όλα σχετικά με την ενεργοποίηση των συζητήσεων και την παροχή φωνής στους χρήστες. Ωστόσο, μερικές φορές έχετε περιεχόμενο που δεν είναι πλέον σχετικό, ή θέλετε να απενεργοποιήσετε προσωρινά τις απαντήσεις, τώρα μπορείτε.

Απλώς μεταβείτε στη <a href="https://fastcomments.com/auth/my-account/customize-widget" target="_blank">σελίδα προσαρμογής</a> και ελέγξτε την επιλογή "Αποτροπή νέων απαντήσεων".

Σημειώστε ότι θα πρέπει να ορίσετε μια διεύθυνση URL, εκτός αν θέλετε να απενεργοποιήσετε τα σχόλια σε ολόκληρη την ιστοσελίδα σας.
