Από το 2021, το JustComments θα κλείσει. Στο FastComments έχουμε διευκολύνει την μετάβαση στην πλατφόρμα μας.

Θα διαπιστώσετε ότι το FastComments παρέχει πολλές από τις ίδιες δυνατότητες που γνωρίζετε και ακόμη περισσότερες.

## Διαφορές Τιμολόγησης

Ενώ το JustComments χρησιμοποιεί ένα σύστημα βασισμένο σε πιστώσεις, το FastComments χρησιμοποιεί ένα κλιμακωτό μοντέλο με τρία διαθέσιμα επίπεδα. Με το FastComments, το πλάνο των $5/μήνα
σας καλύπτει μέχρι 1M φορτώσεις σελίδων τον μήνα. Πέρα από αυτό υπάρχουν τα πλάνα Pro και Enterprise, τα οποία μπορείτε να δείτε στην <a href="https://fastcomments.com/traffic-pricing" target="_blank">σελίδα τιμολόγησης</a>.

## Αν Είστε σε WordPress

Το FastComments έχει ένα ειδικό plugin για το WordPress: <a href="https://wordpress.org/plugins/fastcomments/" target="_blank">https://wordpress.org/plugins/fastcomments/</a>.

Το plugin διευκολύνει την εγκατάσταση, τη ρύθμιση και τη συγχρονισμό. Ωστόσο, το JustComments δεν συγχρονίζει τα σχόλιά σας πίσω στην εγκατάσταση WordPress σας όπως το FastComments
κάνει. Αυτό σημαίνει ότι μετά τη συγχρονισμό θα πρέπει να ακολουθήσετε τα βήματα εξαγωγής και εισαγωγής παρακάτω.

## Εξαγωγή Υπαρχόντων Σχολίων

Ο εξαγωγέας του FastComments θα μεταφέρει τα σχόλιά σας, τα ονόματα χρηστών, τα avatar χρηστών και τις εικόνες inline. Θα μεταφέρουμε τις εικόνες στους διακομιστές μας απρόσκοπτα.

Για να κατεβάσετε τα δεδομένα των σχολίων σας από το JustComments, πηγαίνετε στη σελίδα του Λογαριασμού σας.

## Εισαγωγή στην Διαχείριση του FastComments

Μην ανησυχείτε για το άνοιγμα του παραγόμενου αρχείου από το JustComments.

Αν είστε συνδεδεμένοι μπορείτε να πάτε <a href="https://fastcomments.com/auth/my-account/manage-data/import" target="_blank">εδώ για να εισάγετε το αρχείο</a>.

Επιλέξτε το JustComments από την αναδυόμενη λίστα και ανεβάστε το αρχείο σας.

### Αναγνωριστικά Σελίδας

Κατά την εισαγωγή, θα δείτε την επιλογή να επιλέξετε URL Σελίδας ή "Item ID". Αν δεν είστε σίγουροι ποιο να επιλέξετε, επιλέξτε URL Σελίδας. Αν έχετε μια ενσωμάτωσή με το JustComments
όπου καθορίζετε το Item ID στη ρύθμιση του widget, επιλέξτε Item ID.

### Περιμένετε μερικά λεπτά

Η εισαγωγή του FastComments είναι "ασύγχρονη". Αυτό σημαίνει ότι η μεταφόρτωση του αρχείου και η επεξεργασία του είναι ξεχωριστά βήματα.

Έτσι, ανάλογα με το μέγεθος του αρχείου σας, μπορεί να λάβετε το μήνυμα επιτυχίας αμέσως. Στη σελίδα εισαγωγής, στο κάτω μέρος υπάρχει ένας πίνακας - κάθε γραμμή αντιπροσωπεύει μια προσπάθεια εισαγωγής.
Μπορείτε να ανανεώσετε αυτή τη σελίδα με ασφάλεια για να δείτε την κατάσταση της εισαγωγής σας και πόσες γραμμές έχουν εισαχθεί μέχρι στιγμής.

### Όταν τελειώσει

Θα λάβετε ένα email όταν ολοκληρωθεί η εισαγωγή - είτε είναι επιτυχής είτε όχι. Μπορείτε να κλείσετε τη σελίδα αφού η μεταφόρτωση του αρχείου είναι επιτυχής και βλέπετε την εισαγωγή σας στη σελίδα Εισαγωγών. Θα έχει κατάσταση "Αιτήθηκε" και όταν ξεκινά η κατάσταση θα είναι "Εκτελείται".

Είναι ασφαλές να επανεισάγετε όσες φορές χρειάζεται - ωστόσο αν η εισαγωγή σας αποτύχει τότε <a href="https://fastcomments.com/auth/my-account/help" target="_blank">επικοινωνήστε μαζί μας</a> για να μπορέσουμε να βοηθήσουμε.

### Αντικαταστήστε τον Κωδικό JustComments με τον FastComments

Αν δεν χρησιμοποιείτε το plugin JustComments για WordPress τότε είναι απλώς θέμα αφαίρεσης του μικρού κώδικα που σας παρέχουν και αντικατάστασής του με <a href="https://fastcomments.com/auth/my-account/get-acct-code" target="_blank">τον δικό μας</a>.
Μπορείτε ακόμη να εκτελείτε το JustComments και το FastComments στον ίδιο ιστότοπο προσωρινά για να δοκιμάσετε και να προσαρμόσετε την εμφάνιση και την αίσθηση. <a href="https://fastcomments.com/auth/my-account/help" target="_blank">Μπορούμε να σας βοηθήσουμε με αυτό</a>.

### Μεταφέροντας URLs ταυτόχρονα

Τα σχόλια καθαυτά είναι συνδεδεμένα με τα πεδία "pageUrl" ως προεπιλογή στην εξαγωγή του JustComments, οπότε όσο οι διευθύνσεις URL σας δεν αλλάζουν, η μετάβαση είναι εύκολη. Αν θέλετε να μεταφέρετε URLs και να διατηρήσετε τα
σχόλιά σας τότε <a href="https://fastcomments.com/auth/my-account/help" target="_blank">ενημερώστε μας</a> στέλνοντάς μας τις παλιές και νέες διευθύνσεις URL. Η σελίδα βοήθειας επιτρέπει την μεταφόρτωση μικρών αρχείων κειμένου, οπότε αν
μεταφέρετε περισσότερες από μερικές διευθύνσεις URL δημιουργήστε ένα φύλλο Excel.

## Ανακεφαλαίωση

1. Εξαγωγή των δεδομένων σας
2. Εισαγωγή στο FastComments
3. Αν δεν είστε σε WordPress, Αντικαταστήστε τον κωδικό JavaScript του JustComments με <a href="https://fastcomments.com/auth/my-account/get-acct-code" target="_blank">του FastComments</a>
