# [postlink]Τα Προεπιλεγμένα Αβατάρ του FastComments Χρησιμοποιώντας το Gravatar[/postlink]

### Τι είναι ένα Αβατάρ;

Ένα αβατάρ είναι το εικονίδιο προφίλ που σχετίζεται με κάθε λογαριασμό και εμφανίζεται δίπλα σε κάθε σχόλιο.

### Τι είναι το Gravatar;

<a href="https://en.gravatar.com/" target="_blank">Gravatar</a> αυτοπαρουσιάζεται ως "Ένα Παγκοσμίως Αναγνωρίσιμο Αβατάρ".
Είναι μια υπηρεσία όπου μπορείτε να ορίσετε ένα αβατάρ για μια συγκεκριμένη διεύθυνση email. Όλα τα προϊόντα στα οποία εγγράφεστε χρησιμοποιώντας αυτό το email μπορούν στη συνέχεια
να χρησιμοποιήσουν αυτό το αβατάρ. Αυτό αφαιρεί ένα βήμα για τους ανθρώπους που εγγράφονται στο FastComments, καθώς το αβατάρ τους ρυθμίζεται αυτόματα.

### Ποια είναι η αλλαγή;

Το FastComments θα αντλεί τώρα προεπιλεγμένα αβατάρ από το Gravatar. Εσείς ή οι χρήστες σας δεν χρειάζεται να κάνετε τίποτα για να αποκτήσετε αυτή τη λειτουργία.

Αν οι υπάρχοντες χρήστες σας έχουν ήδη ορίσει το δικό τους αβατάρ - μην ανησυχείτε - δεν θα αντικαταστήσουμε τα προσαρμοσμένα αβατάρ τους.
Επίσης, οι χρήστες μπορούν ακόμα να αντικαταστήσουν το αβατάρ Gravatar στο FastComments.
