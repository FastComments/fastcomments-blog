# [postlink]Συντομεύσεις Πληκτρολογίου για τη Διαχείριση Σχολίων[/postlink]

{{#unless isPost}}
Οι δυναμικοί χρήστες μπορούν τώρα να διαχειρίζονται τα σχόλια αποκλειστικά από το πληκτρολόγιο με ένα πλήρες σύνολο συντομεύσεων για πλοήγηση, ενέργειες και περισσότερα.
{{/unless}}

{{#isPost}}

### Τι Νέo Υπάρχει

Η σελίδα Διαχείρισης Σχολίων τώρα υποστηρίζει ολοκληρωμένες συντομεύσεις πληκτρολογίου. Πλοηγηθείτε μεταξύ των σχολίων, εγκρίνετε, διαγράφετε, αποκλείετε,
σημειώνετε ως ανεπιθύμητο, αλλάζετε το περιεχόμενο, προβάλλετε αρχεία καταγραφής και περισσότερα - όλα χωρίς να αγγίξετε το ποντίκι.

Αυτή είναι μια σημαντική αναβάθμιση για τις ομάδες που διαχειρίζονται μεγάλους όγκους σχολίων. Κάθε ενέργεια που προηγουμένως απαιτούσε μετακίνηση του ποντικιού και κλικ σε κουμπί
είναι τώρα μόλις μια πίεση πλήκτρου μακριά.

### Παράδειγμα Ροής

Άς υποθέσουμε ότι έχετε μία ουρά σχολίων προς αναθεώρηση. Πατήστε **j** για να εστιάσετε στο πρώτο σχόλιο. Πατήστε **c** για να επεκτείνετε το
περιεχόμενο και να δείτε τη γύρω συζήτηση. Φαίνεται καλά, οπότε πατήστε **c** ξανά για να συμπτύξτε το περιεχόμενο, έπειτα **a** για να το εγκρίνετε.
Πατήστε **j** για να μεταβείτε στο επόμενο σχόλιο και να επαναλάβετε. Εάν κάτι φαίνεται λάθος, πατήστε **s** για να το σημειώσετε ως ανεπιθύμητο ή **d** για να
το διαγράψετε. Μπορείτε να εργαστείτε σε ολόκληρη τη σελίδα σχολίων χωρίς να χρειαστεί να αγγίξετε το ποντίκι.

### Πλοήγηση

Πατήστε **j** για να μετακινήσετε την εστίαση στο επόμενο σχόλιο και **k** για να μεταφερθείτε στο προηγούμενο. Το εστιασμένο σχόλιο επισημαίνεται
με ένα μπλε περίγραμμα για να διακρίνεται από τα επιλεγμένα (ελεγμένα) σχόλια.

Πατήστε **x** ή **Enter** για να εναλλάξετε επιλογή στο εστιασμένο σχόλιο. Κρατήστε πατημένο το **Shift** και πατήστε **j** ή **k** για να επιλέξετε
το τρέχον σχόλιο και να μετακινήσετε την εστίαση σε μία κίνηση - χρήσιμο για να επιλέξετε γρήγορα μία σειρά σχολίων.

### Ενέργειες

Με ένα σχόλιο εστιασμένο, οι συντομεύσεις ενός πλήκτρου ενεργοποιούν τις ίδιες ενέργειες με τα κουμπιά της γραμμής εργαλείων:

- **a** - Εναλλαγή έγκρισης / αποδοχής
- **s** - Εναλλαγή ανεπιθύμητου / μη ανεπιθύμητου
- **d** - Διαγραφή
- **u** - Αναίρεση διαγραφής
- **r** - Εναλλαγή αναθεωρημένου / μη αναθεωρημένου
- **e** - Επεξεργασία
- **b** - Αποκλεισμός / διαχείριση αποκλεισμού
- **p** - Εναλλαγή σταθερού / αποσταθεροποίησης
- **f** - Αφαίρεση σημείωσης
- **i** - Προβολή όλων από IP
- **t** - Ρύθμιση παράγοντα εμπιστοσύνης
- **m** - Διαχείριση εσήματων
- **l** - Προβολή αρχείων καταγραφής
- **c** - Εναλλαγή περιεχομένου

### Παγκόσμιες Συντομεύσεις

Ορισμένες συντομεύσεις λειτουργούν ανεξάρτητα από το αν ένα σχόλιο είναι εστιασμένο:

- **Ctrl+a** - Επιλογή όλων των σχολίων
- **Ctrl+z** - Αναίρεση της τελευταίας ενέργειας
- **/** - Εστίαση στο πεδίο αναζήτησης
- **[** και **]** - Πλοήγηση στην προηγούμενη ή επόμενη σελίδα

### Βοήθεια και Οδηγοί

Πατήστε **?** οποιαδήποτε στιγμή για να ανοίξετε ένα παράθυρο βοήθειας που θα αναφέρει κάθε διαθέσιμη συντόμευση.

Πατήστε **g** για να εναλλάξετε τη λειτουργία "οδηγών", η οποία επικάθεται μικρές ετικέτες στα κουμπιά δράσης του εστιασμένου σχολίου που δείχνουν
το αντίστοιχο πλήκτρο συντόμευσης. Αυτό είναι χρήσιμο κατά την εκμάθηση των συντομεύσεων.

Όλες οι συντομεύσεις καταστέλλονται όταν πληκτρολογείτε στο πεδίο αναζήτησης ή σε οποιοδήποτε άλλο πεδίο εισαγωγής, και όταν είναι ανοιχτό ένα παράθυρο διαλόγου.
Πατήστε **Escape** για να θολώσετε μια εστιασμένη είσοδο και να επιστρέψετε στη λειτουργία συντομεύσεων.

### Τεκμηρίωση

Για πλήρη αναφορά, δείτε τον <a href="https://docs.fastcomments.com/guide-moderation.html" target="_blank">Οδηγό Διαχείρισης</a>.

### Συμπέρασμα

Ελπίζουμε οι συντομεύσεις πληκτρολογίου να κάνουν τη διαχείριση μεγάλου όγκου σχολίων πιο γρήγορη και άνετη. Οι συντομεύσεις ακολουθούν τις συμβάσεις
που είναι οικείες στους χρήστες εργαλείων όπως το Gmail και το GitHub, οπότε θα πρέπει να φαίνονται φυσικές από την αρχή.

Ενημερώστε μας παρακάτω αν έχετε οποιεσδήποτε ερωτήσεις ή σχόλια!

Ζητώ συγγνώμη!
