# [postlink]Προσαρμόστε τη Μαύρη Λίστα Σχολίων σας με το FastComments[/postlink]

Για μερικούς, η προεπιλεγμένη μαύρη λίστα λέξεων είναι πολύ περιοριστική ή πολύ ανοιχτή.

{{#isPost}}

Η προεπιλεγμένη μαύρη λίστα λέξεων μπορεί τώρα να επεξεργαστεί ή να αλλάξει εντελώς. Απλώς πηγαίνετε στο Moderate Comments -> Edit Moderation Settings -> Edit Blacklist.

<img 
    src="images/fc-customize-blacklist.png"
    alt="Προσαρμογή Μαύρης Λίστας"
    title="Προσαρμογή Μαύρης Λίστας"
    class='lozad' />

Οι λέξεις σε αυτή τη λίστα θα πρέπει να διαχωρίζονται με κόμματα - αλλά μπορείτε να χρησιμοποιήσετε κενά ή νέες γραμμές αν σας διευκολύνει να τις διαβάσετε.

Οποιεσδήποτε αλλαγές αναμένονται να απαιτήσουν το πολύ πέντε λεπτά για να ενημερωθούν.

{{/isPost}}
