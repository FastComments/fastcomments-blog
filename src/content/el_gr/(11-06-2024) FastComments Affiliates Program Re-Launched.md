### Τι είναι νέο

Το FastComments διαθέτει τώρα ένα εντελώς αυτο-φιλοξενούμενο πρόγραμμα συνεργατών για να βελτιώσει τα ποσοστά μετατροπής και την αξιοπιστία.

### Προηγούμενος Πάροχος

Το FastComments είχε προηγουμένως ένα πρόγραμμα συνεργατών μέσω ενός παρόχου που ονομάζεται Rewardful. Αρχίσαμε να ανησυχούμε για τα ποσοστά μετατροπής μέσω αυτού του προγράμματος και καταλάβαμε ότι οι αποκλειστές διαφημίσεων επηρεάζουν αυτήν την ενσωμάτωσή.

Προφανώς, αυτό είναι ένα μεγάλο ζήτημα, γιατί οι περισσότεροι άνθρωποι χρησιμοποιούν έναν αποκλειστή διαφημίσεων σήμερα.

Σύμφωνα με το Rewardful, το πρόγραμμα λειτουργούσε σωστά, αλλά για να λειτουργήσει σωστά, θα μπορούσαμε να τους πληρώσουμε περισσότερα χρήματα. Δεν ήταν απαγορευτικό, αλλά βρήκαμε τον τρόπο που χειρίζονταν αυτό το πρόβλημα και δεν περιλάμβαναν τα χαρακτηριστικά που είναι απαραίτητα για να λειτουργεί σε όλες τις κατηγορίες της πλατφόρμας τους, απαράδεκτο. Έτσι, ακυρώσαμε γρήγορα αυτή τη συνεργασία. Οποιαδήποτε δεδομένα πελατών FastComments που βρίσκονταν στο Rewardful θα πρέπει τώρα να διαγραφούν κατόπιν αιτήματός μας.

### Επιπτώσεις για Υπάρχοντες Συνεργάτες

Οι υπάρχοντες συνεργάτες έχουν ήδη μεταφερθεί στο νέο σύστημα. Αν είχατε λογαριασμό, μπορείτε να δείτε τα στατιστικά σας για τους συνεργάτες στον
[πίνακα ελέγχου συνεργατών](https://fastcomments.com/auth/my-account/affiliates). Οι πληρωμές έχουν αποκατασταθεί σύμφωνα με το κανονικό πρόγραμμα.

### Οφέλη από τη Δική μας Διαχείριση

Τώρα που διαχειριζόμαστε το πρόγραμμα συνεργατών με τη δική μας τεχνολογία, δεν χρειάζεται να ανησυχούμε για τους αποκλειστές διαφημίσεων ή για άλλα προβλήματα.

Μπορούμε επίσης να παρέχουμε πολύ πιο λεπτομερή στατιστικά γύρω από τις μετατροπές. Μπορούμε επίσης να προσαρμόσουμε αυτήν την εμπειρία αναφοράς με την πάροδο του χρόνου χωρίς να είμαστε υποχρεωμένοι σε άλλο πάροχο.

### Τεκμηρίωση

[Μπορείτε να βρείτε την πλήρη τεκμηρίωση εδώ](https://docs.fastcomments.com/guide-affiliates.html)

Όπως σε όλες τις σημαντικές κυκλοφορίες, είμαστε χαρούμενοι που μπορέσαμε να αφιερώσουμε χρόνο για να βελτιστοποιήσουμε, να δοκιμάσουμε και να κυκλοφορήσουμε σωστά αυτό. Ενημερώστε μας παρακάτω αν ανακαλύψετε οποιοδήποτε πρόβλημα.
