# [postlink]Η Σελίδα Ελέγχου Είναι Τώρα Ζωντανή[/postlink]

{{#unless isPost}}
Η σελίδα ελέγχου του FastComments δείχνει τώρα μια ζωντανή ειδοποίηση όταν φτάνουν νέες σχόλια που ταιριάζουν με τα τρέχοντα φίλτρα σας.
{{/unless}}

{{#isPost}}

### Τι Νέο

Η σελίδα ελέγχου υπήρξε πάντα ένα ισχυρό εργαλείο για τη διαχείριση των σχολίων. Μπορούσατε να φιλτράρετε ανά κατάσταση, να κάνετε αναζήτηση με κείμενο, να φιλτράρετε ανά σελίδα ή χρήστη και να εκτελείτε μαζικές ενέργειες. Αλλά για να δείτε νέα σχόλια, έπρεπε να ανανεώσετε τη σελίδα.

Όχι πια! Η σελίδα ελέγχου δείχνει τώρα πότε φτάνουν νέα σχόλια που ταιριάζουν με τα τρέχοντα φίλτρα σας. Ένα πανό εμφανίζεται στην κορυφή της λίστας σχολίων, λέγοντας σας πόσα νέα σχόλια περιμένουν, και κάνοντας κλικ πάνω του, φορτώνονται εντός σελίδας χωρίς ανανέωση ολόκληρης της σελίδας.

Το μήνυμα "νέα σχόλια για τα φίλτρα σας" θα πρέπει να είναι σχεδόν άμεσο στις περισσότερες περιπτώσεις.

### Αξιολόγηση Φίλτρων

Η μεταφορική στρώση αξιολογεί όλα τα φίλτρα ελέγχου côté server. Αυτό περιλαμβάνει φίλτρα κατάστασης (χρειάζεται αναθεώρηση, χρειάζεται έγκριση, spam, επισημασμένα, αποκλεισμένα), φίλτρα συμβολοσειράς (αναγνωριστικό χρήστη, URL σελίδας, τομέας), και ακόμη και αναζητήσεις κειμένου. Το κείμενο με εισαγωγικά όπως <div class="code">"ακριβής φράση"</div> κάνει μια
ακριβή ευθυγράμμιση υποσυμβολοσειράς που δεν είναι ευαίσθητη σε πεζά-κεφαλαία, ενώ οι όροι χωρίς εισαγωγικά κάνουν θολές ενημερώσεις που δεν είναι ευαίσθητες σε πεζά-κεφαλαία.

Αυτό σημαίνει ότι αν φιλτράρετε για spam σχόλια, δεν θα ενημερωθείτε για τα εγκεκριμένα σχόλια. Αν αναζητάτε σχόλια που περιέχουν μια συγκεκριμένη φράση, μόνο τα σχόλια που ταιριάζουν ενεργοποιούν το πανό.

### Ομάδες Ελέγχου

Εάν ο λογαριασμός σας χρησιμοποιεί ομάδες ελέγχου για να περιορίσει ποιους ελεγκτές μπορούν να δουν ποια σχόλια, αυτές οι περιορισμοί εφαρμόζονται πλήρως για τις ζωντανές ειδοποιήσεις. Η μεταφορική στρώση καλεί πίσω τον server εφαρμογής για να δημιουργήσει το φίλτρο, το οποίο περιλαμβάνει περιορισμούς ομάδας με βάση τις ανατεθειμένες ομάδες του ελεγκτή. Αυτό το φίλτρο αποθηκεύεται στη μνήμη για τη διάρκεια της σύνδεσης WebSocket και ενημερώνεται κάθε φορά που ο ελεγκτής αλλάζει φίλτρα.

### Χαμένες Εκδηλώσεις

Εάν η σύνδεση ενός ελεγκτή διακοπεί προσωρινά (προβλήματα δικτύου, ύπνος φορητού υπολογιστή κ.λπ.), το σύστημα ανακτά αυτόματα τη σύνδεση κατά την επανασύνδεση, ώστε να μην χάσετε καμία εργασία.

### Απόδοση

Σχεδιάσαμε αυτή τη δυνατότητα ώστε να έχει ελάχιστη επίπτωση στον χρόνο φόρτωσης της σελίδας ελέγχου.

### Πολυπεριοχή

Αυτή η δυνατότητα λειτουργεί χωρίς προβλήματα με την παγκοσμίως κατανεμημένη αρχιτεκτονική μας, έτσι ώστε οι παγκοσμίως κατανεμημένες ομάδες ελέγχου να έχουν την ίδια ζωντανή θέα.

### Σε Συμπέρασμα

Ελπίζουμε αυτή η δυνατότητα να εξοικονομήσει χρόνο στους ελεγκτές και να κάνει την εμπειρία ελέγχου πιο ανταποκριτική. Όπως πάντα, ευχαριστούμε τους πελάτες μας για την ανατροφοδότηση και την υποστήριξή τους.

Cheers!
