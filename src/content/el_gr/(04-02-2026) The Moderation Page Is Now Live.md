# [postlink]Η Σελίδα Διαχείρισης Είναι Τώρα Ενεργή[/postlink]

{{#unless isPost}}
Η σελίδα διαχείρισης FastComments εμφανίζει τώρα μια ζωντανή ειδοποίηση όταν φτάνουν νέες σχόλια που πληρούν τα τρέχοντα φίλτρα σας.
{{/unless}}

{{#isPost}}

### Τι Είναι Νέο

Η σελίδα διαχείρισης ήταν πάντα ένα ισχυρό εργαλείο για τη διαχείριση σχολίων. Μπορούσατε να φιλτράρετε κατά κατάσταση, να αναζητήσετε με κείμενο, να φιλτράρετε κατά σελίδα ή χρήστη και να εκτελείτε μαζικές ενέργειες. Αλλά για να δείτε νέα σχόλια έπρεπε να ανανεώνετε τη σελίδα.

Όχι πια! Η σελίδα διαχείρισης δείχνει τώρα πότε φτάνουν νέα σχόλια που ταιριάζουν στα τρέχοντα φίλτρα σας. Ένα πανό εμφανίζεται στην κορυφή της λίστας σχολίων λέγοντάς σας πόσα νέα σχόλια περιμένουν, και με ένα κλικ, τα φορτώνει αυτά inline χωρίς πλήρη ανανέωση της σελίδας.

Το μήνυμα "νέα σχόλια για τα φίλτρα σας" θα πρέπει να είναι σχεδόν άμεσο στις περισσότερες περιπτώσεις.

### Αξιολόγηση Φίλτρων

Η μεταφορική στοιβάδα αξιολογεί όλα τα φίλτρα διαχείρισης στην πλευρά του διακομιστή. Αυτό περιλαμβάνει φίλτρα κατάστασης (χρειάζεται έγκριση, χρειάζεται έγκριση, spam, σηματοδοτημένα, αποκλεισμένα), φίλτρα συμβολοσειράς (ταυτότητα χρήστη, URL σελίδας, τομέας) και μάλιστα αναζητήσεις κειμένου. Το κείμενο σε εισαγωγικά όπως το `"exact phrase"` κάνει μια ακριβή, χωρίς ευαισθησία στη γραφή, αναζήτηση υποσυμβολοσειράς, ενώ οι μη εισαγωγικές όροι κάνουν θολές, χωρίς ευαισθησία στη γραφή, αναζητήσεις.

Αυτό σημαίνει ότι αν φιλτράρετε για σχόλια spam, δεν θα ειδοποιηθείτε για τα εγκεκριμένα σχόλια. Αν αναζητάτε σχόλια που περιέχουν μια συγκεκριμένη φράση, μόνο τα ταιριαστά σχόλια ενεργοποιούν το πανό.

### Ομάδες Διαχείρισης

Αν ο λογαριασμός σας χρησιμοποιεί ομάδες διαχείρισης για να περιορίσει ποιες διαχειρίσεις βλέπουν ποια σχόλια, αυτοί οι περιορισμοί επιδιώκονται πλήρως για τις ζωντανές ειδοποιήσεις. Η μεταφορική στοιβάδα καλεί τον διακομιστή εφαρμογών για να κατασκευάσει το φίλτρο, το οποίο περιλαμβάνει περιορισμούς ομάδας βασισμένους στις ανατεθειμένες ομάδες στον διαχειριστή. Αυτό το φίλτρο αποθηκεύεται στην μνήμη για τη διάρκεια της σύνδεσης WebSocket και ενημερώνεται κάθε φορά που ο διαχειριστής αλλάζει φίλτρα.

### Χαμένες Εκδηλώσεις

Αν η σύνδεση ενός διαχειριστή διακοπεί προσωρινά (προβλήματα δικτύου, ύπνος φορητού υπολογιστή, κ.λπ.), το σύστημα ανακάμπτει αυτόματα κατά την επανασύνδεση, ώστε να μην χάσετε καμία εργασία.

### Απόδοση

Σχεδιάσαμε αυτή τη δυνατότητα ώστε να έχει ελάχιστο αντίκτυπο στο χρόνο φόρτωσης της σελίδας διαχείρισης.

### Πολυ-Περιοχή

Αυτή η δυνατότητα λειτουργεί απρόσκοπτα με την παγκόσμια κατανεμημένη αρχιτεκτονική μας, έτσι ώστε οι παγκόσμια κατανεμημένες ομάδες διαχείρισης να αποκτούν την ίδια ζωντανή θέα.

### Συμπέρασμα

Ελπίζουμε αυτή η δυνατότητα να εξοικονομήσει χρόνο στους διαχειριστές και να καταστήσει την εμπειρία διαχείρισης πιο άμεση. Όπως πάντα, ευχαριστούμε τους πελάτες μας για την ανατροφοδότηση και την υποστήριξή τους.

Εις υγείαν!
