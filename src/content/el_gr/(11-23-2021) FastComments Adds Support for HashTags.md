### Τι Νέο

Κατά την υποβολή σχολίου, οι χρήστες μπορούν πλέον να εισάγουν ένα ή περισσότερα `#hashtags`. Με το κλικ σε ένα `#hashtag` σε ένα σχόλιο, θα σας οδηγήσει σε μια προβολή με άλλα σχόλια
με το hashtag που εμφανίζεται.

Επιπλέον, τα `#hashtags` μπορούν να χρησιμοποιηθούν για να συνδέονται με εξωτερικά κομμάτια περιεχομένου. Μαζί με αυτήν την έκδοση, έχουμε ξεκινήσει το [HashTag API](https://docs.fastcomments.com/guide-api.html#hash-tag-structure)
το οποίο επιτρέπει την προ-συμπλήρωση των hashtags στον λογαριασμό σας.

Κατά την προ-συμπλήρωση hashtags, μπορούμε να ορίσουμε μια URL για κάθε ετικέτα. Αυτό σημαίνει ότι τα `#hashtags` μπορούν επίσης να χρησιμοποιηθούν για να συνδέονται, για παράδειγμα, με έναν αριθμό εισιτηρίου, ή
ένα έγγραφο σε ένα CRM.

Το σύμβολο `#` μπορεί επίσης να προσαρμοστεί, κατόπιν αιτήματος.

### Πώς να το Χρησιμοποιήσετε

Όταν πληκτρολογείτε `#`, και στη συνέχεια την αρχή ενός hashtag, θα εμφανιστεί μια λίστα με αποτελέσματα αναζήτησης.

<div class="text-center">

</div>

Απλά κάντε κλικ στο hashtag που επιθυμείτε να προσθέσετε.

Επιπλέον, μπορείτε να χρησιμοποιήσετε τα πλήκτρα βέλους επάνω και κάτω για να πλοηγηθείτε σε αυτή τη λίστα, και να πατήσετε enter για να επιλέξετε, ή escape για να αποχωρήσετε.

Γνωρίζουμε ότι μπορεί να θέλετε να χρησιμοποιήσετε το σύμβολο `#` χωρίς να προσθέσετε ένα hashtag. Το FastComments θα το ανιχνεύσει αυτό και δεν θα παρεμβαίνει στην εμπειρία πληκτρολόγησής σας.

### Προσθήκη Hash Tags

Εάν το hashtag που θέλετε να χρησιμοποιήσετε δεν βρίσκεται στη λίστα προτάσεων - το FastComments θα δημιουργήσει αυτόματα το hashtag στο παρασκήνιο.

Η αυτόματη δημιουργία `#hashtag` μπορεί να απενεργοποιηθεί σύμφωνα με την τεκμηρίωση [εδώ](https://docs.fastcomments.com/guide-customizations-and-configuration.html#disable-automatic-hashtag-creation).

### Ποιοι το Λαμβάνουν

Όλοι οι τρέχοντες και νέοι πελάτες του FastComments, σε όλες τις κατηγορίες, έχουν τώρα πρόσβαση σε `#hashtags`. Τα `#hashtags` έχουν μεταφερθεί και σε παλαιότερες εκδόσεις του
widget σχολίων.

### Πώς Λειτουργεί η Αυτόματη Συμπλήρωση

Οι ετικέτες που εμφανίζονται στη λίστα αυτόματης συμπλήρωσης προέρχονται μόνο από τον λογαριασμό σας. Τα HashTags δεν μοιράζονται μεταξύ των ενοικιαστών του FastComments.

### Τεκμηρίωση

Υπάρχει ειδική τεκμηρίωση για τα `#hashtags`. Δείτε εδώ: https://docs.fastcomments.com/guide-hashtags.html

### Συμπέρασμα

Όπως και με το υπόλοιπο FastComments, ελπίζουμε ότι θα βρείτε αυτή τη λειτουργία γρήγορη και εύκολη στη χρήση.

Προς υγειά σας!
