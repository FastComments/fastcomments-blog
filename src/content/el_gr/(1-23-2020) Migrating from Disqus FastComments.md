Υπάρχουν πολλά πράγματα που κάνει σωστά το Disqus, αλλά εκεί που υστερεί είναι εκεί που εισέρχεται το FastComments.

## Αν Είστε Σε WordPress

Το FastComments διαθέτει έναν ειδικό plugin για WordPress: <a href="https://wordpress.org/plugins/fastcomments/" target="_blank">https://wordpress.org/plugins/fastcomments/</a>.

Ο plugin διευκολύνει τη διαδικασία εγκατάστασης, ρύθμισης και συγχρονισμού. Εάν ακολουθήσετε αυτό το μονοπάτι, μπορείτε να αγνοήσετε τις υπόλοιπες οδηγίες εδώ.

## Χειροκίνητη, Γενική Εγκατάσταση

### Εξάγετε Τα Υπάρχοντα Σχόλιά Σας

Ο εισαγωγέας του FastComments θα μεταφέρει τα σχόλιά σας, τα ονόματα χρηστών, τους avatar χρηστών και τις inline εικόνες. Θα μεταφέρουμε τις εικόνες στους διακομιστές μας χωρίς πρόβλημα.

Ένα πράγμα που κάνει σωστά το Disqus είναι πώς διαχειρίζεται την εξαγωγή δεδομένων. Θα παρατηρήσετε μετά την παρακολούθηση των βημάτων στον οδηγό τους <a href="https://help.disqus.com/en/articles/1717199-importing-exporting" target="_blank">εδώ</a> ότι το τελικό 
αρχείο που λαμβάνετε μέσω email έχει μια μυστηριώδη επέκταση αρχείου "gz". Εάν είστε λίγο περισσότερο τεχνικά καταρτισμένοι, ίσως να γνωρίζετε ότι αυτό σημαίνει "gzip" και είναι ένας δημοφιλής και αποδοτικός τρόπος για να συμπιέσετε αρχεία.

### Μην ανοίξετε το αρχείο "gz"

Μια προεπιλεγμένη εγκατάσταση Windows 10 δεν θα μπορεί να ανοίξει το αρχείο από το Disqus. Αυτό είναι εντάξει, γιατί δεν είναι απαραίτητο με το FastComments. Το backend μας κατανοεί
αυτό το συμπιεσμένο αρχείο, οπότε το μόνο που έχετε να κάνετε είναι να ακολουθήσετε τις οδηγίες τους <a href="https://help.disqus.com/en/articles/1717199-importing-exporting" target="_blank">εδώ για να λάβετε το εν λόγω αρχείο</a>.

Αφού το κάνετε αυτό, αν είστε συνδεδεμένοι μπορείτε να πάτε <a href="https://fastcomments.com/auth/my-account/manage-data/import" target="_blank">εδώ για να εισάγετε το αρχείο</a>. Επιλέξτε Disqus από το αναπτυσσόμενο μενού και ανεβάστε το αρχείο σας.

### Περιμένετε μερικά λεπτά

Η εισαγωγή του FastComments είναι "ασύγχρονη". Αυτό σημαίνει ότι η διαδικασία ανάρτησης του αρχείου και η επεξεργασία του είναι ξεχωριστά βήματα.

Έτσι, ανάλογα με το μέγεθος του αρχείου σας, μπορεί να λάβετε το μήνυμα επιτυχίας αμέσως. Στην εισαγωγική σελίδα, στο κάτω μέρος, υπάρχει ένας πίνακας - κάθε γραμμή αντιπροσωπεύει μια απόπειρα εισαγωγής.
Μπορείτε να ανανεώσετε ασφαλώς αυτή τη σελίδα για να δείτε την κατάσταση της εισαγωγής σας και πόσες γραμμές έχουν ήδη εισαχθεί.

### Όταν ολοκληρωθεί

Θα λάβετε ένα email όταν ολοκληρωθεί η εισαγωγή - είτε είναι επιτυχής είτε όχι. Μπορείτε να κλείσετε τη σελίδα αφού η ανάρτηση του αρχείου είναι επιτυχής και δείτε την εισαγωγή σας στη σελίδα εισαγωγών. Θα έχει κατάσταση "Ζητήθηκε" και όταν ξεκινήσει θα έχει κατάσταση "Σε εκτέλεση".

Είναι ασφαλές να κάνετε επανεισαγωγή όσες φορές απαιτείται - ωστόσο, αν η εισαγωγή σας αποτύχει, <a href="https://fastcomments.com/auth/my-account/help" target="_blank">επικοινωνήστε μαζί μας</a> για να μπορέσουμε να βοηθήσουμε.

### Αντικαταστήστε τον Κώδικα Disqus με το FastComments

Εάν χρησιμοποιείτε την Γενική Ενσωμάτωσή σας με το Disqus, τότε είναι απλώς θέμα αφαίρεσης του μικρού κομματιού κώδικα που σας παρέχουν και αντικατάστασής του με <a href="https://fastcomments.com/auth/my-account/get-acct-code" target="_blank">τον δικό μας</a>.
Μπορείτε ακόμη και να τρέχετε το Disqus και το FastComments στον ίδιο ιστότοπο προσωρινά για να δοκιμάσετε και να προσαρμόσετε την εμφάνιση και την αίσθηση. <a href="https://fastcomments.com/auth/my-account/help" target="_blank">Μπορούμε να βοηθήσουμε με αυτό</a>.

Εάν δεν χρησιμοποιείτε την Γενική Ενσωμάτωσή σας, τότε οι οδηγίες θα διαφέρουν ανάλογα με την πλατφόρμα σας - για άλλη μια φορά <a href="https://fastcomments.com/auth/my-account/help" target="_blank">επικοινωνήστε μαζί μας</a>.

### Μεταφορά URLs ταυτόχρονα

Τα σχόλια συνδέονται με τα πεδία "σύνδεσμος" στην εξαγωγή του Disqus, οπότε αρκεί οι διευθύνσεις URL σας να μην αλλάξουν για να είναι εύκολη η μεταφορά. Εάν θέλετε να μεταφέρετε διευθύνσεις URL και να διατηρήσετε τα 
σχόλιά σας, τότε <a href="https://fastcomments.com/auth/my-account/help" target="_blank">ενημερώστε μας</a> στέλνοντάς μας τις παλιές και νέες διευθύνσεις URL. Η σελίδα βοήθειας επιτρέπει την ανάρτηση μικρών αρχείων κειμένου, οπότε αν
μεταφέρετε περισσότερες από μερικές διευθύνσεις URL, δημιουργήστε ένα φύλλο Excel.

### Ανακεφαλαίωση

1. Εξάγετε τα δεδομένα σας
2. Εισάγετε στο FastComments
3. Αντικαταστήστε το JavaScript snippet του Disqus με <a href="https://fastcomments.com/auth/my-account/get-acct-code" target="_blank">του FastComments</a>

## Γιατί δε στέλνει το Disqus το αρχείο μου;
Για μεγαλύτερους ιστότοπους, το Disqus ενδέχεται να μην μπορεί να δημιουργήσει το εξαγόμενο αρχείο λόγω τεχνικών περιορισμών από την πλευρά τους. Έχουμε τη δυνατότητα να ανακτήσουμε τα υπάρχοντα σχόλιά σας από τον ιστότοπό σας, ωστόσο αυτό μπορεί να είναι
αρκετά προσαρμοσμένο και λόγω του χρόνου που απαιτεί θα χρειαστεί να έχετε το Πακέτο Υποστήριξης. <a href="https://fastcomments.com/auth/my-account/help" target="_blank">Μπορείτε να ζητήσετε βοήθεια με αυτό εδώ</a>.
