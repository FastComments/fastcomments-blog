# [postlink]Εύκολη Υποστήριξη Κώδικα με FastComments[/postlink]

Ορισμένοι από τους πελάτες μας έχουν ιστότοπους και μπλογκ που σχετίζονται με την ανάπτυξη λογισμικού. Σε αυτές τις κοινότητες είναι συνηθισμένο να μοιράζονται αποσπάσματα κώδικα. Ορίστε πώς το FastComments διευκολύνει αυτό.

{{#isPost}}

Το FastComments επιτρέπει την κοινή χρήση κώδικα απλά επικολλώντας τον στο σχόλιο. Μπορεί να συνδυαστεί με άλλο κείμενο και εικόνες, και δεν απαιτεί να προσθέσετε χειροκίνητα κανενός είδους "tags" κώδικα για να προσθέσετε τη μορφοποίηση. Απλά επικολλήστε!

Ορίστε μια γρήγορη επίδειξη που δείχνει τη λειτουργία του.

<video src="images/fc-code-snippet-demo.mp4" autoplay controls loop=true alt="Επίδειξη Κοινής Χρήσης Αποσπασμάτων Κώδικα" title="Επίδειξη Κοινής Χρήσης Αποσπασμάτων Κώδικα"></video>

### Επιπτώσεις στην Απόδοση

Η προσθήκη αυτής της δυνατότητας έχει προσθέσει μόνο ~150 bytes στο widget της πλευράς του πελάτη καθώς οι περισσότερες λειτουργίες διαχειρίζονται από την πλευρά του διακομιστή. Στην πραγματικότητα, αν το συνδυάσετε με τις πρόσφατες δυνατότητες Markdown, το μέγεθος του widget έχει μειωθεί με αυτές τις πρόσφατες δυνατότητες!

### Πράγματα που Πρέπει να Σημειωθούν

Αν έχετε μια κοινότητα που μοιράζεται έντονα κώδικα, θα πρέπει να σκεφτείτε την αύξηση του μέγιστου μεγέθους σχολίου για να διευκολύνετε την κοινή χρήση αποσπασμάτων κώδικα. Αυτό γίνεται μέσω της καρτέλας "Εξατομίκευση" του πίνακα διαχείρισης.

### Ανίχνευση Κώδικα

Για να διατηρήσουμε το μέγεθος του widget της πλευράς του πελάτη χαμηλό, θα ανιχνεύουμε αυτόματα τον επικολλημένο κώδικα μόνο για γλώσσες παρόμοιες με το C και την Python (Java, JavaScript, CSS, C++, για παράδειγμα) που είτε χρησιμοποιούν "{" είτε κενά για τη ροή ελέγχου. Αν χρησιμοποιείτε συχνά μια γλώσσα που δεν ανιχνεύεται αυτόματα, επικοινωνήστε μαζί μας.

Για να εισαγάγετε με το χέρι ένα μπλοκ κώδικα, γράψτε ένα σχόλιο όπως: ```<code>(defun someLispCode(1, 2, 3))</code>```. Μπορούμε να ανιχνεύσουμε αυτόματα περίπου 150 γλώσσες.

### Μεταφορά Υπαρχόντων Σχολίων

Αν θέλετε να μεταφέρετε σε FastComments και έχετε πολλά σχόλια που περιέχουν αποσπάσματα κώδικα που θα θέλατε να μορφοποιηθούν σωστά, <a href="https://fastcomments.com/auth/my-account/help" target="_blank">μπορούμε να σας βοηθήσουμε</a>.
