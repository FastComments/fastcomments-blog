### <i class="circle">!</i> Αυτό το Άρθρο Περιέχει Τεχνική Ορολογία

#### Τι Νέο Υπάρχει

Στο <a href="https://fastcomments.com">FastComments</a> μόλις κυκλοφορήσαμε μια επέκταση προγράμματος περιήγησης για το Google Chrome. Αυτή η επέκταση θα σας βοηθήσει να κατανοήσετε ποια ρύθμιση
λαμβάνει το widget σχολίων καθώς και γιατί μπορεί να μην εμφανίζεται.

Μπορείτε να βρείτε την επέκταση εδώ: <a href="https://chrome.google.com/webstore/detail/fastcomments-debugger/cadggdemhfkjjghkdbfhonoccnplffjj" target="_blank">https://chrome.google.com/webstore/detail/fastcomments-debugger/cadggdemhfkjjghkdbfhonoccnplffjj</a>

#### Πώς Να Την Χρησιμοποιήσετε

Εάν διαπιστώσετε ότι το widget σχολίων δεν φορτώνει, ή κάποιες ρυθμίσεις που έχετε ορίσει δεν εμφανίζονται, δοκιμάστε να εγκαταστήσετε <a href="https://chrome.google.com/webstore/detail/fastcomments-debugger/cadggdemhfkjjghkdbfhonoccnplffjj">την επέκταση</a>
και να την ανοίξετε στη σελίδα που σας ενδιαφέρει. Για να δείτε το εικονίδιο στη γραμμή εργαλείων σας, ίσως χρειαστεί να κάνετε κλικ στο κομμάτι παζλ στην επάνω δεξιά γωνία για να το καρφιτσώσετε όπως δείχνει:

<div class="text-center">
    <img src="images/fc-extension-pin.png" alt="Πώς να Καρφιτσώσετε την Επέκταση" title="Πώς να Καρφιτσώσετε την Επέκταση" class="lozad" />
</div>

Η επέκταση μπορεί να βοηθήσει στην αποσφαλμάτωση δύο κατηγοριών σφαλμάτων. Η πρώτη, εάν το widget δεν είναι καθόλου συμπεριληφθεί στη σελίδα ή αν υπάρχει μεγάλο ορθογραφικό λάθος στο script.

Η δεύτερη είναι εάν κάποια ρύθμιση δεν φαίνεται να έχει εφαρμοστεί.

Για να αποσφαλματώσετε και τις δύο, ξεκινάτε απλώς κάνοντας κλικ στο εικονίδιο.

Εάν το εργαλείο δεν βρει το widget σχολίων στη σελίδα, θα λάβετε ένα μεγάλο κόκκινο σφάλμα:

<div class="text-center">
    <img src="images/fc-extension-embed-js-not-found.png" alt="Παράδειγμα Embed.js όχι βρέθηκε" title="Παράδειγμα Embed.js όχι βρέθηκε" class="lozad" />
</div>

Εάν έχουν βρεθεί μία ή περισσότερες περιπτώσεις του widget, θα δείτε τις ρυθμίσεις τους να εμφανίζονται ξεχωριστά:

<div class="text-center">
    <img src="images/fc-extension-multiple-instances.png" alt="Παράδειγμα Πολλαπλών Ρυθμίσεων" title="Παράδειγμα Πολλαπλών Ρυθμίσεων" class="lozad" />
</div>

Επιπλέον, ο Debugger FastComments θα εμφανίσει επίσης προειδοποιήσεις για κοινά προβλήματα - όπως το widget να μην εμφανίζεται σε λειτουργία μόνο για ανάγνωση εάν δεν υπάρχουν διαθέσιμα σχόλια.

#### Ρυθμίσεις "Τελικής" Περίπτωσης;

Μπορεί να δείτε ότι ο debugger δείχνει δύο σύνολα ρυθμίσεων - ένα επισημασμένο ως "Περασμένο στο Widget" και ένα επισημασμένο ως "Τελικό".

Η ρύθμιση "Περασμένο στο Widget" είναι, όπως μπορείτε να μαντέψετε, η ρύθμιση που μεταβιβάζετε στο widget. Ορισμένες τιμές παρέχονται έμμεσα από το embed.js όπως η διεύθυνση URL ή ο Τίτλος Σελίδας - και αυτές περιλαμβάνονται.

Η "Τελική" ρύθμιση είναι όλα όσα χρησιμοποιεί το widget για να απεικονιστεί. Αυτό μπορεί να σημαίνει αυτοσυμπληρούμενες τιμές (όπως το id της περίπτωσης), ή <b>τιμές από τη σελίδα διαχείρισης προσαρμογής widget</b>.

#### Πώς Λειτουργεί

Το widget σχολίων αποθηκεύει τη ρύθμισή του σε μια θέση όπου ο debugger μπορεί να τη βρει, και στη συνέχεια ο debugger σαρώνει την τρέχουσα σελίδα στην οποία βρίσκεστε για περιπτώσεις του widget και τις φορτώνει στη μνήμη.

#### Ποιες Πληροφορίες Συλλέγει Αυτή η Επέκταση;

Το FastComments Debug Tool δεν τηλεφωνεί σπίτι ή επικοινωνεί με κανέναν διακομιστή του FastComments. Στην πραγματικότητα, το widget δεν θα ελέγξει καν το περιεχόμενο της σελίδας στην οποία βρίσκεστε
εκτός αν το ανοίξετε.

#### Συμπέρασμα

Ελπίζουμε ότι η παροχή περισσότερων εργαλείων όπως αυτό καθιστά την εγκατάσταση του FastComments πιο εύκολη για κάθε τύπο πελάτη που έχουμε. Ενημερώστε μας στην ενότητα σχολίων παρακάτω
πώς λειτούργησε αυτό για εσάς.

Γνωρίζουμε ότι σε ορισμένες περιπτώσεις μπορεί να χρειαστείτε ακόμη μια βοηθητική χείρα - και για αυτό υπάρχει ακόμη η <a href="https://fastcomments.com/auth/my-account/help" target="_blank">σελίδα υποστήριξης πελατών</a>.

Καλή συνέχεια!
