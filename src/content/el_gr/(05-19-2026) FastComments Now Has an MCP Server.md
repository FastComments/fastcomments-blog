# [postlink]Το FastComments Έχει Τώρα Έναν Διακομιστή MCP[/postlink]

{{#unless isPost}}
Μπορείτε τώρα να συνδέσετε τον Claude Code, Claude Desktop, Cursor ή οποιονδήποτε άλλο πελάτη πρωτοκόλλου Model Context στο FastComments και να κάνετε απευθείας κλήση στην API μας.
{{/unless}}

{{#isPost}}

### Τι είναι Νέο

Το FastComments τώρα παρέχει έναν φιλοξενούμενο [Model Context Protocol](https://modelcontextprotocol.io/) (MCP) διακομιστή. Το MCP είναι το ανοιχτό πρότυπο που χρησιμοποιούν οι βοηθοί κωδικοποίησης AI για να ενεργοποιούν εργαλεία server-side. Συνδέστε τον πελάτη σας σε μία URL και μπορεί να απαριθμήσει τα σχόλια σας, να τα μετριάσει, να διαχειριστεί χρήστες SSO, να διαμορφώσει widgets και οτιδήποτε άλλο κάνει η [API](https://docs.fastcomments.com/guide-api.html).

### Πού Βρίσκεται

Από τον πίνακα ελέγχου του FastComments, ανοίξτε **Ενσωμάτωση, στη συνέχεια Διακομιστής MCP**, ή επισκεφθείτε το απευθείας στη διεύθυνση
[fastcomments.com/auth/my-account/mcp-setup](https://fastcomments.com/auth/my-account/mcp-setup).
Επιλέξτε το API key που θέλετε να χρησιμοποιήσει ο πελάτης και η σελίδα δημιουργεί μία προ-συμπληρωμένη URL καθώς και αποσπάσματα για επικόλληση για τους αρχείου ρυθμίσεων του Claude Code και του Claude Desktop / Cursor.

### Claude Code

Μία εντολή για να εγγραφείτε:

    claude mcp add --transport http fastcomments 'https://fastcomments.com/mcp?tenantId=YOUR_TENANT_ID&API_KEY=YOUR_API_KEY'

Έπειτα, το `/mcp` μέσα σε μία συνεδρία δείχνει τη σύνδεση και απαριθμεί κάθε εργαλείο που εκθέτει ο διακομιστής. Ρωτήστε πράγματα όπως "δείξτε μου τα τελευταία 20 σημειωμένα σχόλια στο /blog/launch", "μπλοκάρετε αυτόν τον σχολιαστή σε όλες τις ιστοσελίδες μου", ή "ποιο είναι το ποσοστό σπαμ μας την τελευταία εβδομάδα". Ο Claude δρομολογεί μέσω της σωστής κλήσης εργαλείου.

### Claude Desktop και Cursor

Προσθέστε αυτό το μπλοκ στη ρύθμιση MCP του πελάτη σας (`claude_desktop_config.json` για Claude Desktop, `mcp.json` για Cursor):

```json
{
  "mcpServers": {
    "fastcomments": {
      "type": "http",
      "url": "https://fastcomments.com/mcp?tenantId=YOUR_TENANT_ID&API_KEY=YOUR_API_KEY"
    }
  }
}
```

Επανεκκινήστε τον πελάτη και ο διακομιστής θα εμφανιστεί δίπλα σε οποιουσδήποτε άλλους διακομιστές MCP που έχετε ρυθμίσει.

### Ασφάλεια

Το API key είναι ενσωματωμένο στη διεύθυνση URL, οπότε αντιμετωπίστε τη διεύθυνση URL ως μυστικό. Μην το επικολλάτε σε δημόσιες συνομιλίες, screenshots ή commits. Αν ένα κλειδί εκτεθεί, αλλάξτε το από τη [σελίδα API Keys](https://fastcomments.com/auth/my-account/api-secret) και ξαναεπικολλήστε το απόσπασμα από τη σελίδα ρύθμισης. Για να περιορίσετε τι μπορεί να κάνει ένας πράκτορας, δημιουργήστε ένα ειδικό API key ανά βοηθό και κλειδώστε το στο σωστό τομέα. Το αναδυόμενο μενού στη σελίδα ρύθμισης σας επιτρέπει να αλλάξετε ποιο κλειδί χρησιμοποιούν τα αποσπάσματα με μία κίνηση.

### Γιατί Το Δημιουργήσαμε

Οι άνθρωποι χρησιμοποιούν ολοένα και περισσότερα εργαλεία ενεργειών για καθημερινές εργασίες που σχετίζονται με προϊόντα SaaS. Οποιοσδήποτε συνδέει μία συνεδρία Claude ή Cursor για να ταξινομήσει τη σειρά μετριασμού του, ή έναν προσαρμοσμένο πράκτορα για να επισημάνει νήματα που αξίζει να καρφιτσωθούν, μπορεί τώρα να παραλείψει πλήρως την επικάλυψη.

### Συμπερασματικά

Ανοίξτε [Ενσωμάτωση, στη συνέχεια Διακομιστής MCP](https://fastcomments.com/auth/my-account/mcp-setup) στον πίνακα ελέγχου σας, αντιγράψτε το απόσπασμα για τον αγαπημένο σας πελάτη MCP και είστε έτοιμοι. Πλήρης τεκμηρίωση είναι στο [LLM Kit guide](https://docs.fastcomments.com/guide-llm-kit.html).

Όπως πάντα, ενημερώστε μας παρακάτω αν αντιμετωπίσετε οτιδήποτε.

Cheers!
