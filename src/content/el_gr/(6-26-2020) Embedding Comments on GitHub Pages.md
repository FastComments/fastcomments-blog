#### Για Προγραμματιστές
Η εγκατάσταση του FastComments σε μια ιστοσελίδα απαιτεί μόνο την εισαγωγή ενός μικρού αποσπάσματος κώδικα. Αυτό σημαίνει ότι λειτουργεί σε στατικές σελίδες χωρίς καμία δυσκολία - συμπεριλαμβανομένης της δωρεάν φιλοξενίας Σελίδων του GitHub.

#### Μια Επίδειξη

Έχουμε ρυθμίσει μια παράδειγμα Σελίδα GitHub <a href="https://fastcomments.github.io/fastcomments-demo.github.io/" target="_blank">εδώ</a>.

<a href="https://github.com/fastcomments/fastcomments-demo.github.io/blob/master/index.html#L29" target="_blank">Εδώ</a> είναι ο σύνδεσμος προς το σημείο όπου έχουμε προσθέσει το απόσπασμα κώδικα για τον
λογαριασμό δοκιμής μας.

#### Ρύθμιση

Θα χρειαστεί να προσθέσετε το domain σας στο GitHub ως έγκυρο domain στον λογαριασμό σας.
