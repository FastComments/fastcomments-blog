# [postlink]Προσαρμογή του Προεπιλεγμένου Avatar[/postlink]

{{#unless isPost}}
Η προεπιλεγμένη εικόνα avatar μπορεί τώρα να προσαρμοστεί.
{{/unless}}

{{#isPost}}

### <i class="circle">!</i> Αυτό το Άρθρο Περιέχει Τεχνική Ορολογία

#### Εισαγωγή

Το προεπιλεγμένο εικονίδιο είναι το ίδιο από τότε που ξεκίνησε το FastComments. Δείτε τη δόξα του προεπιλεγμένου εικονιδίου προφίλ.

<div class="text-center">
    <img src="https://fastcomments.com/images/unknown-person.png" title="Default Profile Icon" alt="Default Profile Icon" />
</div>

Ωστόσο τώρα, μπορείτε να το προσαρμόσετε.

#### Χωρίς Κώδικα

Πρώτα, <a href="https://fastcomments.com/auth/my-account/customize-widget" target="_blank">προσθέστε ή επεξεργαστείτε έναν κανόνα προσαρμογής</a>. Οι κανόνες προσαρμογής μπορούν
να εφαρμοστούν μόνο σε μια σελίδα, ένα τομέα ή σε ολόκληρο τον λογαριασμό σας. Αν θέλετε να αλλάξετε την εικόνα προφίλ παγκοσμίως μπορείτε να αφήσετε το πεδίο ID URL κενό.

Πηγαινετε κάτω στην ενότητα "Προεπιλεγμένο Avatar".

<img 
    src="/images/fc-customize-default-icon-button.png"
    alt="FastComments Hackaday Settings"
    class='lozad' />
    
Κάντε κλικ στο κουμπί για να ανεβάσετε ένα JPG, PNG ή GIF έως 20mb. Θα μετατραπεί σε ένα τετράγωνο 100px.

#### Σε Κώδικα

Αν θέλετε να φιλοξενήσετε το προεπιλεγμένο avatar, και έχετε ήδη μια δημόσια διεύθυνση URL για αυτό, μπορεί να οριστεί σε κώδικα όπως στο δοθέν παράδειγμα.

<div class="code">    window.FastCommentsUI(document.getElementById('fastcomments-widget'), {
        tenantId: 'demo',
        defaultAvatarSrc: 'https://example.com/some-100-x-100-image.png'
    });
</div>

Η εικόνα θα πρέπει να είναι τετράγωνη και το μέγεθός της θα πρέπει να είναι 100px τετράγωνο. Ακολουθώντας αυτούς τους περιορισμούς θα εξασφαλίσετε την καλύτερη εμπειρία.

#### Τυχόν Προβλήματα

1. Η προσαρμογή της προεπιλεγμένης εικόνας προφίλ επηρεάζει μόνο τη δική σας ιστοσελίδα. Δεν επηρεάζει τα email ειδοποίησης ή τους λογαριασμούς χρηστών με κανέναν τρόπο.
2. Τα σχόλια που δημιουργήθηκαν πριν από αυτή την αλλαγή μπορεί να εξακολουθούν να εμφανίζουν την παλιά εικόνα προφίλ. Αν ρυθμίσετε το προεπιλεγμένο εικονίδιο προφίλ και παρατηρήσετε πολλά σχόλια στην ιστοσελίδα σας
χρησιμοποιώντας το παλιό εικονίδιο προφίλ, <a href="https://fastcomments.com/auth/my-account/help" target="_blank"> ενημερώστε μας</a>.

#### Συμπέρασμα

Ελπίζουμε να βρήκατε αυτόν τον οδηγό χρήσιμο και εύκολο στην ανάγνωση. Μη διστάσετε να σχολιάσετε παρακάτω με οποιεσδήποτε ερωτήσεις.

Στην υγειά σας!
