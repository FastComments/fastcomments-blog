# [postlink]Δημοσιεύτηκε το Αρχείο Ελέγχου[/postlink]

{{#unless isPost}}Το FastComments τώρα παρέχει ένα λεπτομερές αρχείο όλων των τροποποιήσεων που έγιναν μέσω της πλατφόρμας.{{/unless}}

{{#isPost}}

### Τι Νέο

Από τον Μάρτιο, το FastComments περιλαμβάνει τώρα λεπτομερή έλεγχο με το πρόγραμμα Pro, και για όλους τους υπάρχοντες πελάτες επιχειρήσεων.

Όλες οι διοικητικές ενέργειες, είτε πρόκειται για πρόσβαση σε δεδομένα, τροποποίηση ενός σχολίου, ή αποδέσμευση ενός χρήστη, παρακολουθούνται και καταγράφονται στο αρχείο ελέγχου.

### Πώς να το Χρησιμοποιήσετε

Το Αρχείο Ελέγχου μπορεί να προσπελαστεί πηγαίνοντας στο `Manage Data` και έπειτα στα `Audit Logs`. Ένας άμεσος σύνδεσμος είναι [εδώ](https://fastcomments.com/auth/my-account/manage-data/audit-logs).

Για να ξεκινήσετε, θα δείτε μια σελίδα όπως η παρακάτω:

<div class="text-center">
    <img src="/images/audit-log-main.png" alt="Audit Log" />
</div>

Από εδώ, μπορείτε να κάνετε αναζήτηση με βάση το όνομα χρήστη, τη διεύθυνση IP, τον τύπο της ενέργειας, ή το πόρο.

Για παράδειγμα, φιλτράροντας κατά πόρο:

<div class="text-center">
    <img src="/images/audit-log-resource.png" alt="Audit Log Resource Selection" />
</div>

Μπορούμε επίσης να εμβαθύνουμε στις λεπτομέρειες μιας καταχώρησης στο αρχείο ελέγχου, και να δούμε ακόμα και το σχετικό αντικείμενο που άλλαξε.

Για παράδειγμα, μπορούμε να εμβαθύνουμε σε μια καταχώρηση για αποκλεισμό ενός χρήστη:

<div class="text-center">
    <img src="/images/audit-log-details.png" alt="Audit Log Entry Details" />
</div>

Εδώ μπορούμε να δούμε ποιος απέκλεισε τον χρήστη (ένας χρήστης με το όνομα χρήστη `winrid`). Μπορούμε να δούμε ότι είναι μόνιμος αποκλεισμός, και έχουμε επίσης το id χρήστη και το email που αποκλείστηκε, μαζί με το κείμενο για το οποίο αποκλείστηκε.

Μπορούμε να δούμε τη διεύθυνση IP που δημιούργησε τον αποκλεισμό, αλλά η διεύθυνση IP του αρχικού σχολιαστή παραλείπεται σκόπιμα για την ιδιωτικότητά του.

Για να δούμε όλες τις ενέργειες από τον `winrid`, μπορούμε απλά να κάνουμε κλικ στο εικονίδιο φίλτρου στα δεξιά, δίπλα στο όνομα χρήστη τους.

### Περιορισμός Πρόσβασης

Δεδομένα στο αρχείο ελέγχου μπορεί να προσπελαστούν μόνο από χρήστες με την άδεια `Manage Data`.

### Διατήρηση

Όλες οι καταχωρήσεις στο αρχείο ελέγχου διατηρούνται για δύο χρόνια. Ωστόσο, το αρχείο ελέγχου διαγράφεται όταν διαγραφεί ο λογαριασμός σας.

### Πρόσβαση API

[Η τεκμηρίωση για το API του Αρχείου Ελέγχου είναι εδώ.](https://docs.fastcomments.com/guide-api.html#audit-log-structure) Θα διαπιστώσετε ότι το αρχείο ελέγχου μπορεί να
ταξινομηθεί και να σελιδοποιηθεί με διάφορους τρόπους ανάλογα με την περίπτωση χρήσης σας.

### Υπάρχοντες Πελάτες

Εάν είστε στο πρόγραμμα Pro ή ισοδύναμο, θα διαπιστώσετε ότι το αρχείο ελέγχου έχει ενεργοποιηθεί αυτόματα! Εάν θέλετε να αποκτήσετε πρόσβαση, παρακαλούμε επικοινωνήστε με τον εκπρόσωπο υποστήριξής σας.

### Τι έχει Αποφευχθεί

Ο μόνος τύπος αλλαγών που αποφεύγεται είναι τα πραγματικά σχόλια και οι ψήφοι τους. Αυτό συμβαίνει επειδή έχουμε διαπιστώσει ότι οι πελάτες μας ενδιαφέρονται κυρίως για την διοικητική πλευρά κατά τον έλεγχο, και όχι απαραίτητα για τον έλεγχο των σχολίων που κάνουν οι άνθρωποι. Αυτοί οι τύποι ενεργειών προσθέτουν επίσης σημαντικό θόρυβο στο αρχείο.

Ωστόσο, αν αυτό είναι απαίτηση, επικοινωνήστε μαζί μας, και μπορούμε να συζητήσουμε την ενεργοποίηση αυτού για εσάς.

### Στην Κατακλείδα

Ελπίζουμε ότι θα βρείτε αυτήν την περιοχή της προσφοράς μας χρήσιμη, το UI εύχρηστο, και το API γρήγορο.

Καλή συνέχεια!
