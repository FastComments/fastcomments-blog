# [postlink]Νέα Ειδοποίηση Συνδεσιμότητας IP[/postlink]

### Τι Νέο

Η FastComments στέλνει πλέον αυτόματα ένα email ειδοποίησης ασφαλείας όταν ανιχνεύουμε είσοδο στον λογαριασμό σας από μία διεύθυνση IP που δεν έχουμε ξαναδεί. Το email περιλαμβάνει:

- Τη **περίπου τοποθεσία** (πόλη και χώρα) της σύνδεσης.
- Τη **διεύθυνση IP** που χρησιμοποιήθηκε.
- Τη **ώρα** της σύνδεσης.
- Έναν άμεσο σύνδεσμο για **να αλλάξετε τον κωδικό σας** αν η είσοδος δεν ήταν δική σας.

Αυτό ισχύει για όλες τις μεθόδους σύνδεσης: σύνδεση με κωδικό πρόσβασης, μαγικούς συνδέσμους και ροές επαλήθευσης δύο παραγόντων.

### Ιδιωτικότητα

Δεν αποθηκεύουμε τη διεύθυνση IP σας. Μόνο αποθηκεύουμε μια κρυπτογραφημένη μορφή για σκοπούς ασφαλείας. Η περίπου τοποθεσία καθορίζεται τη στιγμή της σύνδεσης και δεν αποθηκεύεται.

**Η IP σας δεν μοιράζεται με τρίτο μέρος** για τον προσδιορισμό της τοποθεσίας. Φιλοξενούμε τη δική μας υπηρεσία που εξάγει την περίπου τοποθεσία από ένα (εβδομαδιαία ενημερωμένο) αντίγραφο της βάσης δεδομένων MaxMind.

### Όταν Στέλνονται οι Ειδοποιήσεις

Μία ειδοποίηση αποστέλλεται όταν **όλα** τα παρακάτω είναι αληθή:

- Η σύνδεση ήταν επιτυχής.
- Δεν είναι η πρώτη φορά που ο χρήστης συνδέεται.
- Η διεύθυνση IP δεν έχει παρατηρηθεί σε προηγούμενη επιτυχής σύνδεση για αυτόν τον λογαριασμό.

Αυτό σημαίνει ότι δεν θα λάβετε ειδοποίηση στην πρώτη σας σύνδεση και δεν θα λάβετε επαναλαμβανόμενες ειδοποιήσεις για την ίδια διεύθυνση IP.

### Συμπέρασμα

Αυτό είναι ένα ακόμη βήμα για την ασφάλεια του λογαριασμού σας. Αν λάβετε μία ειδοποίηση που δεν αναγνωρίζετε, προτείνουμε να αλλάξετε αμέσως τον κωδικό σας.

Ενημερώστε μας παρακάτω αν έχετε οποιαδήποτε σχόλια!

Καλή συνέχεια!
