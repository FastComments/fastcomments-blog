## Τι Νέο

Το FastComments μπορεί τώρα να ενσωματωθεί με άλλες εφαρμογές και τρίτους μεταπωλητές μέσω της Λευκής Σήμανσης. Η Λευκή Σήμανση σας επιτρέπει να δημιουργήσετε λογαριασμούς FastComments (που ονομάζονται ενοικιαστές), χρήστες, συντονιστές και πολλά άλλα.

Τώρα προσφέρουμε ένα [συνολικό API](https://docs.fastcomments.com/guide-white-labeling.html#white-labeling-using-the-api) για να αυτοματοποιήσετε την ενσωμάτωση
με τρίτους.

Μπορείτε τώρα, μέσω του API:

- Να δημιουργήσετε πελάτες (παιδικούς ενοικιαστές) που διαχειρίζονται από τον ενοικιαστή σας.
- Να διαχειριστείτε τα πακέτα που είναι διαθέσιμα στους πελάτες σας και πώς θα διαχειριστείτε την χρέωση.
- Να προσθέσετε και να διαχειριστείτε χρήστες στους διαχειριζόμενους ενοικιαστές σας.
- Να προσθέσετε και να διαχειριστείτε συντονιστές στους διαχειριζόμενους ενοικιαστές σας.
- Να προσκαλέσετε συντονιστές και να στείλετε συνδέσμους σύνδεσης σε χρήστες ενοικιαστών.

## Χρέωση

Με την χρέωση, υπάρχουν δύο επιλογές:

1. Μέσω του FastComments Flex, η χρήση των παιδικών ενοικιαστών σας περιλαμβάνεται αυτόματα και χρεώνεται στον γονικό ενοικιαστή. Αυτό μπορεί να παρακολουθείται μέσω της [Σελίδας Ανάλυσης Χρέωσης](https://fastcomments.com/auth/my-account/analytics/billing).
2. Μέσω του FastComments Pro, έχετε πρόσβαση για να δημιουργήσετε έναν σταθερό αριθμό παιδικών ενοικιαστών για μια σταθερή μηνιαία τιμή.

## Αφαίρεση Μάρκας

Οι διαχειριζόμενοι ενοικιαστές μπορούν να έχουν ενεργοποιημένη την αφαίρεση μάρκας, η οποία αφαιρεί τα λογότυπά μας από το widget σχολίων.

## Πώς να αποκτήσετε Λευκή Σήμανση

Αυτή τη στιγμή, τα προγράμματα FastComments Flex και Pro προσφέρουν πρόσβαση στο API Λευκής Σήμανσης.

## Σενάρια και Παραδείγματα

Ένα παράδειγμα σεναρίου που χρησιμοποιεί το API για να ρυθμίσει έναν ενοικιαστή με χρήστες και συντονιστές [είναι διαθέσιμο εδώ](https://github.com/FastComments/fastcomments-code-examples/tree/master/white-labeling).

## Άλλες Χρήσεις

Τα APIs [χρηστών ενοικιαστών](https://docs.fastcomments.com/guide-api.html#tenant-user-structure) και [συντονιστών](https://docs.fastcomments.com/guide-api.html#moderator-structure) μπορούν επίσης να χρησιμοποιηθούν για
να προσθέσετε και να διαχειριστείτε αυτούς τους πόρους στον δικό σας ενοικιαστή, ανεξαρτήτως της λευκής σήμανσης.
