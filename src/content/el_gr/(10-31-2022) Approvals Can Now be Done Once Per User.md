### Τι νέο υπάρχει

Προβληματικά, αν η χειροκίνητη έγκριση ήταν ενεργοποιημένη, κάθε σχόλιο έπρεπε να μετριαστεί. Τώρα, αν ***Μόνο Αυτόματη Έγκριση για Εγκεκριμένους Χρήστες*** είναι ενεργοποιημένο,
ένα σχόλιο χρήστη δεν απαιτεί έγκριση εάν έχει εγκριθεί προηγουμένως.

Αυτό λειτουργεί ως μια μορφή πύλης σε μια κοινότητα, κάτι που μπορεί να είναι επιθυμητό για κοινότητες που θέλουν να έχουν εγκρίσεις αλλά δεν θέλουν να
εγκρίνουν κάθε σχόλιο.

### Πώς να το αποκτήσετε

Μπορείτε να ενεργοποιήσετε αυτή τη δυνατότητα μέσω των [Ρυθμίσεων Μετρίασης Σχολίων](https://fastcomments.com/auth/my-account/moderate-comments/settings), προσβάσιμες μέσω
`Μετρίαση Σχολίων -> Ρυθμίσεις`.

Κάτω από `Έγκριση Σχολίων`, επιλέξτε `Αυτόματη Έγκριση Σχολίων` και στη συνέχεια `Μόνο Αυτόματη Έγκριση για Εγκεκριμένους Χρήστες`.

### Διατήρηση Υπαρχουσών Λειτουργιών

Αν εγκρίνετε χειροκίνητα κάθε σχόλιο σήμερα, και θέλετε να συνεχίσετε να το κάνετε, απλά μην κάνετε τίποτα. Δεν θα ενεργοποιήσουμε αυτή τη δυνατότητα αυτόματα
ή εκ προεπιλογής.

### Στο Συμπέρασμα

Γνωρίζουμε ότι αυτή η δυνατότητα έχει αναμενθεί από κάποιους. Όπως σε όλες τις μεγάλες εκδόσεις, είμαστε χαρούμενοι που μπορέσαμε να αφιερώσουμε χρόνο για να βελτιστοποιήσουμε, να δοκιμάσουμε και να απελευθερώσουμε σωστά αυτή τη δυνατότητα.

Υγεία!
