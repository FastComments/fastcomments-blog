# [postlink]Η Διευκόλυνση της Διαχείρισης Σχολίων[/postlink]

{{#unless isPost}}Μπορείτε τώρα να διαχειρίζεστε τα σχόλια χωρίς να αφήσετε την ιστοσελίδα σας!{{/unless}}

{{#isPost}}

### Τι Νέο

Προηγουμένως, εάν θέλατε να διαχειριστείτε τα σχόλια, μπορούσατε να το κάνετε μόνο από τη σελίδα Διαχείρισης Σχολίων.

Αν και αυτή η σελίδα είχε αρκετές βελτιστοποιήσεις για να διευκολύνει τη διαχείριση, όπως οι μαζικές ενέργειες, υπάρχουν πλεονεκτήματα στο να μην χρειάζεται να αφήσετε την ιστοσελίδα σας. Δηλαδή, όταν οι συντονιστές αρχίζουν τη δουλειά τους μπορούν εύκολα να βλέπουν το περιβάλλον γύρω από κάθε σχόλιο καθώς εργάζονται μέσα στα νήματα σχολίων. Αυτό σημαίνει επίσης ότι μπορείτε να επεξεργάζεστε τα σχόλια **ζωντανά** καθώς εισέρχονται.

Έτσι τώρα, μπορείτε να διαχειρίζεστε απευθείας στην ιστοσελίδα σας χρησιμοποιώντας το FastComments!

Όταν είστε συνδεδεμένοι, οι συντονιστές θα έχουν ένα μικρό εικονίδιο επεξεργασίας στην πάνω δεξιά γωνία κάθε σχολίου. Κάνοντας hover σε αυτό θα εμφανιστεί ένα μενού με μερικές νέες επιλογές.

Αυτές οι επιλογές θα εμφανίζονται επίσης στον Θέαση Νημάτων Σχολίων.

### Τεκμηρίωση

Μπορείτε να βρείτε τεκμηρίωση που καλύπτει ποιες ενέργειες διαχείρισης είναι διαθέσιμες και πώς αυτό επηρεάζει τη ζωντανή σχολιασμό <a href="https://docs.fastcomments.com/guide-moderation.html#comment-actions-from-comment-widget" target="_blank">εδώ</a>.

### Ποιος Το Έχει

Οι πελάτες σε όλα τα επίπεδα έχουν πλέον ενεργοποιημένη αυτή τη δυνατότητα.

### Επίπτωση Απόδοσης

Ο κώδικας και τα στοιχεία που σχετίζονται με τη διαχείριση με αυτόν τον τρόπο κατεβαίνουν μόνο στους περιηγητές των συντονιστών σας - δεν επηρεάζει τους χρόνους φόρτωσης της σελίδας για τους επισκέπτες της ιστοσελίδας σας.

### Συμπερασματικά

Ελπίζουμε να βρείτε ότι αυτό διευκολύνει τη δουλειά σας ως συντονιστής.

Καλή σχολιασμένη διάθεση!
