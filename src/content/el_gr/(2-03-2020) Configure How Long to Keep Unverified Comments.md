# [postlink]Ρυθμίστε πόσο καιρό να κρατάτε μη επαληθευμένα σχόλια[/postlink]

Από προεπιλογή, το FastComments θα κρατάει τα μη επαληθευμένα σχόλιά σας για τρεις ημέρες.

### Τι σημαίνει το "μη επαληθευμένο";

Όταν ένας χρήστης σχολιάζει και αφήνει το email του, το FastComments θα του στείλει ένα email ζητώντας του να επαληθεύσει το σχόλιό του.
Αυτό βοηθά στην καταπολέμηση του spam και αναδεικνύει τα σχόλια που οι συγγραφείς τους θέλουν πραγματικά να εμφανίζονται.

Αν ενεργοποιήσετε <a href="https://blog.fastcomments.com/(2-01-2020)-how-to-allow-anonymous-comments-with-fastcomments.html" target="_blank">Ανώνυμο Σχολιασμό</a>
δεν θα χρειαστεί καν να αφήσουν το email τους!

Αλλά τώρα μπορείτε να ρυθμίσετε πόσο καιρό να κρατάτε αυτά τα σχόλια - ίσως να μην θέλετε καν να τα διαγράψετε αυτόματα! Αυτό είναι πλήρως ρυθμιζόμενο μέσω του Moderate Comments -> Edit Moderation Settings

{{#isPost}}
