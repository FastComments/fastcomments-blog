### Το Πρόβλημα

Έχετε πολύ περιεχόμενο και πολλά σχόλια. Αυτό είναι καλό - η βάση χρηστών σας είναι αρκετά ενεργή ώστε να αλληλεπιδρά μαζί σας. Αλλά τώρα είστε υπερφορτωμένοι 
και χρειάζεστε να επεκτείνετε τη λειτουργία σας.

### Διαχείριση Σχολίων με το FastComments

Τα εργαλεία διαχείρισης σχολίων του FastComments παρέχουν ένα αποτελεσματικό περιβάλλον για την εξέταση των σχολίων, την επισήμανση τους ως spam, ή (αν χρειάζεται) τη διαγραφή τους.

Ο πίνακας ελέγχου είναι επίσης ανταγωνιστικός, οπότε μπορείτε να εξετάσετε και να διαχειριστείτε τα σχόλια ενώ είστε εν κινήσει!

Αλλά η πρόσβαση στη διαχείριση των σχολίων για την ιστοσελίδα σας δεν περιορίζεται μόνο στον προσωπικό σας λογαριασμό. Κάποιος μπορεί να δημιουργήσει τον δικό του λογαριασμό FastComments και να προσκληθεί
στην ιστοσελίδα σας μέσω του πίνακα διαχείρισης, [ή μαζικά μέσω του API](https://docs.fastcomments.com/guide-api.html#moderator-structure). Δεν χρειάζεται καν να γνωρίζει για το FastComments ή να δημιουργήσει λογαριασμό πριν. Ο σύνδεσμος πρόσκλησης που θα σας δείξουμε
πώς να στείλετε θα τους καθοδηγήσει στη διαδικασία δημιουργίας λογαριασμού.

Απλά συνδεθείτε στον λογαριασμό σας και περάστε τον δείκτη του ποντικιού επάνω από Διαχείριση Σχολίων. Θα δείτε μια επιλογή για Διαχείριση Διαχειριστών.

Αν δεν έχετε προσθέσει κανέναν διαχειριστή ακόμα, αυτό το κουμπί θα σας μεταφέρει σε έναν οδηγό που θα σας καθοδηγήσει να προσθέσετε έναν διαχειριστή.

Θα χρειαστεί να παρέχετε μόνο ένα όνομα και ένα email. Αυτόματα θα τους σταλεί ένας σύνδεσμος πρόσκλησης που θα τους καθοδηγήσει επίσης στη δημιουργία λογαριασμού.

Προαιρετικά, μπορείτε επίσης να τοποθετήσετε διαχειριστές και σχόλια σε ξεχωριστές ομάδες. [Μάθετε περισσότερα για τις Ομάδες Διαχείρισης και άλλες δυνατότητες διαχείρισης εδώ.](https://docs.fastcomments.com/guide-moderation.html)

Εάν έχετε μια ομάδα διαχειριστών μπορείτε να δείτε την πρόοδό τους από το ίδιο σημείο - η λίστα Διαχειριστών Σχολίων παρέχει στατιστικά για κάθε διαχειριστή. Αυτό περιλαμβάνει 
τον αριθμό των σχολίων που έχουν επισημανθεί ως ελεγμένα, spam, και διαγραμμένα.

Αυτό είναι όλο. Αφού στείλετε αυτόν τον σύνδεσμο πρόσκλησης, μπορούν να δημιουργήσουν τον λογαριασμό τους και θα έχουν πλήρη πρόσβαση για να διαχειριστούν τα σχόλιά σας. Θα δουν μόνο μια απλοποιημένη
έκδοση της σελίδας Διαχείρισης Σχολίων - δεν παρέχεται δυνατότητα αλλαγής ρυθμίσεων στους διαχειριστές.

Καλή σχολιογράφηση!
