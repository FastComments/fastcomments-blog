# [postlink]Βελτιώσεις στην Ανάρτηση Εικόνων[/postlink]

{{#unless isPost}}
 διάφορες βελτιώσεις στην ανάρτηση εικόνων και τον επεξεργαστή WYSIWYG σε αυτή την ενημέρωση.
{{/unless}}

{{#isPost}}

### Τι νέο υπάρχει

Σήμερα κυκλοφόρησε μια σειρά από διορθώσεις σφαλμάτων για την ανάρτηση εικόνων και τη λειτουργία επεξεργασίας WYSIWYG (ό,τι βλέπεις είναι αυτό που παίρνεις).

1. Η ένδειξη προόδου εμφανίζεται πλέον κατά την ανάρτηση εικόνων.
2. Η επικόλληση εικόνων, όπως οι στιγμιότυπα οθόνης, υποστηρίζεται πλέον (μόνο για τον επεξεργαστή WYSIWYG).
3. Κατά την επεξεργασία σχολίων, οι πλαίσιες σχολίων έχουν πλέον σωστό μέγεθος για να εμφανίζουν όλες τις εικόνες (μόνο για τον επεξεργαστή WYSIWYG).
4. Όταν αφαιρούνται εικόνες, το πλαίσιο σχολίου μειώνεται όπως αναμένεται (μόνο για τον επεξεργαστή WYSIWYG).
