# [postlink]Πώς να Απαιτείτε Έγκριση για Όλα τα Σχόλια[/postlink]

Από προεπιλογή, δεν απαιτείται διαδικασία έγκρισης με το FastComments. Οι χρήστες σας σχολιάζουν και τα σχόλιά τους είναι άμεσα διαθέσιμα στο
όλο το διαδίκτυο (αν και με την ένδειξη μη επαληθευμένου αν δεν έχουν συνδεθεί). Τι γίνεται όμως αν θέλετε να μετριάσετε και
να εγκρίνετε χειροκίνητα κάθε σχόλιο;

{{#isPost}}

Η απενεργοποίηση της αυτόματης έγκρισης των σχολίων σημαίνει ότι εσείς ή οι συντονιστές σας θα πρέπει να εγκρίνουν κάθε σχόλιο πριν εμφανιστεί δημόσια.

Αρχικά, μεταβείτε στη σελίδα Ρυθμίσεων Μέτρησης, μέσω Μετρίαση Σχολίων -> Επεξεργασία Ρυθμίσεων Μέτρησης.

Σε αυτή τη σελίδα υπάρχει μια επιλογή που ονομάζεται "Αυτόματη Έγκριση Σχολίων". Αφαιρέστε την επιλογή και πατήστε αποθήκευση.

Αυτό είναι. Όλα τα σχόλιά σας θα απαιτούν τώρα έγκριση. 

<img 
    src="images/fc-auto-approval.png"
    alt="Ρύθμιση Αυτόματης Έγκρισης"
    title="Ρύθμιση Αυτόματης Έγκρισης"
    class='lozad' />

Για να εγκρίνετε σχόλια, μπορείτε να φιλτράρετε όπως επιθυμείτε και να κάνετε κλικ στην επιλογή "Σημειώστε ως Εγκεκριμένο". Εάν η επιλογή μετατραπεί σε "Σημειώστε ως Μη Εγκεκριμένο", αυτό σημαίνει ότι αποθηκεύτηκε.

<img 
    src="images/fc-mark-approved.png"
    alt="Πώς να Σημειώσετε ένα Σχόλιο ως Εγκεκριμένο"
    title="Πώς να Σημειώσετε ένα Σχόλιο ως Εγκεκριμένο"
    class='lozad' />

Μπορείτε να δείτε πόσα σχόλια εγκρίνουν οι συντονιστές σας μέσω της σελίδας Συντονιστών:
