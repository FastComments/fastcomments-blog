# [postlink]Προσθήκη Υπονοούμενων[/postlink]

{{#unless isPost}}
Οι χρήστες μπορούν τώρα να προσθέτουν υπονοούμενα στα σχόλιά τους, τα οποία είναι κρυφά μέχρι να περάσει το ποντίκι από πάνω τους.
{{/unless}}

{{#isPost}}

### Τι Νέο
Οι χρήστες του FastComments μπορούν τώρα να κάνουν κλικ σε ένα κουμπί "SPOILER" για να προσθέσουν υπονοούμενα στα σχόλιά τους. Μπορούν επίσης να γράψουν ετικέτες `<spoiler>some text</spoiler>` απευθείας αν το επιθυμούν.
Σημειώστε ότι η επιλογή της γραμμής εργαλείων για υπονοούμενα είναι απενεργοποιημένη από προεπιλογή.

### Ενεργοποίηση Υπονοούμενων

Στην Προσαρμογή Γκατζετών μπορείτε απλά να ενεργοποιήσετε την επιλογή `Spoilers` και να πατήσετε αποθήκευση. Τα έγγραφα είναι [εδώ](https://docs.fastcomments.com/guide-customizations-and-configuration.html#enable-spoilers).
Τα έχουμε ενεργοποιήσει για αυτό το άρθρο του blog για να επιδείξουμε τη δυνατότητα.

### Για Προγραμματιστές & Παρεξηγήσεις

Η προεπιλεγμένη στυλιστική για υπονοούμενα υποστηρίζει φωτεινές και σκοτεινές λειτουργίες. Αν γράψετε προσαρμοσμένο CSS για κείμενο με μάσκα, θυμηθείτε να καλύψετε και τη σκοτεινή λειτουργία αν η ιστοσελίδα σας την υποστηρίζει.

### Συμπερασματικά

Όπως σε όλες τις μεγάλες εκδόσεις, είμαστε χαρούμενοι που μπορέσαμε να αφιερώσουμε χρόνο για να βελτιστοποιήσουμε, να δοκιμάσουμε και να κυκλοφορήσουμε σωστά αυτή τη δυνατότητα. Ενημερώστε μας παρακάτω αν ανακαλύψετε οποιοδήποτε πρόβλημα.

Ευχαριστούμε!

{{/isPost}}
