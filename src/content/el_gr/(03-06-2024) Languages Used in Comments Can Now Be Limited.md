# [postlink]Οι γλώσσες που χρησιμοποιούνται στα σχόλια μπορούν τώρα να περιοριστούν[/postlink]

{{#unless isPost}}
Ορισμένες κοινότητες θέλουν να περιορίσουν τις γλώσσες που χρησιμοποιούνται. Αυτό μπορεί τώρα να γίνει με το FastComments.
{{/unless}}

{{#isPost}}

### Τι είναι νέο

Το FastComments έχει πάντα προσπαθήσει να υποστηρίζει όσο το δυνατόν περισσότερες γλώσσες και περιοχές.

Ωστόσο, ορισμένες κοινότητες επιθυμούν να περιορίσουν τις γλώσσες που χρησιμοποιούνται για την επικοινωνία και την υποβολή σχολίων.

Αντί να επιβάλλεται αυτό ως κανόνας και να απαγορεύονται ενδεχομένως οι χρήστες, μπορεί τώρα να ρυθμιστεί ρητά στην πλατφόρμα.

### Πώς να το ρυθμίσετε

Στην ίδια διεπαφή προσαρμογής Widget που πιθανόν να γνωρίζετε, μπορείτε τώρα να επιλέξετε μία ή περισσότερες γλώσσες στις οποίες μπορούν να γραφτούν τα σχόλια.

Η πλατφόρμα θα προσπαθήσει, κατά την υποβολή του σχολίου, να προσδιορίσει τη γλώσσα του σχολίου σε πραγματικό χρόνο. Εάν η εμπιστοσύνη της προσδιορισμένης γλώσσας είναι
πάνω από 70% και ταιριάζει με μια επιτρεπόμενη γλώσσα, τότε το σχόλιο επιτρέπεται.

Εάν το γραμμένο σχόλιο δεν είναι σε γλώσσα που έχει καθοριστεί από τη ρύθμισή σας, ο χρήστης θα δει ένα μήνυμα σφάλματος στη δική του γλώσσα.

### Επιπτώσεις για προγραμματιστές

Αυτή η ρύθμιση θα επηρεάσει και τα σχόλια που αποθηκεύονται μέσω της διεπαφής προγραμματισμού εφαρμογών (API).

Θα λάβετε ένα σφάλμα όπως το παρακάτω:

            status: 'failed',
            reason: `Το σχόλιο φαίνεται ότι είναι μία από αυτές τις γλώσσες: [es], αλλά μόνο [en,fr] επιτρέπονται.`,
            code: 'language-not-allowed',
            translatedError: "Ένα μήνυμα σφάλματος στη γλώσσα του χρήστη."

### Συμπ Conclusion

Όπως σε όλες τις μεγάλες κυκλοφορίες, είμαστε χαρούμενοι που μπορέσαμε να αφιερώσουμε χρόνο για να βελτιστοποιήσουμε, να δοκιμάσουμε και να κυκλοφορήσουμε σωστά αυτή τη δυνατότητα. Ενημερώστε μας
παρακάτω αν ανακαλύψετε τυχόν προβλήματα.

Cheers!

{{/isPost}}
