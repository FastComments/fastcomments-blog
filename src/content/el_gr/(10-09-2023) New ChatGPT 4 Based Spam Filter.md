# [postlink]Νέος φίλτρο spam βασισμένο σε ChatGPT 4.[/postlink]

{{#unless isPost}}
Η FastComments προσθέτει στη λίστα των επιλογών φίλτρων spam.
{{/unless}}

{{#isPost}}

### Τι καινούριο

Η FastComments συνεχίζει να βελτιώνει τον Naïve-Bayes classifier για spam, ωστόσο έχουμε προσθέσει μια νέα επιλογή στο στόλο των ανιχνευτών spam.

Μπορείτε πλέον να ανιχνεύσετε spam χρησιμοποιώντας το ChatGPT 4 που παρέχεται από την OpenAI.

### Απόκτηση

Για να ρυθμίσετε ποιον ανιχνευτή spam να χρησιμοποιήσετε, ελέγξτε τη σελίδα Ρυθμίσεις Μετριασμού στον πίνακα ελέγχου σας. Ο ανιχνευτής spam βασισμένος σε GPT4 χρεώνεται σε `$0.08` ανά `1000` tokens.

Αυτός ο ανιχνευτής spam είναι διαθέσιμος μόνο για πελάτες με σχέδιο Flex, καθώς χρεώνεται με βάση τα tokens που χρησιμοποιούνται.

### Βελτιστοποιήσεις

Πρώτα περνάμε όλο το περιεχόμενο μέσα από τον κοινόχρηστο Naïve-Bayes classifier για να περιορίσουμε τις κλήσεις προς την OpenAI. Αυτό σας εξοικονομεί χρήματα για πολύ προφανές spam. Για τα επιπλέον υπόγεια spam, στη συνέχεια κάνουμε κλήση στην OpenAI για να δούμε αν το περιεχόμενο φαίνεται σαν spam.

### Αποτελέσματα

Παρατηρήσαμε άμεσες βελτιώσεις στις κοινότητες που έχουν ενεργοποιήσει αυτή τη δυνατότητα.

### Τεκμηρίωση

Αυτή η ρύθμιση μπορεί να γίνει μέσω της σελίδας Ρυθμίσεις Μετριασμού στον πίνακα ελέγχου σας.

[Η τεκμηρίωση μπορεί επίσης να βρεθεί εδώ](https://docs.fastcomments.com/guide-moderation.html#spam-detection)

### Στο Συμπέρασμα

Ευχαριστούμε τους πελάτες μας που μας δίνουν συνεχώς feedback ώστε να μπορούμε να σκεφτόμαστε ιδέες όπως αυτή. Ελπίζουμε να συνεχίσετε να αγαπάτε το FastComments.

Η Υγεία σας!
