### Τι νέο υπάρχει

Πριν από τις 14 Νοεμβρίου 2023, κάναμε τις εξής αλλαγές στα webhooks:

#### Ενημερώσεις Webhooks

**Προηγουμένως**, οι ενημερώσεις περιείχαν την τελευταία έκδοση ενός σχολίου. Εάν δύο χρήστες ψήφισαν για ένα σχόλιο σε σύντομο χρονικό διάστημα, αυτό θα
αφύπνιζε δύο webhooks. Ωστόσο, ήταν δυνατό το πρώτο webhook event να περιέχει την **τελευταία έκδοση**, με αποτέλεσμα να έχουμε δύο
ταυτόσημα webhook events:

1. ` ... { votes: 2 } ...`
2. ` ... { votes: 2 } ...`

**Από τις 14 Νοεμβρίου**, κάθε webhook event είναι ιδεατό και περιέχει το αντικείμενο του σχολίου τη στιγμή της αλλαγής. Στο παραπάνω παράδειγμα τώρα θα λάβετε δύο events:

1. ` ... { votes: 1 } ...`
2. ` ... { votes: 2 } ...`

#### Διαγραφή Webhooks

**Προηγουμένως**, τα webhook διαγραφής περιείχαν μόνο το id του σχολίου. Αυτό ήταν ένα αντικείμενο όπως `{ id: string }`.

**Από τις 14 Νοεμβρίου**, τα webhook διαγραφής θα περιέχουν το **όλο** το αντικείμενο του σχολίου τη στιγμή της αφαίρεσης.

### Συμπερασματικά

Όπως σε όλες τις σημαντικές εκδόσεις, είμαστε χαρούμενοι που μπορέσαμε να αφιερώσουμε χρόνο για να βελτιστοποιήσουμε, να δοκιμάσουμε και να κυκλοφορήσουμε σωστά αυτή τη λειτουργία. Ενημερώστε μας
παρακάτω αν ανακαλύψετε οποιαδήποτε προβλήματα.
