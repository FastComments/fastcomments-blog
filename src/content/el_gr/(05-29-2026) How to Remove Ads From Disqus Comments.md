# [postlink]Πώς να Αφαιρέσετε Διαφημίσεις από τα Σχόλια του Disqus[/postlink]

{{#unless isPost}}
Με το δωρεάν πλάνο του Disqus, η ενότητα σχολίων σας έρχεται με διαφημίσεις. Υπάρχουν δύο τρόποι να απαλλαγείτε από αυτές. Ο καλύτερος σας επιτρέπει επίσης να φορτώνετε τα σχόλιά σας πιο γρήγορα και προστατεύει τους αναγνώστες σας από την παρακολούθηση.
{{/unless}}

{{#isPost}}

Το Disqus χρηματοδοτεί το δωρεάν πλάνο του βάζοντας διαφημίσεις στην ενότητα σχολίων σας, ακριβώς κάτω από το περιεχόμενό σας, στο ίδιο σημείο που οι αναγνώστες σας πηγαίνουν για να σας μιλήσουν. Αυτές οι διαφημίσεις δεν είναι δικές σας. Δεν τις επιλέγετε, και δεν κερδίζετε ούτε σεντ από αυτές. Αυτό είναι το αντάλλαγμα για το δωρεάν επίπεδο.

Υπάρχουν δύο αληθινές επιλογές για να τις αφαιρέσετε.

## Επιλογή 1: Αναβαθμίστε το Πλάνο σας στο Disqus

Ο απλός τρόπος είναι να πληρώσετε το Disqus. Τα πληρωμένα πλάνα τους απενεργοποιούν τις διαφημίσεις. Μπορείτε να βρείτε τις τρέχουσες επιλογές στην <a href="https://disqus.com/pricing/" target="_blank">σελίδα τιμολόγησης του Disqus</a>.

Αυτό λειτουργεί, αλλά αξίζει να είμαστε σαφείς για το τι αγοράζετε. Πληρώνετε για να αναιρέσετε ένα μειονέκτημα. Οι διαφημίσεις φεύγουν, και το υπόλοιπο του Disqus παραμένει το ίδιο: η ενότητα σχολίων εξακολουθεί να φορτώνει ένα βαρύ πακέτο από τρίτα scripts, και οι αναγνώστες σας παρακολουθούνται μέσω των ιστότοπων που το ενσωματώνουν. Πληρώνετε για να αφαιρέσετε τις διαφημίσεις, όχι για να ελαφρύνετε ή να κάνετε την ενότητα σχολίων πιο ιδιωτική.

## Επιλογή 2: Μεταβείτε σε μια Πλατφόρμα που Δεν Εμφανίζει Ποτέ Διαφημίσεις

Ο άλλος τρόπος για να αφαιρέσετε τις διαφημίσεις είναι να χρησιμοποιήσετε μια πλατφόρμα σχολιασμού που δεν τις τρέχει εξαρχής. :)

Μόνο για $0.99/μήνα, το FastComments παρέχει λειτουργικότητα βελτιστοποιημένη για απόδοση χωρίς παρακολούθηση ή διαφημίσεις.

## Κρατάτε Κάθε Σχόλιο

Το κομμάτι που ανησυχεί περισσότερο τους ανθρώπους είναι η απώλεια της υπάρχουσας συζήτησής τους. Το FastComments διαθέτει έναν ενσωματωμένο εισαγωγέα του Disqus που μεταφέρει τα σχόλιά σας, τα ονόματα χρηστών, τα avatars, τις εικόνες inline, τις ψήφους και τις χρονικές σφραγίδες. Μεταφέρουμε ακόμη και όλες τις εικόνες σε CDN δικό μας για εσάς.

Αν θέλετε τον πλήρη, λεπτομερή οδηγό, έχουμε έναν ειδικό οδηγό: [Μεταβαίνοντας από το Disqus στο FastComments](/(1-23-2020)-migrating-from-disqus-fastcomments.html). Η σύντομη έκδοση είναι παρακάτω.

## Πώς να Μεταβείτε σε Τρία Βήματα

### 1. Εξαγωγή των σχολίων σας από το Disqus

Ακολουθήστε τα <a href="https://help.disqus.com/en/articles/1717199-importing-exporting" target="_blank">βήματα εξαγωγής του Disqus</a>. Θα σας στείλουν ένα αρχείο με επέκταση "gz". Δεν χρειάζεται να το ανοίξετε ή να το αποσυμπιέσετε, και μια προεπιλεγμένη εγκατάσταση Windows δεν μπορεί έτσι κι αλλιώς. Ο backend μας διαβάζει αυτό το συμπιεσμένο αρχείο απευθείας.

### 2. Εισαγωγή του αρχείου στο FastComments

Αφού είστε συνδεδεμένοι, μεταβείτε στη <a href="https://fastcomments.com/auth/my-account/manage-data/import" target="_blank">σελίδα εισαγωγής</a>, επιλέξτε το Disqus από την αναπτυσσόμενη λίστα και ανεβάστε το αρχείο όπως είναι. Η εισαγωγή τρέχει στο παρασκήνιο; η σελίδα θα ενημερώνεται αυτόματα καθώς τρέχει για να σας δείξει μια ζωντανή καταμέτρηση των σχολίων που ανακαλύφθηκαν και εισήχθησαν. Είναι ασφαλές να κάνετε επανεισαγωγή όσες φορές χρειαστεί.

### 3. Αντικαταστήστε το snippet του Disqus με το δικό μας

Αφαιρέστε τον κωδικό ενσωμάτωσης του Disqus από τις σελίδες σας και τοποθετήστε <a href="https://fastcomments.com/auth/my-account/install-wizard/" target="_blank">το snippet του FastComments</a> αντί. Εφόσον οι διευθύνσεις URL των σελίδων σας παραμένουν οι ίδιες, τα εισαχθέντα σχόλιά σας ευθυγραμμίζονται ακριβώς εκεί που ήταν.

## Σε WordPress;

Αν η ιστοσελίδα σας τρέχει σε WordPress, παραλείψτε τα χειροκίνητα βήματα και εγκαταστήστε το <a href="https://wordpress.org/plugins/fastcomments/" target="_blank">πρόσθετο FastComments WordPress</a>. Αυτό χειρίζεται την εγκατάσταση, τη ρύθμιση και τη συγχρονισμό για εσάς, και μετά μπορείτε να απενεργοποιήσετε το Disqus.

## Δοκιμάστε το Πριν Δεσμευτείτε

Δεν χρειάζεται να διακόψετε τα πάντα ταυτόχρονα. Μπορείτε να τρέχετε το Disqus και το FastComments στην ίδια σελίδα προσωρινά για να συγκρίνετε την εμφάνιση και την αίσθηση πριν κάνετε την πλήρη αλλαγή. Αν χρειάζεστε βοήθεια με οτιδήποτε από αυτά, <a href="https://fastcomments.com/auth/my-account/help" target="_blank">επικοινωνήστε μαζί μας</a> και θα σας βοηθήσουμε.

Δεν είστε σίγουροι για το πόσο θα σας κοστίσει το FastComments; Κάθε πλάνο είναι χωρίς διαφημίσεις, και μπορείτε να δείτε τον αριθμό σας στον <a href="https://fastcomments.com/pricing-calculator" target="_blank">υπολογιστή τιμολόγησης</a>. Αν βρίσκεστε στην ΕΕ, μπορείτε να δημιουργήσετε τον λογαριασμό σας στο <a href="https://eu.fastcomments.com" target="_blank">eu.fastcomments.com</a> ώστε τα δεδομένα σας να παραμείνουν στην ΕΕ.

## Ανακεφαλαίωση

1. Εξαγωγή των δεδομένων σας από το Disqus
2. Εισαγωγή τους στο FastComments
3. Αντικατάσταση του snippet του Disqus με <a href="https://fastcomments.com/auth/my-account/get-acct-code" target="_blank">του FastComments</a>

Αυτό είναι. Καμία διαφήμιση στην ενότητα σχολίων σας, ελαφρύτερες σελίδες και κάθε σχόλιο που είχατε ήδη.

Cheers!
