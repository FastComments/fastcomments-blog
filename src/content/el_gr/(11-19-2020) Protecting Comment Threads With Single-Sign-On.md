# [postlink]Προστασία Σχολίων Χρησιμοποιώντας Single-Sign-On[/postlink]

{{#unless isPost}}
Δεν θεωρείτε δημόσια τα σχόλια σας; Έχετε ρυθμίσει SSO για την αυθεντικοποίηση; Αυτό μπορεί να είναι για εσάς.
{{/unless}}

{{#isPost}}

### <i class="circle">!</i> Αυτό το Άρθρο Περιέχει Τεχνική Ορολογία

#### Εισαγωγή

Η FastComments SSO <a href="/(4-13-2020)-setting-up-sso-with-fastcomments.html" target="_blank">(λεπτομέρειες εδώ)</a> παρέχει στους χρήστες σας έναν τρόπο
να σχολιάζουν χωρίς να χρειάζεται να συνδεθούν σε άλλη πλατφόρμα.

Ωστόσο, αυτό από μόνο του δεν ασφαλίζει τα σχόλια σας, καθώς κατά προεπιλογή τα δεδομένα των σχολίων είναι δημόσιες πληροφορίες - οποιοσδήποτε μπορεί να δει
τη σελίδα μπορεί να δει τα σχόλια. Ωστόσο, αυτή η έκδοση αλλάζει αυτό.

#### Ρύθμιση Χωρίς Κώδικα

Μπορούμε να εμποδίσουμε την προβολή και την αλληλεπίδραση με τα σχόλια μας, όταν έχει ρυθμιστεί το SSO, δημιουργώντας έναν <a href="https://fastcomments.com/auth/my-account/customize-widget" target="_blank">κανόνα προσαρμογής</a>.

Κατά την εκτέλεση αυτού, αναζητήστε SSO, και θα βρείτε αυτή την επιλογή:

<div class="text-center">
    <img src="/images/fc-sso-protection.png" title="Ρυθμιστική Επιλογή για Κλείδωμα Σχολίων μέσω SSO" alt="Ρυθμιστική Επιλογή για Κλείδωμα Σχολίων μέσω SSO" class="lozad" />
</div>

Ενεργοποιήστε το και αποθηκεύστε τον κανόνα προσαρμογής.

#### Προστασία Μόνο για Ορισμένο Τομέα ή Σελίδα

Για να προστατεύσετε μόνο έναν ορισμένο Τομέα ή Σελίδα, θα ρυθμίσουμε απλά τον κανόνα προσαρμογής για να το κάνουμε αυτό.

Στην κορυφή της διεπαφής προσαρμογής, θα βρούμε δύο πεδία, Τομέα και ID URL.

<div class="text-center">
    <img src="/images/fc-sso-protection-by-page.png" title="Ρυθμιστική Επιλογή κατά Σελίδα ή Τομέα" alt="Ρυθμιστική Επιλογή κατά Σελίδα ή Τομέα" class="lozad" />
</div>

Για να προστατεύσετε απλά έναν συγκεκριμένο τομέα, εισάγετε τον τομέα που σας ενδιαφέρει στο πεδίο "domain".

Για να προστατεύσετε μια συγκεκριμένη σελίδα, εισάγετε ένα URL σελίδας στο πεδίο "URL ID". Εάν έχετε μια προσαρμοσμένη ολοκλήρωση με τη FastComments, μπορείτε να εισάγετε έναν τύπο ID εδώ
αντί για ένα URL.

#### Προστασία Πέρα από την Ανάγνωση

Η ενεργοποίηση αυτής της επιλογής θα προστατεύσει τη σελίδα ή τον τομέα από το να σχολιαστεί, εκτός εάν ο χρήστης είναι συνδεδεμένος μέσω SSO.

#### Παγίδες

Οποιοσδήποτε χρήστης έχει δημιουργήσει σχόλια πριν από την ολοκλήρωση του SSO δεν θα μπορεί να τα δει, εκτός αν συνδεθεί μέσω της ολοκλήρωσης του SSO σας.

#### Συμπέρασμα

Ελπίζουμε να βρήκατε αυτόν τον οδηγό χρήσιμο και εύκολο στην ανάγνωση. Μη διστάσετε να σχολιάσετε παρακάτω με οποιαδήποτε ερώτηση.

Στην υγειά σας!
