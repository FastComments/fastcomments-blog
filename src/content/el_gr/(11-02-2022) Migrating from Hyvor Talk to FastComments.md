## Εάν Είστε Σε WordPress

Το FastComments διαθέτει ένα ειδικό plugin για WordPress: <a href="https://wordpress.org/plugins/fastcomments/" target="_blank">https://wordpress.org/plugins/fastcomments/</a>.

Το plugin καθιστά την εγκατάσταση, τη ρύθμιση και τη συγχρονισμό πολύ εύκολα. Εάν ακολουθήσετε αυτήν την κατεύθυνση, μπορείτε να αγνοήσετε τις υπόλοιπες οδηγίες εδώ. Απλώς βεβαιωθείτε
ότι έχετε συγχρονίσει τα σχόλια του Hyvor Talk πίσω στην εγκατάσταση του WordPress σας.

Εάν αυτό δεν λειτουργεί για εσάς όπως αναφέρουν μερικοί από τους πελάτες μας, μπορείτε να χρησιμοποιήσετε το plugin WordPress μας, αλλά να κάνετε εισαγωγή δεδομένων
εξάγοντας από το Hyvor Talk χειροκίνητα. Μπορείτε να βρείτε τις εισαγωγές δεδομένων στο [Διαχείριση Δεδομένων -> Εισαγωγή Σχολίων](https://fastcomments.com/auth/my-account/manage-data/import).

## Χειροκίνητη, Γενική Εγκατάσταση

### Εξαγωγή Των Υπαρχόντων Σχολίων Σας

Ο εισαγωγέας FastComments θα μεταφέρει τις ροές σχολίων σας, τα ονόματα χρηστών, τα avatar χρηστών, τα emoji και τις inline εικόνες. Θα μεταφέρουμε τις εικόνες στους διακομιστές μας απρόσκοπτα.

Από το 2022, το Hyvor Talk δεν εξάγει email ή ψήφους. Εάν μπορείτε να λάβετε μια εξαγωγή από αυτούς με αυτές τις πληροφορίες, μπορούμε να την εισάγουμε.

### Εισαγωγή Του Αρχείου .JSON

Μετά την λήψη της εξαγωγής από το Hyvor, συνδεθείτε στον πίνακα ελέγχου του FastComments σας και πηγαίνετε <a href="https://fastcomments.com/auth/my-account/manage-data/import" target="_blank">εδώ για να εισαγάγετε το αρχείο</a>. Επιλέξτε το Hyvor Talk από το dropdown και ανεβάστε το αρχείο σας.

### Περιμένετε μερικά λεπτά

Η εισαγωγή FastComments είναι "ασύγχρονη". Αυτό σημαίνει ότι η μεταφόρτωση του αρχείου και η επεξεργασία του είναι ξεχωριστά βήματα.

Έτσι, ανάλογα με το μέγεθος του αρχείου σας, μπορεί να λάβετε το μήνυμα επιτυχίας αμέσως. Στην σελίδα εισαγωγής στο κάτω μέρος υπάρχει ένας πίνακας - κάθε γραμμή αντιπροσωπεύει μια απόπειρα εισαγωγής.
Μπορείτε να ανανεώσετε αυτήν τη σελίδα με ασφάλεια για να δείτε την κατάσταση της εισαγωγής σας και πόσες γραμμές έχουν εισαχθεί μέχρι στιγμής.

### Όταν Ολοκληρωθεί

Θα λάβετε ένα email όταν ολοκληρωθεί η εισαγωγή - είτε επιτυχώς είτε όχι. Μπορείτε να κλείσετε την σελίδα αφού η μεταφόρτωση του αρχείου επιτυχεί και δείτε την εισαγωγή σας στην σελίδα Εισαγωγών. Θα έχει κατάσταση όπως "Αιτήθηκε" και όταν ξεκινά, η κατάσταση θα είναι "Εκτελείται".

Είναι ασφαλές να εισάγετε όσες φορές απαιτείται - ωστόσο αν η εισαγωγή σας αποτύχει, τότε <a href="https://fastcomments.com/auth/my-account/help" target="_blank">επικοινωνήστε μαζί μας</a> ώστε να μπορέσουμε να βοηθήσουμε.

### Αντικαταστήστε Τον Κώδικα Hyvor Talk Με Τον FastComments

Εάν χρησιμοποιείτε την εγκατάσταση κώδικα με το Hyvor τότε είναι απλά θέμα αφαίρεσης του μικρού κομματιού κώδικα που σας δίνει και αντικατάστασής του με <a href="https://fastcomments.com/install-wizard" target="_blank">τον δικό μας</a>.
Υποστηρίζουμε επίσης πολλά front end frameworks τα οποία [μπορείτε να βρείτε εδώ](https://fastcomments.com/install-wizard). Μπορείτε ακόμη και να τρέξετε τα Hyvor και FastComments στον ίδιο ιστότοπο προσωρινά για να δοκιμάσετε και να ρυθμίσετε την εμφάνιση και την αίσθηση. <a href="https://fastcomments.com/auth/my-account/help" target="_blank">Μπορούμε να σας βοηθήσουμε με αυτό</a>.

Εάν δεν χρησιμοποιείτε την εγκατάσταση κώδικα, τότε οι οδηγίες θα διαφέρουν ανάλογα με την πλατφόρμα σας - πάλι <a href="https://fastcomments.com/auth/my-account/help" target="_blank">επικοινωνήστε μαζί μας</a>.

Προσφέρουμε έναν οδηγό εγκατάστασης αυτοεξυπηρέτησης [εδώ](https://fastcomments.com/install-wizard).

### Μεταφορά URL Ταυτόχρονα

Τα σχόλια είναι συνδεδεμένα με τα πεδία id της σελίδας στην εξαγωγή, οπότε όσο τα URLs σας δεν αλλάξουν, η αλλαγή είναι εύκολη. Εάν θέλετε να μεταφέρετε URLs και να διατηρήσετε τα σχόλιά σας, τότε <a href="https://fastcomments.com/auth/my-account/help" target="_blank">ει告ήστε μας</a> στέλνοντάς μας τα παλιά και νέα URLs. Η σελίδα βοήθειας επιτρέπει την μεταφόρτωση μικρών αρχείων κειμένου, οπότε αν
μεταφέρετε περισσότερα από μερικά urls, δημιουργήστε ένα αρχείο Excel.

### ΕΕ

Εάν βρίσκεστε στην ΕΕ, θα θελήσετε πιθανώς να δημιουργήσετε τον λογαριασμό σας στο [eu.fastcomments.com](https://eu.fastcomments.com) ώστε τα δεδομένα των πελατών σας να παραμείνουν στην ΕΕ.

### Συνοψίζοντας

1. Εξαγάγετε τα δεδομένα σας
2. Εισάγετε στο FastComments
3. Αντικαταστήστε το Snippet JavaScript του Hyvor με <a href="https://fastcomments.com/install-wizard" target="_blank">του FastComments</a>

## Γιατί Δεν Θα Μου Στείλει Ο Hyvor Το Αρχείο Μου;
Για μεγαλύτερους ιστότοπους, το Hyvor μπορεί να μην είναι σε θέση να δημιουργήσει το εξαγόμενο αρχείο λόγω τεχνικών περιορισμών από την πλευρά τους. Έχουμε τη δυνατότητα να κάνουμε scraping στον υπάρχοντα ιστότοπό σας και να τραβήξουμε τα σχόλια από αυτόν, ωστόσο αυτό μπορεί να είναι 
αρκετά προσαρμοσμένο και λόγω του χρόνου που απαιτεί θα χρειαστεί να έχετε το Πακέτο Υποστήριξης. <a href="https://fastcomments.com/auth/my-account/help" target="_blank">Μπορείτε να ζητήσετε βοήθεια γι' αυτό εδώ</a>.
