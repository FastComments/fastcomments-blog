# [postlink]Προσαρμόζοντας το Κείμενο του Widget[/postlink]

{{#unless isPost}}
Η προσαρμογή του κειμένου του widget είναι μια συχνή ερώτηση υποστήριξης. Τώρα είναι αυτοεξυπηρέτηση.
{{/unless}}

{{#isPost}}

#### Τι Νέο

Προηγουμένως, ο μόνος τρόπος για να προσαρμόσετε το κείμενο στο widget σχολίων ήταν είτε να κάνετε αλλαγές στον κώδικα όταν ενσωματώνετε το widget για να περάσετε τις μεταφράσεις, είτε να υποβάλετε ένα αίτημα υποστήριξης πελατών.

#### Πώς Λειτουργεί

Όπως με άλλες προσαρμογές widget, αυτό τώρα διαχειρίζεται στη σελίδα <a href="https://fastcomments.com/auth/my-account/customize-widget" target="_blank">Προσαρμογές</a>.

Εισάγετε το domain για το οποίο θέλετε να προσαρμόσετε το κείμενο, και είτε μια συγκεκριμένη διεύθυνση URL είτε * για όλες τις σελίδες.

Κοντά στη μέση της σελίδας κάντε κλικ στο "Εμφάνιση Προσαρμοσμένου Κειμένου" και έπειτα "Επιλέξτε Υπέρβαση Μετάφρασης".

Στο αναπτυσσόμενο μενού μπορείτε να αναζητήσετε μεταφράσεις με βάση το προεπιλεγμένο αγγλικό κείμενο στο widget σχολίων για να βρείτε το κείμενο που θέλετε να υπερβείτε.

Καθώς κάνετε αλλαγές, αυτές θα εμφανιστούν στην προεπισκόπηση στο κάτω μέρος της σελίδας (με έντονα γράμματα).

#### Τι Αλλάζει για τις Υπάρχουσες Προσαρμογές Μετάφρασης

Εάν έχετε ήδη προσαρμόσει το κείμενό σας στο widget σχολίων με μία από τις προηγούμενες μεθόδους, δεν θα πρέπει να αλλάξει τίποτα - οι υπερβάσεις σας θα παραμείνουν έτσι όπως είναι.

#### Εν κατακλείδι

Ελπίζουμε ότι παρέχοντας περισσότερα εργαλεία αυτοεξυπηρέτησης όπως αυτό καθιστά ευκολότερη τη ρύθμιση του FastComments για κάθε τύπο πελάτη που έχουμε. Ενημερώστε μας στην ενότητα σχολίων παρακάτω πώς λειτούργησε για εσάς.

Στην υγειά σας!

{{/isPost}}
