# [postlink]Νέο Widget Floating Like Κυκλοφόρησε[/postlink]

{{#unless isPost}}
Ένα νέο widget έχει κυκλοφορήσει που επιτρέπει στους χρήστες να κάνουν like σε σελίδες ή άρθρα.
{{/unless}}

{{#isPost}}

### Τι νέο υπάρχει

Στο FastComments, ενώ οι χρήστες μπορούν να εγγραφούν σε σελίδες απευθείας στο widget των σχολίων κάνοντας κλικ στο καμπανάκι, δεν είχαμε μια λύση για αντιδράσεις σε επίπεδο σελίδας.

Τώρα έχουμε λανσάρει αυτή τη δυνατότητα, με το πρώτο widget να είναι το Floating Likes widget που βλέπετε στην κάτω δεξιά γωνία αυτού του blog.

Μπορείτε να κάνετε like στη σελίδα απευθείας, ακόμη και αν δεν έχετε λογαριασμό στο FastComments.com. Αν είστε συνδεδεμένοι, η αντίδραση συνδέεται με τον λογαριασμό σας.

### SSO

Το widget υποστηρίζει SSO για σκοπούς αυθεντοποίησης, επίσης.

### Τεκμηρίωση

Μπορείτε να βρείτε την τεκμηρίωση στο [docs.fastcomments.com](https://docs.fastcomments.com/guide-page-reacts.html).

### Προσαρμόσιμο

Αυτό το widget δεν χρησιμοποιεί iframe και μπορεί να στιλιστεί απευθείας μέσω CSS.

### Σκοτεινός Θέμα

Από προεπιλογή, το widget θα αλλάξει σε ένα πιο σκοτεινό θέμα εάν οι προτιμήσεις του περιηγητή ή του λειτουργικού συστήματος του χρήστη προτιμούν σκοτεινό θέμα.

### Προσδοκίες

Μακροπρόθεσμα θα θέλαμε να προσθέσουμε διάφορες αντιδράσεις σε επίπεδο σελίδας, και ίσως ακόμα και ψηφοφορίες, αν διαπιστώσουμε ότι η πελατειακή μας βάση είναι ενδιαφερόμενη για τέτοιες δυνατότητες.

Ανυπομονούμε για τα σχόλια από τους χρήστες και την κοινότητα ώστε να βελτιώσουμε αυτό το νέο widget με την πάροδο του χρόνου.

### Συμπέρασμα

Γνωρίζουμε ότι αυτή η δυνατότητα έχει αναμενθεί εδώ και καιρό από ορισμένους. Όπως σε όλες τις μεγάλες κυκλοφορίες, είμαστε χαρούμενοι που μπορέσαμε να αφιερώσουμε χρόνο για να βελτιστοποιήσουμε, να δοκιμάσουμε και να κυκλοφορήσουμε σωστά αυτό το στοιχείο.

Στην υγειά σας!

{{/isPost}}
