# [postlink]Το FastComments αποκτά Αβατάρ[/postlink]

Μετά την κυκλοφορία, αυτή ήταν η πιο ζητηθείσα δυνατότητα, οπότε ιδού.

Αν είστε συνδεδεμένοι και σχολιάζετε, ή βλέπετε τα υπάρχοντα σχόλιά σας, θα δείτε ότι τώρα υπάρχει μια κενή φωτογραφία προφίλ δίπλα στο όνομά σας.

{{#isPost}}
Αυτό είναι σε λειτουργία κλικ και θα σας μεταφέρει στον λογαριασμό σας όπου μπορείτε να ρυθμίσετε ένα Αβατάρ.
Τα αβατάρ είναι αναγνωρισμένα σε 40x40 pixel και δεχόμαστε JPG και PNG.

Μόλις ρυθμίσετε ένα αβατάρ, όλα τα παλαιά σας σχόλια θα ενημερωθούν.

Επίσης, είμαστε επιεικείς και δεν θα σπαταλήσουμε το εύρος ζώνης σας. Τα αβατάρ δεν φορτώνονται εκτός αν εμφανιστούν στην οθόνη.
{{/isPost}}
