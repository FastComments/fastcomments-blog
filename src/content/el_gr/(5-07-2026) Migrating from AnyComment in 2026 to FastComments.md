## Το AnyComment Ζει Μέσα Στο WordPress

Το AnyComment είναι ένα plugin του WordPress. Σε αντίθεση με αυτόνομες πλατφόρμες όπως το Disqus ή το Hyvor Talk, δεν τρέχει το δικό του backend - τα σχόλια σας αποθηκεύονται απευθείας στη βάση δεδομένων του WordPress, στον ίδιο πίνακα `wp_comments` που χρησιμοποιεί ο κύριος πυρήνας του WordPress. Αυτό είναι εξαιρετικά νέα για τη μετάβαση: δεν υπάρχει ξεχωριστή εξαγωγή AnyComment να διαχειριστείτε, και δεν χρειάζεται να συντονιστείτε με κάποιον τρίτο διακομιστή. Τα δεδομένα σας είναι ήδη στον δικό σας διακομιστή.

Αυτό σημαίνει ότι η τυπική ροή μετάβασης WordPress του FastComments χειρίζεται το AnyComment χωρίς επιπλέον βήματα.

## Επιλογή 1: Το Plugin FastComments για WordPress (Συνιστάται)

Η πιο εύκολη διαδρομή είναι να εγκαταστήσετε το <a href="https://wordpress.org/plugins/fastcomments/" target="_blank">plugin FastComments για WordPress</a>. Το plugin σας καθοδηγεί στη σύνδεση της εγκατάστασης WordPress με το FastComments, και αντιγράφει αυτόματα τα υπάρχοντα δεδομένα σχολίων σας. Δεν χρειάζεται να κατεβάσετε ή να ανεβάσετε τίποτα χειροκίνητα. Τα δεδομένα αντιγράφονται από τη βάση δεδομένων του WordPress στους διακομιστές μας, οπότε αυτό θα αποφορτίσει τους διακομιστές σας μόλις ολοκληρωθεί η μετάβαση.

Οι περισσότερες μεταβάσεις ολοκληρώνονται σε λίγα λεπτά.

Αφού μεταφερθούν τα δεδομένα, απενεργοποιήστε το AnyComment. Τα σχόλια θα συνεχίσουν να παρέχονται από το FastComments, και το plugin κρατά τη βάση δεδομένων του WordPress σας συγχρονισμένη ως εφεδρικό (αν ενεργοποιήσετε αυτή τη δυνατότητα), έτσι ώστε να έχετε πάντα τον έλεγχο των δεδομένων σας.

## Επιλογή 2: Εξαγωγή XML από WordPress

Αν προτιμάτε να μεταβείτε χειροκίνητα, ή έχετε ήδη μετακινήσει τον ιστότοπό σας από το WordPress και απλώς έχετε ένα backup XML, χρησιμοποιήστε την ενσωματωμένη εξαγωγή του WordPress.

1. Στον πίνακα διαχείρισης του WordPress σας, μεταβείτε στο **Εργαλεία -> Εξαγωγή** και κατεβάστε το αρχείο XML.
2. Συνδεθείτε στον πίνακα ελέγχου FastComments και πηγαίνετε <a href="https://fastcomments.com/auth/my-account/manage-data/import" target="_blank">εδώ για να εισάγετε το αρχείο</a>.
3. Επιλέξτε **WordPress (.xml)** από την αναπτυσσόμενη λίστα και ανεβάστε το αρχείο σας.

Δεδομένου ότι το AnyComment γράφει στον πίνακα `wp_comments`, κάθε νήμα AnyComment βρίσκεται σε αυτό το XML δίπλα στα άλλα σχόλια του WordPress σας. Ο εισαγωγέας τα ταιριάζει αυτόματα με την σωστή δημοσίευση.

## Αντικατάσταση Του Widget AnyComment

Αν χρησιμοποιήσατε την Επιλογή 1, το plugin FastComments για WordPress ήδη αντικαθιστά το AnyComment στον ιστότοπό σας - απλώς απενεργοποιήστε το AnyComment αφού ολοκληρωθεί η μετάβαση.

Αν χρησιμοποιήσατε την Επιλογή 2 και διατηρείτε την εγκατάσταση του WordPress, εγκαταστήστε το plugin FastComments στη συνέχεια για να χειριστεί την απόδοση του widget και τη συνεχή συγχρονισμό. Αν μετακινείστε εντελώς από το WordPress, εισάγετε το <a href="https://fastcomments.com/install-wizard" target="_blank">snippet εγκατάστασης</a> στον νέο σας ιστότοπο - υποστηρίζουμε πολλές frontend frameworks που <a href="https://fastcomments.com/install-wizard" target="_blank">μπορείτε να βρείτε εδώ</a>. <a href="https://fastcomments.com/auth/my-account/help" target="_blank">Μπορούμε να βοηθήσουμε με αυτό</a>.
