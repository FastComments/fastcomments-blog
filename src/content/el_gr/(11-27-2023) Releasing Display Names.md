### Τι Είναι Νέο

Το FastComments πάντα υποστήριζε ονόματα εμφάνισης για χρήστες SSO. Ωστόσο, αυτό δεν είχε ποτέ εκτεθεί σε κανονικούς σχολιαστές. Σήμερα αλλάζει αυτό!

Στη σελίδα [Λεπτομέρειες Λογαριασμού](https://fastcomments.com/auth/my-account/edit-details) μπορείτε τώρα να συμπληρώσετε ένα νέο πεδίο `Display Name`. Το `Display Name` θα
εμφανίζεται στα σχόλιά σας και στα προφίλ σας.

### Emojis

Μέχρι τώρα, οι χρήστες είχαν δυσκολία να χρησιμοποιούν emojis σε ονόματα χρήστη καθώς πρέπει να εισάγουν το όνομα χρήστη τους για να συνδεθούν. Ωστόσο, τώρα το `Display Name` μπορεί να περιλαμβάνει emojis.

### Μοναδικότητα και Κατάχρηση

Ο σκοπός του `Display Name` είναι ότι δεν είναι μοναδικό. Εάν δύο χρήστες του FastComments.com, σε δύο πολύ διαφορετικές μεγάλες κοινότητες έχουν παρόμοια ονόματα, θα θέλαμε
να μπορούν απλά να ορίσουν το προτιμώμενο όνομα εμφάνισης τους. Ωστόσο, δεν μπορούμε να περιορίσουμε το που σχολιάζουν οι άνθρωποι με τον παγκόσμιο λογαριασμό FastComments τους, οπότε αυτό μπορεί
μερικές φορές να προκαλέσει σύγχυση.

Το να έχετε ένα παρόμοιο όνομα με άλλο χρήστη ή έναν διαχειριστή, από μόνο του, δεν αποτελεί περίπτωση κατάχρησης. Ωστόσο, αν το καταχραστείτε αυτό με την πρόθεση να μιμηθείτε άλλους χρήστες, ενδέχεται
να απενεργοποιήσουμε αυτή τη δυνατότητα στον λογαριασμό σας και θα είστε περιορισμένοι στο να χρησιμοποιείτε μόνο το `username`.

### Σύνδεση

Εξακολουθείτε να χρησιμοποιείτε το όνομα χρήστη σας για να συνδεθείτε - **όχι το όνομα εμφάνισης σας!**

### Για Προγραμματιστές & Επιπλοκές

Τα κανονικά αντικείμενα `User` που εκτίθενται μέσω του API έχουν τώρα `displayName`.

### Συμπερασματικά

Όπως σε όλες τις μεγάλες εκδόσεις, είμαστε χαρούμενοι που μπορέσαμε να αφιερώσουμε χρόνο για να βελτιστοποιήσουμε, να δοκιμάσουμε και να εκδώσουμε σωστά αυτή τη δυνατότητα. Ενημερώστε μας
παρακάτω αν ανακαλύψετε οποιαδήποτε προβλήματα.

Στην Υγειά σας!
