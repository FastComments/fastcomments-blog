# [postlink]Η Αναζήτηση Πλήρους Κειμένου Έρχεται στο FastComments[/postlink]

Οι διαχειριστές τώρα έχουν τη δυνατότητα να αναζητούν μια συνδυασμένη σειρά ονομάτων χρηστών και κειμένου σχολίων μέσω της σελίδας διαχείρισης.

{{#isPost}}

Απλά πηγαίνετε στη σελίδα Διαχείριση Σχολίων και εισάγετε κάποιο κείμενο στο πλαίσιο αναζήτησης στο κέντρο για να ξεκινήσετε να αναζητάτε σχόλια μέσω λέξεων και φράσεών.

<img 
    src="images/fc-search-comments.png"
    alt="Παράδειγμα Αναζήτησης Σχολίων"
    title="Παράδειγμα Αναζήτησης Σχολίων"
    class='lozad' />
    
Πιστεύουμε ότι αυτό θα βοηθήσει τους διαχειριστές να βρουν περιεχόμενο που έχουν δει προηγουμένως, αλλά δεν έχουν αποθηκεύσει σύνδεσμο γι' αυτό.

Επίσης, εάν κάνετε κλικ στο κουμπί "Μετά" κατά την αναζήτηση, μπορείτε να μοιραστείτε ή να αποθηκεύσετε τη διεύθυνση URL στο πρόγραμμα περιήγησής σας. Πηγαίνοντας σε αυτήν την ίδια διεύθυνση URL αργότερα θα δείτε τα ίδια αποτελέσματα αναζήτησης και τη σελίδα στην οποία ήσασταν.

Δεδομένου ότι αυτή είναι "αναζήτηση πλήρους κειμένου", μια μηχανή αναζήτησης προσπαθεί να βρει τους καλύτερους ταιριαστούς όρους βάσει των όσων πληκτρολογείτε - αυτό δεν είναι όπως η αναζήτηση σε έγγραφο Word, για παράδειγμα. Έτσι, μπορεί να παρατηρήσετε ότι πρέπει να ολοκληρώσετε κάποιες λέξεις για να λάβετε αποτελέσματα.

Καλή διαχείριση!

{{/isPost}}
