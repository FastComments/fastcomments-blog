# [postlink]Αναβάθμιση Διαχείρισης Ενοικιαστών[/postlink]

{{#unless isPost}}
Το FastComments έχει βελτιώσει την αλληλεπίδραση για τη διαχείριση των ενοικιαστών σας!
{{/unless}}

{{#isPost}}

### Τι Νέο

Από τις 11 Δεκεμβρίου 2023, το FastComments διαθέτει πλέον τη δυνατότητα αλλαγής ενοικιαστών χωρίς να μιμείστε άλλους χρήστες ενοικιαστών.

### Στοχευόμενο Κοινό

Αυτή η αναβάθμιση δεν απευθύνεται σε σχολιαστές ή διαχειριστές ιστότοπου που διαχειρίζονται έναν μόνο ιστότοπο. Οφελεί τους διαχειριστές που διαχειρίζονται πολλούς ιστότοπους ή ενοικιαστές μόνο.

### Επιπτώσεις για τους Διαχειριστές

Προηγουμένως, η δημιουργία ενοικιαστών και η αλλαγή χρηστών ήταν μια ταλαιπωρία. Μερικά κοινά σημεία δυσκολίας ήταν:

1. Έπρεπε να δημιουργήσετε έναν χρήστη όταν δημιουργούσατε τον ενοικιαστή σας, καθώς αυτός ο χρήστης "κατείχε" τον ενοικιαστή.
2. Αυτό σήμαινε ότι έπρεπε να χρησιμοποιείτε νέο email για κάθε ιστότοπο, ακόμη και αν θέλατε να χρησιμοποιήσετε ένα email/λογαριασμό για πολλούς υπο-ενοικιαστές.
3. Η αλλαγή ενοικιαστών απαιτούσε μιμητισμό, πράγμα που σήμαινε ότι οι χρήστες σας έπρεπε να διπλασιαστούν σε όλους τους ενοικιαστές, με τις ίδιες άδειες, και ούτω καθεξής.

Τώρα:

1. Δεν απαιτείται η δημιουργία χρήστη κατά τη δημιουργία ενός ενοικιαστή.
2. Οποιοσδήποτε χρήστης Super Admin με τις κατάλληλες άδειες έχει τις ίδιες άδειες σε όλους τους υπο-ενοικιαστές.
3. Υπάρχει ένα κουμπί `Switch` στη λίστα ενοικιαστών που διατηρεί τον τρέχοντα χρήστη και τις άδειές σας, αλλά αλλάζει τον ενοικιαστή στον οποίο είστε συνδεδεμένοι.
4. Εάν είστε διαχειριστής ενός μητρικού ενοικιαστή, έχετε άδειες συντονιστή σε υπο-ενοικιαστές όταν χρησιμοποιείτε το widget σχολίων.

Τα γεγονότα καταγράφονται ακόμη στο Audit Log για τον ενοικιαστή στον οποίο έχετε μεταφερθεί.

### Μιμητισμός

Μπορείτε να μιμηθείτε ακόμη χρήστες. Μεταβείτε σε έναν ενοικιαστή και επισκεφθείτε τη σελίδα `Users`.

### Για τους Προγραμματιστές & Παράξενες Περίπτωσεις

Κατά τη χρήση του API `POST /tenants`, το πεδίο `email` είναι πλέον προαιρετικό. Η τεκμηρίωση έχει ενημερωθεί.

### Μελλοντικές Αναβαθμίσεις

Στο μέλλον, θα υπάρχουν περισσότερες διοικητικές ενέργειες στη λίστα ενοικιαστών, όπως η διαγραφή ενοικιαστών και ούτω καθεξής.

### Εν Σύνοψη

Όπως σε όλες τις μεγάλες εκδόσεις, είμαστε glad που μπορέσαμε να αφιερώσουμε χρόνο για να βελτιστοποιήσουμε, να δοκιμάσουμε και να κυκλοφορήσουμε σωστά αυτή τη δυνατότητα. Ενημερώστε μας
παρακάτω αν ανακαλύψετε τυχόν προβλήματα.

Ευχαριστώ!
