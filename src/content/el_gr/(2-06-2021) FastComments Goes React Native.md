### Τι Νέο

Το FastComments διαθέτει πλέον στοιχεία React Native που είναι ισοδύναμα με το widget VanillaJS μας.

Αποφασίσαμε να ονομάσουμε τη βιβλιοθήκη `fastcomments-react-native`. Μπορείτε να βρείτε το αποθετήριο GitHub με τον πηγαίο κώδικα <a href="https://github.com/FastComments/fastcomments-react-native" target="_blank">εδώ</a>.

Το αποθετήριο περιέχει επίσης παραδείγματα χρήσης του widget.

Είναι διαθέσιμο στο NPM, το οποίο μπορείτε να βρείτε <a href="https://www.npmjs.com/package/fastcomments-react-native" target="_blank">εδώ</a>.

### Πώς Λειτουργεί

Ο τρόπος που λειτουργεί αυτό το νέο στοιχείο είναι ότι το συστατικό React είναι μια "συσκευασία" γύρω από το υπάρχον widget FastComments.
 
Αυτό σημαίνει ότι αν προσθέσουμε χαρακτηριστικά στα υπάρχοντα στοιχεία μας ή διορθώσουμε σφάλματα, θα ωφεληθείτε αμέσως!

### Χρόνος Ανάπτυξης

Τα widgets React Native μπορούν να προστεθούν στην εφαρμογή σας μόλις σε λίγα λεπτά με ελάχιστο απαιτούμενο χρόνο ανάπτυξης.

### SSO

Όλες οι βιβλιοθήκες React Native υποστηρίζουν SSO από το κουτί, και η διαδικασία ολοκλήρωσης SSO είναι η ίδια με τα στοιχεία Vanilla JS. Ισχύουν επίσης τα υπάρχοντα παραδείγματα κώδικα backend.

### Προσαρμοσμένο Στυλ;

Τα στοιχεία React Native μπορούν να στυλιζαριστούν χρησιμοποιώντας CSS μέσω του UI Προσαρμογής Widget, όπως τα web-based widgets.
