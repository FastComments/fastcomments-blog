# [postlink]Μετάβαση από τα Σχόλια Muut σε FastComments[/postlink]

{{#unless isPost}}
Διαβάστε αυτή την ανάρτηση για να δείτε πώς να μεταναστεύσετε πλήρως από το Muut στο FastComments.

Είναι απλά θέμα αφαίρεσης του μικρού αποσπάσματος που σας παρέχει το Muut και αντικατάστασής του με <a href="https://fastcomments.com/auth/my-account/get-acct-code" target="_blank">το δικό μας</a>.
Μπορείτε ακόμη να τρέξετε το Muut και το FastComments ταυτόχρονα στον ίδιο ιστότοπο προσωρινά για να δοκιμάσετε και να προσαρμόσετε την εμφάνιση και την αίσθηση. <a href="https://fastcomments.com/auth/my-account/help" target="_blank">Μπορούμε να βοηθήσουμε με αυτό</a>.
{{/unless}}

{{#isPost}}
### Εξαγωγή των Υπάρχοντων Σχολίων σας

Ο εισαγωγέας FastComments θα μεταφέρει τα σχόλιά σας, τα ονόματα χρηστών, τα avatar χρηστών και τις ενσωματωμένες εικόνες. Θα μεταφέρουμε τις εικόνες στους διακομιστές μας χωρίς καμία δυσκολία.

Για να εξάγετε τα δεδομένα Σχολίων και Χρηστών από το Muut από τον πίνακα διαχείρισης τους, πηγαίνετε στην κατηγορία Ενσωματώσεις και έπειτα στην Εξαγωγή JSON.

Θα παρατηρήσετε ότι το αρχείο που θα λάβετε μέσω email έχει μια μυστήρια "gz" επέκταση αρχείου. Αν είστε λίγο παραπάνω τεχνικά καταρτισμένος, μπορεί να ξέρετε ότι αυτό σημαίνει "gzip" και είναι ένας δημοφιλής και αποδοτικός τρόπος για να συμπιέσετε αρχεία. 

### Μην ανοίξετε το αρχείο "gz"

Μια προεπιλεγμένη εγκατάσταση Windows 10 δεν θα μπορεί να ανοίξει το αρχείο από το Muut. Αυτό είναι εντάξει, γιατί δεν χρειάζεται να το κάνετε με το FastComments. Το backend μας καταλαβαίνει αυτό το συμπιεσμένο αρχείο.

Μετά από αυτό, αν είστε συνδεδεμένοι μπορείτε να πάτε <a href="https://fastcomments.com/auth/my-account/manage-data/import" target="_blank">εδώ για να εισαγάγετε το αρχείο</a>. Επιλέξτε Muut από την αναπτυσσόμενη λίστα και ανεβάστε το αρχείο σας.

Θα χρειαστεί να εισαγάγετε τη διεύθυνση URL της ιστοσελίδας σας. Αυτό συμβαίνει γιατί το Muut δεν μας παρέχει αυτή την πληροφορία στην εξαγωγή.

### Περιμένετε λίγα λεπτά

Η εισαγωγή FastComments είναι "ασύγχρονη". Αυτό σημαίνει ότι η μεταφόρτωση του αρχείου και η επεξεργασία του είναι ξεχωριστά βήματα.

Έτσι, ανάλογα με το μέγεθος του αρχείου σας, μπορεί να λάβετε το μήνυμα επιτυχίας αμέσως. Στη σελίδα εισαγωγής στο κάτω μέρος υπάρχει ένας πίνακας - κάθε γραμμή αντιπροσωπεύει μια απόπειρα εισαγωγής.
Μπορείτε να ανανεώσετε αυτή τη σελίδα με ασφάλεια για να δείτε την κατάσταση της εισαγωγής σας και πόσες γραμμές έχουν εισαχθεί μέχρι τώρα.

### Όταν ολοκληρωθεί

Θα λάβετε ένα email όταν ολοκληρωθεί η εισαγωγή - ανεξάρτητα αν ήταν επιτυχής ή όχι. Μπορείτε να κλείσετε τη σελίδα μετά την επιτυχία της μεταφόρτωσης του αρχείου και να δείτε την εισαγωγή σας στη σελίδα εισαγωγών. Θα έχει κατάσταση όπως "Αιτήθηκε" και όταν ξεκινήσει η κατάσταση θα είναι "Εκτέλεση".

Είναι ασφαλές να επαναφέρετε όσο φορές απαιτείται - ωστόσο αν η εισαγωγή σας αποτύχει τότε <a href="https://fastcomments.com/auth/my-account/help" target="_blank">επικοινωνήστε μαζί μας</a> για να μπορέσουμε να βοηθήσουμε.

### Αντικαταστήστε τον Κωδικό Muut με το FastComments

Είναι απλά θέμα αφαίρεσης του μικρού αποσπάσματος που σας παρέχει το Muut και αντικατάστασής του με <a href="https://fastcomments.com/auth/my-account/get-acct-code" target="_blank">το δικό μας</a>.
Μπορείτε ακόμη να τρέξετε το Muut και το FastComments ταυτόχρονα στον ίδιο ιστότοπο προσωρινά για να δοκιμάσετε και να προσαρμόσετε την εμφάνιση και την αίσθηση. <a href="https://fastcomments.com/auth/my-account/help" target="_blank">Μπορούμε να βοηθήσουμε με αυτό</a>.

### Μεταφορά των Διευθύνσεων URL ταυτόχρονα

Τα σχόλια συνδέονται με τα πεδία "σύνδεσμος" στην εξαγωγή του Muut, οπότε εφόσον οι διευθύνσεις σας δεν αλλάξουν, η μετάβαση είναι εύκολη. Αν θέλετε να μεταφέρετε τις διευθύνσεις URL και να διατηρήσετε τα σχόλια σας, τότε <a href="https://fastcomments.com/auth/my-account/help" target="_blank">ενημερώστε μας</a> στέλνοντάς μας τις παλιές και νέες διευθύνσεις URL. Η σελίδα βοήθειας επιτρέπει την ανάρτηση μικρών κειμένων, οπότε αν μεταφέρετε περισσότερες από μερικές διευθύνσεις URL δημιουργήστε ένα φύλλο Excel.

### Ανακεφαλαίωση

1. Εξαγάγετε τα δεδομένα σας
2. Εισάγετε στο FastComments
3. Αντικαταστήστε το απόσπασμα JavaScript του Muut με το <a href="https://fastcomments.com/auth/my-account/get-acct-code" target="_blank">FastComments's</a>

## Γιατί δεν μου στέλνει το Muut το αρχείο μου;
Για μεγαλύτερους ιστότοπους, το Muut μπορεί να μην είναι σε θέση να δημιουργήσει το εξαγόμενο αρχείο λόγω τεχνικών περιορισμών από την πλευρά τους. Έχουμε τη δυνατότητα να "ξεσκονίσουμε" την υπάρχουσα ιστοσελίδα σας και να συλλέξουμε τα σχόλια από αυτήν, αλλά αυτό μπορεί να είναι αρκετά προσαρμοσμένο και λόγω του χρόνου που απαιτείτε, θα χρειαστεί να έχετε το Πακέτο Υποστήριξης. <a href="https://fastcomments.com/auth/my-account/help" target="_blank">Μπορείτε να ζητήσετε βοήθεια με αυτό εδώ</a>.
