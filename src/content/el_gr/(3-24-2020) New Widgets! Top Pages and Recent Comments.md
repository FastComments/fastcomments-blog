# [postlink]Νέα Widget! Κορυφαίες Σελίδες και Πρόσφατα Σχόλια[/postlink]

Σήμερα παρουσιάζουμε δύο νέους τύπους widget, φέρνοντας τον συνολικό αριθμό υποστηριζόμενων widget σε τρία: Αριθμός Σχολίων, Πρόσφατα Σχόλια και Κορυφαίες Σελίδες.

{{#isPost}}

Εάν είστε συνδεδεμένοι, μπορείτε να δείτε τη λίστα <a href="https://fastcomments.com/auth/my-account/widgets" target="_blank">εδώ</a>, διαφορετικά μεταβείτε στον Λογαριασμό Μου -> Λάβετε Κωδικό Widget -> Άλλα Widget.

Λάβετε υπόψη ότι όλα τα widget μας έχουν πολύ μινιμαλιστικό στυλ - είναι σχεδιασμένα να είναι γρήγορα και εύκολα να προσαρμοστούν στις ανάγκες σας.

Το widget Πρόσφατα Σχόλια θα εμφανίζει τα δέκα κορυφαία εγκεκριμένα σχόλια, ταξινομημένα κατά ημερομηνία.

Το widget Κορυφαίες Σελίδες θα εμφανίζει τις 30 κορυφαίες σελίδες ταξινομημένες κατά αριθμό σχολίων.

Οι τιμές για αυτά τα widget είναι προκαθορισμένες πριν φορτώσετε τη σελίδα, **και** αποθηκεύονται στη μνήμη cache για να εξασφαλίσουν την καλύτερη εμπειρία για τους χρήστες σας. Για τεκμηρίωση γύρω από τα widget, μεταβείτε στη λίστα widget και επιλέξτε ένα. Από εκεί θα δείτε τον κωδικό που μπορείτε να προσθέσετε στον ιστότοπό σας, καθώς και οποιαδήποτε ρύθμιση αποδέχεται το widget.

Εάν διαπιστώσετε ότι αυτά τα widget δεν παρέχουν τις πληροφορίες που χρειάζεστε ή θα θέλατε βοήθεια με το στυλ τους, μη διστάσετε να <a href="https://fastcomments.com/auth/my-account/help" target="_blank">επικοινωνήσετε</a>.

{{/isPost}}
