# [postlink]Ενεργοποίηση Email Ειδοποιήσεων Για Συνδρομές[/postlink]

{{#unless isPost}}
Η FastComments τώρα στέλνει email στους χρήστες για σελίδες στις οποίες είναι συνδρομητές.
{{/unless}}

{{#isPost}}

### Τι Είναι Νέο

Με την FastComments, οι χρήστες μπορούν να κάνουν κλικ στο εικονίδιο του καμπανάκι σε ένα widget σχολίων για να εγγραφούν σε αυτή τη σελίδα. Θα τους ειδοποιήσουμε
όταν νέες σχόλια προστεθούν σε αυτή τη σελίδα, ή όταν υπάρχει ένα νέο σχόλιο σε μια αλυσίδα απαντήσεων στην οποία συμμετέχουν.

Το κάναμε αυτό για χρόνια, ωστόσο δεν στέλναμε email για αυτές τις ειδοποιήσεις - ήταν μόνο ορατές στον πίνακα ελέγχου μας.

Τώρα θα στέλνουμε email για αυτές τις ειδοποιήσεις.

### Νέοι Τύποι Email

Τώρα στέλνουμε δύο νέους τύπους email. Ο πρώτος είναι όταν έχετε μόνο μερικές ειδοποιήσεις. Απλά θα παραθέσουμε
τα νέα σχόλια στο email για εσάς.

Ο δεύτερος τύπος email που θα λάβετε περιέχει συνδέσμους σε κάθε σελίδα για την οποία λάβατε ειδοποιήσεις. Αυτό στέλνεται
όταν λαμβάνετε πολλές ειδοποιήσεις.

Αυτά τα email ειδοποιήσεων είναι μόνο για νέα σχόλια σε σελίδες στις οποίες είστε συνδρομητές.

### Πρόγραμμα Ειδοποιήσεων μέσω Email

Τα email ειδοποιήσεων αποστέλλονται κάθε ώρα για να αποτραπεί η υπερβολική αποστολή email. Η εξαίρεση είναι αν η σελίδα που τίθεται υπόψη έχει πάνω από δέκα χιλιάδες
σχόλια, οπότε θα στείλουμε ειδοποιήσεις για αυτή τη σελίδα μόνο μία φορά την ημέρα.

### Για Ιδιοκτήτες Ιστοσελίδων - Προσαρμογή των Emails

Μια νέα πρότυπη email για τα `Νέα Σχόλια Συνδρομής` θα είναι σύντομα διαθέσιμη για προσαρμογή.

### Για Προγραμματιστές & Γενικές Επισημάνσεις

Για Χρήστες SSO, δεν θα στέλνουμε τα email ειδοποιήσεων από προεπιλογή. Θα πρέπει να περάσετε μια νέα σημαία: `optedInSubscriptionNotifications: true` στο αντικείμενο χρήστη, ή
στο payload του widget σχολίων, για να ενεργοποιήσετε αυτές τις νέες ειδοποιήσεις.

### Σε Συμπέρασμα

Όπως με όλες τις μεγάλες ενημερώσεις, είμαστε χαρούμενοι που μπορέσαμε να αφιερώσουμε χρόνο για να βελτιστοποιήσουμε, να δοκιμάσουμε και να κυκλοφορήσουμε σωστά αυτή τη δυνατότητα. Ενημερώστε μας
παρακάτω αν ανακαλύψετε οποιαδήποτε προβλήματα.

Στη υγειά σας!
