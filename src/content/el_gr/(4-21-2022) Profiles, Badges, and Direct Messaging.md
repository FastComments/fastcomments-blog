# [postlink]Προφίλ, Διακριτικά και Άμεσο Μήνυμα[/postlink]

{{#unless isPost}}Σε αυτή την έκδοση έχουμε πολλές δυνατότητες που ανταμείβουν τις κοινότητες που χρησιμοποιούν το FastComments και τους βοηθούν να επικοινωνούν.{{/unless}}

{{#isPost}}

### Τι Νέο
Το FastComments ανακοίνωσε επίσημα τα Προφίλ Χρηστών και τις σχετικές δυνατότητες.

### Προφίλ Χρηστών

Το προφίλ ενός χρήστη αποτελείται από το όνομά τους, αν είναι online, τη πρόσφατη δραστηριότητά τους και ένα μέρος για να σχολιάσουν στο προφίλ τους.

Τα διακριτικά τους και η πρόοδος στην απόκτηση διακριτικών εμφανίζονται επίσης εδώ.

<div class="text-center">
    <video src="/images/profile.webm" autoplay loop muted alt="FastComments User Profile" title="FastComments User Profile"></video>
</div>

Οι χρήστες μπορούν να δουν το δικό τους προφίλ κάνοντας κλικ στο avatar τους σε μια συνομιλία, ή [επισκεπτόμενοι τη σελίδα του λογαριασμού τους](https://fastcomments.com/auth/my-account).

Επιπλέον, μπορείτε να στείλετε άμεσο μήνυμα σε έναν χρήστη από το προφίλ τους.

### Μόντελ Προφίλ Χρήστη
Όταν επιλέγεται το avatar ενός χρήστη, θα ανοίξει το Μόντελ Προφίλ. Αυτό έχει προστεθεί επίσης για τους μoderators στη σελίδα Διαχείρισης Σχολίων.

### Ανταμοιβή Χρηστών Με Διακριτικά

Τα διακριτικά είναι μια άλλη πρόσφατη προσθήκη στο FastComments που σας επιτρέπει να ανταμείβετε τους χρήστες της κοινότητάς σας. Η δημιουργία ενός διακριτικού είναι εύκολη:

<div class="text-center">
    <img src="/images/create-badge.png" alt="Create a Badge" title="Create a Badge" />
</div>

Τα διακριτικά μπορούν να απονεμηθούν με βάση τις ακόλουθες συνθήκες:

- Αριθμός Σχολίων
- Αριθμός Ψηφισμάτων (Up-Votes)
- Αριθμός Απαντήσεων που Λάβατε
- Αριθμός Σχολίων που Έχουν Καρφιτσωθεί
- Βετεράνος (χρόνος από το πρώτο σχόλιο)
- Σχολιασμός Αργά τη Νύχτα
- Γρήγορος Χρόνος Απάντησης
- Απάντηση σε Συγκεκριμένη Σελίδα
- Moderators - Αριθμός Σχολίων που Διαγράφηκαν
- Moderators - Αριθμός Σχολίων που Εγκρίθηκαν
- Moderators - Αριθμός Σχολίων που Δεν Εγκρίθηκαν
- Moderators - Αριθμός Σχολίων που Εξετάστηκαν
- Moderators - Αριθμός Σχολίων που Σημειώθηκαν ως Spam
- Moderators - Αριθμός Σχολίων που Σημειώθηκαν ως Μη Spam

Όπως μπορείτε να δείτε, υπάρχουν πολλές καταστάσεις για τις οποίες μπορούμε να ανταμείψουμε τους χρήστες.

Μπορούμε να ορίσουμε πολλαπλά *επίπεδα* διακριτικών. Για παράδειγμα, θα μπορούσαμε να προσθέσουμε διακριτικά `Αριθμού Σχολίων` με όρια 10, 100 και 1000 σχολίων. Καθώς οι χρήστες
αλληλεπιδρούν στην κοινότητά σας, τα διακριτικά που κερδίζουν θα **συσσωρεύονται** με την πάροδο του χρόνου.

Ας μην ξεχνάμε τους *Moderators*. Σε πολλές κοινότητες, αυτή είναι μια εργασία χωρίς αναγνώριση. Δώστε στους moderators σας κάτι να επιδείξουν ρυθμίζοντας διακριτικά Moderators για αυτούς.

Υποστηρίζουμε επίσης μερικές μοναδικές περιπτώσεις, όπως τα διακριτικά `Νυχτερίδα` και `Γρήγορος Χρόνος Απάντησης` για να αναγνωρίσουμε μέλη που είναι ενεργά αργά τη νύχτα ή απαντούν
γρήγορα. Το `Γρήγορος Χρόνος Απάντησης` λειτουργεί καλύτερα όταν χρησιμοποιούνται οι ζωντανές δυνατότητες του FastComments.

Όταν δημιουργούνται νέα διακριτικά, θα προστεθούν αυτόματα στους υπάρχοντες μέλη της κοινότητάς σας που πληρούν τις προϋποθέσεις.

### Άμεσo Μήνυμα

Οι πελάτες έχουν ζητήσει τη δυνατότητα να στέλνουν Άμεσα Μηνύματα σε χρήστες από την εκκίνηση του FastComments. Μετά από δύο χρόνια, είμαστε ευτυχείς να ανακοινώσουμε επιτέλους αυτή τη δυνατότητα και να ευχαριστήσουμε
όσους περίμεναν υπομονετικά.

<div class="text-center">
    <video src="/images/dm-example.webm" autoplay loop muted alt="FastComments Direct Messaging" title="FastComments Direct Messaging"></video>
</div>

Η δυνατότητα Άμεσου Μηνύματος επιτρέπει στα μέλη μιας κοινότητας να στέλνουν μηνύματα ο ένας στον άλλο ιδιωτικά, αλλά είναι επίσης ένα χρήσιμο εργαλείο για τους Moderators και τους Διαχειριστές να επικοινωνούν με τους σχολιαστές.

Όπως οι περισσότερες δυνατότητες του FastComments, όλες οι δυνατότητες Άμεσου Μηνύματος είναι ζωντανές.

#### Ασύγχρονο Άμεσο Μήνυμα

Όταν στείλετε μήνυμα σε έναν χρήστη, μπορεί να είναι εκτός σύνδεσης. Σε αυτή την περίπτωση, θα στείλουμε στον χρήστη ένα email που θα τους ενημερώνει ότι έχουν νέο μήνυμα.

#### Άμεσο Μήνυμα μέσω Email

Όταν ένας χρήστης λαμβάνει μήνυμα μέσω email, έχει τη δυνατότητα να απαντήσει μέσω αυτού του email.

<div class="text-center">
    <img src="/images/email-dm.png" alt="DM Email" title="DM Email" />
</div>

Όταν το κάνει, η απάντησή του θα εμφανιστεί **ζωντανά** στον άλλο χρήστη στην καρτέλα Μηνυμάτων του προφίλ τους.

### Ασφαλές SSO

Τα Προφίλ Χρηστών του FastComments είναι πλήρως συμβατά με τις υπάρχουσες εγκαταστάσεις Ασφαλούς SSO.

Πράγματα όπως οι ρυθμίσεις απορρήτου του χρήστη μπορούν επίσης να ρυθμιστούν μέσω της σημαίας `isProfileActivityPrivate` στο φορτίο SSO.

Από προεπιλογή, το `isProfileActivityPrivate` είναι true.

### Υπάρχοντες Πελάτες

Έχουμε ολοκληρώσει την κυκλοφορία αυτής της δυνατότητας σε όλους τους υπάρχοντες πελάτες σε όλες τις κατηγορίες.

### Ρύθμιση Απορρήτου

Με την προβολή του προφίλ σας, υπάρχουν επιλογές για:

- Να απενεργοποιήσετε το σχολιασμό στο προφίλ σας.
- Να εμποδίσετε την δραστηριότητά σας να είναι ορατή σε άλλους.
- Να αποτρέψετε άλλους από το να σας στείλουν άμεσα μηνύματα.

### Συλλεγόμενα Δεδομένα

Τα Προφίλ Χρηστών δημιουργούνται με βάση τα δεδομένα που έχουμε σήμερα, όπως το όνομά τους, ποια σχόλια έχουν αφήσει πρόσφατα κ.λπ.

Δεν θα συλλέξουμε πρόσθετα δεδομένα για να επεκτείνουμε τα προφίλ των χρηστών. Μας αρέσει να σκεφτόμαστε τα Προφίλ Χρηστών περισσότερο όπως ένα τυπικό προφίλ χρήστη σε ένα φόρουμ.

### Συμπερασματικά

Πιστεύουμε ότι αυτές οι δυνατότητες επεκτείνουν σημαντικά το FastComments ως πλατφόρμα και ελπίζουμε να τις βρείτε χρήσιμες και εύχρηστες.

Στην υγειά σας!
