## Τι Είναι Νέο

Προηγουμένως, οι συντονιστές έβλεπαν όλα τα σχόλια στον λογαριασμό σας όταν επισκέπτονταν τη σελίδα Διαχείρισης Σχολίων. Αν είχατε πολλούς τύπους περιεχομένου, όλοι οι συντονιστές
έβλεπαν την ακριβή ίδια λίστα. Για μεγαλύτερες ιστοσελίδες, αυτό θα μπορούσε να προκαλέσει κάποια σύγχυση σχετικά με το ποιοι συντονιστές διαχειρίζονται τι.

Τώρα παρέχουμε τη δυνατότητα να εμφανίζονται μόνο τα σχόλια που απαιτούν διαχείριση με βάση την ομάδα του συντονιστή.

## Πώς Τα Σχόλια Τοποθετούνται Σε Ομάδες

Αρχικά, πρέπει να ρυθμιστεί μια προσαρμοσμένη ολοκλήρωση ώστε όταν αφήνονται σχόλια, το FastComments να γνωρίζει σε ποια ομάδα θα πρέπει να ανήκει το αποτέλεσμα του σχολίου.

[Δείτε εδώ πώς να ρυθμίσετε την ολοκλήρωση.](https://docs.fastcomments.com/guide-customizations-and-configuration.html#moderation-group-ids)

## Πρόσθετες Ομάδες

Απλά πηγαίνοντας στη νέα [Σελίδα Ομάδων Διαχείρισης](https://fastcomments.com/auth/my-account/moderate-comments/moderation-groups), μπορούμε να ορίσουμε ομάδες με τα δικά τους ονόματα, να δούμε τα παραγόμενα ids που απαιτούνται για την ολοκλήρωση, και να παρακολουθήσουμε ακόμη και
τον αριθμό των συντονιστών και των σχολίων σε κάθε ομάδα.

Μπορείτε στη συνέχεια να επεξεργαστείτε έναν συντονιστή όπως θα κάνατε κανονικά, και μια νέα επιλογή θα είναι διαθέσιμη για να επιλέξετε τις ομάδες τους.

Σημειώστε ότι τόσο οι συντονιστές όσο και τα σχόλια μπορούν να ανήκουν σε περισσότερες από μία ομάδες.

Για λεπτομερή τεκμηρίωση σχετικά με την προσθήκη ομάδων, [ρίξτε μια ματιά στην τεκμηρίωση!](https://docs.fastcomments.com/guide-moderation.html#moderation-groups).

## Μεταφορά Υπαρχόντων Σχολίων

Μέσω αίτησης υποστήριξης, το FastComments.com μπορεί να μεταφέρει τα σχόλιά σας στις επιθυμητές ομάδες - απλά ενημερώστε μας!

## Επαλήθευση της Ολοκλήρωσης

Μπορείτε να επαληθεύσετε ότι η ολοκλήρωση έχει ρυθμιστεί σωστά ελέγχοντας τον Αριθμό Σχολίων στη σελίδα Ομάδων Διαχείρισης.

## Επίπτωση Σε Υπάρχοντες Πελάτες

Αν είστε υπάρχων πελάτης που χρησιμοποιεί το FastComments, η δυνατότητα αυτή είναι ήδη διαθέσιμη στον λογαριασμό σας.

Αν δεν κάνετε καμία αλλαγή για να χρησιμοποιήσετε αυτή τη δυνατότητα, οι συντονιστές σας δεν θα βιώσουν καμία αλλαγή στη ροή εργασίας τους.

## Συμπέρασμα

Ελπίζουμε να βρήκατε αυτή την ενημέρωση και την τεκμηρίωσή της χρήσιμη. Καλή σχολιαστική εμπειρία!
