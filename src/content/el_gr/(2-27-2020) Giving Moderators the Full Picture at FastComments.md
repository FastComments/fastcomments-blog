# [postlink]Δίνοντας στους Συντονιστές την Πλήρη Εικόνα στο FastComments[/postlink]

Συχνά η διαφορά μεταξύ του να χαρακτηριστεί κάτι ως spam και της έγκρισής του μπορεί να είναι το πλαίσιο - έτσι το FastComments παρέχει
ένα εύκολο τρόπο να μεταβείτε από ένα μεμονωμένο σχόλιο σε μια ακριβή θέση σε μια συνομιλία.

{{#isPost}}

Όταν μετριάζετε τα σχόλια, μπορείτε τώρα να κάνετε κλικ στον σύνδεσμο "Δείτε το Σχόλιο" στην κάτω δεξιά γωνία ενός σχολίου.
Αν έχει απαντήσεις, το κείμενο θα αλλάξει για να υποδείξει τον αριθμό των απαντήσεων. Ωστόσο, κάνοντας κλικ σε αυτό θα σας μεταφέρει
στο σχόλιο.

<img 
    src="images/fc-moderator-view-comment.png"
    alt="Πώς να Δείτε ένα Σχόλιο με Πλαίσιο"
    title="Πώς να Δείτε ένα Σχόλιο με Πλαίσιο"
    class='lozad' />

Αυτό θα σας μεταφέρει στον Εξερευνητή Συνομιλιών Σχολίων και θα μετακυλήσει γρήγορα τον περιηγητή σας σε αυτό το ακριβές σχόλιο.

Δεν υπάρχουν αργές κινήσεις ή πράγματα που να σας αποσπούν την προσοχή από τη δουλειά σας. Γρήγορα πηδάει στο σχόλιο που προσπαθείτε να δείτε.

<img 
    src="images/fc-moderator-view-comment-thread.png"
    alt="Εξερευνητής Συνομιλιών Σχολίων"
    title="Εξερευνητής Συνομιλιών Σχολίων"
    class='lozad' />

Έχετε υπόψη ότι αυτή είναι μια ζωντανή αναπαράσταση της συνομιλίας σχολίων. Οποιεσδήποτε απαντήσεις, ψήφοι ή ενέργεια που αναλάβετε θα είναι αμέσως ζωντανές.

Καλή μετρίαση!

{{/isPost}}
