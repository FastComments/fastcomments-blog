# [postlink]Ενημέρωση Συχνότητας Ειδοποιήσεων μέσω Email[/postlink]

{{#unless isPost}}
Οι χρήστες μπορούν πλέον να ελέγχουν πόσο συχνά λαμβάνουν ειδοποιήσεις μέσω email για νέα σχόλια και απαντήσεις, κάθε λεπτό, ωριαία περιγραφή ή ημερήσια περιγραφή, με μια μοναδική κοινή ρύθμιση, συν παρακάτω υπερκαλύψεις ανά συνδρομή.
{{/unless}}

{{#isPost}}

### Τι Νέο

Έχουμε προσθέσει μια ρύθμιση **Συχνότητας Ειδοποιήσεων Νέου Σχολίου** που ελέγχει πόσο συχνά λαμβάνετε ειδοποιήσεις μέσω email τόσο για ειδοποιήσεις απαντήσεων όσο και για ειδοποιήσεις ενοικιαστών (νέο σχόλιο). Οι τρεις επιλογές είναι:

- **Κάθε λεπτό** - λαμβάνετε ένα email μόλις φτάσουν νέα σχόλια (ελέγχεται κάθε λεπτό).
- **Ωριαία περιγραφή** - λαμβάνετε μια σωρευμένη περίληψη νέων σχολίων μια φορά την ώρα.
- **Ημερήσια περιγραφή** - λαμβάνετε μια σωρευμένη περίληψη νέων σχολίων μία φορά την ημέρα.

Αυτή η ρύθμιση είναι διαθέσιμη τόσο για τους διαχειριστές ενοικιαστών όσο και για τους σχολιαστές και ισχύει για όλα τα emails ειδοποιήσεων σχολίων. Σημειώστε ότι οι ειδοποιήσεις @mention αποστέλλονται πάντα αμέσως ανεξάρτητα από αυτή τη ρύθμιση.

Μπορείτε επίσης να παρακάμψετε τη συχνότητα σε επίπεδο συνδρομής στον πίνακα Συνδρομών, για πιο λεπτομερή έλεγχο στις ατομικές σελίδες.

### Πώς να το Ρυθμίσετε

1. Μεταβείτε στις [Ρυθμίσεις Ειδοποιήσεων](https://fastcomments.com/auth/my-account/edit-notifications).
2. Χρησιμοποιήστε το αναπτυσσόμενο μενού **Συχνότητας Ειδοποιήσεων Νέου Σχολίου** για να ρυθμίσετε την προτιμώμενη συχνότητα.
3. Προαιρετικά, παρακάμψτε τη συχνότητα για ατομικές συνδρομές στον πίνακα Συνδρομών.
4. Πατήστε **Αποθήκευση Αλλαγών**.

Η προεπιλογή είναι **Κάθε λεπτό**, η οποία ταιριάζει στη προηγούμενη συμπεριφορά.

### Υποστήριξη API

Η συχνότητα ειδοποιήσεων είναι επίσης διαθέσιμη μέσω του API. Η ρύθμιση επιπέδου χρήστη είναι το πεδίο `notificationFrequency`, και οι παρακάμψεις ανά συνδρομή μπορούν να οριστούν μέσω του πεδίου `notificationFrequency` της συνδρομής. Δείτε την [τεκμηρίωση API](https://docs.fastcomments.com/guide-api.html) για λεπτομέρειες.

### Σε Συμπέρασμα

Αυτό παρέχει στους χρήστες έλεγχο πάνω στο inbox τους χωρίς να χρειάζεται να αποσυνδεθούν εντελώς από τις σελίδες. Μια ρύθμιση καλύπτει τώρα τόσο τις ειδοποιήσεις απαντήσεων όσο και τις ειδοποιήσεις ενοικιαστών, διευκολύνοντας τη διαχείριση.

Πείτε μας παρακάτω αν έχετε σχόλια!
