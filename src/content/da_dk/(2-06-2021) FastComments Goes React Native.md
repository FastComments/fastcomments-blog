### Hvad Er Nyt

FastComments har nu React Native komponenter, der er på niveau med vores VanillaJS widget.

Vi besluttede at kalde biblioteket `fastcomments-react-native`. Du kan finde GitHub-repositoriet med kildekoden <a href="https://github.com/FastComments/fastcomments-react-native" target="_blank">her</a>.

Repoet indeholder også eksempler på anvendelse af widgetten.

Det er tilgængeligt på NPM, som du kan finde <a href="https://www.npmjs.com/package/fastcomments-react-native" target="_blank">her</a>.

### Hvordan Det Fungerer

Måden, denne nye komponent fungerer på, er, at React-komponenten er et omslag omkring den eksisterende FastComments widget.
 
Det betyder, at hvis vi tilføjer funktioner til vores eksisterende komponenter eller retter fejl, vil du stadig få glæde af det med det samme!

### Udviklingstid

React Native widgets kan tilføjes til din applikation på bare minutter med lidt udviklingstid krævet.

### SSO

Alle React Native biblioteker understøtter SSO uden ekstra konfiguration, og SSO integrationsprocessen er den samme som for Vanilla JS komponenterne. Den eksisterende
backendkode gælder også.

### Tilpasset Styling?

React Native komponenterne kan styles ved hjælp af CSS via Widget Tilpasnings UI, ligesom de web-baserede widgets.
