# [postlink]Introduktion af Kommentator Navneformater[/postlink]

{{#unless isPost}}
Dette er en lille funktion, som vi mener vil gavne nogle fællesskaber.
{{/unless}}

{{#isPost}}

### Hvad er nyt

FastComments viser som standard dit brugernavn, som du indtastede det. Tilsvarende, hvis dit brugernavn blev sendt til os via SSO, vil vi vise det. Vi understøtter
visning af **etiketter** ved siden af dit brugernavn, men det suppleres ikke dit brugernavn.

Nu kan lejere konfigurere, på et sitespecifikt eller side-specifikt niveau, hvordan brugernavne vil blive vist. For eksempel, givet brugernavnet "allen rex", kan vi i stedet
automatisk formatere det som:

- Stort begyndelsesbogstav (vis allen rex som Allen Rex)
- Sidste Initial (vis Allen Rex som Allen R.)
- Alle Initialer (vis Allen Red som A. R.)
- Vis "Anonym"

### Dokumentation

Dette kan opsættes via tilpasningssiden i dit administrationspanel.

[Dokumentation kan også findes her](https://docs.fastcomments.com/guide-customizations-and-configuration.html#user-name-formats)

### Afslutning

Vi takker vores kunder, der giver os løbende feedback, så vi kan tænke på idéer som denne. Vi håber, du
fortsætter med at elske FastComments.

Skål!
