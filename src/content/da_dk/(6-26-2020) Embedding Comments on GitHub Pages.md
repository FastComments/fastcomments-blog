# [postlink]Indlejring af kommentarer på GitHub Pages[/postlink]

{{#unless isPost}}
Vil du tilføje kommentarer til en statisk side, som GitHub Pages?
{{/unless}}

{{#isPost}}
#### Til udviklere
Installation af FastComments på en webside kræver kun at indsætte et lille kodeudsnit. Dette betyder, at det fungerer på statiske sider uden besvær - herunder GitHubs gratis Pages hosting.

#### En demo

Vi har opsat en eksempel GitHub Page <a href="https://fastcomments.github.io/fastcomments-demo.github.io/" target="_blank">her</a>.

<a href="https://github.com/fastcomments/fastcomments-demo.github.io/blob/master/index.html#L29" target="_blank">Her</a> er linket til, hvor vi har tilføjet kodeudsnittet til vores
testkonto.
