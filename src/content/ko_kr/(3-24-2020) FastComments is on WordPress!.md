# [postlink]FastComments가 WordPress에 출시되었습니다![/postlink]

다른 WordPress 댓글 플러그인에서 전환할 생각이신가요? FastComments를 사용해 보세요!

{{#isPost}}

많은 노력을 기울인 끝에, 우리는 WordPress 플러그인을 출시하게 된 것을 자랑스럽게 생각합니다. 여기에서 찾을 수 있습니다:
<a href="https://wordpress.org/plugins/fastcomments/" target="_blank">https://wordpress.org/plugins/fastcomments/</a>.

FastComments가 기본 WordPress 댓글보다 성능이 훨씬 좋다는 것을 알게 되었습니다. 특히 큰 댓글 스레드에서 더욱 그렇습니다.

FastComments는 WordPress 대신 댓글 스레드를 렌더링하므로 서버의 부담을 덜어주어 WordPress 사이트의 속도를 높입니다.

벤치마크는 거짓말을 하지 않습니다 - 부하가 없는 서버에서 100개의 댓글이 달린 게시물에서 기본 WordPress 설치는 초당 86개의 요청을 처리할 수 있습니다.

FastComments를 활성화하면 그 숫자가 **400 이상**으로 증가합니다.
