# [postlink]웹후크 구조 변경 사항[/postlink]

{{#unless isPost}}
웹후크 구조에 대한 일부 동작 관련 변경 사항을 적용했습니다.
{{/unless}}

{{#isPost}}

### 신규 사항

2023년 11월 14일 이전에, 웹후크에 다음과 같은 변경 사항을 적용했습니다:

#### 웹후크 업데이트

**이전에는**, 업데이트에는 댓글의 최신 버전이 포함되었습니다. 두 사용자가 댓글에 빠른 시간 안에 투표하면 두 개의 웹후크가 트리거되었습니다. 그러나 첫 번째 웹후크 이벤트가 **최신 버전**을 포함할 수 있어 두 개의 동일한 웹후크 이벤트가 발생할 수 있었습니다:

1. ` ... { votes: 2 } ...`
2. ` ... { votes: 2 } ...`

**11월 14일부터** 각 웹후크 이벤트는 멱등성이 있으며 변경 시점의 댓글 객체를 포함합니다. 위 예시에서 이제 두 개의 이벤트를 받게 됩니다:

1. ` ... { votes: 1 } ...`
2. ` ... { votes: 2 } ...`

#### 웹후크 삭제

**이전에는**, 삭제 웹후크는 댓글 ID만 포함되었습니다. 이것은 `{ id: string }`과 같은 객체였습니다.

**11월 14일부터**, 삭제 웹후크는 제거 시점의 **전체** 댓글 객체를 포함합니다.
