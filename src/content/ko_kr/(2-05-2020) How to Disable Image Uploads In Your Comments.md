# [postlink]댓글에서 이미지 업로드 비활성화하는 방법[/postlink]

FastComments를 사용하면 전체 사이트 또는 특정 페이지의 댓글 섹션에서 이미지 업로드를 비활성화할 수 있습니다.

{{#isPost}}
이렇게 하려면 대시보드에서 <a href="https://fastcomments.com/auth/my-account/customize-widget" target="_blank">사용자화</a>로 이동하여 사용자화 규칙을 추가하세요.

설정 활성화는 다음과 같이 보입니다:

<img 
    src="images/fc-disable-image-uploads.png"
    alt="이미지 업로드 비활성화"
    title="이미지 업로드 비활성화"
    class='lozad' />

... 그리고 저장을 클릭하세요.
이 페이지에서 도메인을 정의해야 하며, 이미 자동으로 채워져 있지 않은 경우에는 입력해야 합니다. URL ID 칸은 비워 두어 전체 사이트에 규칙을 적용하세요.

이 페이지에서는 이미지 업로드가 비활성화된 예시입니다.
