#### 개발자를 위한 안내
웹 페이지에 FastComments를 설치하려면 작은 코드 스니펫을 삽입하면 됩니다. 이는 GitHub의 무료 Pages 호스팅을 포함하여 정적 페이지에서도 문제없이 작동함을 의미합니다.

#### 데모

예제 GitHub Page를 <a href="https://fastcomments.github.io/fastcomments-demo.github.io/" target="_blank">여기</a>에 설정했습니다.

<a href="https://github.com/fastcomments/fastcomments-demo.github.io/blob/master/index.html#L29" target="_blank">여기</a>는 테스트 계정을 위한 코드 스니펫을 추가한 링크입니다.

#### 설정

GitHub 도메인을 계정의 유효한 도메인으로 추가해야 합니다.
