# [postlink]FastComments로 쉽게 코드 조각 지원하기[/postlink]

우리 고객 중 일부는 소프트웨어 개발 관련 사이트와 블로그를 운영하고 있습니다. 이러한 커뮤니티에서는 코드 조각을 공유하는 것이 일반적입니다. FastComments가 이를 어떻게 쉽게 만드는지 살펴보겠습니다.

{{#isPost}}

FastComments는 댓글 상자에 코드를 붙여넣기만 하면 공유할 수 있도록 합니다. 다른 텍스트와 이미지와 혼합할 수 있으며, 포맷을 추가하기 위해 수동으로 "코드" 태그를 추가할 필요가 없습니다. 간단히 붙여넣기만 하면 됩니다!

여기에 그 작동을 보여주는 간단한 데모가 있습니다.

<video src="images/fc-code-snippet-demo.mp4" autoplay controls loop=true alt="코드 조각 공유 데모" title="코드 조각 공유 데모"></video>

### 성능 영향

이 기능을 추가하면서 클라이언트 측 위젯에 약 150바이트만 추가되었습니다. 대부분의 기능은 서버 측에서 처리되기 때문입니다. 실제로, 최근에 추가된 Markdown 기능과 함께 이를 bundling하면 위젯 크기가 실제로 줄어들었습니다!

### 유의할 점

코드를 많이 공유하는 커뮤니티라면 코드 조각 공유를 쉽게 하도록 최대 댓글 크기를 늘리는 것을 고려해야 합니다. 이는 관리 대시보드의 "사용자 지정" 탭을 통해 가능합니다.

### 코드 감지

클라이언트 측 위젯 크기를 줄이기 위해 "{" 또는 공백을 제어 흐름으로 사용하는 C와 Python 유사 언어(예: Java, JavaScript, CSS, C++)에 대해서만 붙여넣기된 코드를 자동으로 감지합니다. 자동 감지되지 않는 언어를 자주 사용하고 있다면, 저희에게 연락해 주세요.

코드 블록을 수동으로 삽입하려면 다음과 같은 댓글을 작성하세요: ```<code>(defun someLispCode(1, 2, 3))</code>```. 우리는 약 150개 언어를 자동으로 감지할 수 있습니다.

### 기존 댓글 마이그레이션

FastComments로 마이그레이션하고 올바르게 포맷된 코드 조각이 포함된 많은 댓글이 있다면, <a href="https://fastcomments.com/auth/my-account/help" target="_blank">도와드릴 수 있습니다</a>.
