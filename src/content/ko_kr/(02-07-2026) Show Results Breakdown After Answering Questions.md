# [postlink]질문에 답변한 후 결과 분석 표시[/postlink]

{{#unless isPost}}
FastComments는 이제 사용자가 평가 또는 설문 질문에 답변한 후 모든 응답의 집계된 분석을 보여줄 수 있습니다.
{{/unless}}

{{#isPost}}

### 새로운 기능

별점 평가, NPS, 좋아요/싫어요, 또는 슬라이더 질문을 통해 피드백을 수집할 때, 이제 사용자가 답변한 직후에 집계된 결과 분석을 보여줄 수 있습니다.

이전에 사용자는 질문에 답변하고 자신의 선택이 강조 표시되는 것만 보았습니다. 이제 새로운 **질문에 답변한 후 결과 분석 표시** 설정으로, 해당 페이지의 다른 모든 사용자가 어떻게 응답했는지를 볼 수 있습니다.

### 어떻게 보일까요

**별점 평가**의 경우, 분석에는 평균 평점이 부분 채워진 별로 표시되며, 별별 분포 막대 차트와 비율, 총 응답 수가 포함됩니다.

**NPS 질문**의 경우, NPS 점수와 함께 후원자, 중립자 및 반대자 비율이 색상으로 구분된 막대와 함께 표시됩니다.

**좋아요/싫어요**의 경우, 사용자는 좋아요와 싫어요 간의 비율 분포를 봅니다.

**슬라이더 질문**의 경우, 평균 값과 총 응답 수가 표시됩니다.

하위 질문마다 개별적 분석도 제공됩니다.

### 활성화 방법

1. [사용자 UI 맞춤 설정](https://fastcomments.com/auth/my-account/customize-widget)으로 이동합니다.
2. 맞춤 설정 규칙을 편집하거나 새 규칙을 생성합니다.
3. 질문 설정 아래에서 **질문에 답변한 후 결과 분석 표시**를 체크합니다.
4. 저장합니다.

이게 전부입니다. 사용자가 답변을 제출한 후 해당 페이지에서 분석이 나타납니다.

### 성능

분석 데이터는 페이지별로 집계되어 서버 측에 캐시되므로, 상당한 부하를 추가하지 않습니다. 사용자가 새로운 답변을 제출하면 캐시가 갱신되어 분석이 즉시 최신 데이터를 반영합니다.

### 다크 모드

분석은 다크 모드를 완전히 지원합니다. 웹사이트가 어두운 배경을 사용할 경우, 막대, 라벨 및 텍스트 색상이 자동으로 적응합니다.

### 결론

이 기능은 사용자가 답변한 후 즉각적인 사회적 증거와 맥락을 제공하며, 평가 및 설문 참여를 증가시킬 수 있습니다. 네 가지 질문 유형 모두에서 작동하며 활성화를 위해 코드 변경이 필요 없습니다.
