# [postlink]댓글 관리 댓글 검색 업데이트[/postlink]

{{#unless isPost}}
FastComments는 이제 댓글 관리 페이지에서 페이지와 사용자별로 검색할 수 있습니다.
{{/unless}}

{{#isPost}}

### 새로운 기능

이전에는 FastComments가 댓글 관리 페이지에서 구문이나 정확한 문자열 일치로 검색할 수 있도록 했습니다.

이제 다음과 같은 방법으로 댓글을 검색할 수 있습니다:

- 모호한 단어 검색: **cats love**
- 정확한 구문 일치: **"I love cats."**
- **_새로운!_** 페이지 제목으로: **page:"Page Title"**
  - 자동 완성을 지원합니다.
  - 자동 완성 제안 목록에 각 페이지의 댓글 수를 표시합니다.
- **_새로운!_** 페이지 URL로: **page:"https://example.com/some-page"**
  - 자동 완성을 지원합니다.
  - 자동 완성 제안 목록에 각 페이지의 댓글 수를 표시합니다.
- **_새로운!_** 사용자로: **user:"Bob"**
  - 자동 완성을 지원합니다.

검색 결과는 댓글 관리 페이지에서 페이지 URL를 공유하여 다른 관리자나 보조자와 공유할 수 있습니다. "Go" 버튼을 누르면 검색 필드 값이 브라우저의 URL에 포함됩니다.

### 얻는 방법

이 기능은 모든 고객에게 배포되었습니다!

### 문서

문서 사이트가 업데이트되었으며, 이 기능에 대한 전용 문서가 [여기에서 유지됩니다](https://docs.fastcomments.com/guide-moderation.html#search).
