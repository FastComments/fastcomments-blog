# [postlink]새로운 IP 로그인 경고[/postlink]

{{#unless isPost}}
FastComments는 이제 새로운 IP 주소에서 로그인 감지 시 보안 경고 이메일을 발송하여 사용자가 무단 접근을 감지할 수 있도록 도와줍니다. 이메일에는 대략적인 위치가 포함됩니다.
{{/unless}}

{{#isPost}}

### 새로운 소식

FastComments는 이제 사용자의 계정에 대한 로그인 시 감지된 IP 주소가 이전에 본 적이 없는 경우 자동으로 보안 경고 이메일을 발송합니다. 이메일에는 다음이 포함됩니다:

- 로그인 위치의 **대략적인 위치**(도시 및 국가).
- 사용된 **IP 주소**.
- 로그인 **시간**.
- 로그인한 것이 아닌 경우 **비밀번호 변경**을 위한 직접 링크.

이는 모든 로그인 방법: 비밀번호 기반 로그인, 매직 링크 및 2단계 인증 흐름에 적용됩니다.

### 개인정보 보호

우리는 당신의 원시 IP 주소를 저장하지 않습니다. 보안 비교를 위한 목적에서만 변화된 형태를 저장합니다. 대략적인 위치는 로그인 시 결정되며 저장되지 않습니다.

**귀하의 IP는 위치를 판단하기 위해 제3자와 공유되지 않습니다.** 우리는 MaxMind 데이터베이스의 (매주 업데이트되는) 복사본에서 대략적인 위치를 추출하는 자체 서비스를 운영합니다.

### 경고가 전송되는 경우

다음 **모든** 조건이 충족되면 경고가 전송됩니다:

- 로그인이 성공적이었다.
- 사용자의 최초 로그인은 아니다.
- 해당 계정에 대해 이전에 성공적인 로그인에서 해당 IP 주소가 확인되지 않았다.

이는 여러분이 첫 로그인 시에는 알림을 받지 않으며, 동일한 IP에 대한 반복적인 알림을 받지 않음을 의미합니다.

### 결론

이는 귀하의 계정을 안전하게 유지하기 위한 또 다른 단계입니다. 인식하지 못하는 경고를 받고 계시다면, 즉시 비밀번호를 변경하는 것을 권장합니다.
