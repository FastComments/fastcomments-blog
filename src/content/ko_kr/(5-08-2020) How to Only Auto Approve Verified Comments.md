# [postlink]검증된 댓글만 자동 승인하는 방법[/postlink]

자동 승인이 활성화되어 있지만 사이트나 블로그에 검증된 댓글만 표시하고 싶으신가요?

{{#isPost}}

방법은 다음과 같습니다: <a href="https://fastcomments.com/auth/my-account/moderate-comments/settings" target="_blank">댓글 관리 -> 관리 설정 편집</a>로 이동하세요.

거기에서, 체크가 되어 있지 않다면 **댓글 자동 승인**을 체크하고 - 그 다음 **검증된 댓글만 자동 승인**을 체크한 후 **저장**을 클릭하세요.

<img 
    src="images/fc-verification-approves-comment.png"
    alt="검증된 경우 자동 승인을 하도록 설정하는 옵션 하이라이트"
    title="검증된 경우 자동 승인을 하도록 설정하는 옵션 하이라이트"
    class='lozad' />

이제 이메일로 검증되지 않은 댓글은 댓글 작성자가 검증할 때까지 표시되지 않습니다. 댓글을 남길 때 이메일을 확인하라는 친절한 메시지도 표시됩니다.

이 설정을 변경하는 것은 즉시 적용되지만, 과거 댓글에는 적용되지 않습니다.

즐거운 댓글 작성 되세요!
