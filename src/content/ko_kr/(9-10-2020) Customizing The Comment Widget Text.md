# [postlink]위젯 텍스트 사용자 지정[/postlink]

{{#unless isPost}}
위젯 텍스트 사용자 지정은 자주 묻는 지원 질문입니다. 이제는 자가 서비스입니다.
{{/unless}}

{{#isPost}}

#### 새 소식

이전에는 댓글 위젯의 텍스트를 사용자 지정하는 유일한 방법은 위젯을 삽입할 때 코드 변경을 하여 번역을 전달하거나 고객 지원 티켓을 제출하는 것이었습니다.

#### 작동 방식

다른 위젯 사용자 지정과 마찬가지로, 이제는 <a href="https://fastcomments.com/auth/my-account/customize-widget" target="_blank">사용자 지정</a> 페이지에서 처리됩니다.

텍스트를 사용자 지정하려는 도메인을 입력하고, 특정 URL 또는 모든 페이지에 대한 *을 입력하십시오.

페이지 중간 근처에서 "사용자 정의 텍스트 표시"를 클릭한 다음 "번역 재정의 선택"을 클릭합니다.

드롭다운에서 댓글 위젯의 기본 영어 텍스트로 번역을 검색하여 재정의하려는 텍스트를 찾을 수 있습니다.

변경을 진행하면 페이지 하단의 미리 보기에서 변경 사항이 나타납니다(하이라이트됨).

#### 기존 번역 사용자 지정 시 변경 사항

이전 방법 중 하나로 댓글 위젯에서 이미 텍스트를 사용자 지정한 경우 변경 사항이 없어야 합니다 - 당신의 재정의는 그대로 유지되어야 합니다.
