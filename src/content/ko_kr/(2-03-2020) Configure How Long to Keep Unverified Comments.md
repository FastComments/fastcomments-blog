# [postlink]검증되지 않은 댓글을 얼마나 오래 보관할지 구성하기[/postlink]

기본적으로 FastComments는 검증되지 않은 댓글을 3일 동안 보관합니다.

### "검증되지 않음"은 무슨 뜻인가요?

사용자가 댓글을 달고 이메일을 남기면 FastComments는 그들에게 댓글을 검증해 달라는 이메일을 보냅니다.  
이는 스팸과 싸우는 데 도움이 되며, 작성자가 실제로 보여주고 싶어하는 댓글을 강조합니다.

<a href="https://blog.fastcomments.com/(2-01-2020)-how-to-allow-anonymous-comments-with-fastcomments.html" target="_blank">익명 댓글 작성</a>을 활성화하면 댓글을 작성할 때 이메일을 남기지 않아도 됩니다!

하지만 이제 댓글을 얼마나 오래 보관할지 구성할 수 있습니다 - 아마도 자동으로 제거하지 않기를 원할 수도 있습니다! 이는 모두 Moderate Comments -> Edit Moderation Settings에서 구성 가능합니다.

{{#isPost}}
