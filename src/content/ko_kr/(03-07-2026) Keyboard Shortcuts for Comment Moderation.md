# [postlink]댓글 관리용 키보드 단축키[/postlink]

{{#unless isPost}}
파워 유저는 이제 내비게이션, 액션 및 기타 작업을 위한 전체 단축키 세트를 사용하여 키보드만으로 댓글을 완전히 관리할 수 있습니다.
{{/unless}}

{{#isPost}}

### 새로운 기능

댓글 관리 페이지는 이제 종합적인 키보드 단축키를 지원합니다. 마우스를 사용하지 않고도 댓글 사이를 탐색하고, 승인하고, 삭제하고, 차단하고, 스팸으로 표시하고, 컨텍스트를 전환하고, 로그를 볼 수 있습니다.

이 기능은 많은 댓글을 관리하는 팀에 매우 중요합니다. 이전에 마우스를 이동하고 버튼을 클릭해야 했던 모든 작업이 이제는 단일 키 누름으로 가능해졌습니다.

### 예시 흐름

검토할 댓글 대기열이 있다고 가정해보세요. **j**를 눌러 첫 번째 댓글에 포커스를 맞춥니다. **c**를 눌러 해당 댓글의 컨텍스트를 확장하여 주변 스레드를 확인합니다. 괜찮으니 **c**를 다시 눌러 컨텍스트를 축소하고, **a**를 눌러 승인합니다. **j**를 눌러 다음 댓글로 이동하고 반복합니다. 뭔가 잘못된 것 같으면 **s**를 눌러 스팸으로 표시하거나 **d**를 눌러 삭제합니다. 마우스를 전혀 사용하지 않고도 전체 페이지의 댓글을 처리할 수 있습니다.

### 내비게이션

**j**를 눌러 다음 댓글로 포커스를 이동하고, **k**를 눌러 이전 댓글로 이동합니다. 포커스가 맞춰진 댓글은 선택된 (체크된) 댓글과 구분하기 위해 파란색 윤곽선으로 강조 표시됩니다.

**x** 또는 **Enter**를 눌러 포커스가 맞춰진 댓글의 선택을 전환합니다. **Shift**를 누른 채 **j** 또는 **k**를 눌러 현재 댓글을 선택하고 포커스를 이동시키는 동작을 빠르게 수행할 수 있습니다. 

### 액션

댓글에 포커스가 맞춰져 있을 때, 단일 키 단축키가 툴바 버튼과 동일한 액션을 트리거합니다:

- **a** - 승인 / 비승인 전환
- **s** - 스팸 / 비스팸 전환
- **d** - 삭제
- **u** - 삭제 취소
- **r** - 검토됨 / 검토되지 않음 전환
- **e** - 편집
- **b** - 차단 / 차단 관리
- **p** - 고정 / 고정 해제 전환
- **f** - 플래그 해제
- **i** - 모든 IP 보기
- **t** - 신뢰 계수 설정
- **m** - 배지 관리
- **l** - 로그 보기
- **c** - 컨텍스트 전환

### 글로벌 단축키

댓글에 포커스가 맞춰져 있는지 여부에 관계없이 몇 가지 단축키가 작동합니다:

- **Ctrl+a** - 모든 댓글 선택
- **Ctrl+z** - 마지막 작업 실행 취소
- **/** - 검색 입력으로 포커스 이동
- **[** 및 **]** - 이전 또는 다음 페이지로 이동

### 도움 및 가이드

언제든지 **?**를 눌러 사용 가능한 모든 단축키를 나열하는 도움말 대화 상자를 엽니다.

**g**를 눌러 "가이드" 모드를 전환하면 포커스가 맞춰진 댓글의 액션 버튼에 해당 단축키를 보여주는 작은 배지가 표시됩니다. 이는 단축키를 배우는 데 유용합니다.

모든 단축키는 검색 상자나 다른 입력 필드에서 입력할 때와 대화 상자가 열려 있을 때는 억제됩니다. **Escape**를 눌러 포커스를 맞춘 입력을 블러 처리하고 단축키 모드로 돌아갑니다.
