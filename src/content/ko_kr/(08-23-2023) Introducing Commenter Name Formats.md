# [postlink]댓글 작성자 이름 형식 소개[/postlink]

{{#unless isPost}}
이것은 일부 커뮤니티에 도움이 될 것이라고 생각되는 작은 기능 출시입니다.
{{/unless}}

{{#isPost}}

### 새로운 기능

FastComments는 기본적으로 입력한 대로 사용자 이름을 표시합니다. 마찬가지로, 사용자 이름이 SSO를 통해 전달된 경우 해당 이름을 표시합니다. 우리는 사용자 이름과 함께 표시될 **레이블**을 지원하지만, 이는 사용자 이름을 보완하지 않습니다.

이제 테넌트는 사이트 전체나 페이지 특정 수준에서 사용자 이름 표시 방법을 구성할 수 있습니다. 예를 들어, 사용자 이름 "allen rex"가 주어지는 경우, 이를 자동으로 다음과 같이 형식화할 수 있습니다:

- 대문자 (allen rex를 Allen Rex로 표시)
- 성의 첫 글자 (Allen Rex를 Allen R.로 표시)
- 모든 이니셜 (Allen Red를 A. R.로 표시)
- "익명" 표시

### 문서

이는 관리 대시보드의 사용자화 페이지를 통해 설정할 수 있습니다.

[문서는 여기에서도 확인할 수 있습니다](https://docs.fastcomments.com/guide-customizations-and-configuration.html#user-name-formats)

### 결론

지속적인 피드백을 주시는 고객들에게 감사드리며, 이러한 아이디어를 생각할 수 있게 해주셔서 감사합니다. 여러분이 FastComments를 계속 사랑해주시기를 바랍니다.
