# [postlink]FastComments is op WordPress![/postlink]

Overweeg je om over te schakelen van een andere WordPress reactieplugin? Probeer FastComments!

{{#isPost}}

We zijn trots aan te kondigen dat we, na veel werk, onze WordPress plugin hebben gelanceerd. Je kunt het hier vinden:
<a href="https://wordpress.org/plugins/fastcomments/" target="_blank">https://wordpress.org/plugins/fastcomments/</a>.

We hebben ontdekt dat FastComments veel beter presteert dan de standaard WordPress reacties, vooral voor grotere discussie-threads.

Het zal je WordPress site versnellen door de belasting van je server te verlichten - aangezien wij de discussie-threads renderen in plaats van WordPress.

De benchmarks liegen niet - bij 100 reacties op een post die draait op een server zonder belasting kan een standaard WordPress installatie 86 verzoeken per seconde verwerken voordat deze in elkaar stort.

Met FastComments ingeschakeld, springt dat aantal omhoog naar **meer dan 400**.

<img 
    src="images/fc-wordpress-stats.png"
    alt="Benchmark Resultaten"
    title="Benchmark Resultaten"

We hopen dat mensen de plugin als eenvoudig te gebruiken, aanpasbaar, en over het algemeen "gewoon werkt" ervaren. Het zou ook naadloos je WordPress reacties heen en weer moeten synchroniseren zonder druk op je servers te zetten.
