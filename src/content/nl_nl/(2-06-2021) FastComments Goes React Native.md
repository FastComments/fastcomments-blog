### <i class="circle">!</i> Dit Artikel Bevat Technisch Jargon

### Wat is Nieuw

FastComments heeft nu React Native componenten die gelijkwaardig zijn aan onze VanillaJS widget.

We hebben besloten de bibliotheek `fastcomments-react-native` te noemen. Je kunt de GitHub-repository met de source code <a href="https://github.com/FastComments/fastcomments-react-native" target="_blank">hier</a> vinden.

De repository bevat ook voorbeeldgebruik van de widget.

Het is op NPM, wat je <a href="https://www.npmjs.com/package/fastcomments-react-native" target="_blank">hier</a> kunt vinden.

### Hoe het Werkt

De manier waarop deze nieuwe component werkt, is dat de React-component een wrapper is rond de bestaande FastComments widget.
 
Dit betekent dat als we functies toevoegen aan onze bestaande componenten of bugs oplossen, je daar meteen van profiteert!

### Ontwikkeltijd

De React Native widgets kunnen in slechts enkele minuten aan je applicatie worden toegevoegd met weinig ontwikkeltijd.

### SSO

Alle React Native bibliotheken ondersteunen SSO direct, en het SSO-integratieproces is hetzelfde als bij de Vanilla JS componenten. De bestaande
backend codevoorbeelden zijn ook van toepassing.

### Aangepaste Stijling?

De React Native componenten kunnen worden gestyled met CSS via de Widget Aanpassings-UI, net als de webgebaseerde widgets.
