#### Voor Ontwikkelaars
Het installeren van FastComments op een webpagina vereist alleen het invoegen van een kleine codefragment. Dit betekent dat het werkt op statische pagina's zonder enige rompslomp - inclusief GitHub's gratis Pages hosting.

#### Een Demo

We hebben een voorbeeld GitHub-pagina <a href="https://fastcomments.github.io/fastcomments-demo.github.io/" target="_blank">hier</a> opgezet.

<a href="https://github.com/fastcomments/fastcomments-demo.github.io/blob/master/index.html#L29" target="_blank">Hier</a> is de link naar waar we de codefragment hebben toegevoegd voor ons
testaccount.
