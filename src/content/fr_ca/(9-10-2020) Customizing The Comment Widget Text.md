# [postlink]Personnaliser le texte du widget[/postlink]

{{#unless isPost}}
La personnalisation du texte du widget est une question fréquente en matière de support. Maintenant, c'est en libre-service.
{{/unless}}

{{#isPost}}

#### Quoi de Neuf

Auparavant, la seule façon de personnaliser le texte dans le widget de commentaires était soit de modifier le code lors de l’intégration du widget pour passer les traductions, soit de soumettre un ticket au support client.

#### Comment Ça Fonctionne

Comme pour d'autres personnalisations de widget, cela se fait maintenant sur la page des <a href="https://fastcomments.com/auth/my-account/customize-widget" target="_blank">Personnalisations</a>.

Entrez le domaine pour lequel vous souhaitez personnaliser le texte, ainsi qu'une URL spécifique ou * pour toutes les pages.

Vers le milieu de la page, cliquez sur "Afficher le texte personnalisé" puis sur "Sélectionner l'écrasement de traduction".

Dans le menu déroulant, vous pouvez rechercher des traductions par le texte anglais par défaut dans le widget de commentaires pour trouver le texte que vous souhaitez écraser.

Lorsque vous apportez des modifications, elles apparaîtront dans l'aperçu en bas de la page (comme indiqué).

#### Qu'est-ce qui change pour les personnalisations de traduction existantes

Si vous avez déjà personnalisé votre texte dans le widget de commentaires avec l'une des méthodes précédentes, rien ne devrait changer - vos écrasements devraient rester tels quels.

#### En Conclusion

Nous espérons qu'en fournissant plus d'outils en libre-service comme celui-ci, la configuration de FastComments soit plus facile pour chaque type de client que nous avons. Faites-nous savoir dans la section des commentaires ci-dessous 
comment cela a fonctionné pour vous.

À la vôtre !
