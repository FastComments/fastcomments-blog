# [postlink]Avatars par défaut de FastComments utilisant Gravatar[/postlink]

### Qu'est-ce qu'un avatar ?

Un avatar est l'icône de profil associée à chaque compte et affichée à côté de chaque commentaire.

### Qu'est-ce que Gravatar ?

<a href="https://en.gravatar.com/" target="_blank">Gravatar</a> se décrit comme "Un avatar reconnu mondialement".
C'est un service où vous pouvez définir un avatar pour une adresse e-mail donnée. Tous les produits auxquels vous vous inscrivez en utilisant cette adresse e-mail peuvent alors
utiliser cet avatar. Cela supprime une étape pour les personnes qui s'inscrivent à FastComments puisque leur avatar est configuré automatiquement.

### Alors quel est le changement ?

FastComments utilisera maintenant des avatars par défaut provenant de Gravatar. Vous ou vos utilisateurs n'avez rien à faire pour bénéficier de cette fonctionnalité.

Si vos utilisateurs existants ont déjà défini leur propre avatar - pas de souci - nous n’allons pas écraser leurs avatars personnalisés.
De plus, les utilisateurs peuvent toujours remplacer l'avatar Gravatar dans FastComments.
