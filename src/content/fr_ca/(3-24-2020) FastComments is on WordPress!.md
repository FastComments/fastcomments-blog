# [postlink]FastComments est sur WordPress ![/postlink]

Vous envisagez de passer d'un autre plugin de commentaires WordPress ? Essayez FastComments !

{{#isPost}}

Nous sommes fiers d'annoncer qu'après beaucoup de travail, nous avons lancé notre plugin WordPress. Vous pouvez le trouver ici :
<a href="https://wordpress.org/plugins/fastcomments/" target="_blank">https://wordpress.org/plugins/fastcomments/</a>.

Nous avons constaté que FastComments fonctionne beaucoup mieux que les commentaires par défaut de WordPress, en particulier pour les longues discussions.

Cela accélérera votre site WordPress en allégeant la charge sur votre serveur - puisque nous rendrons les fils de commentaires au lieu de WordPress.

Les benchmarks ne mentent pas - avec 100 commentaires sur un article fonctionnant sur un serveur sans charge, une installation vanilla de WordPress peut gérer 86 requêtes par seconde avant de s'effondrer.

Avec FastComments activé, ce nombre passe à **plus de 400**.

    src="images/fc-wordpress-stats.png"

Nous espérons que les utilisateurs trouveront le plugin facile à utiliser, à personnaliser, et dans l'ensemble "ça fonctionne tout simplement". Il devrait synchroniser les commentaires WordPress de manière fluide sans mettre de pression sur vos serveurs.
