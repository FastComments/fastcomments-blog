# [postlink]Comment approuver automatiquement uniquement les commentaires vérifiés[/postlink]

Avez-vous activé l'approbation automatique, mais souhaitez-vous que seuls les commentaires vérifiés apparaissent sur votre site ou blog?

{{#isPost}}

Voici comment faire : Allez sur <a href="https://fastcomments.com/auth/my-account/moderate-comments/settings" target="_blank">Modérer les commentaires -> Modifier les paramètres de modération</a>.

À partir de là, cochez **Approuver automatiquement les commentaires** s'il n'est pas déjà coché - puis **N'approuver que les commentaires vérifiés automatiquement** et cliquez sur **enregistrer**.

<img 
    src="images/fc-verification-approves-comment.png"
    alt="L'option d'approbation automatique lorsque vérifié mise en surbrillance"
    title="L'option d'approbation automatique lorsque vérifié mise en surbrillance"
    class='lozad' />

Maintenant, tout commentaire qui n'est pas vérifié par email ne sera pas affiché tant que le commentateur ne l'aura pas vérifié. On lui affichera également un message amical
lorsqu'il laissera son commentaire, lui indiquant qu'il doit vérifier son email.

Changer ce paramètre est immédiat, cependant cela ne s'appliquera pas pour les commentaires passés.

Bon commentaire!

{{/isPost}}
