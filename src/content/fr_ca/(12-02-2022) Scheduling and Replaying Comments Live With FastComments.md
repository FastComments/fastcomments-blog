### Quoi de Neuf

FastComments peut maintenant être utilisé pour planifier automatiquement et lire les commentaires sur une période donnée.

Cela est utile pour les webinaires ou d'autres événements en direct.

### Comment Cela Fonctionne

Avec les Commentaires Planifiés, il suffit de télécharger un fichier CSV décrivant les commentaires que vous souhaitez publier. Ce CSV inclut où et quand publier ces commentaires. Chaque commentaire peut avoir son propre nom d'utilisateur et avatar, et est publié **en direct**.

Vous pouvez rejouer le même fil de commentaires encore et encore, ce qui est utile pour les webinaires ou les scénarios de formation.

### Comment l'Obtenir & Documentation

Tous les clients de tous les niveaux ont accès aux Commentaires Planifiés. [Vous pouvez trouver la documentation sur les Commentaires Planifiés ici.](https://docs.fastcomments.com/guide-scheduled-comments.html).

Vous pouvez trouver l'[interface des Commentaires Planifiés ici](https://fastcomments.com/auth/my-account/manage-data/import-scheduled).
