#### Public cible

Cet article s'adresse à toute personne souhaitant ajouter un support pour les commentaires à son site Webflow.

#### Introduction

Installer FastComments sur un site Webflow est aussi simple que sur n'importe quel autre site - avec seulement une particularité unique aux fournisseurs d'hébergement comme Webflow.

#### Les étapes

1. Ajoutez le domaine de votre site Webflow à votre compte - le widget ne se chargera pas sans cela. C'est le domaine utilisé lorsque vous cliquez sur "Publier" dans Webflow, par exemple "https://your-superb-project.webflow.io".
2. En mode Design, allez dans "Ajouter des éléments" en haut à gauche, puis faites défiler jusqu'en bas pour "Composants". Cliquez sur "Intégrer" - ajoutez le snippet de code FastComments que vous pouvez trouver <a href="https://fastcomments.com/auth/my-account/get-acct-code" target="_blank">ici</a>.

#### Particularités

FastComments sécurise votre installation en validant les demandes par nom de domaine. Si vous souhaitez que le widget de commentaire s'affiche pour votre site Webflow avant d'avoir un nom de domaine dédié, **vous devrez ajouter "weblfow.io" comme domaine à votre compte <a href="https://fastcomments.com/auth/my-account/configure-domains" target="_blank">ici</a>.**

Pour une sécurité accrue, vous pouvez définir le nom de domaine complet, par exemple "your-awesome-site.webflow.io".
