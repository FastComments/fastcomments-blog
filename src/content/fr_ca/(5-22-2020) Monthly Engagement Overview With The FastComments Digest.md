# [postlink]Aperçu de l'engagement mensuel avec le FastComments Digest[/postlink]

Vous voulez un rappel régulier des commentaires qui pourraient nécessiter de l'attention, sans recevoir des courriels chaque heure ?

{{#isPost}}

Maintenant, vous le pouvez ! Dans le cadre de ce déploiement, par défaut, tous les propriétaires de site et modérateurs recevront le FastComments Digest, mensuel, à partir de juin.

Pour vous désinscrire, ou pour configurer la fréquence, allez à <a href="https://fastcomments.com/auth/my-account/edit-notifications" target="_blank">Mon Compte -> Notifications</a>.

De là, sous **Calendrier du Digest**, vous pouvez choisir une option comme **Hebdomadaire** ou **Désactivé** et cliquer sur **enregistrer**.

En termes de ce à quoi s'attendre dans le courriel - notre objectif était d'inclure deux choses.

### Métriques d'engagement

Le FastComments Digest comprend des compteurs pour les Nouveaux Commentaires, Nouveaux Utilisateurs, et Nouveaux Spams. Tous ces indicateurs sont relatifs à la date d'envoi du dernier courriel digest.

Chaque indicateur inclut également une flèche vers le haut ou vers le bas pour indiquer le changement depuis le dernier courriel.

### Commentaires récents

Le courriel Digest inclut jusqu'à trois commentaires récents qui peuvent être directement approuvés ou auxquels il est possible de répondre en cliquant sur des boutons dans le courriel. Approuver depuis ce courriel marque également les commentaires comme examinés. En ce moment, ce courriel est statique - cependant, nous avons une surprise à venir dans une future version !

### Montrez-le moi

Comme le reste de FastComments, le courriel Digest est léger, facile à lire et très fonctionnel.

### En Conclusion

Nous espérons que vous trouverez ce courriel programmé utile et amical pour vous rappeler les commentaires qui pourraient nécessiter une approbation. Nous espérons également que si vous ne souhaitez pas utiliser cette fonctionnalité, vous la trouverez facile à désactiver.

Si vous constatez quelque chose manquant dans ce courriel, ou si vous avez des problèmes de compatibilité avec votre client de courriel, veuillez contacter votre représentant du service à la clientèle.
