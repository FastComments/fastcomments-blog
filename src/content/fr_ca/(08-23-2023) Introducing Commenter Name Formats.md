# [postlink]Présentation des formats de nom de commentateur[/postlink]

{{#unless isPost}}
Il s'agit d'un petit lancement de fonctionnalité qui, selon nous, sera bénéfique pour certaines communautés.
{{/unless}}

{{#isPost}}

### Quoi de neuf

FastComments affiche par défaut votre nom d'utilisateur, tel que vous l'avez saisi. De même, si votre nom d'utilisateur a été transmis via SSO, nous l'afficherons. Nous supportons les
**étiquettes** à afficher aux côtés de votre nom d'utilisateur, mais elles ne complètent pas votre nom d'utilisateur.

Désormais, les locataires peuvent configurer, à l'échelle du site ou de manière spécifique à la page, la façon dont les noms d'utilisateur seront affichés. Par exemple, pour le nom d'utilisateur "allen rex", nous pouvons plutôt le
formater automatiquement comme suit :

- Mettre en majuscule (afficher allen rex comme Allen Rex)
- Initiale du dernier nom (afficher Allen Rex comme Allen R.)
- Toutes les initiales (afficher Allen Red comme A. R.)
- Afficher "Anonyme"

### Documentation

Cela peut être configuré via la page Personnaliser dans votre tableau de bord d'administration.

[La documentation peut également être consultée ici](https://docs.fastcomments.com/guide-customizations-and-configuration.html#user-name-formats)

### En conclusion

Nous remercions nos clients qui nous donnent des retours continus afin que nous puissions penser à des idées comme celle-ci. Nous espérons que vous
continuerez à aimer FastComments.

Santé !
