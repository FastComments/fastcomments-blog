# [postlink]Comment désactiver les téléchargements d'images dans vos commentaires[/postlink]

Avec FastComments, vous pouvez désactiver les téléchargements d'images dans la section des commentaires de votre site Web, que ce soit pour l'ensemble du site ou juste pour une page particulière.

{{#isPost}}
Pour ce faire, allez dans <a href="https://fastcomments.com/auth/my-account/customize-widget" target="_blank">Personnaliser</a> dans le tableau de bord et ajoutez une règle de personnalisation.

Activer le paramètre ressemble à ceci :

<img 
    src="images/fc-disable-image-uploads.png"
    alt="Désactiver les téléchargements d'images"
    title="Désactiver les téléchargements d'images"
    class='lozad' />

... puis cliquez sur enregistrer.
Sur cette page, vous devez définir votre domaine s'il n'est pas déjà rempli. Laissez le champ ID URL vide pour appliquer la règle à l'ensemble de votre site.

Cette page a des téléchargements d'images désactivés en exemple.

{{/isPost}}
