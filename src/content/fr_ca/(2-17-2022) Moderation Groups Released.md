# [postlink]Groupes de Modération Émis[/postlink]

{{#unless isPost}}
Vous avez plus d'un type de contenu et souhaitez que différentes équipes de modérateurs gèrent leur propre contenu ? Vous pouvez maintenant le faire avec FastComments !
{{/unless}}

{{#isPost}}

## Quoi de Neuf

Auparavant, les modérateurs voyaient tous les commentaires dans votre compte en visitant la page Modérer les Commentaires. Si vous aviez de nombreux types de contenu, tous les modérateurs
voyageaient la même liste exacte. Pour les sites plus importants, cela pouvait entraîner une certaine confusion quant à quel modérateur modérait quoi.

Nous offrons maintenant la possibilité d'afficher uniquement les commentaires nécessitant une modération en fonction du groupe du modérateur.

## Comment les Commentaires Sont Classés Dans des Groupes

Tout d'abord, une intégration personnalisée doit être configurée afin que, lorsque des commentaires sont laissés, FastComments sache dans quel groupe le commentaire résultant doit être placé.

[Consultez ici pour savoir comment configurer l'intégration.](https://docs.fastcomments.com/guide-customizations-and-configuration.html#moderation-group-ids)

## Ajout de Groupes

En vous rendant simplement sur la nouvelle [Page des Groupes de Modération](https://fastcomments.com/auth/my-account/moderate-comments/moderation-groups), nous pouvons définir des groupes avec leurs propres noms, voir les ids générés nécessaires pour l'intégration, et même
surveiller le nombre de modérateurs et de commentaires dans chaque groupe.

Vous pouvez ensuite éditer un modérateur comme vous le feriez normalement, et une nouvelle option sera disponible pour sélectionner leurs groupes.

Notez que les modérateurs et les commentaires peuvent appartenir à plus d'un groupe.

Pour une documentation détaillée sur l'ajout de groupes, [consultez la documentation !](https://docs.fastcomments.com/guide-moderation.html#moderation-groups).

## Migration des Commentaires Existants

Via une demande de support, FastComments.com peut migrer vos commentaires vers vos groupes souhaités - faites-le nous savoir !

## Vérification de l'Intégration

Vous pouvez vérifier que l'intégration est correctement configurée en vérifiant le Compte de Commentaires dans la page des Groupes de Modération.

## Impact sur les Clients Existants

Si vous êtes un client existant utilisant FastComments, la fonctionnalité a déjà été mise à disposition de votre compte.

Si vous ne faites aucun changement pour utiliser cette fonctionnalité, vos modérateurs ne connaîtront aucun changement dans leur flux de travail.

## En Conclusion

Nous espérons que vous avez trouvé cette mise à jour et sa documentation utile. Bonne modération des commentaires !
