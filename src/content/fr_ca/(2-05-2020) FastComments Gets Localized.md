# [postlink]FastComments devient localisé[/postlink]

La localisation n'est pas une petite étape - et elle est arrivée chez FastComments. Le widget de commentaires intégrable est maintenant entièrement localisé en anglais (en-us), espagnol (es-es), et français (fr-fr).

Vous remarquerez que nous n'avons pas ralentis non plus. Généralement, lorsque la plupart des applications ajoutent le support de la localisation, le code commence à gonfler. Dans notre cas, le script côté client a en fait diminué de taille alors que nous avons également effectué un certain ménage et d'autres optimisations en même temps.

Si vous souhaitez que FastComments soit traduit dans votre langue, ou si vous avez remarqué un problème de traduction, <a href="https://fastcomments.com/auth/my-account/help">faites-le nous savoir</a>.
