# [postlink]Modération du Contenu d'Image avec FastComments[/postlink]

{{#unless isPost}}
Gardez votre section de commentaires sécurisée avec cette mise à jour.
{{/unless}}

{{#isPost}}
FastComments prend désormais en charge le blocage des images en fonction de leur contenu.

Les images téléchargées dans les sections de commentaires peuvent maintenant être automatiquement filtrées en fonction de différents niveaux de vulgarité. Cela vous donne plus de contrôle sur ce qui apparaît dans les discussions de votre communauté.

Vous pouvez choisir parmi quatre niveaux de modération différents :

- **Désactivé** : Pas de modération (défaut)
- **Bas** : Bloque le contenu sexuel explicite et violent
- **Moyen** : Bloque également le contenu de haine, de harcèlement et de auto-mutilation  
- **Élevé** : Bloque en plus le contenu légèrement suggestif

Cette fonctionnalité fonctionne pour toutes les images nouvellement téléchargées sur toutes les sections de commentaires de votre site, offrant une modération cohérente sans nécessiter de révision manuelle.

#### Documentation

Vous pouvez trouver la documentation ici : [https://docs.fastcomments.com/guide-customizations-and-configuration.html#image-content-moderation](https://docs.fastcomments.com/guide-customizations-and-configuration.html#image-content-moderation).
