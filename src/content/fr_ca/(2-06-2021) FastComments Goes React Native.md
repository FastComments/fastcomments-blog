### Quoi de neuf

FastComments dispose maintenant de composants React Native qui sont à égalité fonctionnelle avec notre widget VanillaJS.

Nous avons décidé d'appeler la bibliothèque `fastcomments-react-native`. Vous pouvez trouver le dépôt GitHub avec le code source <a href="https://github.com/FastComments/fastcomments-react-native" target="_blank">ici</a>.

Le dépôt contient également des exemples d'utilisation du widget.

Il est sur NPM, que vous pouvez trouver <a href="https://www.npmjs.com/package/fastcomments-react-native" target="_blank">ici</a>.

### Comment cela fonctionne

Le fonctionnement de ce nouveau composant est que le composant React est un wrapper autour de l'widget FastComments existant.

Cela signifie que si nous ajoutons des fonctionnalités à nos composants existants ou corrigeons des bogues, vous en bénéficierez immédiatement !

### Temps de développement

Les widgets React Native peuvent être ajoutés à votre application en quelques minutes avec peu de temps de développement requis.

### SSO

Toutes les bibliothèques React Native prennent en charge le SSO par défaut, et le processus d'intégration SSO est le même que pour les composants Vanilla JS. Les exemples de code backend existants s'appliquent également.

### Personnalisation du style ?

Les composants React Native peuvent être stylisés en utilisant CSS via l'interface de personnalisation du widget, de la même manière que pour les widgets basés sur le web.
