# [postlink]Einbettbare Kommentare für Webflow[/postlink]

{{#unless isPost}}
Anleitungen, einschließlich eines Videos, zur Installation von FastComments auf einer Webflow.io-Website.
{{/unless}}

{{#isPost}}

#### Zielgruppe

Dieser Artikel richtet sich an alle, die Unterstützung für Kommentare zu ihrer Webflow-Website hinzufügen möchten.

#### Einführung

Die Installation von FastComments auf einer Webflow-Website ist so einfach wie die Installation auf jeder anderen - mit nur einem speziellen Punkt, der für Hosting-Anbieter wie Webflow einzigartig ist.

#### Die Schritte

1. Fügen Sie die Domain Ihrer Webflow-Website zu Ihrem Konto hinzu - das Widget wird ohne sie nicht geladen. Dies ist die Domain, die verwendet wird, wenn Sie in Webflow auf "Veröffentlichen" klicken, zum Beispiel "https://your-superb-project.webflow.io".
2. Gehen Sie im Designmodus auf "Elemente hinzufügen" in der oberen linken Ecke, und scrollen Sie dann ganz nach unten zu "Komponenten". Klicken Sie auf "Einbetten" - fügen Sie den FastComments-Code-Snippet hinzu, den Sie <a href="https://fastcomments.com/auth/my-account/get-acct-code" target="_blank">hier</a> finden können.

#### Besondere Hinweise

FastComments sichert Ihre Installation, indem es Anfragen nach Domainnamen validiert. Wenn Sie möchten, dass das Kommentar-Widget für Ihre Webflow-Website angezeigt wird, bevor Sie einen eigenen Domainnamen haben, **müssen Sie "weblfow.io" als Domain zu Ihrem Konto hinzufügen <a href="https://fastcomments.com/auth/my-account/configure-domains" target="_blank">hier</a>.**

Für erhöhte Sicherheit können Sie den gesamten Domainnamen definieren, zum Beispiel "your-awesome-site.webflow.io".
