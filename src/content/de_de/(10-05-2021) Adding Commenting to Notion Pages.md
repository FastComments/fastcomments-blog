# [postlink]Kommentare zu Notion-Seiten hinzufügen[/postlink]

{{#unless isPost}}
Möchten Sie Kommentare zu Ihren Notion-Seiten hinzufügen? Wir haben das für Sie.
{{/unless}}

{{#isPost}}

## Neuigkeiten

FastComments ist jetzt ein Embed.ly-Anbieter und stellt somit APIs und Dienste zur Verfügung, die es ermöglichen, Kommentarstränge in Anwendungen einzubetten, die [oEmbed](https://oembed.com) unterstützen.

Das bedeutet, dass Sie jetzt ganz einfach Live-Kommentare zu einer mit Notion, Google Sites oder Nimbus Note erstellten Website hinzufügen können.

## So verwenden Sie es - Eine kurze Demo

Wenn Sie in Ihrem Tenant angemeldet sind, können Sie einfach [fastcomments.com/create](https://fastcomments.com/create) besuchen, um einen Gesprächslink zu erstellen, der dann zu Ihrer Notion-Seite oder einer anderen Anwendung hinzugefügt werden kann, die mit embed.ly integriert ist.

Sie können es selbst ausprobieren, indem Sie Ihren Demobindungslink verwenden, [fastcomments.com/c/demo](https://fastcomments.com/c/demo):

<video src="images/fastcomments-notion.mp4" controls alt="FastComments und Notion Demo" title="FastComments und Notion Demo"></video>

Das eingebettete Kommentar-Widget kann dann genau wie jede andere Instanz von FastComments angepasst werden.

Zum Beispiel, wie im obigen Video gezeigt, können wir das Kommentar-Widget so anpassen, dass der Benutzer nur seinen Namen eingeben muss und nicht seine E-Mail-Adresse, um zu kommentieren oder abzustimmen.

## Update 2023 - Unterstützung für Super.so

Jetzt können Sie Kommentare zu Super.so-Websites hinzufügen, die auf Notion-Dokumenten basieren, mit FastComments! [Sie finden die Dokumentation hier.](https://docs.fastcomments.com/guide-installation-super-so.html)

## Google Sites

Google Sites wird ebenfalls unterstützt, und der Ablauf ist ähnlich.

1. [Erstellen Sie ein Gespräch zum Einbetten.](https://fastcomments.com/create)
2. Navigieren Sie beim Bearbeiten Ihrer Google-Website zu `Einfügen` -> `Einbetten`. Wählen Sie `Nach URL`.
3. Fügen Sie die URL aus Schritt 1 ein.
4. Klicken Sie auf `Einfügen`. Fertig!

## Nimbus Note

Nimbus Note wird ebenfalls unterstützt, und der Ablauf ist ähnlich.

1. [Erstellen Sie ein Gespräch zum Einbetten.](https://fastcomments.com/create)
2. Fügen Sie beim Bearbeiten Ihrer Notiz einen Einbettblock hinzu.
3. Verwenden Sie die URL aus Schritt 1 als URL zum Einbetten.
4. Klicken Sie auf `Einfügen`. Fertig!

### Wer erhält es

Alle aktuellen und neuen FastComments-Kunden, auf allen Ebenen, haben jetzt Zugriff auf die oEmbed-Unterstützung.

### Ein vorhandenes Gespräch einbetten

Um ein vorhandenes Gespräch einzubetten, geben Sie einfach den Wert der <a href="https://docs.fastcomments.com/guide-customizations-and-configuration.html#url-id" target="_blank">URL-ID</a>, die an das Kommentar-Widget übergeben wurde, auf der Seite "Gespräch erstellen" als Gesprächsnamen ein.
