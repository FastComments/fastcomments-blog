# [postlink]Einen Kommentar-Thread mit FastComments schließen[/postlink]

Haben Sie eine Diskussion, die Sie schließen möchten? Die Option ist hier!

{{#isPost}}

FastComments dreht sich alles darum, Diskussionen zu ermöglichen und den Nutzern eine Stimme zu geben. Manchmal haben Sie jedoch Inhalte, die nicht mehr relevant sind, oder Sie möchten vorübergehend Antworten deaktivieren, und das ist jetzt möglich.

Gehen Sie einfach zur <a href="https://fastcomments.com/auth/my-account/customize-widget" target="_blank">Anpassungsseite</a> und aktivieren Sie die Option "Neue Antworten verhindern".

Bitte beachten Sie, dass Sie eine URL definieren sollten, es sei denn, Sie möchten das Kommentieren auf Ihrer gesamten Seite deaktivieren.
