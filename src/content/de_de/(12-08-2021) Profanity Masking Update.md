# [postlink]Update zur Maskierung von Schimpfwörtern[/postlink]

{{#unless isPost}}
FastComments Schimpfwort-Erkennung und Maskierung erhält ein Upgrade
{{/unless}}

{{#isPost}}

### Neuigkeiten

Zuvor erlaubte FastComments nur die Definition einer einzigen globalen Wortschwärze auf Ihrem gesamten Konto.

Diese Liste von Wörtern, die maskiert oder bei Bedarf zur Spam-Erkennung verwendet werden sollten, konnte nicht pro Sprache definiert werden. Sie konnte außerdem nur eine Liste von Wörtern enthalten und keine Phrasen.

Jetzt wurden folgende Upgrades vorgenommen:

1. Die Schwarze Liste kann pro Gebietsschema (Sprache + Region) definiert werden.
2. Neben Wörtern können jetzt auch Phrasen maskiert oder zur Spam-Erkennung verwendet werden.
3. Das Aktualisieren der Konfiguration zur Schimpfwort-Erkennung erfolgt jetzt *live* und *sofort*. Zuvor konnte es bis zu fünf Minuten dauern, bis Änderungen wirksam wurden.

### So nutzen Sie es

Der Link zur Wortschwarze Liste hat sich nicht geändert und bleibt unter [/auth/my-account/moderate-comments/blacklist](https://fastcomments.com/auth/my-account/moderate-comments/blacklist) und
ist weiterhin zugänglich über `Moderate Comments -> Moderation Settings -> Edit Word Blacklist`.

Auf dieser Seite sehen Sie jetzt ein Dropdown-Menü für die Sprache, um die Schwarze Liste pro Gebietsschema anzupassen.

Nach Auswahl eines Gebietsschemas können Sie Wörter oder Phrasen definieren, die maskiert werden sollen.

### Einschränkungen

Pro Gebietsschema können bis zu 5.000 Wörter oder Phrasen definiert werden. Wenn Sie diese Grenze erhöhen möchten, kontaktieren Sie den Support.

### Es ist aktiviert!

Alle Änderungen, die Sie an der Liste von Wörtern oder Phrasen zur Maskierung vornehmen, werden jetzt sofort im System angewendet.

### Wer es bekommt

Alle aktuellen und neuen FastComments-Kunden, auf allen Ebenen, haben jetzt Zugriff auf die neuen Funktionen der *Wort* und *Phrasen* Schwarze Liste.

### Fazit

Wie der Rest von FastComments hoffen wir, dass Sie dieses Feature schnell und einfach zu bedienen finden.

Prost!
