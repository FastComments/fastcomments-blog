# [postlink]Verbesserungen beim Hochladen von Bildern[/postlink]

{{#unless isPost}}
Verschiedene Verbesserungen beim Hochladen von Bildern und im WYSIWYG-Editor in diesem Update.
{{/unless}}

{{#isPost}}

### Neuigkeiten

Heute wurde eine Reihe von Bugfixes für das Hochladen von Bildern und den WYSIWYG (was Sie sehen, ist was Sie bekommen) Bearbeitungsmodus veröffentlicht.

1. Eine Fortschrittsanzeige wird jetzt beim Hochladen von Bildern angezeigt.
2. Das Einfügen von Bildern, wie Screenshots, wird nun unterstützt (nur WYSIWYG-Editor).
3. Beim Bearbeiten von Kommentaren sind die Kommentarfelder jetzt richtig dimensioniert, um alle Bilder anzuzeigen (nur WYSIWYG-Editor).
4. Beim Entfernen von Bildern verkleinert sich das Kommentarfeld wie erwartet (nur WYSIWYG-Editor). 

### Fazit

Danke an unsere geduldigen Kunden, die einige dieser Probleme angesprochen haben. Wir hoffen, dass Sie weiterhin gerne mit FastComments arbeiten.
