# [postlink]Webhook-Ereignisse können jetzt abgebrochen werden[/postlink]

{{#unless isPost}}
Ereignisse in der Webhook-Verarbeitungswarteschlange können abgebrochen werden, sodass sie nicht mehr erneut versucht werden.
{{/unless}}

{{#isPost}}

### Was gibt es Neues

Früher wurde das Abbrechen von Webhook-Ereignissen über ein Support-Ticket erledigt. Stattdessen können sie jetzt im Selbstservice abgebrochen werden, indem Sie zur
[Protokollseite](https://fastcomments.com/auth/my-account/manage-data/webhooks/logs) gehen und auf das `x` neben dem Ereignis klicken.

Darüber hinaus wurde die Benutzeroberfläche der Webhook-Protokolle erheblich beschleunigt.

### Fazit

Wir hoffen, dass Sie dieses Update hilfreich fanden und weiterhin gerne mit FastComments arbeiten.

{{/isPost}}
