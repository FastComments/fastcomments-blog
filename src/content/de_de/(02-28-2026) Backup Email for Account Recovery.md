# [postlink]Backup-E-Mail für die Kontowiederherstellung[/postlink]

{{#unless isPost}}
FastComments unterstützt jetzt das Hinzufügen einer verifizierten Backup-E-Mail zu Ihrem Konto und bietet Ihnen eine alternative Möglichkeit, den Zugriff wiederherzustellen, falls Sie Ihre primäre E-Mail verlieren.
{{/unless}}

{{#isPost}}

### Was gibt's Neues

Sie können jetzt eine **Backup-E-Mail** zu Ihrem FastComments-Konto hinzufügen. Wenn Sie jemals den Zugriff auf Ihre primäre E-Mail verlieren, kann Ihre Backup-E-Mail verwendet werden, um Ihr Konto über einen unserer bestehenden Anmeldeabläufe wiederherzustellen.

Dieses Feature ist für alle regulären (nicht-SSO) Benutzer verfügbar.

### So funktioniert es

1. Gehen Sie zu Ihrer [Kontodetails](https://fastcomments.com/auth/my-account/edit-details) Seite.
2. Klicken Sie auf den Link **Backup-E-Mail definieren** unter dem E-Mail-Feld.
3. Geben Sie Ihre Backup-E-Mail-Adresse ein und senden Sie sie ab.
4. Überprüfen Sie Ihren Posteingang auf eine Bestätigungs-E-Mail und klicken Sie auf den Link zur Bestätigung.

Sobald verifiziert, wird Ihre Backup-E-Mail gespeichert und ist bereit zur Nutzung, falls Sie sie jemals benötigen. Sie können sie jederzeit auf derselben Seite ändern oder entfernen.

### So funktioniert die Wiederherstellung

Ihre Backup-E-Mail integriert sich in alle vorhandenen Wiederherstellungsmöglichkeiten:

- **Benutzername vergessen** - Geben Sie Ihre Backup-E-Mail auf der Seite für vergessene Benutzernamen ein, und Ihr Benutzername wird an diese Adresse gesendet.
- **Magic Link-Anmeldung** - Geben Sie Ihre Backup-E-Mail als E-Mail-Adresse beim Anmelden ein. Ein Magic Link wird an Ihre Backup-E-Mail gesendet.
- **Passwortanmeldung** - Sie können Ihre Backup-E-Mail im Benutzername/E-Mail-Feld eingeben, um Ihr Konto zu finden.

Es war kein neuer Wiederherstellungsablauf erforderlich. Ihre Backup-E-Mail funktioniert einfach überall dort, wo Ihre primäre E-Mail für Authentifizierungszwecke funktioniert.

### Was es nicht ist

Ihre Backup-E-Mail wird **nur** für die Kontowiederherstellung verwendet. Sie wird keine Benachrichtigungen, Digest-E-Mails oder andere Kommunikationen erhalten. Sie ist ein Sicherheitsnetz, kein zweites Postfach.

### Zum Schluss

Den Zugriff auf ein E-Mail-Konto zu verlieren, ist stressig genug, ohne auch noch den Zugriff auf alle damit verbundenen Dienste zu verlieren. Wir hoffen, dass diese kleine Ergänzung Ihnen Ruhe und Sicherheit gibt.
