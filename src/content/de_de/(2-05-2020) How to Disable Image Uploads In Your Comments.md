# [postlink]So deaktivieren Sie Bild-Uploads in Ihren Kommentaren[/postlink]

Mit FastComments können Sie die Bild-Uploads im Kommentarbereich Ihrer Website entweder für die gesamte Website oder nur für eine bestimmte Seite deaktivieren.

{{#isPost}}
Gehen Sie dazu im Dashboard zu <a href="https://fastcomments.com/auth/my-account/customize-widget" target="_blank">Anpassen</a> und fügen Sie eine Anpassungsregel hinzu.

Das Aktivieren der Einstellung sieht so aus:

<img 
    src="images/fc-disable-image-uploads.png"
    alt="Bild-Uploads deaktivieren"
    title="Bild-Uploads deaktivieren"
    class='lozad' />

... und klicken Sie auf speichern.
Auf dieser Seite müssen Sie Ihre Domain angeben, falls sie nicht bereits ausgefüllt ist. Lassen Sie die URL-ID leer, um die Regel auf Ihre gesamte Website anzuwenden.

Diese Seite hat Bild-Uploads aus Gründen des Beispiels deaktiviert.

{{/isPost}}
