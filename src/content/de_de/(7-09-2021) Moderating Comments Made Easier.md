# [postlink]Moderation von Kommentaren Vergleicht Es Einfacher[/postlink]

{{#unless isPost}}Sie können jetzt Kommentare moderieren, ohne Ihre Seite zu verlassen!{{/unless}}

{{#isPost}}

### Was ist Neu

Früher konnten Sie Kommentare nur von der Seite zur Kommentar Moderation aus moderieren.

Obwohl diese Seite eine Reihe von Optimierungen hatte, um die Moderation zu erleichtern, wie z. B. Massenaktionen, gibt es Vorteile, 
nicht Ihre Seite verlassen zu müssen. Nämlich, wenn Moderatoren zur Arbeit gehen, können sie den Kontext um jeden Kommentar 
easily sehen, während sie durch die Kommentarstränge arbeiten. Das bedeutet auch, dass Sie Kommentare **live** bearbeiten können, 
sobald sie eingehen.

Jetzt können Sie also direkt auf Ihrer Seite mit FastComments moderieren!

Wenn Sie angemeldet sind, haben Moderatoren ein kleines Bearbeitungssymbol oben rechts in jedem Kommentar. Wenn Sie darüber fahren, 
wird ein Menü mit einigen neuen Optionen angezeigt.

<div class="text-center">
    <img class="lozad" src="images/fc-moderate-comment-actions.png" alt="Aktionen zur Kommentar Moderation" title="Aktionen zur Kommentar Moderation" />
</div>

Diese Optionen werden auch im Kommentarstrangansichtsmodul angezeigt.

### Dokumentation

Sie finden die Dokumentation, die die verfügbaren Moderationsaktionen und deren Auswirkungen auf das Live-Kommentieren abdeckt, <a href="https://docs.fastcomments.com/guide-moderation.html#comment-actions-from-comment-widget" target="_blank">hier</a>.

### Wer Es Bekommt

Kunden auf allen Ebenen haben jetzt diese Funktion aktiviert.

### Leistungsbeeinträchtigung

Der Code und die Ressourcen, die mit der Moderation auf diese Weise verbunden sind, werden nur in die Browser Ihrer Moderatoren heruntergeladen - 
es hat keine Auswirkungen auf die Ladezeiten der Besucher Ihrer Seite.

### Abschließend

Wir hoffen, dass Sie finden, dass dies Ihre Arbeit als Moderator erleichtert.

Viel Spaß beim Kommentieren!
