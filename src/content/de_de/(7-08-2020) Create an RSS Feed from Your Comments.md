# [postlink]Erstellen Sie einen RSS-Feed aus Ihren Kommentaren[/postlink]

{{#unless isPost}}
RSS ist nach wie vor lebendig und gesund, und jetzt unterstützt FastComments es. Wissen Sie nicht, was RSS ist? Keine Sorge, wir werden Sie informieren.
{{/unless}}

{{#isPost}}

<a href="https://en.wikipedia.org/wiki/RSS" target="_blank">RSS</a> ist eine Möglichkeit, sich bei vielen verschiedenen Websites über einen einzigen Nachrichtenaggregator anzumelden.

Das bedeutet, dass Sie jetzt **Ihre Kommentare bei FastComments über eine RSS-App auf Ihrem Desktop oder Mobiltelefon überwachen können.**

#### So verwenden Sie es

Sie können jetzt Ihren RSS-Feed von der speziellen <a href="https://fastcomments.com/auth/my-account/rss" target="_blank">RSS-Feed-Seite</a> in Ihrem Dashboard verwalten. Aktivieren Sie einfach den Feed, kopieren Sie Ihre eindeutige RSS-URL und fügen Sie diese in Ihren bevorzugten RSS-Aggregator ein.

Der Feed enthält alle Kommentare der letzten 30 Tage.

Das war's. Viel Spaß beim Kommentieren!

{{/isPost}}
