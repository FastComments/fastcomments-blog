# [postlink]Kommentare auf GitHub Pages einbetten[/postlink]

{{#unless isPost}}
Möchten Sie Kommentare zu einer statischen Seite hinzufügen, wie GitHub Pages?
{{/unless}}

{{#isPost}}
#### Für Entwickler
Die Installation von FastComments auf einer Webseite erfordert nur das Einfügen eines kleinen Code-Snippets. Das bedeutet, es funktioniert auf statischen Seiten ohne Probleme - einschließlich GitHubs kostenlosem Pages-Hosting.

#### Eine Demo

Wir haben eine Beispiel-GitHub-Seite <a href="https://fastcomments.github.io/fastcomments-demo.github.io/" target="_blank">hier</a> eingerichtet.

<a href="https://github.com/fastcomments/fastcomments-demo.github.io/blob/master/index.html#L29" target="_blank">Hier</a> ist der Link zu der Stelle, wo wir das Code-Snippet für unser
Testkonto hinzugefügt haben.

#### Einrichtung

Sie müssen Ihre GitHub-Domain als gültige Domain in Ihrem Konto hinzufügen.
