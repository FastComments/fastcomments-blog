# [postlink]Volltextsuche kommt zu FastComments[/postlink]

Moderatoren haben jetzt die Möglichkeit, eine Kombination aus Benutzernamen und Kommentartext über die Moderationsseite zu durchsuchen.

{{#isPost}}

Gehen Sie einfach zur Seite Kommentare moderieren und geben Sie etwas Text in das Suchfeld in der Mitte ein, um Kommentare nach 
Wörtern und Phrasen zu suchen.

<img 
    src="images/fc-search-comments.png"
    alt="Beispiel für das Durchsuchen von Kommentaren"
    title="Beispiel für das Durchsuchen von Kommentaren"
    class='lozad' />
    
Wir glauben, dass dies Moderatoren helfen wird, Inhalte zu finden, die sie zuvor gesehen haben, aber keinen Link dazu gespeichert haben.

Wenn Sie während der Suche auf die Schaltfläche "Los" klicken, können Sie die URL in Ihrem Browser teilen oder speichern. Wenn Sie später zu dieser gleichen URL gehen, werden die gleichen Suchergebnisse und die Seite, auf der Sie sich befanden, angezeigt.

Da dies eine "Volltextsuche" ist, versucht eine Suchmaschine, die besten Übereinstimmungen basierend auf dem, was Sie eingeben, zu finden – das ist zum Beispiel nicht wie das Durchsuchen eines Word-Dokuments. Daher kann es sein, dass Sie einige Wörter vervollständigen müssen, um Übereinstimmungen zu erhalten.

Viel Spaß beim Moderieren!
