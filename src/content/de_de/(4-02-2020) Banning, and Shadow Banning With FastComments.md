# [postlink]Nutzer sperren und Schatten-Sperren mit FastComments[/postlink]

### Übersicht

Das Moderations-Toolkit von FastComments hat jetzt die Möglichkeit, einen Benutzer zu sperren oder Schatten zu sperren. Wenn Sie von einer anderen Plattform kommen, können Sie Benutzer über die E-Mail auf der <a href="https://fastcomments.com/auth/my-account/moderate-comments/banned-users" target="_blank">Seite der gesperrten Benutzer</a> sperren.

### Nebenwirkungen der Sperrung

Das Sperren eines Benutzers von der Moderationsseite löscht auch den entsprechenden Kommentar, unabhängig davon, ob Sie sich für eine Schatten-Sperrung entscheiden oder nicht.

Alle Sperren sind derzeit dauerhaft, können jedoch über die <a href="https://fastcomments.com/auth/my-account/moderate-comments/banned-users" target="_blank">Seite der gesperrten Benutzer</a> rückgängig gemacht werden.

### Moderatoren

Moderatoren haben die Möglichkeit, zu sperren, allerdings können sie derzeit nicht auf die Liste der gesperrten Benutzer zugreifen oder Benutzer per E-Mail sperren. Dies werden wir für zukünftige Versionen berücksichtigen.

Das Metrik "Gesperrte Benutzer" wurde ebenfalls zur Moderatoren-Seite hinzugefügt, damit Sie sehen können, wie viele Benutzer ein bestimmter Moderator gesperrt hat.

### Schatten-Sperren

Wenn Sie einen Benutzer von Ihrer Seite mit FastComments schatten-sperren, wird es so erscheinen, als wären ihre Stimmen und Kommentare für sie erfolgreich, jedoch werden die Ergebnisse nicht gespeichert (es sei denn, sie aktualisieren die Seite, da wir es nur für diese Sitzung "fälschen").
Das Sperren und Schatten-Sperren kann unterschiedliche Auswirkungen auf Ihre Community haben, also denken Sie darüber nach, bevor Sie auf die Schaltfläche Sperren klicken. Gesperrte Benutzer könnten ausflippen und 
schatten-gesperrte Benutzer könnten Unterstützungstickets erstellen und fragen, warum ihre Kommentare nicht gespeichert werden.

Insgesamt hoffen wir, dass dies für die Kunden, die es benötigen, hilfreich ist, aber es sollte mit Bedacht verwendet werden. Prost!
