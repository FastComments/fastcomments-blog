# [postlink]Neues Floating Like Widget veröffentlicht[/postlink]

{{#unless isPost}}
Ein neues Widget wurde veröffentlicht, mit dem Benutzer Seiten oder Artikel liken können.
{{/unless}}

{{#isPost}}

### Neuheiten

Bei FastComments können Benutzer Seiten direkt im Kommentar-Widget abonnieren, indem sie auf die Glocke klicken, aber wir hatten bisher keine Lösung für Seitenebene-Reaktionen.

Wir haben nun dieses Feature eingeführt, wobei das erste Widget das Floating Likes Widget ist, das Sie unten rechts in diesem Blog sehen.

Sie können die Seite direkt liken, auch wenn Sie kein FastComments.com-Konto haben. Wenn Sie angemeldet sind, ist die Reaktion an Ihr Konto gebunden.

### SSO

Das Widget unterstützt ebenfalls SSO zur Authentifizierung.

### Dokumentation

Sie finden die Dokumentation unter [docs.fastcomments.com](https://docs.fastcomments.com/guide-page-reacts.html).

### Anpassbar

Dieses Widget verwendet kein iframe und kann direkt über CSS gestaltet werden.

### Dunkler Modus

Standardmäßig wechselt das Widget in ein dunkleres Thema, wenn die Vorlieben des Benutzers im Browser oder Betriebssystem den Dunkelmodus bevorzugen.

### Ausblick

Langfristig möchten wir verschiedene Reaktionen auf Seitenebene hinzufügen, und vielleicht sogar Abstimmungen, wenn wir feststellen, dass unsere Kundenbasis an solchen Funktionen interessiert ist.

Wir freuen uns auf das Feedback von Benutzern und der Gemeinschaft, um dieses neue Widget im Laufe der Zeit zu verbessern.

### Fazit

Wir wissen, dass dieses Feature von einigen lange erwartet wurde. Wie bei allen größeren Veröffentlichungen freuen wir uns, dass wir uns die Zeit nehmen konnten, dieses Element zu optimieren, zu testen und ordnungsgemäß zu veröffentlichen.

Prost!
