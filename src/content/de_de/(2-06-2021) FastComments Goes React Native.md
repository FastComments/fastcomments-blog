# [postlink]FastComments Geht auf React Native[/postlink]

{{#unless isPost}}
Müssen Sie Live-Kommentare in eine mit React Native entwickelte Anwendung einbetten? Wir haben die Lösung.
{{/unless}}

{{#isPost}}

### <i class="circle">!</i> Dieser Artikel Beinhaltet Technischen Jargon

### Was ist Neu

FastComments hat jetzt React Native Komponenten, die mit unserem VanillaJS-Widget gleichwertig sind.

Wir haben uns entschieden, die Bibliothek `fastcomments-react-native` zu nennen. Sie können das GitHub-Repository mit dem Quellcode <a href="https://github.com/FastComments/fastcomments-react-native" target="_blank">hier</a> finden.

Das Repository enthält auch ein Beispiel für die Verwendung des Widgets.

Es ist auf NPM verfügbar, das Sie <a href="https://www.npmjs.com/package/fastcomments-react-native" target="_blank">hier</a> finden können.

### Wie es Funktioniert

Die Funktionsweise dieser neuen Komponente besteht darin, dass die React-Komponente ein Wrapper um das bestehende FastComments-Widget ist.

Das bedeutet, dass Sie sofort von neuen Funktionen oder Bugfixes profitieren können, die wir zu unseren bestehenden Komponenten hinzufügen.

### Entwicklungszeit

Die React Native-Widgets können in nur wenigen Minuten mit minimalem Entwicklungsaufwand zu Ihrer Anwendung hinzugefügt werden.

### SSO

Alle React Native-Bibliotheken unterstützen SSO sofort, und der SSO-Integrationsprozess ist derselbe wie bei den Vanilla JS-Komponenten. Die vorhandenen
Backend-Codebeispiele gelten ebenfalls.

### Benutzerdefinierte Stile?

Die React Native-Komponenten können über die Widget-Anpassungsbenutzeroberfläche mit CSS gestylt werden, genau wie die webbasierten Widgets.
