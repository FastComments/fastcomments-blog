# [postlink]FastComments のデフォルトアバターを Gravatar を利用して設定する[/postlink]

### アバターとは？

アバターは、すべてのアカウントに関連付けられたプロフィールアイコンであり、すべてのコメントの隣に表示されます。

### Gravatar とは？

<a href="https://en.gravatar.com/" target="_blank">Gravatar</a> は「グローバルに認識されたアバター」としてブランドを展開しています。
これは、特定のメールアドレスに対してアバターを定義できるサービスです。そのメールアドレスを使用して登録した製品は、そのアバターを利用できます。これにより、FastComments にサインアップする人々が自動的にアバターを設定できるようになり、手間が省けます。

### それでは、変更点は何ですか？

FastComments は今後、デフォルトのアバターを Gravatar から取得します。あなたやユーザーは、この機能を得るために何もする必要はありません。

既存のユーザーがすでに自分のアバターを定義している場合でも、心配はいりません。カスタムアバターが上書きされることはありません。
また、ユーザーは FastComments 内で Gravatar アバターを上書きすることもできます。
