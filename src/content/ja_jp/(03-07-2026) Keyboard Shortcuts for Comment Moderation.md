# [postlink]コメントモデレーションのためのキーボードショートカット[/postlink]

{{#unless isPost}}
パワーユーザーは、ナビゲーション、アクションなどのためのフルセットのショートカットを使って、キーボードだけでコメントを完全にモデレーションできるようになりました。
{{/unless}}

{{#isPost}}

### 新機能

コメントモデレーションページは、包括的なキーボードショートカットをサポートするようになりました。コメント間を移動し、承認、削除、バン、
スパムとしてマーク、コンテキストの切り替え、ログの表示などを行い、すべてマウスに触れることなく実行できます。

これは、多くのコメントをモデレーションするチームにとって大きな意味を持ちます。以前はマウスを動かしてボタンをクリックする必要があったすべてのアクションが、
今では1回のキー入力で実行できます。

### 例の流れ

レビューするコメントのキューがあるとしましょう。**j**を押して最初のコメントにフォーカスを当てます。**c**を押してその
コンテキストを展開し、周囲のスレッドを表示します。良さそうなので、再度**c**を押してコンテキストを折り畳み、次に**a**を押して承認します。
**j**を押して次のコメントに移動し、繰り返します。何かおかしければ、**s**を押してスパムとしてマークするか、**d**を押して
削除します。マウスに手を伸ばすことなく、コメントのページ全体を処理できます。

### ナビゲーション

**j**を押して次のコメントにフォーカスを移動し、**k**を押して前のコメントに移動します。フォーカスされたコメントは
選択された（チェックされた）コメントと区別するために青いアウトラインでハイライトされます。

**x**または**Enter**を押してフォーカスされたコメントの選択を切り替えます。**Shift**を押しながら**j**または**k**を押すことで
現在のコメントを選択しながらフォーカスを移動できます - コメントの連続を迅速に選択するのに便利です。

### アクション

コメントにフォーカスが当たっている状態で、単一キーのショートカットはツールバーのボタンと同じアクションをトリガーします：

- **a** - 承認 / 承認取り消しを切り替え
- **s** - スパム / スパムでないを切り替え
- **d** - 削除
- **u** - 削除取り消し
- **r** - レビュー済み / 未レビューを切り替え
- **e** - 編集
- **b** - バン / バン管理
- **p** - ピン / ピン解除を切り替え
- **f** - フラグ解除
- **i** - IPからの全てを表示
- **t** - 信頼要素を設定
- **m** - バッジを管理
- **l** - ログを表示
- **c** - コンテキストを切り替え

### グローバルショートカット

いくつかのショートカットはコメントにフォーカスが当たっているかどうかに関わらず機能します：

- **Ctrl+a** - すべてのコメントを選択
- **Ctrl+z** - 最後のアクションを元に戻す
- **/** - 検索入力にフォーカスを当てる
- **[** と **]** - 前のページまたは次のページに移動

### ヘルプとガイダー

いつでも**?**を押して、利用可能なショートカットのリストを表示するヘルプダイアログを開きます。

**g**を押すと「ガイダー」モードが切り替わり、フォーカスされたコメントのアクションボタン上に対応するショートカットキーを
示す小さなバッジがオーバーレイされます。これはショートカットを学んでいる間に便利です。

ショートカットは検索ボックスや他の入力フィールドに入力中、またはダイアログが開いているときには無効になります。
**Escape**を押してフォーカスされた入力をぼかし、ショートカットモードに戻ります。

完全なリファレンスは、<a href="https://docs.fastcomments.com/guide-moderation.html" target="_blank">モデレーションガイド</a>を参照してください。

### 結論

キーボードショートカットが高ボリュームのモデレーションをより迅速かつ快適にすることを願っています。これらのショートカットは
GmailやGitHubのようなツールのユーザーに馴染みのあるコンベンションに従っているため、すぐに自然に感じるはずです。
