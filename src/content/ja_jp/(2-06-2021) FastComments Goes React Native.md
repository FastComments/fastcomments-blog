# [postlink]FastCommentsがReact Nativeに対応しました[/postlink]

React Nativeで開発されたアプリケーションにライブコメント機能を埋め込みたいですか？私たちがお手伝いします。

{{#isPost}}

### <i class="circle">!</i> この記事には技術用語が含まれています

### 新着情報

FastCommentsは現在、VanillaJSウィジェットと機能的に同等のReact Nativeコンポーネントを提供しています。

このライブラリを`fastcomments-react-native`と名付けました。ソースコードを含むGitHubリポジトリは<a href="https://github.com/FastComments/fastcomments-react-native" target="_blank">こちら</a>にあります。

リポジトリにはウィジェットの使用例も含まれています。

NPMにも登録されており、こちらで見つけることができます<a href="https://www.npmjs.com/package/fastcomments-react-native" target="_blank">こちら</a>。

### 仕組み

この新しいコンポーネントの仕組みは、Reactコンポーネントが既存のFastCommentsウィジェットをラップしているというものです。

これは、既存のコンポーネントに機能を追加したりバグを修正した場合でも、すぐにその恩恵を受けられることを意味します！

### 開発時間

React Nativeウィジェットは、数分で簡単にアプリケーションに追加でき、多くの開発時間は必要ありません。

### SSO

すべてのReact Nativeライブラリは、初めからSSOをサポートしており、SSO統合プロセスはVanilla JSコンポーネントと同じです。既存のバックエンドコードの例も適用できます。

### カスタムスタイル？

React Nativeコンポーネントは、ウェブベースのウィジェットと同様に、ウィジェットカスタマイズUIを通じてCSSでスタイリングできます。

### 代替案

あなた自身のUIを構築することもできますが

### 結論

`fastcomments-react-native`ライブラリと今後のライブラリをリリースすることで、私たちは開発者が現代の開発手法を使用しながらFastCommentsをより簡単に採用できるようにすることを期待しています。
