# [postlink]FastCommentsによる簡単なコードスニペットサポート[/postlink]

私たちの顧客の中には、ソフトウェア開発に関連するサイトやブログを持つ方々がいます。これらのコミュニティでは、コードスニペットを共有することが一般的です。FastCommentsがそれを簡単にする方法を紹介します。

{{#isPost}}

FastCommentsでは、コメントボックスにコードを単純に貼り付けることで共有することができます。テキストや画像と混ぜて使用することができ、「コード」タグを手動で追加してフォーマットを整える必要はありません。ただ貼り付けるだけです！

こちらはそれを実演した簡単なデモです。

### パフォーマンスへの影響

この機能を追加したことで、クライアントサイドのウィジェットには約150バイトしか追加されていません。ほとんどの機能はサーバー側で処理されているためです。実際、最近追加されたMarkdown機能と組み合わせると、ウィジェットのサイズは実際に減少しています！

### 注目すべき点

コードを頻繁に共有するコミュニティがある場合は、コードスニペットを共有しやすくするために、最大コメントサイズを増やすことを検討するべきです。これは管理ダッシュボードの「カスタマイズ」タブから行えます。

### コード検出

クライアントサイドのウィジェットのサイズを抑えるために、「{」または空白を制御フローに使用するCおよびPythonに似た言語（例：Java、JavaScript、CSS、C++）に対してのみ、自動的に貼り付けられたコードを検出します。もし、自動検出されない言語を頻繁に使用している場合は、お気軽にお問い合わせください。

コードブロックを手動で挿入するには、次のようにコメントを記述します: ```<code>(defun someLispCode(1, 2, 3))</code>```. 約150の言語を自動検出できます。

### 既存のコメントの移行

FastCommentsに移行したいが、正しくフォーマットされたコードスニペットがたくさんあるコメントがある場合は、<a href="https://fastcomments.com/auth/my-account/help" target="_blank">お手伝いできます</a>。

以上です！この機能はFastComments上のすべてのオンラインコミュニティに展開されました。
