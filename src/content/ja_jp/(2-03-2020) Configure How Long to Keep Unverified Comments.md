# [postlink]未確認コメントの保持期間を設定する[/postlink]

デフォルトでは、FastCommentsは未確認のコメントを3日間保持します。

### 「未確認」とは何ですか？

ユーザーがコメントを残し、メールアドレスを入力すると、FastCommentsはそのコメントを確認するように要求するメールを送信します。  
これはスパムを防ぐ手助けをし、著者が実際に表示したいコメントを際立たせます。

<a href="https://blog.fastcomments.com/(2-01-2020)-how-to-allow-anonymous-comments-with-fastcomments.html" target="_blank">匿名コメント</a>を有効にすると、メールアドレスを入力する必要すらありません！

しかし、これらのコメントをどれくらいの期間保持するかを設定できるようになりました。自動的に削除したくない場合もあるでしょう！これはすべて、コメントの管理 -> モデレーション設定を編集から設定可能です。

{{#isPost}}

デフォルトでは、コメントモデレーション設定は自動削除が有効になっており、保持期間が3日間になっているかもしれません。
