# [postlink]GitHub Pagesへのコメント埋め込み[/postlink]

{{#unless isPost}}
静的サイト、例えばGitHub Pagesにコメントを追加したいですか？
{{/unless}}

{{#isPost}}
#### 開発者向け
FastCommentsをウェブページにインストールするには、小さなコードスニペットを挿入するだけで済みます。これは、GitHubの無料のPagesホスティングを含む静的ページで簡単に機能します。

#### デモ

例としてGitHub Pageを<a href="https://fastcomments.github.io/fastcomments-demo.github.io/" target="_blank">こちら</a>に設定しました。

<a href="https://github.com/fastcomments/fastcomments-demo.github.io/blob/master/index.html#L29" target="_blank">こちら</a>は、テストアカウント用にコードスニペットを追加した場所へのリンクです。

#### 設定

あなたのGitHubドメインをアカウントの有効なドメインとして追加する必要があります。
