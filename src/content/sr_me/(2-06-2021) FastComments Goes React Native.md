### Šta je Novo

FastComments sada ima React Native komponente koje su u skladu sa funkcionalnostima našeg VanillaJS dodatka.

Odlučili smo da biblioteku nazovemo `fastcomments-react-native`. Možete pronaći GitHub repozitorij sa izvorni kodom <a href="https://github.com/FastComments/fastcomments-react-native" target="_blank">ovde</a>.

Repozitorij takođe sadrži primer upotrebe dodatka.

Dostupan je na NPM-u, koji možete pronaći <a href="https://www.npmjs.com/package/fastcomments-react-native" target="_blank">ovde</a>.

### Kako Funkcioniše

Način na koji ovaj novi komponent funkcioniše je da je React komponenta omotač oko postojećeg FastComments dodatka.
 
To znači da ako dodamo nove funkcije našim postojećim komponentama ili ispravimo greške, vi ćete odmah imati koristi!

### Vreme Razvoja

React Native dodaci mogu biti dodati u vašu aplikaciju za samo nekoliko minuta uz malo potrebnog vremena za razvoj.

### SSO

Sve React Native biblioteke podržavaju SSO odmah, a proces integracije SSO je isti kao i kod Vanilla JS komponenti. Postojeći
primeri backend koda se takođe primenjuju.

### Prilagođeni Stil?

React Native komponente mogu biti stilizovane koristeći CSS putem UI za prilagođavanje dodataka, kao i web-based dodaci.
