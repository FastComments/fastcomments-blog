# [postlink]Ugradnja komentara na GitHub Pages[/postlink]

{{#unless isPost}}
Želite da dodate komentare na statički sajt, kao što su GitHub Pages?
{{/unless}}

{{#isPost}}
#### Za programere
Instalacija FastComments na web stranicu zahteva samo umetanje malog koda. Ovo znači da funkcioniše na statičkim stranicama bez ikakvih problema - uključujući GitHub-ovo besplatno hosting rešenje za stranice.

#### Demonstracija

Postavili smo primer GitHub stranice <a href="https://fastcomments.github.io/fastcomments-demo.github.io/" target="_blank">ovde</a>.

<a href="https://github.com/fastcomments/fastcomments-demo.github.io/blob/master/index.html#L29" target="_blank">Evo</a> je link na kojem smo dodali kod za naš
testni nalog.
