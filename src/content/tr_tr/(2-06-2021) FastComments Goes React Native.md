### <i class="circle">!</i> Bu Makale Teknik Terimler İçeriyor

### Yenilikler

FastComments artık VanillaJS widget'ımızla aynı özelliklere sahip React Native bileşenlerine sahip.

Kütüphaneyi `fastcomments-react-native` olarak adlandırmaya karar verdik. Kaynak koduyla GitHub deposunu <a href="https://github.com/FastComments/fastcomments-react-native" target="_blank">burada</a> bulabilirsiniz.

Depo ayrıca widget'ın örnek kullanımını da içeriyor.

NPM'de mevcut, burada <a href="https://www.npmjs.com/package/fastcomments-react-native" target="_blank">bulabilirsiniz</a>.

### Nasıl Çalışır

Bu yeni bileşenin çalışma şekli, React bileşeninin mevcut FastComments widget'ının etrafında bir sarmalayıcı olmasıdır.

Bu, mevcut bileşenlerimize yeni özellikler ekler veya hataları giderirsek, hemen faydalanabileceğiniz anlamına geliyor!

### Geliştirme Süresi

React Native widget'ları, sadece birkaç dakika içinde ve az geliştirme süresi gerektirecek şekilde uygulamanıza eklenebilir.

### SSO

Tüm React Native kütüphaneleri kutudan çıkar çıkmaz SSO'yu destekler ve SSO entegrasyon süreci, Vanilla JS bileşenleriyle aynıdır. Mevcut arka uç kod örnekleri de geçerlidir.

### Özel Stil?

React Native bileşenleri, web tabanlı widget'lar gibi Widget Özelleştirme UI aracılığıyla CSS kullanılarak stillendirilebilir.

### Alternatifler

Kendi UI'nizi oluşturabilirsiniz 

### Sonuç Olarak

`fastcomments-react-native` kütüphanesini ve gelecekteki kütüphaneleri yayınlayarak, geliştiricilerin modern geliştirme metodolojilerini kullanırken FastComments'ı benimsemelerini kolaylaştırmayı umuyoruz.
