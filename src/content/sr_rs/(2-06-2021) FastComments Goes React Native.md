### Šta je novo

FastComments sada ima React Native komponente koje su u skladu sa našim VanillaJS widgetom.

Odlučili smo da biblioteku nazovemo `fastcomments-react-native`. Izvorni kod možete pronaći u GitHub repozitorijumu <a href="https://github.com/FastComments/fastcomments-react-native" target="_blank">ovde</a>.

Repozitorijum takođe sadrži primer korišćenja widgeta.

Dostupan je na NPM, koji možete pronaći <a href="https://www.npmjs.com/package/fastcomments-react-native" target="_blank">ovde</a>.

### Kako to funkcioniše

Način na koji ova nova komponenta funkcioniše je da je React komponenta omot oko postojećeg FastComments widgeta.

To znači da ako dodamo funkcionalnosti našim postojećim komponentama ili ispravimo greške, vi ćete odmah imati koristi!

### Vreme razvoja

React Native widgeti mogu biti dodati u vašu aplikaciju za samo nekoliko minuta uz malo potrebnog vremena za razvoj.

### SSO

Sve React Native biblioteke podržavaju SSO odmah, a proces integracije SSO je isti kao i za Vanilla JS komponente. Postojeći
primeri backend koda se takođe primenjuju.

### Prilagođavanje stila?

React Native komponente se mogu stilizovati koristeći CSS putem UI za prilagođavanje widgeta, isto kao i web widgeti.
