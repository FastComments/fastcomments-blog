התקנת FastComments בדף אינטרנט מחייבת רק הוספת קטע קוד קטן. זה אומר שזה עובד על דפים סטטיים בלי בעיות - כולל את אירוח הדפים החינמי של GitHub.

#### דמו

הקמנו דף GitHub לדוגמה <a href="https://fastcomments.github.io/fastcomments-demo.github.io/" target="_blank">כאן</a>.
