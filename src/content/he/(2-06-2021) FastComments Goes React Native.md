### <i class="circle">!</i> מאמר זה מכיל מושגים טכניים

### מה חדש

FastComments עכשיו יש לו רכיבי React Native ששווים בערכם לאלה של הווידג'ט VanillaJS שלנו.

החלטנו לקרוא לספריה `fastcomments-react-native`. אתה יכול למצוא את מאגר GitHub עם קוד המקור <a href="https://github.com/FastComments/fastcomments-react-native" target="_blank">כאן</a>.

במאגר יש גם דוגמאות לשימוש בווידג'ט.

זה נמצא ב-NPM, אותו תוכל למצוא <a href="https://www.npmjs.com/package/fastcomments-react-native" target="_blank">כאן</a>.

### איך זה עובד

הדרך שבה רכיב חדש זה עובד היא שהרכיב של React הוא עטיפה סביב הווידג'ט הקיים של FastComments.

זה אומר שאם נוסיף תכונות לרכיבים הקיימים שלנו או נתקן באגים, אתה עדיין תיהנה מיד!

### זמן פיתוח

הווידג'טים של React Native יכולים להתווסף לאפליקציה שלך בתוך דקות עם מעט זמן פיתוח הנדרש.

### SSO

כל הספריות של React Native תומכות ב-SSO באופן מובנה, ותהליך ההשתלבות של SSO הוא אותו תהליך כמו רכיבי ה-JS Vanilla. דוגמאות קוד קצה קיימות תקפות גם כאן.

### סגנון מותאם אישית?

רכיבי React Native ניתן לעצב באמצעות CSS דרך ממשק ההתאמה של הווידג'ט, כמו בווידג'טים מבוססי הווב.
