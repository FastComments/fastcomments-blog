# [postlink]הוספת צ'אט חי לאפליקציות מקוריות עם FastComments React Native[/postlink]

{{#unless isPost}}
לאחר חודשים של עבודה, אנו שמחים להודיע כי fastcomments-react-native-sdk כעת בגרסה 1.0 זמינה לציבור.
{{/unless}}

{{#isPost}}

### <i class="circle">!</i> מאמר זה מכיל מילים טכניות

### מה חדש

ב-FastComments, השקנו דרך חדשה לבניית אפליקציות React Native עם FastComments. בעבר, הייתם משתמשים במעטפת react-native שלנו סביב
ספריית VanillaJS. זה הציג כמה בעיות זיכרון ועיכוב בגלל הצורך להוריד תצוגת אינטרנט - דפדפן באפליקציה שלכם - רק כדי לטעון את הספרייה שלנו.

עכשיו כבר **עברנו לגמרי** את FastComments ל-React Native, וכתבנו עורך WYSIWYG משלו כדי לגמרי להעלים את הצורך בתצוגות אינטרנט.

הספרייה תומכת בעורות וסגנונות שונים. לדוגמה, הנה עור Erebus שהוא עור כהה בסגנון צ'אט חי:

<div class="text-center">
    <img src="images/skin-erebus.png" alt="Erebus Skin" title="Erebus Skin" />
</div>

הספרייה זמינה [ב-GitHub ב-FastComments/fastcomments-react-native-sdk](https://github.com/FastComments/fastcomments-react-native-sdk) כמו גם
[ב-NPM](https://www.npmjs.com/package/fastcomments-react-native-sdk).

### ניתן להתאמה אישית

הספרייה המקומית החדשה מיועדת להיות ניתנת להתאמה אישית בדיוק כמו הספרייה הקיימת באינטרנט.

אתם יכולים:

- להתאים אישית את כל הנכסים והאייקונים שהספרייה משתמשת בהם.
- להתאים אישית את העיצוב של כל רכיב בספרייה על ידי העברת אובייקט Styles מהחלק העליון.
- להתאים אישית את כל התרגומים.
- להשתמש באחד מהעורות המוגדרים מראש או שהקהילה פיתחה.
- להשתמש בהרבה מהאפשרויות הקיימות של הספרייה באינטרנט.

### איך זה עובד

הספרייה היא כתיבה מחדש מלאה של לקוח FastComments ב-TypeScript ו-React Native.

### תכנית ארוכת טווח

בטווח הארוך, אנו נתחזק גם את ספריית VanillaJS וגם את ספריית react-native במקביל, כמו גם כל ספרייה מקומית עתידית שאנו עשויים לפתח.

### מה הצעד הבא?

נשמח לקבל פידבק מהמשתמשים ומהקהילה כדי לשפר את הספרייה הזו עם הזמן, בדיוק כמו שעשינו עם מערכת האינטרנט.

### לסיכום

אנו יודעים שהספרייה הזו הייתה ממתינה זמן רב עבור כמה. כמו בכל השקות גדולות, אנחנו שמחים שיכולנו לקחת את הזמן לייעל, לבדוק ו לשחרר את הספרייה הזו בצורה נכונה.
