### מה חדש

לפני ה-14 בנובמבר 2023, ביצענו את השינויים הבאים ב-webhooks:

#### עדכון Webhooks

**בעבר**, העדכונים כללו את גרסת הבסיס האחרונה של תגובה. אם שני משתמשים הצביעו על תגובה ברצף מהיר, זה היה
מפעיל שני webhooks. עם זאת, היה אפשרי שאירוע ה-webhook הראשון יכיל את **גרסת הבסיס האחרונה**, דבר שגרם לשני
אירועי webhook זהים:

1. ` ... { votes: 2 } ...`
2. ` ... { votes: 2 } ...`

**נכון ל-14 לנובמבר**, כל אירוע webhook הוא אידמפטנטי ומכיל את אובייקט התגובה בזמן השינוי. בדוגמה למעלה עכשיו תקבל שני אירועים:

1. ` ... { votes: 1 } ...`
2. ` ... { votes: 2 } ...`

#### מחיקת Webhooks

**בעבר**, webhooks למחיקה הכילו רק את מזהה התגובה. זה היה אובייקט כמו `{ id: string }`.

**נכון ל-14 לנובמבר**, webhooks למחיקה יכילו את **כל** אובייקט התגובה בזמן ההסרה.

### לסיכום

כמו בכל השקות עיקריות, אנחנו שמחים שיכולנו לקחת את הזמן כדי לייעל, לבדוק ולשחרר את התכונה הזו כמו שצריך. יידעו אותנו
למטה אם תגלו בעיות כלשהן.

לחיים!
