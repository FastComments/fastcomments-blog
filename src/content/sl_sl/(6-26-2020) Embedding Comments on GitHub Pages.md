#### Za razvijalce
Namestitev FastComments na spletno stran zahteva le vstavljanje majhne kode. To pomeni, da deluje na statičnih straneh brez skrbi - vključno z brezplačnim gostovanjem strani GitHub.

#### Prikaz

Pripravili smo primer GitHub strani <a href="https://fastcomments.github.io/fastcomments-demo.github.io/" target="_blank">tukaj</a>.

<a href="https://github.com/fastcomments/fastcomments-demo.github.io/blob/master/index.html#L29" target="_blank">Tukaj</a> je povezava do mesta, kjer smo dodali kodo za našo
testno račun.
