# [postlink]Komentarji administratorjev in moderatorjev so zdaj avtomatsko odobreni in pregledani[/postlink]

Manjša novica - a dobrodošla za mnoge!

{{#isPost}}

Prej, če je bila avtomatska odobritev onemogočena, so komentarji, ki so jih napisali administratorji ali moderatorji, še vedno potrebovali odobritev. Poleg tega so bili označeni kot "Nepregledani".

S sotrednjo različico bodo vsi komentarji moderatorjev in administratorjev označeni kot Odobreni in Pregledani.
