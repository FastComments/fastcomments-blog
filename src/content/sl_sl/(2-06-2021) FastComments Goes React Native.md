### Kaj je novega

FastComments zdaj vključuje komponente za React Native, ki so na enaki ravni funkcionalnosti kot naš VanillaJS vtičnik.

Odločili smo se, da knjižnico poimenujemo `fastcomments-react-native`. GitHub repozitorij z izvorno kodo lahko najdete <a href="https://github.com/FastComments/fastcomments-react-native" target="_blank">tukaj</a>.

Repozitorij vsebuje tudi primer uporabe vtičnika.

Na voljo je na NPM, ki ga lahko najdete <a href="https://www.npmjs.com/package/fastcomments-react-native" target="_blank">tukaj</a>.

### Kako deluje

Način, kako ta nova komponenta deluje, je, da je React komponenta ovitek okoli obstoječega vtičnika FastComments.

To pomeni, da če dodamo funkcionalnosti našim obstoječim komponentam ali popravimo napake, boste še vedno takoj imeli koristi!

### Čas razvoja

Vtičniki React Native se lahko dodajo vaši aplikaciji v le nekaj minutah z malo potrebnega razvoja.

### SSO

Vse knjižnice React Native podpirajo SSO kar se tiče izvorne funkcionalnosti, proces integracije SSO pa je enak tistemu za komponente Vanilla JS. Obstajajoči
primeri back-end kode se prav tako uporabljajo.

### Po meri oblikovanje?

Komponente React Native lahko oblikujete z uporabo CSS prek uporabniškega vmesnika za prilagajanje vtičnikov, enako kot vtičnike, ki temeljijo na spletu.
