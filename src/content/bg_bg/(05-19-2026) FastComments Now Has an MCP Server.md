# [postlink]FastComments сега разполага с MCP сървър[/postlink]  

{{#unless isPost}}  
Сега можете да насочите Claude Code, Claude Desktop, Cursor или всеки друг клиент на Model Context Protocol към FastComments и да извикате нашето API директно.  
{{/unless}}  

{{#isPost}}  

### Какво ново  

FastComments сега предлага хостван [Model Context Protocol](https://modelcontextprotocol.io/) (MCP) сървър. MCP е отвореният стандарт, който AI кодови асистенти използват, за да извикват инструменти на сървъра. Насочете клиента си към един URL и той може да изброява вашите коментари, да ги модерира, да управлява потребители със SSO, да конфигурира виджети и всичко друго, което прави [API](https://docs.fastcomments.com/guide-api.html).  

### Къде се намира  

От вашето табло на FastComments, отворете **Интегрирайте, след това MCP сървър**, или посетете го директно на  
[fastcomments.com/auth/my-account/mcp-setup](https://fastcomments.com/auth/my-account/mcp-setup).  
Изберете кой API ключ искате клиентът да използва, а страницата генерира предварително попълнен URL плюс кодове за копиране и поставяне за конфигурационните файлове на Claude Code и Claude Desktop / Cursor.  

### Claude Code  

Една команда за регистриране:  

    claude mcp add --transport http fastcomments 'https://fastcomments.com/mcp?tenantId=YOUR_TENANT_ID&API_KEY=YOUR_API_KEY'  

След това `/mcp` в сесия показва връзката и изброява всеки инструмент, който сървърът предоставя. Запитайте неща като "покажи ми последните 20 маркирани коментара на /blog/launch", "блокирай този коментатор на всички мои сайтове" или "какъв е нашият процент на улавяне на спам за последната седмица". Claude насочва през правилния инструментален извик.  

### Claude Desktop и Cursor  

Добавете този блок в конфигурацията на MCP на клиента си (`claude_desktop_config.json` за Claude Desktop, `mcp.json` за Cursor):  

```json  
{  
  "mcpServers": {  
    "fastcomments": {  
      "type": "http",  
      "url": "https://fastcomments.com/mcp?tenantId=YOUR_TENANT_ID&API_KEY=YOUR_API_KEY"  
    }  
  }  
}  
```  

Рестартирайте клиента и сървърът ще се появи до всеки друг MCP сървър, който сте конфигурирали.  

### Сигурност  

API ключът е вграден в URL, така че третирайте самия URL като тайна. Не го поставяйте в публични чатове, екранни снимки или комити. Ако ключ бъде изложен, променете го от [страницата с API ключове](https://fastcomments.com/auth/my-account/api-secret) и повторно копирайте фрагмента от страницата за настройка. За да ограничите какво може да направи агентът, генерирайте отделен API ключ за всеки асистент и го заключете към правилния домейн. Падащото меню на страницата за настройка ви позволява да смените кой ключ използват фрагментите с едно кликване.  

### Защо го изградихме  

Хората все повече използват агентни инструменти за ежедневни задачи, свързани с работа с SaaS продукти. Всеки, който свързва сесия на Claude или Cursor, за да триажира своята модерационна опашка или за да повърне специфични теми, на които си заслужава да се обърне внимание, сега може напълно да пропусне обвивния слой.  

### В заключение  

Отворете [Интегрирайте, след това MCP сървър](https://fastcomments.com/auth/my-account/mcp-setup) на вашето табло, копирайте фрагмента за любимия си MCP клиент и сте готови. Пълната документация е в [LLM Kit ръководство](https://docs.fastcomments.com/guide-llm-kit.html).  

Както винаги, не се колебайте да ни уведомите по-долу, ако срещнете нещо.  

Наздраве!
