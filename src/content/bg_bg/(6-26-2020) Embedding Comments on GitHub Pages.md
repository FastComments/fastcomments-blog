# [postlink]Вграждане на коментари в GitHub Pages[/postlink]

{{#unless isPost}}
Искате ли да добавите коментари към статичен сайт, като GitHub Pages?
{{/unless}}

{{#isPost}}
#### За Разработчици
Инсталирането на FastComments на уеб страница изисква само вмъкване на малък кодов фрагмент. Това означава, че работи на статични страници без никакви усложнения - включително безплатния хостинг на Pages на GitHub.

#### Демонстрация

Създадохме примерна GitHub страница <a href="https://fastcomments.github.io/fastcomments-demo.github.io/" target="_blank">тук</a>.

<a href="https://github.com/fastcomments/fastcomments-demo.github.io/blob/master/index.html#L29" target="_blank">Тук</a> е линкът, където сме добавили кода за нашия
тестови акаунт.

#### Настройка

Трябва да добавите вашия GitHub домейн като валиден домейн в акаунта си.
