### <i class="circle">!</i> Тази статия съдържа технически жаргон

### Какво ново

FastComments сега предлага компоненти за React Native, които са на равнището на нашия VanillaJS виджет.

Решихме да наречем библиотеката `fastcomments-react-native`. Можете да намерите GitHub репозито с изходния код <a href="https://github.com/FastComments/fastcomments-react-native" target="_blank">тук</a>.

Репозито също съдържа пример за използване на виджета.

Той е наличен на NPM, който можете да откриете <a href="https://www.npmjs.com/package/fastcomments-react-native" target="_blank">тук</a>.

### Как работи

Новият компонент работи като обвиващ елемент около съществуващия виджет на FastComments.

Това означава, че ако добавим нови функции към нашите съществуващи компоненти или поправим бъгове, вие веднага ще се възползвате!

### Време за разработка

Виджетите за React Native могат да бъдат добавени към вашето приложение само за няколко минути с минимално време за разработка.

### SSO

Всички библиотеки за React Native поддържат SSO по подразбиране, а процесът на интеграция на SSO е същият като при компонентите на Vanilla JS. Съществуващите примери за бекенд код важат също.

### Персонализиране на стила?

Компонентите за React Native могат да бъдат стилно оформени с CSS чрез потребителския интерфейс за персонализиране на виджети, по същия начин както уеб-базираните виджети.

### Алтернативи

### В заключение

С издаването на библиотеката `fastcomments-react-native`, и бъдещи библиотеки, ние се надяваме да улесним разработчиците да приемат FastComments, докато ползват съвременни методологии за разработка.
