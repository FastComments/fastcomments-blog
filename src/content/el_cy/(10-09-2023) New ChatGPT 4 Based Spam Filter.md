# [postlink]Νέος φίλτρο ανεπιθύμητης αλληλογραφίας βασισμένο στο ChatGPT 4.[/postlink]

{{#unless isPost}}
Η FastComments προσθέτει στη λίστα των επιλογών φίλτρων ανεπιθύμητης αλληλογραφίας.
{{/unless}}

{{#isPost}}

### Τι είναι καινούριο

Η FastComments συνεχίζει να βελτιώνει τον ταξινόμο ανεπιθύμητης αλληλογραφίας Naïve-Bayes, ωστόσο προσθέσαμε μια νέα επιλογή στο στόλο των ανιχνευτών ανεπιθύμητης αλληλογραφίας.

Τώρα μπορείτε να ανιχνεύσετε ανεπιθύμητη αλληλογραφία χρησιμοποιώντας το ChatGPT 4 που παρέχεται από την OpenAI.

### Πώς να το ρυθμίσετε

Για να ρυθμίσετε ποιον ανιχνευτή ανεπιθύμητης αλληλογραφίας να χρησιμοποιήσετε, ελέγξτε τη σελίδα Ρυθμίσεων Μετριασμού στον πίνακα ελέγχου διαχείρισης σας. Ο ανιχνευτής ανεπιθύμητης αλληλογραφίας βασισμένος στο GPT4 χρεώνεται με `$0.08` για κάθε `1000` τοκέν.

Αυτός ο ανιχνευτής ανεπιθύμητης αλληλογραφίας είναι διαθέσιμος μόνο για πελάτες σε σχέδιο Flex, καθώς χρεώνεται με βάση τα τοκέν που χρησιμοποιούνται.

### Βελτιστοποιήσεις

Αρχικά περνάμε όλο το περιεχόμενο μέσω του κοινού ταξινόμου Naïve-Bayes για να περιορίσουμε τις κλήσεις στην OpenAI. Αυτό σας εξοικονομεί χρήματα για πολύ προφανή ανεπιθύμητη αλληλογραφία. Για την επιπλέον ύπουλη ανεπιθύμητη αλληλογραφία, στη συνέχεια καλούμε την OpenAI για να δούμε αν το περιεχόμενο μοιάζει με ανεπιθύμητη αλληλογραφία.

### Αποτελέσματα

Παρατηρήσαμε άμεσες βελτιώσεις στις κοινότητες που έχουν ενεργοποιήσει αυτή τη δυνατότητα.

### Τεκμηρίωση

Αυτό μπορεί να ρυθμιστεί μέσω της σελίδας Ρυθμίσεων Μετριασμού στον πίνακα ελέγχου διαχείρισης σας.

[Η τεκμηρίωση μπορεί επίσης να βρεθεί εδώ](https://docs.fastcomments.com/guide-moderation.html#spam-detection)

### Συμπέρασμα

Ευχαριστούμε τους πελάτες μας που μας δίνουν συνεχή ανατροφοδότηση ώστε να μπορούμε να σκεφτόμαστε ιδέες όπως αυτή. Ελπίζουμε να συνεχίσετε να αγαπάτε τη FastComments.

Καλή συνέχεια!
