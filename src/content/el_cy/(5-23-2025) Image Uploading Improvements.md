# [postlink]Βελτιώσεις Μεταφόρτωσης Εικόνων[/postlink]

{{#unless isPost}}
Διάφορες βελτιώσεις στη μεταφόρτωση εικόνων και στον επεξεργαστή WYSIWYG σε αυτήν την έκδοση.
{{/unless}}

{{#isPost}}

### Τι Νέα

Σήμερα έχουν κυκλοφορήσει μια σειρά από διορθώσεις σφαλμάτων για τη μεταφόρτωση εικόνων και τη λειτουργία επεξεργασίας WYSIWYG (αυτό που βλέπετε είναι αυτό που παίρνετε).

1. Η ένδειξη προόδου εμφανίζεται τώρα κατά τη μεταφόρτωση εικόνων.
2. Η επικόλληση εικόνων, όπως οι στιγμιότυπες οθόνες, είναι πλέον υποστηριζόμενη (μόνο στον επεξεργαστή WYSIWYG).
3. Όταν επεξεργάζεστε σχόλια, τα πλαίσια σχολίων είναι πλέον σωστά μεγέθους για να δείχνουν όλες τις εικόνες (μόνο στον επεξεργαστή WYSIWYG).
4. Όταν αφαιρούνται εικόνες, το πλαίσιο σχολίου συρρικνώνεται όπως αναμένεται (μόνο στον επεξεργαστή WYSIWYG).
