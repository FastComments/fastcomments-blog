# [postlink]FastComments Gets @mentions[/postlink]

{{#unless isPost}}
Έχετε ποτέ θελήσει να @αναφέρετε έναν ή περισσότερους χρήστες σε ένα σχόλιο; Τώρα μπορείτε.
{{/unless}}

{{#isPost}}

### Τι Είναι Νέο

Στο παρελθόν, το FastComments ειδοποιούσε τους χρήστες αν απαντούσατε στα σχόλιά τους, αλλά δεν υπήρχε τρόπος να αναφέρετε πολλούς ανθρώπους σε ένα μόνο σχόλιο. Επίσης, αυτές οι ειδοποιήσεις αποστέλλονταν σε δόσεις κάθε ώρα. Με το `@mentions`, είναι πλέον δυνατό να αναφέρετε έναν ή περισσότερους χρήστες και το FastComments θα τους ειδοποιήσει αμέσως μέσω email.

### Πώς να το Χρησιμοποιήσετε

Όταν πληκτρολογείτε `@`, και στη συνέχεια την αρχή του ονόματος ενός χρήστη, θα εμφανιστεί μια λίστα αποτελεσμάτων αναζήτησης.

<div class="text-center">
    <img src="images/fc-mentions.png" alt="Mentions Usage" title="@mentions Demo" />
</div>

Απλά κάντε κλικ στο όνομα του χρήστη που θέλετε να αναφέρετε.

Επιπλέον, μπορείτε να χρησιμοποιήσετε τα πλήκτρα βέλους πάνω και κάτω για να περιηγηθείτε σε αυτή τη λίστα, και πατήστε enter για να επιλέξετε, ή escape για να φύγετε.

Γνωρίζουμε ότι μπορεί να θέλετε να χρησιμοποιήσετε το σύμβολο `@` χωρίς να αναφέρετε έναν χρήστη. Το FastComments θα το ανιχνεύσει αυτό και δεν θα παρεμβαίνει στην εμπειρία πληκτρολόγησής σας.

### Ποιοι Έχουν Πρόσβαση

Όλοι οι τρέχοντες και νέοι πελάτες του FastComments, σε όλα τα επίπεδα, έχουν τώρα πρόσβαση στα `@mentions`. Τα `@mentions` έχουν επιστρέψει σε παλαιότερες εκδόσεις του widget σχολίων, επίσης.

### Πώς Λειτουργεί η Αυτόματη Συμπλήρωση

Οι χρήστες που εμφανίζονται στη λίστα αυτόματης συμπλήρωσης προσδιορίζονται από:

- Χρήστες που έχουν σχολιάσει στην ίδια σελίδα ή νήμα.
- Χρήστες που έχουν δημιουργήσει τους λογαριασμούς τους στο FastComments μέσω της ίδιας ιστοσελίδας.
- Διαχειριστές της τρέχουσας ιστοσελίδας.
- Χρήστες SSO που ανήκουν στο τρέχον ενοικιαστή.

### Επεξεργασία Σχολίων

Οι αναφορές μπορούν να προστεθούν σε υπάρχοντα σχόλια μέσω της επεξεργασίας τους. Ωστόσο, η εφαρμογή `Moderate Comments` δεν ολοκληρώνει αυτόματα τα `@mentions`.

### Μελλοντικές Σκέψεις

Στο μέλλον, οι χρήστες που έχουν ανοιχτό το FastComments σε ένα παράθυρο προγράμματος περιήγησης θα λάβουν ειδοποίηση όταν αναφέρονται, αν έχουν επιλέξει τις ειδοποιήσεις.

### Σε Συμπέρασμα

Όπως και το υπόλοιπο FastComments, ελπίζουμε να βρείτε αυτή τη δυνατότητα γρήγορη και εύκολη στη χρήση.

Cheers!
