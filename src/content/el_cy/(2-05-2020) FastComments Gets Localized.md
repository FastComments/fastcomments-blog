# [postlink]Το FastComments Λαμβάνει Τοπική Προσαρμογή[/postlink]

Η τοπική προσαρμογή δεν είναι μικρό βήμα - και έχει φτάσει στο FastComments. Το ενσωματωμένο widget σχολίων είναι πλέον πλήρως τοπικοποιημένο στα Αγγλικά (en-us), Ισπανικά (es-es) και Γαλλικά (fr-fr).

Θα παρατηρήσετε ότι δεν έχουμε επιβραδυνθεί. Γενικά, όταν οι περισσότερες εφαρμογές προσθέτουν υποστήριξη τοπικής προσαρμογής, η διόγκωση αρχίζει να γίνεται αισθητή. Στην περίπτωση μας, το script πλευράς πελάτη έχει στην πραγματικότητα μειωθεί σε μέγεθος καθώς έχουμε κάνει κάποια καθαριότητα και άλλες βελτιστοποιήσεις ταυτόχρονα.

Αν θέλετε να μεταφραστεί το FastComments στη γλώσσα σας, ή έχει εντοπίσει ένα πρόβλημα μετάφρασης, <a href="https://fastcomments.com/auth/my-account/help">ενημερώστε μας</a>.
