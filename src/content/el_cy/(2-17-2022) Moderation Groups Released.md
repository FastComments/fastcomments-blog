# [postlink]Απελευθέρωση Ομάδων Μετριασμού[/postlink]

## Τι είναι Νέο

Προηγουμένως, οι μετριαστές έβλεπαν όλα τα σχόλια στον λογαριασμό σας όταν επισκέπτονταν τη σελίδα Μετριασμού Σχολίων. Αν είχατε πολλούς τύπους περιεχομένου, όλοι οι μετριαστές
θα έβλεπαν την ακριβή ίδια λίστα. Για μεγαλύτερες ιστοσελίδες, αυτό θα μπορούσε να προκαλέσει κάποια σύγχυση σχετικά με το ποιοι μετριαστές μετριάζουν τι.

Τώρα παρέχουμε τη δυνατότητα να εμφανίζονται μόνο τα σχόλια που απαιτούν μετριασμό με βάση την ομάδα του μετριαστή.

## Πώς τοποθετούνται τα Σχόλια σε Ομάδες

Πρώτα, πρέπει να ρυθμιστεί μια προσαρμοσμένη ενσωμάτωση ώστε όταν αφήνονται σχόλια, το FastComments να ξέρει ποια ομάδα πρέπει να πάει το προκύπτον σχόλιο.

[Δείτε εδώ πώς να ρυθμίσετε την ενσωμάτωση.](https://docs.fastcomments.com/guide-customizations-and-configuration.html#moderation-group-ids)

## Προσθήκη Ομάδων

Απλά πηγαίνοντας στη νέα [Σελίδα Ομάδων Μετριασμού](https://fastcomments.com/auth/my-account/moderate-comments/moderation-groups), μπορούμε να ορίσουμε ομάδες με τα δικά τους ονόματα, να δούμε τα παραγόμενα ids που απαιτούνται για την ενσωμάτωση και ακόμα
να παρακολουθήσουμε τον αριθμό των μετριαστών και των σχολίων σε κάθε ομάδα.

Μπορείτε να επεξεργαστείτε έναν μετριαστή όπως θα κάνατε κανονικά, και μια νέα επιλογή θα είναι διαθέσιμη για να επιλέξετε τις ομάδες τους.

Σημειώστε ότι τόσο οι μετριαστές όσο και τα σχόλια μπορούν να ανήκουν σε περισσότερες από μία ομάδα.

Για λεπτομερή τεκμηρίωση σχετικά με την προσθήκη ομάδων, [ρίξτε μια ματιά στα έγγραφα!](https://docs.fastcomments.com/guide-moderation.html#moderation-groups).

## Μεταφορά Υπαρχόντων Σχολίων

Μέσω αίτησης υποστήριξης, το FastComments.com μπορεί να μεταφέρει τα σχόλια σας σε ομάδες που επιθυμείτε - απλά ενημερώστε μας!

## Επιβεβαίωση της Ενσωμάτωσης

Μπορείτε να επιβεβαιώσετε ότι η ενσωμάτωσή σας είναι σωστά ρυθμισμένη ελέγχοντας τον αριθμό σχολίων στη σελίδα Ομάδων Μετριασμού.

## Επιπτώσεις στους Υπάρχοντες Πελάτες

Αν είστε υπάρχων πελάτης που χρησιμοποιεί το FastComments, η δυνατότητα έχει ήδη γίνει διαθέσιμη στον λογαριασμό σας.

Αν δεν κάνετε καμία αλλαγή για να χρησιμοποιήσετε αυτήν τη δυνατότητα, οι μετριαστές σας δεν θα βιώσουν καμία αλλαγή στη ροή εργασίας τους.

## Συμπέρασμα

Ελπίζουμε να βρήκατε αυτήν την ενημέρωση και την τεκμηρίωσή της χρήσιμες. Καλή σχολιαστική εμπειρία!
