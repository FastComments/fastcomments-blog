# [postlink]Η FastComments είναι τώρα διαθέσιμη στο WordPress![/postlink]

Σκέφτεστε να αλλάξετε από άλλο plugin σχολίων για WordPress; Δοκιμάστε την FastComments!

{{#isPost}}

Είμαστε περήφανοι να ανακοινώσουμε ότι, μετά από πολύ δουλειά, λανσάραμε το plugin μας για WordPress. Μπορείτε να το βρείτε εδώ:
<a href="https://wordpress.org/plugins/fastcomments/" target="_blank">https://wordpress.org/plugins/fastcomments/</a>.

Έχουμε διαπιστώσει ότι η FastComments αποδίδει πολύ καλύτερα από τα προεπιλεγμένα σχόλια του WordPress, ειδικά για μεγαλύτερα νήματα σχολίων.

Θα επιταχύνει τον ιστότοπό σας στο WordPress μειώνοντας το φορτίο στον server σας - αφού θα αποδώσουμε τα νήματα σχολίων αντί του WordPress.

Τα αποτελέσματα των δοκιμών δεν ψεύδονται - με 100 σχόλια σε μία ανάρτηση που εκτελείται σε server χωρίς φορτίο, μια απλή εγκατάσταση του WordPress μπορεί να εξυπηρετήσει 86 αιτήσεις ανά δευτερόλεπτο πριν «πέσει».

Με την ενεργοποιημένη FastComments, αυτός ο αριθμός εκτινάσσεται σε **πάνω από 400**.

Ελπίζουμε οι χρήστες να βρουν το plugin εύκολο στη χρήση, στην προσαρμογή και γενικά «να λειτουργεί απλά». Θα πρέπει να συγχρονίζει αβίαστα τα σχόλια του WordPress σας προς τα εμπρός και πίσω χωρίς να επιβαρύνει τους server σας.
