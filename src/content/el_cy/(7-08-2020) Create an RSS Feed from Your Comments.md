# [postlink]Δημιουργήστε μια Ροή RSS από τα Σχόλιά σας[/postlink]

{{#unless isPost}}
Η RSS είναι ακόμα ζωντανή και καλά, και τώρα η FastComments την υποστηρίζει. Δεν ξέρετε τι είναι η RSS; Μην ανησυχείτε, θα σας ενημερώσουμε.
{{/unless}}

{{#isPost}}

<a href="https://en.wikipedia.org/wiki/RSS" target="_blank">RSS</a> είναι ένας τρόπος για να εγγραφείτε σε πολλές διαφορετικές ιστοσελίδες χρησιμοποιώντας έναν μόνο aggregator ειδήσεων.

Αυτό σημαίνει ότι μπορείτε τώρα **να παρακολουθείτε τα σχόλιά σας στη FastComments χρησιμοποιώντας μια εφαρμογή RSS στον υπολογιστή ή το κινητό σας.**

#### Πώς να το Χρησιμοποιήσετε

Μπορείτε τώρα να διαχειριστείτε τη ροή RSS σας από την αφιερωμένη <a href="https://fastcomments.com/auth/my-account/rss" target="_blank">σελίδα Ροής RSS</a> στον πίνακα ελέγχου σας. Απλά ενεργοποιήστε τη ροή, αντιγράψτε το μοναδικό σας RSS URL και επικολλήστε το στον αγαπημένο σας aggregator RSS.

Η ροή θα περιλαμβάνει όλα τα σχόλια των τελευταίων 30 ημερών.

Αυτά είναι όλα! Καλή σχολιαστική εμπειρία!

{{/isPost}}
