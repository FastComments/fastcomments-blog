# [postlink]Ευκολότερη Διαχείριση Σχολίων[/postlink]

{{#unless isPost}}Μπορείτε τώρα να διαχειρίζεστε τα σχόλια χωρίς να εγκαταλείπετε την ιστοσελίδα σας!{{/unless}}

{{#isPost}}

### Τι Νέο

Προηγουμένως, αν θέλατε να διαχειριστείτε τα σχόλια, μπορούσατε να το κάνετε μόνο από τη σελίδα Διαχείρισης Σχολίων.

Αν και αυτή η σελίδα είχε αρκετές βελτιώσεις για να διευκολύνει τη διαχείριση, όπως οι μαζικές ενέργειες, υπάρχουν πλεονεκτήματα
στο να μην χρειάζεται να φύγετε από την ιστοσελίδα σας. Συγκεκριμένα, όταν οι διαχειριστές ξεκινούν τη δουλειά τους, μπορούν
εύκολα να δουν το περιβάλλον γύρω από κάθε σχόλιο ενώ εργάζονται σε αυτές τις συζητήσεις. Αυτό σημαίνει επίσης ότι μπορείτε
να επεξεργαστείτε τα σχόλια **ζωντανά** καθώς έρχονται.

Έτσι, τώρα μπορείτε να διαχειρίζεστε απευθείας στην ιστοσελίδα σας χρησιμοποιώντας το FastComments!

Όταν είστε συνδεδεμένοι, οι διαχειριστές θα έχουν ένα μικρό εικονίδιο επεξεργασίας στην επάνω δεξιά γωνία κάθε σχολίου. Το να περάσετε το ποντίκι πάνω από αυτό θα
δείξει ένα μενού με μερικές νέες επιλογές.

Αυτές οι επιλογές θα εμφανίζονται επίσης στον Εξερευνητή Συζητήσεων Σχολίων.

### Τεκμηρίωση

Μπορείτε να βρείτε τεκμηρίωση που καλύπτει ποιες ενέργειες διαχείρισης είναι διαθέσιμες και πώς αυτό επηρεάζει τα ζωντανά σχόλια <a href="https://docs.fastcomments.com/guide-moderation.html#comment-actions-from-comment-widget" target="_blank">εδώ</a>.

### Ποιος Το Έχει

Οι πελάτες σε όλα τα επίπεδα έχουν τώρα ενεργοποιημένο αυτό το χαρακτηριστικό.

### Επίδραση στην Απόδοση

Ο κώδικας και οι πόροι που σχετίζονται με τη διαχείριση με αυτόν τον τρόπο κατεβαίνουν μόνο στους περιηγητές των διαχειριστών σας - δεν επηρεάζουν
 τους χρόνους φόρτωσης της σελίδας για τους επισκέπτες της ιστοσελίδας σας.

### Συμπέρασμα

Ελπίζουμε αυτό να διευκολύνει τη δουλειά σας ως διαχειριστής.

Καλή σχολιαστική δράση!
