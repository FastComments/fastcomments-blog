#### Για Προγραμματιστές
Η εγκατάσταση του FastComments σε μια ιστοσελίδα απαιτεί μόνο την προσθήκη ενός μικρού αποσπάσματος κώδικα. Αυτό σημαίνει ότι λειτουργεί σε στατικές σελίδες χωρίς καμία ταλαιπωρία - περιλαμβάνοντας τη δωρεάν φιλοξενία Σελίδων του GitHub.

#### Ένα Δείγμα

Έχουμε ρυθμίσει μια παράδειγμα Σελίδας GitHub <a href="https://fastcomments.github.io/fastcomments-demo.github.io/" target="_blank">εδώ</a>.

<a href="https://github.com/fastcomments/fastcomments-demo.github.io/blob/master/index.html#L29" target="_blank">Εδώ</a> είναι ο σύνδεσμος προς το σημείο όπου έχουμε προσθέσει το απόσπασμα κώδικα για τον 
λογαριασμό δοκιμών μας.

#### Ρυθμίσεις

Θα πρέπει να προσθέσεις το domain σου στο GitHub ως έγκυρο domain στον λογαριασμό σου.
