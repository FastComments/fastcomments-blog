### Τι είναι νέο

Το FastComments διαθέτει τώρα στοιχεία React Native που είναι ισοδύναμα σε δυνατότητες με το widget VanillaJS μας.

Αποφασίσαμε να ονομάσουμε τη βιβλιοθήκη `fastcomments-react-native`. Μπορείτε να βρείτε το αποθετήριο GitHub με τον πηγαίο κώδικα <a href="https://github.com/FastComments/fastcomments-react-native" target="_blank">εδώ</a>.

Το αποθετήριο περιέχει επίσης παραδείγματα χρήσης του widget.

Είναι διαθέσιμο στο NPM, το οποίο μπορείτε να βρείτε <a href="https://www.npmjs.com/package/fastcomments-react-native" target="_blank">εδώ</a>.

### Πώς λειτουργεί

Ο τρόπος λειτουργίας αυτού του νέου στοιχείου είναι ότι το React στοιχείο είναι μια περιτύλιξη γύρω από το υπάρχον widget FastComments.

Αυτό σημαίνει ότι αν προσθέσουμε δυνατότητες στα υπάρχοντα στοιχεία μας ή διορθώσουμε σφάλματα, θα ωφεληθείτε αμέσως!

### Χρόνος ανάπτυξης

Τα widgets React Native μπορούν να προστεθούν στην εφαρμογή σας σε μόλις λίγα λεπτά με ελάχιστο χρόνο ανάπτυξης.

### SSO

Όλες οι βιβλιοθήκες React Native υποστηρίζουν SSO εξ ορισμού, και η διαδικασία ενσωμάτωσης SSO είναι η ίδια με τα στοιχεία Vanilla JS. Οι υπάρχουσες
παραδείγματα κώδικα backend ισχύουν επίσης.

### Προσαρμοσμένο στυλ;

Τα στοιχεία React Native μπορούν να στυλιζαριστούν χρησιμοποιώντας CSS μέσω της διεπαφής προσαρμογής widget, όπως και τα widgets που είναι βασισμένα στο διαδίκτυο.
