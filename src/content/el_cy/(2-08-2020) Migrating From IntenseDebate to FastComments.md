## Αν είστε σε WordPress

Το FastComments διαθέτει ένα ειδικό plugin για WordPress: <a href="https://wordpress.org/plugins/fastcomments/" target="_blank">https://wordpress.org/plugins/fastcomments/</a>.

Το plugin καθιστά την εγκατάσταση, τη ρύθμιση και την συγχρονισμό πολύ εύκολα. Αν ακολουθήσετε αυτή τη διαδρομή μπορείτε να αγνοήσετε τις υπόλοιπες οδηγίες εδώ.

## Χειροκίνητη, Γενική Εγκατάσταση

### Εξαγάγετε τα Υπάρχοντα Σχόλιά σας

Για να εξάγετε τα δεδομένα Σχόλιου και Χρήστη του ιστότοπού σας από το IntenseDebate από τον πίνακα διαχείρισης τους, μεταβείτε σε Sites -> Ο Ιστότοπός σας -> Εργαλεία -> XML Εξαγωγή.

Θα παρατηρήσετε ότι το αρχείο που θα λάβετε μέσω email έχει μια μυστήρια "gz" επέκταση αρχείου. Αν είστε λίγο παραπάνω τεχνικά καταρτισμένος μπορεί να ξέρετε ότι αυτό σημαίνει "gzip" και είναι ένας δημοφιλής και αποτελεσματικός τρόπος συμπίεσης αρχείων.

### Μην ανοίξετε το αρχείο "gz"

Μια προεπιλεγμένη εγκατάσταση Windows 10 δεν θα μπορέσει να ανοίξει το αρχείο από το IntenseDebate. Αυτό είναι εντάξει, γιατί δεν χρειάζεται να το κάνετε με το FastComments. Ο backend μας καταλαβαίνει αυτό το συμπιεσμένο αρχείο.

Μετά από αυτό, αν είστε συνδεδεμένοι μπορείτε να πάτε <a href="https://fastcomments.com/auth/my-account/manage-data/import" target="_blank">εδώ για να εισάγετε το αρχείο</a>. Επιλέξτε το IntenseDebate από τη λίστα και ανεβάστε το αρχείο σας.

### Περιμένετε λίγα λεπτά

Η εισαγωγή στο FastComments είναι "ασύγχρονη". Αυτό σημαίνει ότι η αποστολή του αρχείου και η επεξεργασία του είναι ξεχωριστά βήματα.

Έτσι, ανάλογα με το μέγεθος του αρχείου σας, μπορεί να λάβετε το μήνυμα επιτυχίας αμέσως. Στη σελίδα εισαγωγής στο κάτω μέρος υπάρχει ένα πίνακας - κάθε γραμμή αντιπροσωπεύει μια απόπειρα εισαγωγής.
Μπορείτε με ασφάλεια να ανανεώσετε αυτή τη σελίδα για να δείτε την κατάσταση της εισαγωγής σας και πόσες γραμμές έχουν εισαχθεί μέχρι τώρα.

### Όταν ολοκληρωθεί

Θα λάβετε ένα email όταν η εισαγωγή είναι ολοκληρωμένη - είτε είναι επιτυχής είτε όχι. Μπορείτε να κλείσετε τη σελίδα μετά την επιτυχία ανάρτησης του αρχείου και να δείτε την εισαγωγή σας στη σελίδα Εισαγωγών. Θα έχει κατάσταση σαν "Αιτήθηκε" και όταν ξεκινήσει, η κατάσταση θα είναι "Εκτελείται".

Είναι ασφαλές να εισάγετε ξανά όσες φορές απαιτείται - ωστόσο αν η εισαγωγή σας αποτύχει τότε <a href="https://fastcomments.com/auth/my-account/help" target="_blank">επικοινωνήστε μαζί μας</a> ώστε να μπορέσουμε να βοηθήσουμε.

### Αντικαταστήστε τον Κωδικό IntenseDebate με τον FastComments

Είναι απλά θέμα να αφαιρέσετε το μικρό απόσπασμα που σας δίνει το IntenseDebate και να το αντικαταστήσετε με <a href="https://fastcomments.com/auth/my-account/get-acct-code" target="_blank">τον δικό μας</a>.
Μπορείτε ακόμη και να τρέξετε το IntenseDebate και το FastComments στον ίδιο ιστότοπο προσωρινά για να δοκιμάσετε και να προσαρμόσετε την εμφάνιση και την αίσθηση. <a href="https://fastcomments.com/auth/my-account/help" target="_blank">Μπορούμε να βοηθήσουμε με αυτό</a>.

Αν χρησιμοποιείτε widgets του IntenseDebate και θέλετε να τα αντικαταστήσετε, τότε <a href="https://fastcomments.com/auth/my-account/help" target="_blank">επικοινωνήστε μαζί μας εδώ</a>.

### Μεταφορά URLs ταυτόχρονα

Τα σχόλια είναι συνδεδεμένα με τα πεδία "σύνδεσμος" στην εξαγωγή του IntenseDebate, έτσι όσο τα URLs σας δεν αλλάξουν, η αλλαγή είναι εύκολη. Αν θέλετε να μεταφέρετε URLs και να διατηρήσετε τα
σχόλιά σας, τότε <a href="https://fastcomments.com/auth/my-account/help" target="_blank">ενημερώστε μας</a> στέλνοντάς μας τα παλιά και νέα URLs. Η σελίδα βοήθειας επιτρέπει την αποστολή μικρών κειμένων αρχείων, οπότε αν
μεταφέρετε περισσότερα από μερικά URLs, δημιουργήστε ένα Excel φύλλο.

### Ανακεφαλαίωση

1. Εξάγετε τα δεδομένα σας
2. Εισάγετε στο FastComments
3. Αντικαταστήστε το απόσπασμα JavaScript του IntenseDebate με το <a href="https://fastcomments.com/auth/my-account/get-acct-code" target="_blank">FastComments</a>

## Γιατί το IntenseDebate δεν μου στέλνει το αρχείο μου;
Για μεγαλύτερους ιστότοπους, το IntenseDebate μπορεί να μην είναι σε θέση να δημιουργήσει το εξαγόμενο αρχείο λόγω τεχνικών περιορισμών από την πλευρά τους. Έχουμε την ικανότητα να "scrape" τον υπάρχοντα ιστότοπό σας και να αποσπάσουμε τα σχόλια από αυτόν, ωστόσο αυτό μπορεί να είναι
αρκετά προσαρμοσμένο και λόγω του χρόνου που απαιτεί θα χρειαστεί να έχετε το Πακέτο Υποστήριξης. <a href="https://fastcomments.com/auth/my-account/help" target="_blank">Μπορείτε να ζητήσετε βοήθεια με αυτό εδώ</a>.
