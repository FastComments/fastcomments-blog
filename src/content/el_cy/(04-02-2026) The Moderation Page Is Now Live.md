# [postlink]Η Σελίδα Διαχείρισης Είναι Τώρα Ζωντανή[/postlink]

{{#unless isPost}}
Η σελίδα διαχείρισης του FastComments τώρα δείχνει μια ζωντανή ειδοποίηση όταν φτάνουν νέα σχόλια που ταιριάζουν με τα τρέχοντα φίλτρα σας.
{{/unless}}

{{#isPost}}

### Τι Είναι Νέο

Η σελίδα διαχείρισης ήταν πάντα ένα ισχυρό εργαλείο για τη διαχείριση των σχολίων. Μπορούσατε να φιλτράρετε κατά κατάσταση, να αναζητήσετε με κείμενο, να φιλτράρετε κατά σελίδα
ή χρήστη, και να εκτελέσετε μαζικές ενέργειες. Αλλά για να δείτε νέα σχόλια έπρεπε να ανανεώσετε τη σελίδα.

Όχι πια! Η σελίδα διαχείρισης τώρα δείχνει πότε φτάνουν νέα σχόλια που ταιριάζουν με τα τρέχοντα φίλτρα σας. Εμφανίζεται μια ειδοποίηση στην κορυφή της λίστας σχολίων που σας λέει πόσα νέα σχόλια περιμένουν, και πατώντας την φορτώνει τα σχόλια στο inline χωρίς πλήρη ανανέωση της σελίδας.

Το μήνυμα "νέα σχόλια για τα φίλτρα σας" θα πρέπει να είναι σχεδόν άμεσο κάτω από τις περισσότερες συνθήκες.

### Αξιολόγηση Φίλτρων

Η μεταφορική στρώση αξιολογεί όλα τα φίλτρα διαχείρισης από την πλευρά του διακομιστή. Αυτό περιλαμβάνει φίλτρα κατάστασης (χρειάζεται έλεγχο, χρειάζεται έγκριση, spam, επισημανθέν,
απαγόρευση), φίλτρα συμβολοσειρών (αναγνωριστικό χρήστη, URL σελίδας, τομέας), και ακόμη και αναζήτηση κειμένου. Το κείμενο που παρατίθεται όπως το `"ακριβής φράση"` κάνει
ακριβή αντιστοίχηση υποσυμβολοσειράς χωρίς διάκριση πεζών-κεφαλαίων, ενώ οι μη παρατιθέμενοι όροι κάνουν ασαφή αντιστοίχηση χωρίς διάκριση πεζών-κεφαλαίων.

Αυτό σημαίνει ότι αν φιλτράρετε για σχόλια spam, δεν θα λαμβάνετε ειδοποιήσεις για εγκεκριμένα σχόλια. Αν αναζητάτε σχόλια
που περιέχουν μια συγκεκριμένη φράση, μόνο τα αντίστοιχα σχόλια ενεργοποιούν την ειδοποίηση.

### Ομάδες Διαχείρισης

Αν ο λογαριασμός σας χρησιμοποιεί ομάδες διαχείρισης για να περιορίσει ποιοι διαχειριστές μπορούν να δουν ποια σχόλια, οι περιορισμοί αυτοί επιβάλλονται πλήρως
για τις ζωντανές ειδοποιήσεις. Η μεταφορική στρώση καλεί τον διακομιστή εφαρμογής για να δημιουργήσει το φίλτρο, το οποίο περιλαμβάνει περιορισμούς ομάδων
βάσει των ανατεθειμένων ομάδων του διαχειριστή. Αυτό το φίλτρο αποθηκεύεται στην μνήμη για τη διάρκεια της σύνδεσης WebSocket και ενημερώνεται όποτε
ο διαχειριστής αλλάζει φίλτρα.

### Χαμένες Εκδηλώσεις

Αν η σύνδεση ενός διαχειριστή διακοπεί προσωρινά (προβλήματα δικτύου, ύπνος φορητού υπολογιστή κ.λπ.), το σύστημα συγχρονίζεται αυτόματα κατά την επανασύνδεση ώστε
να μην χάσετε καμία εργασία.

### Απόδοση

Σχεδιάσαμε αυτή τη δυνατότητα ώστε να έχει ελάχιστη επίδραση στον χρόνο φόρτωσης της σελίδας διαχείρισης.

### Πολυ-Περιοχή

Αυτή η δυνατότητα λειτουργεί ομαλά με την παγκόσμια κατανεμημένη αρχιτεκτονική μας, ώστε οι παγκοσμίως κατανεμημένες ομάδες διαχείρισης να έχουν την ίδια ζωντανή εικόνα.

### ΣυμπConclusion

Ελπίζουμε αυτή η δυνατότητα να εξοικονομήσει χρόνο στους διαχειριστές και να κάνει την εμπειρία διαχείρισης πιο άμεση. Όπως πάντα, ευχαριστούμε τους πελάτες μας για
την ανατροφοδότηση και την υποστήριξή τους.
