### Τι Νέο

Πριν την 14η Νοεμβρίου 2023, έχουμε κάνει τις εξής αλλαγές στα webhooks:

#### Ενημέρωση Webhooks

**Προηγουμένως**, οι ενημερώσεις περιλάμβαναν την τελευταία έκδοση ενός σχολίου. Αν δύο χρήστες ψήφιζαν ένα σχόλιο σε γρήγορη διαδοχή, αυτό θα
έκανε trigger σε δύο webhooks. Ωστόσο, ήταν δυνατό το πρώτο γεγονός webhook να περιέχει την **τελευταία έκδοση**, με αποτέλεσμα να έχουμε δύο
identical γεγονότα webhook:

1. ` ... { votes: 2 } ...`
2. ` ... { votes: 2 } ...`

**Από την 14η Νοεμβρίου** κάθε γεγονός webhook είναι ιδεατό και περιέχει το αντικείμενο του σχολίου τη στιγμή της αλλαγής. Στο παραπάνω παράδειγμα θα λάβετε τώρα δύο γεγονότα:

1. ` ... { votes: 1 } ...`
2. ` ... { votes: 2 } ...`

#### Διαγραφή Webhooks

**Προηγουμένως**, τα webhook διαγραφής περιλάμβαναν μόνο το ID του σχολίου. Αυτό ήταν ένα αντικείμενο όπως `{ id: string }`.

**Από την 14η Νοεμβρίου**, τα webhook διαγραφής θα περιλαμβάνουν το **ολόκληρο** αντικείμενο του σχολίου τη στιγμή της αφαίρεσης.

### Στο Συμπέρασμα

Όπως σε όλες τις μεγάλες κυκλοφορίες, είμαστε χαρούμενοι που μπορέσαμε να αφιερώσουμε χρόνο για να βελτιστοποιήσουμε, να δοκιμάσουμε και να κυκλοφορήσουμε σωστά αυτή τη δυνατότητα. Ενημερώστε μας παρακάτω εάν ανακαλύψετε οποιαδήποτε προβλήματα.

Ευχαριστούμε!
