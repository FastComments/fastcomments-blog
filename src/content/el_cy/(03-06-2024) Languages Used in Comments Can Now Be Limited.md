# [postlink]Οι Γλώσσες που Χρησιμοποιούνται στα Σχόλια Μπορούν Τώρα να Περιοριστούν[/postlink]

{{#unless isPost}}
Ορισμένες κοινότητες θέλουν να περιορίσουν τις γλώσσες που χρησιμοποιούνται. Αυτό μπορεί τώρα να γίνει με το FastComments.
{{/unless}}

{{#isPost}}

### Τι είναι Νέο

Το FastComments πάντα προσπαθούσε να υποστηρίζει όσο το δυνατόν περισσότερες γλώσσες και περιοχές.

Ωστόσο, ορισμένες κοινότητες θέλουν να περιορίσουν τις γλώσσες που χρησιμοποιούνται για επικοινωνία και για να γράφουν σχόλια.

Αντί να επιβάλλεται αυτό ως κανόνας και να απαγορεύονται πιθανά χρήστες, μπορεί τώρα να ρυθμιστεί ρητά στην πλατφόρμα.

### Πώς να το Ρυθμίσετε

Στο ίδιο UI Εξατομίκευσης Widget που πιθανόν να γνωρίζετε, μπορείτε τώρα να επιλέξετε μία ή περισσότερες γλώσσες στις οποίες μπορούν να γραφούν σχόλια.

Η πλατφόρμα, κατά την υποβολή του σχολίου, θα προσπαθήσει να καθορίσει τη γλώσσα του σχολίου σε πραγματικό χρόνο. Εάν η εμπιστοσύνη της καθορισμένης γλώσσας είναι
πάνω από 70%, και ταιριάζει με κάποια επιτρεπτή γλώσσα, τότε το σχόλιο θα επιτρέπεται.

Αν το γραμμένο σχόλιο δεν είναι σε γλώσσα που έχει οριστεί από τη ρύθμισή σας, ο χρήστης θα δει ένα μήνυμα σφάλματος στη δική του γλώσσα.

### Επίδραση στους Προγραμματιστές

Αυτή η ρύθμιση θα επηρεάσει και τα σχόλια που αποθηκεύονται μέσω του API.

Θα λάβετε ένα σφάλμα όπως το παρακάτω:

            status: 'failed',
            reason: `Το σχόλιο φαίνεται ότι είναι σε μία από αυτές τις γλώσσες: [es], αλλά επιτρέπονται μόνο οι [en,fr].`,
            code: 'language-not-allowed',
            translatedError: "Ένα μήνυμα σφάλματος στη γλώσσα του χρήστη."

### Στην Σύνοψη

Όπως με όλες τις κύριες εκδόσεις, είμαστε χαρούμενοι που μπορέσαμε να αφιερώσουμε χρόνο για να βελτιστοποιήσουμε, να δοκιμάσουμε και να κυκλοφορήσουμε σωστά αυτή τη δυνατότητα. Ενημερώστε μας
παρακάτω αν ανακαλύψετε οποιαδήποτε προβλήματα.

Ευχαριστώ!

{{/isPost}}
