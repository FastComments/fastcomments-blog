### Τι Είναι Νέο

Όταν σχολιάζουν, οι χρήστες μπορούν τώρα να εισάγουν ένα ή περισσότερα `#hashtags`. Κάνοντας κλικ σε ένα `#hashtag` σε ένα σχόλιο θα σας μεταφέρει σε μια προβολή με άλλα σχόλια
με το hashtag που εμφανίζεται.

Επιπλέον, τα `#hashtags` μπορούν να χρησιμοποιηθούν για σύνδεση με εξωτερικά κομμάτια περιεχομένου. Παράλληλα με αυτή την έκδοση, έχουμε λανσάρει το [HashTag API](https://docs.fastcomments.com/guide-api.html#hash-tag-structure)
το οποίο επιτρέπει την προ-συμπλήρωση hashtags στον λογαριασμό σας.

Όταν προ-συμπληρώνουμε hashtags, μπορούμε να ορίσουμε μια διεύθυνση URL για κάθε ετικέτα. Αυτό σημαίνει ότι τα `#hashtags` μπορούν επίσης να χρησιμοποιηθούν για σύνδεση με, για παράδειγμα, έναν αριθμό εισιτηρίου ή
ένα έγγραφο σε ένα CRM.

Ο συμβολισμός `#` μπορεί επίσης να προσαρμοστεί κατόπιν αιτήματος.

### Πώς Να Το Χρησιμοποιήσετε

Όταν πληκτρολογείτε `#`, και στη συνέχεια την αρχή ενός hashtag, θα εμφανιστεί μια λίστα αποτελεσμάτων αναζήτησης.

<div class="text-center">
    <img src="images/fc-hashtags.png" alt="HashTags Usage" title="#hashtags Demo" />
</div>

Απλά κάντε κλικ στο hashtag που επιθυμείτε να προσθέσετε.

Επιπλέον, μπορείτε να χρησιμοποιήσετε τα πλήκτρα βέλους προς τα επάνω και προς τα κάτω για να πλοηγηθείτε σε αυτή τη λίστα, και να πατήσετε enter για να επιλέξετε, ή escape για να φύγετε.

Γνωρίζουμε ότι μπορεί να θελήσετε να χρησιμοποιήσετε το σύμβολο `#` χωρίς να προσθέσετε ένα hashtag. Το FastComments θα το ανιχνεύσει αυτό και δεν θα παρεμβαίνει στην εμπειρία πληκτρολόγησής σας.

### Προσθήκη Hash Tags

Αν το hashtag που θέλετε να χρησιμοποιήσετε δεν είναι στη λίστα προτάσεων - το FastComments θα δημιουργήσει αυτόματα το hashtag στο παρασκήνιο.

Η αυτόματη δημιουργία `#hashtag` μπορεί να απενεργοποιηθεί όπως αναφέρεται στην τεκμηρίωση [εδώ](https://docs.fastcomments.com/guide-customizations-and-configuration.html#disable-automatic-hashtag-creation).

### Ποιοι Το Λαμβάνουν

Όλοι οι υπάρχοντες και νέοι πελάτες του FastComments, σε όλα τα επίπεδα, έχουν τώρα πρόσβαση σε `#hashtags`. Τα `#hashtags` έχουν επιστραφεί σε παλαιότερες εκδόσεις του
widget σχολίων, επίσης.

### Πώς Λειτουργεί Η Αυτόματη Συμπλήρωση

Οι ετικέτες που εμφανίζονται στη λίστα αυτόματης συμπλήρωσης προέρχονται μόνο από τον λογαριασμό σας. Τα HashTags δεν μοιράζονται μεταξύ των ενοικιαστών του FastComments.

### Τεκμηρίωση

Υπάρχει ειδική τεκμηρίωση για τα `#hashtags`. Δείτε εδώ: https://docs.fastcomments.com/guide-hashtags.html

### Στην Επίλογο

Όπως και ο υπόλοιπος FastComments, ελπίζουμε ότι θα βρείτε αυτή τη δυνατότητα γρήγορη και εύκολη στη χρήση.

Στην υγειά σας!
