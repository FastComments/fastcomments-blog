# [postlink]Ενημέρωση Συχνότητας Ειδοποιήσεων μέσω Email[/postlink]

{{#unless isPost}}
Οι χρήστες μπορούν τώρα να ελέγχουν πόσο συχνά λαμβάνουν ειδοποιήσεις μέσω email για νέα σχόλια και απαντήσεις, κάθε λεπτό, ωριαίο συνοπτικό ή ημερήσιο συνοπτικό, με ξεχωριστές ρυθμίσεις για ειδοποιήσεις απαντήσεων και διαχειριστών, καθώς και υπερβάσεις ανά συνδρομή.
{{/unless}}

{{#isPost}}

### Τι είναι Νέο

Έχουμε προσθέσει ρυθμίσεις **Συχνότητας Ειδοποιήσεων για Απαντήσεις** και **Συχνότητας Ειδοποιήσεων για Διαχειριστές** που ελέγχουν πόσο συχνά λαμβάνετε ειδοποιήσεις μέσω email για απαντήσεις και νέα σχόλια, αντίστοιχα. Οι τρεις επιλογές για κάθε μία είναι:

- **Κάθε λεπτό** - λαμβάνετε ένα email μόλις φτάσουν νέα σχόλια (ελέγχεται κάθε λεπτό).
- **Ωριαίο συνοπτικό** - λαμβάνετε μια συνοπτική αναφορά νέων σχολίων μία φορά την ώρα.
- **Ημερήσιο συνοπτικό** - λαμβάνετε μια συνοπτική αναφορά νέων σχολίων μία φορά την ημέρα.

Η Συχνότητα Ειδοποιήσεων για Απαντήσεις είναι διαθέσιμη σε όλους τους χρήστες και προεπιλεγμένη είναι η **Κάθε λεπτό**. Η Συχνότητα Ειδοποιήσεων για Διαχειριστές είναι διαθέσιμη σε διαχειριστές του site και προεπιλεγμένη είναι η **Ωριαίο συνοπτικό**. Σημειώστε ότι οι ειδοποιήσεις @mention αποστέλλονται πάντα αμέσως ανεξάρτητα από αυτές τις ρυθμίσεις.

Μπορείτε επίσης να υπερβείτε τη συχνότητα σε επίπεδο συνδρομής στον πίνακα Συνδρομών, για πιο λεπτομερή έλεγχο σε ατομικές σελίδες.

### Πώς να το Ρυθμίσετε

1. Μεταβείτε στις [Ρυθμίσεις Ειδοποιήσεων](https://fastcomments.com/auth/my-account/edit-notifications).
2. Χρησιμοποιήστε τις αναπτυσσόμενες λίστες **Συχνότητας Ειδοποιήσεων για Απαντήσεις** και **Συχνότητας Ειδοποιήσεων για Διαχειριστές** για να ορίσετε τις προτιμώμενες συχνότητες σας.
3. Προαιρετικά, υπερβείτε τη συχνότητα για ατομικές συνδρομές στον πίνακα Συνδρομών.
4. Κάντε κλικ στο **Αποθήκευση Αλλαγών**.

### Υποστήριξη API

Το πεδίο `notificationFrequency` στο αντικείμενο χρήστη ελέγχει τη συχνότητα ειδοποιήσεων για απαντήσεις, και το πεδίο `adminNotificationFrequency` ελέγχει τη συχνότητα ειδοποιήσεων για διαχειριστές. Υπερβάσεις ανά συνδρομή μπορούν να οριστούν μέσω του πεδίου `notificationFrequency` της συνδρομής. Δείτε την [τεκμηρίωση API](https://docs.fastcomments.com/guide-api.html) για λεπτομέρειες.

### Στο Συμπέρασμα

Αυτό δίνει στους χρήστες έλεγχο πάνω στο inbox τους χωρίς να χρειάζεται να καταργήσουν τη συνδρομή τους σε σελίδες εντελώς.

Ενημερώστε μας παρακάτω αν έχετε οποιαδήποτε σχόλια!

Χαιρετισμούς!
