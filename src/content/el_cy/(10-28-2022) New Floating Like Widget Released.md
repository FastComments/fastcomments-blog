# [postlink]Νέο Widget Σαν Floating Like Κυκλοφόρησε[/postlink]

{{#unless isPost}}
Ένα νέο widget έχει κυκλοφορήσει που επιτρέπει στους χρήστες να κάνουν like σε σελίδες ή άρθρα.
{{/unless}}

{{#isPost}}

### Τι Νέο

Στο FastComments, ενώ οι χρήστες μπορούν να εγγραφούν σε σελίδες απευθείας στο widget σχολίων κάνοντας κλικ στο καμπανάκι, δεν είχαμε λύση για αντιδράσεις σε επίπεδο σελίδας.

Αυτό το χαρακτηριστικό έχει τώρα λανσαριστεί, με το πρώτο widget να είναι το Floating Likes widget που βλέπετε στο κάτω δεξί μέρος αυτού του blog.

Μπορείτε να κάνετε like στη σελίδα απευθείας, ακόμη και αν δεν έχετε λογαριασμό στο FastComments.com. Εάν είστε συνδεδεμένοι, η αντίδραση συνδέεται με τον λογαριασμό σας.

### SSO

Το widget υποστηρίζει SSO για σκοπούς ταυτοποίησης, επίσης.

### Τεκμηρίωση

Μπορείτε να βρείτε την τεκμηρίωση στο [docs.fastcomments.com](https://docs.fastcomments.com/guide-page-reacts.html).

### Προσαρμόσιμο

Αυτό το widget δεν χρησιμοποιεί iframe και μπορεί να στιλιζαριστεί απευθείας μέσω CSS.

### Σκοτεινή Λειτουργία

Εξ ορισμού, το widget θα αλλάξει σε μια πιο σκούρα θεματική αν οι προτιμήσεις του προγράμματος περιήγησης ή του λειτουργικού συστήματος του χρήστη προτιμούν τη σκοτεινή λειτουργία.

### Προοπτικές

Μακροπρόθεσμα, θα θέλαμε να προσθέσουμε διαφορετικές αντιδράσεις σε επίπεδο σελίδας, και ίσως ακόμα και ψηφοφορία, αν διαπιστώσουμε ότι η πελατεία μας ενδιαφέρεται για τέτοια χαρακτηριστικά.

Θα περιμένουμε σχόλια από τους χρήστες και την κοινότητα για να βελτιώσουμε αυτό το νέο widget με την πάροδο του χρόνου.

### Στο Συμπέρασμα

Γνωρίζουμε ότι αυτό το χαρακτηριστικό έχει αναμενόμενη επιτυχία από μερικούς. Όπως με όλες τις μεγάλες κυκλοφορίες, είμαστε χαρούμενοι που μπορέσαμε να αφιερώσουμε χρόνο για να βελτιστοποιήσουμε, να δοκιμάσουμε και να κυκλοφορήσουμε σωστά αυτό το στοιχείο.

Άντε!
