# [postlink]Ενσωμάτωση Σχολίων σε Ιστότοπο GatsbyJS με FastComments[/postlink]

{{#unless isPost}}
Μπορείτε τώρα να χρησιμοποιήσετε το FastComments για να ενσωματώσετε σχόλια σε έναν ιστότοπο που έχει κατασκευαστεί με το Gatsby!
{{/unless}}

{{#isPost}}

### <i class="circle">!</i> Αυτό το Άρθρο Περιέχει Τεχνική Ορολογία

#### Τι Νέο

Πρόσφατα, λανσάραμε το <a href="https://blog.fastcomments.com/(8-12-2020)-fastcomments-goes-react.html" target="_blank">React component</a> για το FastComments.

Αυτό ανοίγει πολλές ευκαιρίες - συμπεριλαμβανομένης της ενσωμάτωσης του FastComments με το Gatsby.

#### Αν Έχετε Γράψει το Δικό Σας Wrapper

Αν έχετε γράψει το δικό σας React wrapper για το FastComments - εξετάστε το ενδεχόμενο να το αντικαταστήσετε με το νέο μας. Θα διατηρήσουμε αυτή την αφαίρεση για εσάς - και περιέχει πολλά
χρήσιμα πράγματα όπως <a href="https://github.com/FastComments/fastcomments-react/blob/master/example/src/DarkModeApp.tsx#L8" target="_blank">αυτόματη ανανέωση σε αλλαγές ρυθμίσεων</a>.

#### Πώς Λειτουργεί

Εφόσον το FastComments έχει ένα React component ως wrapper γύρω από το βασικό μας widget - είναι απλώς θέμα χρήσης της βιβλιοθήκης μας React για να δημιουργήσετε το widget από την πλευρά του πελάτη.

Υποστηρίζουμε πλήρως αυτό το component και θα το διατηρούμε ενημερωμένο.

#### Παραδείγματα;

Έχουμε αρχίσει να δημιουργούμε παραδείγματα, τα οποία μπορείτε να βρείτε <a href="https://github.com/FastComments/fastcomments-gatsbyjs-example" target="_blank">εδώ</a>.

#### Νυχτερινή Λειτουργία

Ένα από τα παραδείγματα που έχουμε είναι η "νυχτερινή λειτουργία" - εμφάνιση του FastComments σε έναν ιστότοπο με μαύρο (ή πολύ σκούρο) φόντο: <a href="https://github.com/FastComments/fastcomments-react/blob/master/example/src/DarkModeApp.tsx" target="_blank">https://github.com/FastComments/fastcomments-react/blob/master/example/src/DarkModeApp.tsx</a>.

Επιπλέον, εφόσον το widget ανταποκρίνεται σε αλλαγές ρυθμίσεων, αυτό σημαίνει ότι μπορείτε να έχετε μια λειτουργία για να εναλλάσσετε το θέμα του ιστότοπού σας και να λέτε εύκολα στο fastcomments-react να ενημερωθεί.

#### Συμπέρασμα

Ελπίζουμε να βρείτε την ενσωμάτωσή του FastComments στον ιστότοπό σας Gatsby γρήγορη και εύκολη.

Ευχαριστούμε!
