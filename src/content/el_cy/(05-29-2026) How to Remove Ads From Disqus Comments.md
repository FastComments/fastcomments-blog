# [postlink]Πώς να αφαιρέσετε τις διαφημίσεις από τα σχόλια του Disqus[/postlink]

{{#unless isPost}}
Στο δωρεάν σχέδιο του Disqus, η περιοχή των σχολίων σας περιλαμβάνει διαφημίσεις. Υπάρχουν δύο τρόποι για να τις αφαιρέσετε. Ο καλύτερος τρόπος κάνει επίσης τα σχόλια σας να φορτώνουν πιο γρήγορα και εμποδίζει τους αναγνώστες σας να παρακολουθούνται.
{{/unless}}

{{#isPost}}

Το Disqus πληρώνει για το δωρεάν σχέδιο του βάζοντας διαφημίσεις στην περιοχή των σχολίων σας, ακριβώς κάτω από το περιεχόμενό σας, στο ίδιο σημείο που οι αναγνώστες σας πηγαίνουν για να μιλήσουν μαζί σας. Αυτές οι διαφημίσεις δεν είναι δικές σας. Δεν τις επιλέγετε και δεν κερδίζετε ούτε σεντ από αυτές. Αυτό είναι το αντάλλαγμα για το δωρεάν επίπεδο.

Υπάρχουν δύο πραγματικοί τρόποι για να τις αφαιρέσετε.

## Επιλογή 1: Αναβαθμίστε το σχέδιο σας στο Disqus

Η απλή λύση είναι να πληρώσετε το Disqus. Τα πληρωμένα σχέδιά τους ανοιγοκλείνουν τις διαφημίσεις. Μπορείτε να βρείτε τις τρέχουσες επιλογές στη <a href="https://disqus.com/pricing/" target="_blank">σελίδα τιμολόγησης του Disqus</a>.

Αυτό λειτουργεί, αλλά αξίζει να δώσετε προσοχή σε αυτό που αγοράζετε. Πληρώνετε για να αναιρέσετε ένα μειονέκτημα. Οι διαφημίσεις εξαφανίζονται, και το υπόλοιπο του Disqus παραμένει το ίδιο: η περιοχή των σχολίων εξακολουθεί να φορτώνει ένα βαρύ πακέτο τρίτων σεναρίων, και οι αναγνώστες σας εξακολουθούν να παρακολουθούνται στα διάφορα sites που το ενσωματώνουν. Πληρώνετε για να αφαιρέσετε τις διαφημίσεις, όχι για να κάνετε την περιοχή των σχολίων ελαφρύτερη ή πιο ιδιωτική.

## Επιλογή 2: Μεταβείτε σε μια πλατφόρμα που δεν εμφανίζει διαφημίσεις

Ο άλλος τρόπος για να αφαιρέσετε τις διαφημίσεις είναι να χρησιμοποιήσετε μια πλατφόρμα σχολίων που δεν τις εκτελεί από την αρχή. :)

Για μόλις 0,99 $/μήνα, η FastComments παρέχει βαθιά βελτιστοποιημένη λειτουργικότητα χωρίς παρακολούθηση ή διαφημίσεις.

## Διατηρείτε κάθε σχόλιο

Το μέρος που ανησυχούν οι άνθρωποι περισσότερο είναι η απώλεια της υπάρχουσας συζήτησής τους. Η FastComments έχει έναν ενσωματωμένο εισαγωγέα Disqus που μεταφέρει τα σχόλια, τα ονόματα χρηστών, τις εικόνες χρηστών, τις εικόνες εντός των κειμένων, τις ψήφους και τις χρονικές σφραγίδες. Μεταφέρουμε ακόμη και όλες τις εικόνες στον δικό μας CDN για εσάς.

Αν θέλετε τον πλήρη, λεπτομερή οδηγό, έχουμε έναν ειδικό οδηγό: [Μετακόμιση Από το Disqus Στο FastComments](/(1-23-2020)-migrating-from-disqus-fastcomments.html). Η σύντομη έκδοση είναι παρακάτω.

## Πώς να μεταβείτε σε τρία βήματα

### 1. Εξάγετε τα σχόλια σας από το Disqus

Ακολουθήστε τα <a href="https://help.disqus.com/en/articles/1717199-importing-exporting" target="_blank">βήματα εξαγωγής του Disqus</a>. Θα σας στείλουν ένα αρχείο με επέκταση "gz". Δεν χρειάζεται να το ανοίξετε ή να το αποσυμπιέσετε, και μια προεπιλεγμένη εγκατάσταση Windows δεν μπορεί ούτως ή άλλως. Η υποδομή μας διαβάζει αυτό το συμπιεσμένο αρχείο απευθείας.

### 2. Εισάγετε το αρχείο στο FastComments

Αφού συνδεθείτε, πηγαίνετε στη <a href="https://fastcomments.com/auth/my-account/manage-data/import" target="_blank">σελίδα εισαγωγής</a>, επιλέξτε το Disqus από το αναπτυσσόμενο μενού και ανεβάστε το αρχείο όπως είναι. Η εισαγωγή τρέχει στο παρασκήνιο; η σελίδα θα ενημερώνεται αυτόματα καθώς εκτελείται για να σας δείξει μια ζωντανή μέτρηση του αριθμού των σχολίων που ανακαλύφθηκαν και εισήχθησαν. Είναι ασφαλές να κάνετε ξανά την εισαγωγή όσες φορές χρειαστεί.

### 3. Αντικαταστήστε τον κώδικα ενσωμάτωσης του Disqus με τον δικό μας

Αφαιρέστε τον κώδικα ενσωμάτωσης του Disqus από τις σελίδες σας και τοποθετήστε <a href="https://fastcomments.com/auth/my-account/install-wizard/" target="_blank">τον κώδικα ενσωμάτωσης της FastComments</a> αντί αυτού. Όσο παραμένουν οι διευθύνσεις URL των σελίδων σας ίδιες, τα εισαγόμενα σχόλιά σας ευθυγραμμίζονται ακριβώς εκεί που ήταν.

## Χρησιμοποιείτε το WordPress;

Αν ο ιστότοπός σας τρέχει σε WordPress, παραλείψτε τα χειροκίνητα βήματα και εγκαταστήστε το <a href="https://wordpress.org/plugins/fastcomments/" target="_blank">πρόσθετο FastComments για WordPress</a>. Διαχειρίζεται την εγκατάσταση, ρύθμιση και συγχρονισμό για εσάς, και στη συνέχεια μπορείτε να απενεργοποιήσετε το Disqus.

## Δοκιμάστε το Πριν Αναλάβετε

Δεν χρειάζεται να ενεργοποιήσετε ταυτόχρονα τα πάντα. Μπορείτε να τρέξετε το Disqus και την FastComments στην ίδια σελίδα προσωρινά για να συγκρίνετε την εμφάνιση και την αίσθηση πριν κάνετε τη πλήρη μετάβαση. Αν χρειάζεστε βοήθεια με οποιοδήποτε από αυτά, <a href="https://fastcomments.com/auth/my-account/help" target="_blank">επικοινωνήστε μαζί μας</a> και θα σας βοηθήσουμε.

Δεν είστε σίγουροι τι θα σας κοστίσει η FastComments; Κάθε σχέδιο είναι χωρίς διαφημίσεις, και μπορείτε να ελέγξετε τον αριθμό σας στον <a href="https://fastcomments.com/pricing-calculator" target="_blank">υπολογιστή τιμών</a>. Αν βρίσκεστε στην ΕΕ, μπορείτε να δημιουργήσετε τον λογαριασμό σας στο <a href="https://eu.fastcomments.com" target="_blank">eu.fastcomments.com</a> ώστε τα δεδομένα σας να παραμείνουν στην ΕΕ.

## Ανακεφαλαίωση

1. Εξάγετε τα δεδομένα σας από το Disqus
2. Εισάγετε τα στο FastComments
3. Αντικαταστήστε τον κώδικα του Disqus με <a href="https://fastcomments.com/auth/my-account/get-acct-code" target="_blank">τον κώδικα της FastComments</a>

Αυτό είναι. Χωρίς διαφημίσεις στην περιοχή των σχολίων σας, ελαφρύτερες σελίδες, και κάθε σχόλιο που ήδη είχατε.

Στην υγειά σας!
