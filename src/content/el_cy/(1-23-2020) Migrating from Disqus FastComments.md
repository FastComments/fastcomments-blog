Υπάρχουν πολλά πράγματα που το Disqus κάνει σωστά, αλλά εκεί που δεν τα καταφέρνει, το FastComments αναλαμβάνει.

## Εάν Είστε Σε WordPress

Το FastComments έχει ένα ειδικό plugin για WordPress: <a href="https://wordpress.org/plugins/fastcomments/" target="_blank">https://wordpress.org/plugins/fastcomments/</a>.

Το plugin καθιστά την εγκατάσταση, τη ρύθμιση και τη συγχρονισμό πολύ εύκολα. Αν ακολουθήσετε αυτή τη διαδρομή, μπορείτε να αγνοήσετε τις υπόλοιπες οδηγίες εδώ.

## Χειροκίνητη, Γενική Εγκατάσταση

### Εξαγωγή Των Υπαρχόντων Σχολίων Σας

Ο εισαγωγέας του FastComments θα μεταφέρει τα σχόλια σας, τα ονόματα χρηστών, τα άβαταρ χρηστών και τις ενσωματωμένες εικόνες. Θα μεταφέρουμε τις εικόνες στους διακομιστές μας χωρίς προβλήματα.

Ένα πράγμα που το Disqus κάνει σωστά είναι πώς χειρίζεται την εξαγωγή δεδομένων. Θα παρατηρήσετε αφού ακολουθήσετε τα βήματα στον οδηγό τους <a href="https://help.disqus.com/en/articles/1717199-importing-exporting" target="_blank">εδώ</a> ότι το αρχείο που θα λάβετε μέσω email έχει μια μυστήρια επέκταση αρχείου "gz". Εάν είστε λίγο πιο τεχνικά savvy, ίσως γνωρίζετε ότι αυτό σημαίνει "gzip" και είναι ένας δημοφιλής και αποτελεσματικός τρόπος για να συμπιέσετε αρχεία.

### Μην ανοίξετε το αρχείο "gz"

Μια τυπική εγκατάσταση των Windows 10 δεν θα μπορέσει να ανοίξει το αρχείο από το Disqus. Αυτό είναι εντάξει, γιατί δεν χρειάζεται να το κάνετε με το FastComments. Η υποδομή μας κατανοεί αυτό το συμπιεσμένο αρχείο, οπότε το μόνο που έχετε να κάνετε είναι να ακολουθήσετε τις οδηγίες τους <a href="https://help.disqus.com/en/articles/1717199-importing-exporting" target="_blank">εδώ για να αποκτήσετε το εν λόγω αρχείο</a>.

Αφού το κάνετε αυτό, εάν είστε συνδεδεμένοι μπορείτε να πάτε <a href="https://fastcomments.com/auth/my-account/manage-data/import" target="_blank">εδώ για να εισαγάγετε το αρχείο</a>. Επιλέξτε το Disqus από το αναπτυσσόμενο μενού και ανεβάστε το αρχείο σας.

### Περιμένετε μερικά λεπτά

Η εισαγωγή του FastComments είναι "ασύγχρονη". Αυτό σημαίνει ότι η ανάρτηση του αρχείου και η επεξεργασία του είναι ξεχωριστά βήματα.

Έτσι, ανάλογα με το μέγεθος του αρχείου σας, μπορεί να λάβετε το μήνυμα επιτυχίας αμέσως. Στη σελίδα εισαγωγής στο κάτω μέρος υπάρχει ένας πίνακας - κάθε γραμμή αντιπροσωπεύει μια απόπειρα εισαγωγής.
Μπορείτε να ανανεώσετε αυτή τη σελίδα με ασφάλεια για να δείτε την κατάσταση της εισαγωγής σας και πόσες γραμμές έχουν εισαχθεί έως τώρα.

### Όταν τελειώσει

Θα λάβετε ένα email όταν ολοκληρωθεί η εισαγωγή - είτε είναι επιτυχής είτε όχι. Μπορείτε να κλείσετε τη σελίδα αφού επιτύχει η ανάρτηση του αρχείου και να δείτε την εισαγωγή σας στη σελίδα Εισαγωγών. Θα έχει κατάσταση όπως "Ζήτησε" και όταν ξεκινήσει η κατάσταση θα είναι "Εκτελείται".

Είναι ασφαλές να επαναλαμβάνετε την εισαγωγή όσες φορές απαιτείται - ωστόσο, εάν αποτύχει η εισαγωγή σας <a href="https://fastcomments.com/auth/my-account/help" target="_blank">επικοινωνήστε μαζί μας</a> ώστε να μπορούμε να βοηθήσουμε.

### Αντικαταστήστε τον Κωδικό Disqus με το FastComments

Εάν χρησιμοποιείτε την Γενική Ενσωμάτωσης Ιστοσελίδας με το Disqus, τότε απλώς πρέπει να αφαιρέσετε το μικρό απόσπασμα που σας δίνουν και να το αντικαταστήσετε με <a href="https://fastcomments.com/auth/my-account/get-acct-code" target="_blank">δικό μας</a>.
Μπορείτε ακόμη και να τρέχετε το Disqus και το FastComments στον ίδιο ιστότοπο προσωρινά για να δοκιμάσετε και να ρυθμίσετε την εμφάνιση και την αίσθηση. <a href="https://fastcomments.com/auth/my-account/help" target="_blank">Μπορούμε να σας βοηθήσουμε με αυτό</a>.

Εάν δεν χρησιμοποιείτε την Γενική Ενσωμάτωσης Ιστοσελίδας, τότε οι οδηγίες θα διαφέρουν ανάλογα με την πλατφόρμα σας - ξανά <a href="https://fastcomments.com/auth/my-account/help" target="_blank">επικοινωνήστε μαζί μας</a>.

### Μεταφορά URL ταυτόχρονα

Τα σχόλια αυτά καθ’ αυτά συνδέονται με τα πεδία "σύνδεσμος" στην εξαγωγή του Disqus, οπότε εφόσον οι διευθύνσεις URL σας δεν αλλάξουν, η αλλαγή είναι εύκολη. Εάν θέλετε να μεταφέρετε διευθύνσεις URL και να διατηρήσετε τα σχόλιά σας, τότε <a href="https://fastcomments.com/auth/my-account/help" target="_blank">επικοινωνήστε μαζί μας</a> στέλνοντάς μας τις παλιές και νέες διευθύνσεις URL. Η σελίδα βοήθειας επιτρέπει την ανέβασμα μικρών αρχείων κειμένου, οπότε αν μεταφέρετε περισσότερες από μερικές διευθύνσεις URL, δημιουργήστε ένα φύλλο Excel.

### Ανακεφαλαίωση

1. Εξαγάγετε τα δεδομένα σας
2. Εισάγετε στο FastComments
3. Αντικαταστήστε το απόσπασμα JavaScript του Disqus με το <a href="https://fastcomments.com/auth/my-account/get-acct-code" target="_blank">FastComments's</a>

## Γιατί Το Disqus Δεν Μου Στέλνει Το Αρχείο Μου;
Για μεγαλύτερους ιστότοπους, το Disqus ενδέχεται να μην μπορεί να δημιουργήσει το εξαγόμενο αρχείο λόγω τεχνικών περιορισμών από την πλευρά τους. Έχουμε τη δυνατότητα να σκραπάρουμε τον υπάρχοντα ιστότοπό σας και να αντλήσουμε τα σχόλια από αυτόν, ωστόσο αυτό μπορεί να είναι
αρκετά προσαρμοσμένο και λόγω του χρόνου που απαιτεί θα χρειαστεί να έχετε το Πακέτο Υποστήριξης. <a href="https://fastcomments.com/auth/my-account/help" target="_blank">Μπορείτε να ζητήσετε βοήθεια με αυτό εδώ</a>.
