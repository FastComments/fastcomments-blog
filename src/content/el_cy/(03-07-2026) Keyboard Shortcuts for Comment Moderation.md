# [postlink]Συντομεύσεις Πληκτρολογίου για τον Έλεγχο Σχολίων[/postlink]

{{#unless isPost}}
Οι χρήστες με προχωρημένες γνώσεις μπορούν τώρα να ελέγχουν τα σχόλια εξ ολοκλήρου από το πληκτρολόγιο με ένα πλήρες σύνολο συντομεύσεων για πλοήγηση, ενέργειες και άλλα.
{{/unless}}

{{#isPost}}

### Τι Νέο

Η σελίδα Ελέγχου Σχολίων υποστηρίζει τώρα εκτενείς συντομεύσεις πληκτρολογίου. Πλοηγηθείτε μεταξύ των σχολίων, εγκρίνετε, διαγράφετε, απαγορεύετε,
σημειώστε ως ανεπιθύμητο, αλλάξτε πλαίσιο, δείτε αρχεία καταγραφής και άλλα - όλα χωρίς να αγγίξετε το ποντίκι.

Αυτό είναι σημαντικό για τις ομάδες που ελέγχουν μεγάλους όγκους σχολίων. Κάθε ενέργεια που προηγουμένως απαιτούσε μετακίνηση του ποντικιού και κλικ σε κουμπί
είναι τώρα μόλις ένα πάτημα πλήκτρου μακριά.

### Παράδειγμα Ροής

Ας πούμε ότι έχετε μια ουρά σχολίων προς ανασκόπηση. Πατήστε **j** για να εστιάσετε στο πρώτο σχόλιο. Πατήστε **c** για να επεκτείνετε
το πλαίσιο του και να δείτε τη γύρω συζήτηση. Φαίνεται καλή, οπότε πατήστε **c** ξανά για να καταρρεύσετε το πλαίσιο, μετά **a** για να το εγκρίνετε.
Πατήστε **j** για να μεταβείτε στο επόμενο σχόλιο και επαναλάβετε. Αν κάτι φαίνεται περίεργο, πατήστε **s** για να το σημειώσετε ως ανεπιθύμητο ή **d** για
να το διαγράψετε. Μπορείτε να δουλέψετε σε ολόκληρη τη σελίδα σχολίων χωρίς να φτάσετε ποτέ στο ποντίκι.

### Πλοήγηση

Πατήστε **j** για να μετακινήσετε την εστίαση στο επόμενο σχόλιο και **k** για να μεταβείτε στο προηγούμενο. Το εστιασμένο σχόλιο είναι επισημασμένο
με μπλε περίγραμμα για να διακρίνεται από τα επιλεγμένα (επιταγμένα) σχόλια.

Πατήστε **x** ή **Enter** για να ενεργοποιήσετε την επιλογή στο εστιασμένο σχόλιο. Κρατήστε πατημένο το **Shift** και πατήστε **j** ή **k** για να επιλέξετε
το τρέχον σχόλιο και να μετακινήσετε την εστίαση με μία κίνηση - χρήσιμο για γρήγορη επιλογή σειράς σχολίων.

### Ενέργειες

Με ένα σχόλιο εστιασμένο, οι συντομεύσεις ενός πλήκτρου ενεργοποιούν τις ίδιες ενέργειες με τα κουμπιά της γραμμής εργαλείων:

- **a** - Ενεργοποίηση έγκρισης / ακύρωσης έγκρισης
- **s** - Ενεργοποίηση ανεπιθύμητου / μη ανεπιθύμητου
- **d** - Διαγραφή
- **u** - Ανάκτηση διαγραφής
- **r** - Ενεργοποίηση ελέγχου / μη ελέγχου
- **e** - Επεξεργασία
- **b** - Απαγόρευση / διαχείριση απαγόρευσης
- **p** - Ενεργοποίηση καρφώματος / αποκαρφώματος
- **f** - Αποσημείωση
- **i** - Δείτε όλα από IP
- **t** - Ορισμός παράγοντα εμπιστοσύνης
- **m** - Διαχείριση εμβλημάτων
- **l** - Δείτε τα αρχεία καταγραφής
- **c** - Εναλλαγή πλαισίου

### Παγκόσμιες Συντομεύσεις

Λίγες συντομεύσεις λειτουργούν ανεξάρτητα από το αν ένα σχόλιο είναι εστιασμένο:

- **Ctrl+a** - Επιλογή όλων των σχολίων
- **Ctrl+z** - Ανάκτηση της τελευταίας ενέργειας
- **/** - Εστίαση στη γραμμή αναζήτησης
- **[** και **]** - Πλοήγηση στην προηγούμενη ή επόμενη σελίδα

### Βοήθεια και Οδηγοί

Πατήστε **?** ανά πάσα στιγμή για να ανοίξετε ένα διάλογο βοήθειας που παραθέτει κάθε διαθέσιμη συντόμευση.

Πατήστε **g** για να ενεργοποιήσετε τη λειτουργία "οδηγών", η οποία τοποθετεί μικρά εμβλήματα στα κουμπιά ενέργειας του εστιασμένου σχολίου που δείχνουν
το αντίστοιχο πλήκτρο συντόμευσης. Αυτό είναι χρήσιμο κατά την εκμάθηση των συντομεύσεων.

Όλες οι συντομεύσεις καταστέλλονται όταν πληκτρολογείτε στη γραμμή αναζήτησης ή σε οποιοδήποτε άλλο πεδίο εισόδου, και όταν είναι ανοιχτός ένας διάλογος.
Πατήστε **Escape** για να αποστραγγίσετε μια εστιασμένη είσοδο και να επιστρέψετε στη λειτουργία συντομεύσεων.

### Τεκμηρίωση

Για πλήρη αναφορά, δείτε τον <a href="https://docs.fastcomments.com/guide-moderation.html" target="_blank">Οδηγό Ελέγχου</a>.

### Στο Συμπέρασμα

Ελπίζουμε οι συντομεύσεις πληκτρολογίου να κάνουν τον έλεγχο σε υψηλό όγκο γρηγορότερο και πιο ευχάριστο. Οι συντομεύσεις ακολουθούν τις συμβάσεις
γνωστές στους χρήστες εργαλείων όπως το Gmail και το GitHub, οπότε θα πρέπει να αισθάνονται φυσικές αμέσως.

Πείτε μας παρακάτω αν έχετε οποιαδήποτε ερώτηση ή σχόλιο!

Ευχαριστούμε!
