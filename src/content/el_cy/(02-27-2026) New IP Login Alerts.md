### Τι είναι Νέο

Το FastComments τώρα στέλνει αυτόματα ένα email ειδοποίησης ασφαλείας όταν ανιχνεύουμε σύνδεση στον λογαριασμό σας από διεύθυνση IP την οποία δεν έχουμε δει πριν. Το email περιλαμβάνει:

- Την **περίπου τοποθεσία** (πόλη και χώρα) της σύνδεσης.
- Τη **διεύθυνση IP** που χρησιμοποιήθηκε.
- Τη **χρονική στιγμή** της σύνδεσης.
- Έναν άμεσο σύνδεσμο για **να αλλάξετε τον κωδικό σας** αν η σύνδεση δεν έγινε από εσάς.

Αυτό ισχύει για όλες τις μεθόδους σύνδεσης: σύνδεση με βάση τον κωδικό, μαγικοί σύνδεσμοι και ροές δύο παραγόντων.

### Ιδιωτικότητα

Δεν αποθηκεύουμε τη διεύθυνση IP σας σε πρώιμο στάδιο. Αποθηκεύουμε μόνο μια παραποιημένη μορφή για σκοπούς σύγκρισης ασφαλείας. Η περίπου τοποθεσία προσδιορίζεται τη στιγμή της σύνδεσης και δεν αποθηκεύεται.

**Η διεύθυνση IP σας δεν κοινοποιείται σε τρίτα μέρη** για να προσδιορίσει την τοποθεσία. Διατηρούμε τη δική μας υπηρεσία που εξάγει την περίπου τοποθεσία από μια (βδομαδιαία ενημέρωση) αντίγραφο της βάσης δεδομένων MaxMind.

### Όταν Αποστέλλονται οι Ειδοποιήσεις

Μια ειδοποίηση αποστέλλεται όταν **όλα** τα παρακάτω είναι αληθή:

- Η σύνδεση ήταν επιτυχής.
- Δεν είναι η πρώτη σύνδεση του χρήστη.
- Η διεύθυνση IP δεν έχει παρατηρηθεί σε προηγούμενη επιτυχή σύνδεση για αυτόν τον λογαριασμό.

Αυτό σημαίνει ότι δεν θα λάβετε ειδοποίηση στην πρώτη σας σύνδεση, και δεν θα λάβετε επαναλαμβανόμενες ειδοποιήσεις για την ίδια διεύθυνση IP.

### ΣυμπConclusion

Αυτό είναι ένα ακόμα βήμα προς τη διατήρηση της ασφάλειας του λογαριασμού σας. Αν λάβετε μια ειδοποίηση που δεν αναγνωρίζετε, σας προτείνουμε να αλλάξετε αμέσως τον κωδικό σας.

Ενημερώστε μας παρακάτω αν έχετε οποιαδήποτε σχόλια!
