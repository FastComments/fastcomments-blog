# [postlink]Η Αποσφαλμάτωση της Εγκατάστασης FastComments σας έγινε πιο εύκολη[/postlink]

{{#unless isPost}}

### <i class="circle">!</i> Αυτό το Άρθρο Περιέχει Τεχνική Ορολογία

#### Τι Νέο

Στο <a href="https://fastcomments.com">FastComments</a> μόλις κυκλοφορήσαμε μια επέκταση για τον περιηγητή Google Chrome. Αυτή η επέκταση θα σας βοηθήσει στην κατανόηση του πώς
λαμβάνει τις ρυθμίσεις το widget σχολίων καθώς και γιατί μπορεί να μην εμφανίζεται.

Μπορείτε να βρείτε την επέκταση εδώ: <a href="https://chrome.google.com/webstore/detail/fastcomments-debugger/cadggdemhfkjjghkdbfhonoccnplffjj" target="_blank">https://chrome.google.com/webstore/detail/fastcomments-debugger/cadggdemhfkjjghkdbfhonoccnplffjj</a>

#### Πώς να τη Χρησιμοποιήσετε

Αν διαπιστώσετε ότι το widget σχολίων δεν φορτώνεται, ή κάποιες ρυθμίσεις που έχετε ορίσει δεν εμφανίζονται, δοκιμάστε να εγκαταστήσετε <a href="https://chrome.google.com/webstore/detail/fastcomments-debugger/cadggdemhfkjjghkdbfhonoccnplffjj">την επέκταση</a>
και να την ανοίξετε στη σελίδα που σας ενδιαφέρει. Για να δείτε το εικονίδιο στην εργαλειοθήκη σας, ίσως χρειαστεί να κάνετε κλικ στο κομμάτι πάζλ στην επάνω δεξιά γωνία για να το καρφιτσώσετε όπως φαίνεται:

<div class="text-center">
    <img src="images/fc-extension-pin.png" alt="Πώς να Καρφιτσώσετε την Επέκταση" title="Πώς να Καρφιτσώσετε την Επέκταση" class="lozad" />
</div>

Η επέκταση μπορεί να βοηθήσει στην αποσφαλμάτωση δύο κατηγοριών σφαλμάτων. Η πρώτη, εάν το widget δεν περιλαμβάνεται καθόλου στη σελίδα ή εάν υπάρχει σοβαρό λάθος στο σενάριο.

Η δεύτερη είναι εάν κάποια ρύθμιση δεν φαίνεται να έχει αποτέλεσμα.

Για να αποσφαλματώσετε και τις δύο περιπτώσεις, ξεκινάτε απλά κάνοντας κλικ στο εικονίδιο.

Εάν το εργαλείο δεν βρει το widget σχολίων στη σελίδα, θα λάβετε ένα μεγάλο κόκκινο σφάλμα:

<div class="text-center">
    <img src="images/fc-extension-embed-js-not-found.png" alt="Παράδειγμα Embed.js δεν βρέθηκε" title="Παράδειγμα Embed.js δεν βρέθηκε" class="lozad" />
</div>

Εάν έχουν βρεθεί μία ή περισσότερες περιπτώσεις του widget, θα δείτε τις ρυθμίσεις τους να εμφανίζονται ξεχωριστά:

<div class="text-center">
    <img src="images/fc-extension-multiple-instances.png" alt="Παράδειγμα Πολλαπλής Ρύθμισης" title="Παράδειγμα Πολλαπλής Ρύθμισης" class="lozad" />
</div>

Επιπλέον, το FastComments Debugger θα εμφανίζει επίσης προειδοποιήσεις για κοινά προβλήματα - όπως το widget που δεν εμφανίζεται σε λειτουργία μόνο για ανάγνωση αν δεν υπάρχουν διαθέσιμα σχόλια.

#### Ρυθμίσεις "Τελικής" Περίπτωσης;

Μπορείτε να δείτε ότι ο αποσφαλματωτής δείχνει δύο σύνολα ρυθμίσεων - ένα με την ένδειξη "Περασμένο στο Widget" και ένα με την ένδειξη "Τελικό".

Η ρύθμιση "Περασμένο στο Widget" είναι, όπως μπορεί να μαντέψετε, η ρύθμιση που περνάτε στο widget. Ορισμένες τιμές μεταβιβάζονται σιωπηλά από το embed.js, όπως η Διεύθυνση URL ή ο Τίτλος Σελίδας - και αυτές περιλαμβάνονται.

Η ρύθμιση "Τελικό" είναι όλα όσα χρησιμοποιεί το widget για να σχεδιάσει τον εαυτό του. Αυτό μπορεί να σημαίνει αυτοσυμπληρούμενες τιμές (όπως το id της περίπτωσης), ή <b>τιμές από τη σελίδα διαχείρισης προσαρμογής του widget</b>.

#### Πώς Λειτουργεί

Το widget σχολίων αποθηκεύει τη ρύθμισή του σε ένα μέρος όπου ο αποσφαλματωτής μπορεί να την βρει, και έπειτα ο αποσφαλματωτής σαρώσει την τρέχουσα σελίδα στην οποία βρίσκεστε για περιπτώσεις του widget και τις φορτώνει στη μνήμη.

#### Ποιες Πληροφορίες Συλλέγει Αυτή η Επέκταση;

Το FastComments Debug Tool δεν επικοινωνεί με κανέναν από τους διακομιστές FastComments. Στην πραγματικότητα, το widget δεν ελέγχει καν το περιεχόμενο της σελίδας στην οποία βρίσκεστε
εκτός αν το ανοίξετε.

#### Στο Συμπέρασμα

Ελπίζουμε ότι παρέχοντας περισσότερα εργαλεία όπως αυτό καθιστά την εγκατάσταση του FastComments πιο εύκολη για κάθε τύπο πελάτη που έχουμε. Ενημερώστε μας στην ενότητα σχολίων παρακάτω
πως αυτό δούλεψε για εσάς.

Γνωρίζουμε ότι σε ορισμένες περιπτώσεις μπορεί να χρειάζεστε ακόμα μια χείρα βοηθείας - και γι' αυτό υπάρχει ακόμα η <a href="https://fastcomments.com/auth/my-account/help" target="_blank">σελίδα υποστήριξης πελατών</a>.

Εις υγειάν!
