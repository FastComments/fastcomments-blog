# [postlink]Εύκολη Υποστήριξη Κώδικα Με FastComments[/postlink]

Κάποιοι από τους πελάτες μας έχουν ιστότοπους και ιστολόγια σχετιζόμενα με την ανάπτυξη λογισμικού. Σε αυτές τις κοινότητες είναι σύνηθες να μοιράζονται κομμάτια κώδικα. Ακολουθεί πώς το FastComments διευκολύνει αυτή τη διαδικασία.

{{#isPost}}

Το FastComments επιτρέπει την κοινή χρήση κώδικα απλά επικολλώντας τον στο πεδίο σχολίων. Μπορεί να αναμιγνύεται με άλλο κείμενο και εικόνες, και δεν απαιτεί να προσθέσετε χειροκίνητα οποιαδήποτε "tags" κώδικα 
για να προσθέσετε τη μορφοποίηση. Απλά επικολλήστε!

Ακολουθεί μια γρήγορη επίδειξη που δείχνει τη λειτουργία του.

<video src="images/fc-code-snippet-demo.mp4" autoplay controls loop=true alt="Δημοσίευση Κώδικα Επίδειξη" title="Δημοσίευση Κώδικα Επίδειξη"></video>

### Επιπτώσεις Απόδοσης

Η προσθήκη αυτής της δυνατότητας έχει προσθέσει μόνο ~150 byte στο widget πλευράς πελάτη καθώς οι περισσότερες από τις λειτουργίες χειρίζονται από την πλευρά του διακομιστή. Στην πραγματικότητα, αν το συνδυάσετε με τις πρόσφατες δυνατότητες Markdown 
που προστέθηκαν, το μέγεθος του widget στην πραγματικότητα έχει μειωθεί με αυτές τις πρόσφατες δυνατότητες!

### Πράγματα που Πρέπει να Σημειώσετε

Αν έχετε μια κοινότητα που μοιράζεται κώδικα σε μεγάλο βαθμό, θα πρέπει να εξετάσετε το ενδεχόμενο να αυξήσετε το μέγιστο μέγεθος σχολίου για να διευκολύνετε την κοινή χρήση κομματιών κώδικα. Αυτό γίνεται μέσω της καρτέλας "Προσαρμογή" 
του πίνακα διαχείρισης.

### Ανίχνευση Κώδικα

Για να κρατήσουμε το μέγεθος του widget στην πλευρά του πελάτη χαμηλό, θα ανιχνεύουμε αυτόματα μόνο τον επικολλημένο κώδικα για γλώσσες όπως C και Python (Java, JavaScript, CSS, C++, για παράδειγμα) που είτε χρησιμοποιούν "{" είτε κενά για ροή ελέγχου.
Αν χρησιμοποιείτε συχνά μια γλώσσα που δεν ανιχνεύεται αυτόματα, επικοινωνήστε μαζί μας.

Για να εισάγετε χειροκίνητα ένα μπλοκ κώδικα, γράψτε ένα σχόλιο όπως: ```<code>(defun someLispCode(1, 2, 3))</code>```. Μπορούμε να ανιχνεύσουμε αυτόματα περίπου 150 γλώσσες. 

### Μεταφορά Υπαρχόντων Σχολίων

Αν θέλετε να μεταφέρετε σε FastComments και έχετε πολλά σχόλια που περιέχουν κομμάτια κώδικα τα οποία θέλετε να μορφοποιηθούν σωστά, <a href="https://fastcomments.com/auth/my-account/help" target="_blank">μπορούμε να βοηθήσουμε</a>.
