# [postlink]Προσαρμόστε τη Μαύρη Λίστα Σχολίων Σας Με το FastComments[/postlink]

Για κάποιους, η προεπιλεγμένη μαύρη λίστα λέξεων είναι πολύ περιοριστική ή πολύ ανοιχτή.

{{#isPost}}

Η προεπιλεγμένη μαύρη λίστα λέξεων μπορεί τώρα να επεξεργαστεί ή να αλλάξει εντελώς. Απλά πηγαίνετε στο Moderate Comments -> Edit Moderation Settings -> Edit Blacklist.

<img 
    src="images/fc-customize-blacklist.png"
    alt="Προσαρμογή Μαύρης Λίστας"
    title="Προσαρμογή Μαύρης Λίστας"
    class='lozad' />

Οι λέξεις σε αυτή τη λίστα θα πρέπει να είναι διαχωρισμένες με κόμματα - αλλά μπορείτε να χρησιμοποιήσετε κενά ή νέες γραμμές αν αυτό σας διευκολύνει στην ανάγνωση.

Όποιες αλλαγές γίνονται μπορούν να αναμένονται να πάρουν το πολύ πέντε λεπτά για να ενημερωθούν.

{{/isPost}}
