# [postlink]Ενημέρωση Συγκράτησης - Σημαιάκια στη Σελίδα Συγκράτησης[/postlink]

{{#unless isPost}}
Οι διαχειριστές μπορούν τώρα να απονέμουν ή να αφαιρούν σημαίακια από τους χρήστες απευθείας από τη σελίδα συγκράτησης σχολίων, χωρίς να χρειάζεται να πάνε
στον θεατή νήματος σχολίων.
{{/unless}}

{{#isPost}}

### Τι Νέο

Παλαιότερα, όταν οι διαχειριστές ήθελαν να απονείμουν σημαίακια, πήγαιναν στον Θεατή Νήματος Σχολίων, ή απονέμουν τα σημαίακια απευθείας από το εργαλείο σχολίων που είναι εγκατεστημένο στον ιστότοπό σας ή στην εφαρμογή σας.

Ωστόσο, πολλοί διαχειριστές απλώς θέλουν να περνούν το χρόνο τους στη σελίδα Συγκράτησης Σχολίων.

Για να διευκολύνουμε τα πράγματα για όλους, τα σημαίακια των χρηστών εμφανίζονται τώρα σε κάθε σχόλιο στη σελίδα Συγκράτησης Σχολίων.

Επιπλέον, υπάρχουν τώρα επιλογές μενού για την απονομή και την αφαίρεση σημαίakéων για έναν χρήστη, απευθείας σε κάθε μενού σχολίου.

Αυτές οι ενέργειες είναι ζωντανές, ακριβώς όπως από το εργαλείο σχολίων ή τον θεατή νήματος, και όλα τα σχόλια για τον χρήστη στη λίστα συγκράτησης
θα ενημερώνονται όταν προστεθούν ή αφαιρεθούν σημαίακια.

### Μέγιστος Αριθμός Σημαιάκων που Εμφανίζονται

Εμφανίζονται έως `πέντε` σημαίακια. Μετά από αυτό, εμφανίζεται ένας δείκτης ότι ο χρήστης έχει περισσότερα από πέντε σημαίακια.

### Πώς να το Αποκτήσετε

Αυτή η δυνατότητα έχει κυκλοφορήσει σε όλους τους πελάτες!

### Στο Συμπέρασμα

Όπως όλες οι μεγάλες εκδόσεις, είμαστε χαρούμενοι που μπορέσαμε να αφιερώσουμε χρόνο για να βελτιστοποιήσουμε, να δοκιμάσουμε και να κυκλοφορήσουμε σωστά αυτή τη δυνατότητα. Ενημερώστε μας
παρακάτω αν ανακαλύψετε οποιαδήποτε προβλήματα.

Καλή συνέχεια!
