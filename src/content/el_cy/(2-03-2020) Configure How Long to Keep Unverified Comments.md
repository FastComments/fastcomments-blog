# [postlink]Ρυθμίστε Πόσο Καιρό να Διατηρείτε τα Μη Επαληθευμένα Σχόλια[/postlink]

Από προεπιλογή η FastComments θα διατηρεί τα μη επαληθευμένα σχόλιά σας για τρεις ημέρες.

### Τι σημαίνει "μη επαληθευμένο";

Όταν ένας χρήστης σχολιάζει και αφήνει το email του, η FastComments θα του στείλει ένα email ζητώντας του να επαληθεύσει το σχόλιό του.
Αυτό βοηθά στον αγώνα κατά του spam και αναδεικνύει τα σχόλια που οι συγγραφείς τους πραγματικά θέλουν να εμφανίζονται.

Αν ενεργοποιήσετε <a href="https://blog.fastcomments.com/(2-01-2020)-how-to-allow-anonymous-comments-with-fastcomments.html" target="_blank">Ανώνυμο Σχολιασμό</a>
δεν θα χρειαστεί καν να αφήσουν το email τους!

Αλλά μπορείτε τώρα να ρυθμίσετε πόσο καιρό θα διατηρούνται αυτά τα σχόλια - μπορεί και να μην θέλετε καν να τα αφαιρέσετε αυτόματα! Αυτό είναι πλήρως ρυθμιζόμενο μέσω των Ρυθμίσεων Μυθιστορημάτων Σχόλιων -> Επεξεργασία Ρυθμίσεων Μυθιστορημάτων

{{#isPost}}
