# [postlink]Σχόλια Για Το Jamstack[/postlink]

{{#unless isPost}}
Το παλιό γίνεται καινούργιο. Η προκατασκευή και η εξυπηρέτηση ιστοσελίδων από ένα CDN ονομάζεται πλέον "JamStack" - αλλά δεν χρειάζεται να είναι εντελώς στατικές!
{{/unless}}

{{#isPost}}

### <i class="circle">!</i> Αυτό το Άρθρο Περιέχει Τεχνική Ορολογία

<a href="https://jamstack.org/" target="_blank">Το Jamstack</a> είναι ένας τρόπος για να γράψεις μια ιστοσελίδα που δεν εξαρτάται από έναν παραδοσιακό web server. Αντίθετα, εξυπηρετούνται από servers που ονομάζουμε "CDN" για την καλύτερη εμπειρία του χρήστη (φορτώνουν πολύ γρήγορα). Η απόδοση είναι μια τεράστια προτεραιότητα για τις ιστοσελίδες που γράφονται με το Jamstack - ακριβώς όπως η FastComments.

#### Γιατί Η FastComments Λειτουργεί Με Το Jamstack

Η FastComments απαιτεί μόνο ένα μικρό (< 11kb επί του παρόντος) κομμάτι κώδικα στο κάτω μέρος της στατικής σελίδας στην οποία θέλεις να προσθέσεις σχόλια. Έτσι, η εγκατάσταση της FastComments σε μια τυπική ιστοσελίδα Jamstack διαρκεί μόνο λίγα δευτερόλεπτα, όπως σε οποιαδήποτε στατική ιστοσελίδα.

#### Πιθανά Ζητήματα

Κατανοούμε ότι οι ιστοσελίδες Jamstack δεν ξεκινούν πάντα ως κάτι στατικό. Μπορεί να έχουν κατασκευαστεί με Preact, React ή Vue, για παράδειγμα. Γραμμένο σε vanillа JS, κατασκευάσαμε το widget της FastComments για μέγιστη διαλειτουργικότητα ώστε να λειτουργεί με όλα τα μεγάλα frameworks. Ωστόσο, αν αντιμετωπίσεις προβλήματα με τη FastComments και το framework της επιλογής σου, τότε
<a href="https://fastcomments.com/auth/my-account/help" target="_blank">επικοινώνησε μαζί μας</a>.

#### Στην Κατακλείδα

Ελπίζουμε ότι θα διαπιστώσεις ότι η FastComments επιτρέπει στους χρήστες σου να αλληλεπιδρούν με την ιστοσελίδα σου, να παρέχουν ανατροφοδότηση και να αφήνουν σχόλια με τρόπους που κανονικά δεν θα μπορούσαν σε μια στατική ιστοσελίδα. Για παράδειγμα, αυτή η σελίδα είναι προκατασκευασμένη όπως μια παραδοσιακή ιστοσελίδα Jamstack, και μπορείς να δεις την FastComments να λειτουργεί ευτυχώς στο κάτω μέρος.

Ευχαριστώ!
