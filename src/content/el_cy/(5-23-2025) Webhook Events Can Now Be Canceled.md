# [postlink]Οι Εκδηλώσεις Webhook Μπορούν Τώρα να Ακυρωθούν[/postlink]

{{#unless isPost}}
Οι εκδηλώσεις στην ουρά επεξεργασίας webhook μπορούν να ακυρωθούν ώστε να μην επιχειρούν ξανά.
{{/unless}}

{{#isPost}}

### Τι Είναι Νέο

Προηγουμένως, η ακύρωση εκδηλώσεων webhook γινόταν μέσω υποστήριξης. Τώρα μπορούν να ακυρωθούν αυτοεξυπηρετούμενα πηγαίνοντας στη
[σελίδα αρχείων καταγραφής](https://fastcomments.com/auth/my-account/manage-data/webhooks/logs) και πατώντας το `x` δίπλα στην εκδήλωση.

Επιπλέον, η διεπαφή καταγραφών webhook έχει γίνει πολύ πιο γρήγορη.

### Συμπέρασμα

Ελπίζουμε να βρήκατε αυτή την ενημέρωση χρήσιμη και να συνεχίσετε να απολαμβάνετε τη συνεργασία σας με το FastComments.

{{/isPost}}
