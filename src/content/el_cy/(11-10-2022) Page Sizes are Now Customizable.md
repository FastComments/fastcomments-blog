# [postlink]Οι Σελίδες Σχολίων Είναι Τώρα Προσαρμόσιμες[/postlink]

{{#unless isPost}}
Το FastComments είχε μια σταθερή σελίδα με 30 σχόλια από την εκκίνηση. Αυτό τώρα είναι προσαρμόσιμο!
{{/unless}}

{{#isPost}}

### Τι Νέο

Τα νήματα σχολίων σας μπορούν τώρα να έχουν διαφορετικά μεγέθη σελίδας, και αυτό μπορεί τώρα να προσαρμοστεί παγκοσμίως σε ολόκληρο το λογαριασμό σας, ή σε επίπεδο ιστότοπου, ή ακόμη και σελίδας. Αυτό μπορεί να είναι επιθυμητό αν θέλετε ορισμένες σελίδες να έχουν λιγότερα ή περισσότερα σχόλια που να εμφανίζονται πριν ο χρήστης χρειαστεί να αλλάξει σελίδα.

Αυτό επηρεάζει επίσης την ατελείωτη κύλιση, και οποιαδήποτε πλαίσια ή ενσωματώσεις που αποδίδουν σχόλια από το FastComments.

Τα μεγέθη σελίδων μπορεί να κυμαίνονται από `10` έως `200`. Η προεπιλογή παραμένει στα `30`.

### Πώς να το Αποκτήσετε

Απλώς μεταβείτε στο αξιόπιστο [Widget Customization UI](https://fastcomments.com/auth/my-account/customize-widget) και καθορίστε το μέγεθος της σελίδας σας.

Σημειώστε ότι το FastComments προ-υπολογίζει τα σχόλια που εμφανίζονται σε κάθε σελίδα, οπότε η αλλαγή αυτού του παραμέτρου θα πυροδοτήσει το σύστημά μας να ξαναϋπολογίσει όλα τα νήματα σχολίων σας. Αυτό δεν θα προκαλέσει κανέναν χρόνο ανενεργού, αλλά μπορεί να πάρει μερικά λεπτά για να εμφανιστεί η αλλαγή.

### Τεκμηρίωση

Η ενημερωμένη τεκμηρίωση σχετικά με τα μεγέθη σελίδων μπορεί να βρεθεί στο [docs.fastcomments.com](https://docs.fastcomments.com/guide-customizations-and-configuration.html#page-size).

### Καταστροφικές Αλλαγές

Στο πλαίσιο αυτής της αλλαγής, δύο μεταφράσεις που καθορίστηκαν στο widget σχολίων έχουν μετονομαστεί. Τα `PREV_30` και `NEXT_30` είναι τώρα `PREV` και `NEXT`, αντίστοιχα, και περιέχουν μια μεταβλητή `[count]`.

Αν έχετε καθορίσει μεταφράσεις για τα κουμπιά πλοήγησης μέσω του Widget Customization UI, τότε έχουμε ήδη μεταφέρει αυτές για εσάς. Αν περνάτε τις μεταφράσεις μέσω κάποιου είδους ενσωμάτωσης μέσω κώδικα, θα χρειαστεί να ενημερώσετε αυτές:

- `PREV_30` -> `PREV`
- `NEXT_30` -> `NEXT`

### Στην Κατακλείδα

Όπως σε όλες τις σημαντικές εκδόσεις, είμαστε χαρούμενοι που μπορέσαμε να αφιερώσουμε χρόνο για να βελτιστοποιήσουμε, να δοκιμάσουμε και να κυκλοφορήσουμε σωστά αυτή τη δυνατότητα. Ενημερώστε μας παρακάτω αν ανακαλύψετε τυχόν προβλήματα.

Στην υγειά σας!
