# [postlink]Ενημέρωση Διαχείρισης Ενοικιαστών[/postlink]

{{#unless isPost}}
Το FastComments έχει βελτιώσει την αλληλεπίδραση για τη διαχείριση των ενοικιαστών σας!
{{/unless}}

{{#isPost}}

### Τι Νέο

Από τις 11 Δεκεμβρίου 2023, το FastComments διαθέτει τη δυνατότητα αλλαγής ενοικιαστών χωρίς να γίνεται μίμηση άλλων χρηστών ενοικιαστών.

### Στοχευόμενο Κοινό

Αυτή η ενημέρωση δεν αφορά σχολιαστές ή διαχειριστές ιστοσελίδων που διαχειρίζονται μία μόνο ιστοσελίδα. Αυτό επωφελεί τους διαχειριστές που διαχειρίζονται πολλές ιστοσελίδες ή ενοικιαστές μόνο.

### Επιπτώσεις για τους Διαχειριστές

Προηγουμένως, η δημιουργία ενοικιαστών και η αλλαγή χρηστών ήταν δύσκολη υπόθεση. Ορισμένα κοινά σημεία προβλημάτων ήταν:

1. Έπρεπε να δημιουργήσετε έναν χρήστη κατά τη δημιουργία του ενοικιαστή σας, καθώς αυτός ο χρήστης "κατείχε" τον ενοικιαστή.
2. Αυτό σήμαινε ότι έπρεπε να χρησιμοποιείτε μια νέα διεύθυνση ηλεκτρονικού ταχυδρομείου για κάθε ιστοσελίδα, ακόμα και αν θέλατε να χρησιμοποιήσετε μία διεύθυνση/λογαριασμό για πολλούς υπο-ενοικιαστές.
3. Η αλλαγή ενοικιαστών απαιτούσε μίμηση, γεγονός που σημαίνει ότι οι χρήστες σας έπρεπε να διπλασιαστούν σε όλους τους ενοικιαστές, με τις ίδιες άδειες, κ.λπ.

Τώρα:

1. Δεν απαιτείται η δημιουργία χρήστη κατά τη δημιουργία ενός ενοικιαστή.
2. Οποιοσδήποτε χρήστης Super Admin με τις κατάλληλες άδειες έχει τις ίδιες άδειες σε όλους τους υπο-ενοικιαστές.
3. Υπάρχει ένα κουμπί `Switch` στη λίστα των ενοικιαστών που διατηρεί τον τρέχοντα χρήστη και τις άδειές σας, αλλά αλλάζει τον ενοικιαστή που έχετε συνδεθεί.
4. Αν είστε διαχειριστής ενός γονικού ενοικιαστή, έχετε άδειες συντονιστή στους υπο-ενοικιαστές όταν χρησιμοποιείτε το widget σχολίων.

Τα συμβάντα καταγράφονται ακόμη στο Audit Log για τον ενοικιαστή που έχετε αλλάξει.

### Μίμηση

Μπορείτε να μιμηθείτε χρήστες. Αλλάξτε σε έναν ενοικιαστή και πηγαίνετε στη σελίδα `Users`.

### Για Προγραμματιστές & Παράξενα

Κατά τη χρήση του API `POST /tenants`, το πεδίο `email` είναι πλέον προαιρετικό. Η τεκμηρίωση έχει ενημερωθεί.

### Μελλοντικές Ενημερώσεις

Στο μέλλον θα υπάρχουν περισσότερες διοικητικές ενέργειες στη λίστα των ενοικιαστών, όπως η διαγραφή ενοικιαστών και τα λοιπά.

### Συμπέρασμα

Όπως όλες οι μεγάλες εκδόσεις, είμαστε χαρούμενοι που μπορέσαμε να αφιερώσουμε χρόνο για να βελτιστοποιήσουμε, να δοκιμάσουμε και να κυκλοφορήσουμε σωστά αυτή τη δυνατότητα. Ενημερώστε μας
παρακάτω αν ανακαλύψετε τυχόν προβλήματα.

Καλές γιορτές!
