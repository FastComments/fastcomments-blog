# [postlink]Προσαρμογή του Κειμένου του Widget[/postlink]

{{#unless isPost}}
Η προσαρμογή του κειμένου του widget είναι μια σύνηθες ερώτηση υποστήριξης. Τώρα είναι αυτοεξυπηρετούμενη.
{{/unless}}

{{#isPost}}

#### Τι είναι Νέο

Προτού, η μόνη μέθοδος για να προσαρμόσετε το κείμενο στο widget σχολίων ήταν είτε να κάνετε αλλαγές στον κώδικα κατά την ενσωμάτωση του widget για να περάσετε τις μεταφράσεις, είτε να υποβάλετε ένα αίτημα υποστήριξης πελατών.

#### Πώς Λειτουργεί

Όπως και οι άλλες προσαρμογές widget, αυτό τώρα διαχειρίζεται στη <a href="https://fastcomments.com/auth/my-account/customize-widget" target="_blank">σελίδα Προσαρμογών</a>.

Εισάγετε το domain για το οποίο θέλετε να προσαρμόσετε το κείμενο, και είτε μια συγκεκριμένη URL ή * για όλες τις σελίδες.

Κοντά στη μέση της σελίδας, κάντε κλικ στο "Εμφάνιση Προσαρμοσμένου Κειμένου" και στη συνέχεια "Επιλογή Μετάφρασης Υπέρβασης".

Στο dropdown μπορείτε να αναζητήσετε μεταφράσεις με το προεπιλεγμένο αγγλικό κείμενο στο widget σχολίων για να βρείτε το κείμενο που θέλετε να υπερκαλύψετε.

Καθώς κάνετε αλλαγές, θα εμφανίζονται στην προεπισκόπηση στο κάτω μέρος της σελίδας (όπως υπογραμμίζεται).

#### Τι Αλλάζει για τις Υπο υπάρχουσες Προσαρμογές Μετάφρασης

Αν έχετε ήδη προσαρμόσει το κείμενό σας στο widget σχολίων με μία από τις προηγούμενες μεθόδους δεν θα πρέπει να αλλάξει τίποτα - οι υπερκαλύψεις σας θα παραμείνουν όπως είναι.
