### Τι Νέο

Έχετε ποτέ επιστρέψει σε μια σελίδα και θέλετε να συνεχίσετε να διαβάζετε την ενότητα σχολίων όπου είχατε μείνει; Το FastComments τώρα στυλιζάρει τα σχόλια που χάσατε, κάνοντάς το εύκολο.

### Πώς να το Αποκτήσετε

Κάθε σχόλιο που προκάλεσε μια ειδοποίηση εντός της εφαρμογής (απαντήσεις, απαντήσεις στην ίδια συζήτηση, ή σχόλια σε μια σελίδα που είστε εγγεγραμμένος), θα επισημαίνεται αυτόματα με τον avatar του χρήστη να φωτίζεται ελαφρώς. Το χρώμα μπορεί να προσαρμοστεί μέσω CSS χρησιμοποιώντας την κλάση `is-unread`. Ο προεπιλεγμένος στυλισμός υποστηρίζει επίσης τη σκοτεινή λειτουργία.

Επιπλέον, μια νέα κλάση CSS `24hr` έχει προστεθεί στα στοιχεία σχολίων που έχουν δημοσιευτεί τις τελευταίες 24 ώρες. Μπορείτε να τη χρησιμοποιήσετε για να στυλιζάρετε και τα σχόλια.

### Γιατί να Επισημάνετε τον Avatar;

Αναγνωρίζουμε ότι μπορεί να είναι προτιμότερο να επισημαίνεται ολόκληρο το σχόλιο. Ωστόσο, αυτό έχει ανησυχίες από αναγνωσιμότητα και στυλιστικές απόψεις με τους πελάτες μας που έχουν προσαρμόσει την πλατφόρμα ώστε να ταιριάζει με το προϊόν τους. Διαπιστώσαμε ότι η ελαφριά επισήμανση του avatar είναι μια αρκετά διακριτική αλλαγή στις περισσότερες περιπτώσεις. Αν επιθυμείτε, η λάμψη μπορεί να αφαιρεθεί και να αλλάξει χρησιμοποιώντας κανόνες προσαρμογής widget.

### Στο Συμπέρασμα

Γνωρίζουμε ότι αυτή η λειτουργία έχει αναμενθεί εδώ και καιρό από κάποιους. Περάσαμε χρόνο για να επιλέξουμε ένα στυλ που δεν είναι πολύ ενοχλητικό αλλά είναι ακόμα χρήσιμο, και ενδέχεται να κάνουμε περαιτέρω προσαρμογές με βάση την ανατροφοδότηση.

Καλή συνέχεια!
