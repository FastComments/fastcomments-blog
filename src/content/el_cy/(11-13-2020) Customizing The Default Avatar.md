# [postlink]Προσαρμογή Του Προεπιλεγμένου Avatar[/postlink]

{{#unless isPost}}
Η προεπιλεγμένη εικόνα avatar μπορεί τώρα να προσαρμοστεί.
{{/unless}}

{{#isPost}}

### <i class="circle">!</i> Αυτό Το Άρθρο Περιέχει Τεχνική Ορολογία

#### Εισαγωγή

Το προεπιλεγμένο εικονίδιο παραμένει το ίδιο από την κυκλοφορία του FastComments. Δείτε τη δόξα του προεπιλεγμένου εικονιδίου προφίλ.

<div class="text-center">
    <img src="https://fastcomments.com/images/unknown-person.png" title="Default Profile Icon" alt="Default Profile Icon" />
</div>

Ωστόσο τώρα, μπορείτε να το προσαρμόσετε.

#### Χωρίς Κωδικό

Πρώτα, <a href="https://fastcomments.com/auth/my-account/customize-widget" target="_blank">προσθέστε ή επεξεργαστείτε έναν κανόνα προσαρμογής</a>. Οι κανόνες προσαρμογής μπορούν να εφαρμοστούν μόνο σε μια σελίδα, ένα τομέα ή σε ολόκληρο τον λογαριασμό σας. Αν θέλετε να αλλάξετε την εικόνα προφίλ σε παγκόσμιο επίπεδο, μπορείτε να αφήσετε το πεδίο ID URL κενό.

Κάντε κύλιση προς τα κάτω στην ενότητα "Προεπιλεγμένο Avatar".

<img 
    src="/images/fc-customize-default-icon-button.png"
    alt="FastComments Hackaday Settings"
    class='lozad' />
    
Κάντε κλικ στο κουμπί για να ανεβάσετε μια εικόνα JPG, PNG ή GIF με μέγιστο μέγεθος 20mb. Θα αναδιασταλληθεί σε τετράγωνο 100px.

#### Σε Κωδικό

Αν επιθυμείτε να φιλοξενήσετε το προεπιλεγμένο avatar και έχετε ήδη μια δημόσια διεύθυνση URL γι' αυτό, μπορεί να οριστεί στον κωδικό όπως στο δοθέν παράδειγμα.

<div class="code">    window.FastCommentsUI(document.getElementById('fastcomments-widget'), {
        tenantId: 'demo',
        defaultAvatarSrc: 'https://example.com/some-100-x-100-image.png'
    });
</div>

Η εικόνα θα πρέπει να είναι τετράγωνη και το μέγεθός της θα πρέπει να είναι 100px τετράγωνο. Ακολουθώντας αυτούς τους περιορισμούς θα εξασφαλίσετε την καλύτερη εμπειρία.

#### Παράγοντες Κινδύνου

1. Η προσαρμογή της προεπιλεγμένης εικόνας προφίλ επηρεάζει μόνο τον ιστότοπό σας. Δεν επηρεάζει τα email ειδοποιήσεων ή τους λογαριασμούς των χρηστών με οποιοδήποτε τρόπο.
2. Τα σχόλια που δημιουργήθηκαν πριν από αυτή την αλλαγή μπορεί να εξακολουθούν να εμφανίζουν την παλιά εικόνα προφίλ. Εάν προσαρμόσετε το προεπιλεγμένο εικονίδιο προφίλ και παρατηρήσετε πολλά σχόλια στον ιστότοπό σας που χρησιμοποιούν το παλιό εικονίδιο προφίλ, <a href="https://fastcomments.com/auth/my-account/help" target="_blank">ενημερώστε μας</a>.

#### Συμπέρασμα

Ελπίζουμε να βρήκατε αυτόν τον οδηγό χρήσιμο και εύκολο στην ανάγνωση. Μη διστάσετε να σχολιάσετε παρακάτω με τυχόν ερωτήσεις.

Γεια σας!
