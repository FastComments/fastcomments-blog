# [postlink]Μετακίνηση από τα Σχόλια Muut στα FastComments[/postlink]

{{#unless isPost}}
Διαβάστε αυτή την ανάρτηση για να δείτε πώς να μεταφέρετε πλήρως τα σχόλια από το Muut στα FastComments.

Είναι απλώς θέμα να αφαιρέσετε το μικρό κομμάτι κώδικα που σας δίνει το Muut και να το αντικαταστήσετε με <a href="https://fastcomments.com/auth/my-account/get-acct-code" target="_blank">το δικό μας</a>.
Μπορείτε επίσης να τρέξετε το Muut και το FastComments στον ίδιο ιστότοπο προσωρινά για να δοκιμάσετε και να προσαρμόσετε την εμφάνιση και την αίσθηση. <a href="https://fastcomments.com/auth/my-account/help" target="_blank">Μπορούμε να βοηθήσουμε με αυτό</a>.
{{/unless}}

{{#isPost}}
### Εξαγωγή των Υπαρχόντων Σχολίων σας

Ο εισαγωγέας του FastComments θα μεταφέρει τα σχόλιά σας, τα ονόματα χρηστών, τα άβαταρ χρηστών και τις inline εικόνες. Θα μεταφέρουμε τις εικόνες στους διακομιστές μας χωρίς προβλήματα.

Για να εξαγάγετε τα δεδομένα Σχολίων και Χρηστών του ιστότοπού σας από το Muut από τον πίνακα διαχείρισης τους, μεταβείτε στην Ενσωμάτωσή και στη συνέχεια στην Εξαγωγή JSON.

Θα παρατηρήσετε ότι το αρχείο που λαμβάνετε μέσω email έχει μια μυστηριώδη κατάληξη αρχείου "gz". Αν είστε λίγο πιο τεχνικά καταρτισμένοι, ίσως να γνωρίζετε ότι αυτό σημαίνει "gzip" και είναι ένας δημοφιλής και αποδοτικός τρόπος για να συμπιέσετε αρχεία.

### Μην ανοίξετε το αρχείο "gz"

Μια προεπιλεγμένη εγκατάσταση Windows 10 δεν θα μπορεί να ανοίξει το αρχείο από το Muut. Αυτό είναι εντάξει, διότι δεν χρειάζεται να το κάνετε με το FastComments. Ο backend μας κατανοεί αυτό το συμπιεσμένο αρχείο.

Μετά από αυτό, εάν είστε συνδεδεμένοι μπορείτε να πάτε <a href="https://fastcomments.com/auth/my-account/manage-data/import" target="_blank">εδώ για να εισαγάγετε το αρχείο</a>. Επιλέξτε Muut από το αναδυόμενο μενού και ανεβάστε το αρχείο σας.

Θα χρειαστεί να εισαγάγετε το URL του ιστότοπού σας. Αυτό οφείλεται στο γεγονός ότι το Muut δεν μας παρέχει αυτές τις πληροφορίες στην εξαγωγή.

### Περιμένετε μερικά λεπτά

Η εισαγωγή του FastComments είναι "ασύγχρονη". Αυτό σημαίνει ότι η φόρτωση του αρχείου και η επεξεργασία του είναι ξεχωριστά βήματα.

Έτσι, ανάλογα με το μέγεθος του αρχείου σας, μπορεί να λάβετε το μήνυμα επιτυχίας αμέσως. Στη σελίδα εισαγωγής στο κάτω μέρος υπάρχει ένας πίνακας - κάθε γραμμή αντιπροσωπεύει μια προσπάθεια εισαγωγής.
Μπορείτε με ασφάλεια να ανανεώσετε αυτή τη σελίδα για να δείτε την κατάσταση της εισαγωγής σας και πόσες γραμμές έχουν εισαχθεί μέχρι στιγμής.

### Όταν ολοκληρωθεί

Θα λάβετε ένα email όταν η εισαγωγή ολοκληρωθεί - είτε είναι επιτυχής είτε όχι. Μπορείτε να κλείσετε τη σελίδα αφότου η φόρτωση του αρχείου είναι επιτυχής και δείτε την εισαγωγή σας στη σελίδα Εισαγωγών. Θα έχει μια κατάσταση όπως "Αιτήθηκε" και όταν ξεκινήσει η κατάσταση θα είναι "Σε εκτέλεση".

Είναι ασφαλές να επαναφέρετε όσες φορές απαιτείται - ωστόσο, αν η εισαγωγή σας αποτύχει, τότε <a href="https://fastcomments.com/auth/my-account/help" target="_blank">επικοινωνήστε μαζί μας</a> ώστε να μπορέσουμε να βοηθήσουμε.

### Αντικαταστήστε τον Κώδικα Muut με FastComments

Είναι απλώς θέμα να αφαιρέσετε το μικρό κομμάτι κώδικα που σας δίνει το Muut και να το αντικαταστήσετε με <a href="https://fastcomments.com/auth/my-account/get-acct-code" target="_blank">το δικό μας</a>.
Μπορείτε επίσης να τρέξετε το Muut και το FastComments στον ίδιο ιστότοπο προσωρινά για να δοκιμάσετε και να προσαρμόσετε την εμφάνιση και την αίσθηση. <a href="https://fastcomments.com/auth/my-account/help" target="_blank">Μπορούμε να βοηθήσουμε με αυτό</a>.

### Μετακίνηση URLs ταυτόχρονα

Τα ίδια τα σχόλια είναι δεμένα με τα πεδία "σύνδεσμος" στην εξαγωγή Muut, οπότε εφόσον τα URLs σας δεν αλλάξουν, η μεταφορά είναι εύκολη. Εάν θέλετε να μεταφέρετε URLs και να διατηρήσετε τα σχόλια σας, τότε <a href="https://fastcomments.com/auth/my-account/help" target="_blank">ενημερώστε μας</a> στέλνοντάς μας τα παλιά και νέα URLs. Η σελίδα βοήθειας επιτρέπει την αποστολή μικρών αρχείων κειμένου, οπότε αν μεταφέρετε περισσότερα από μερικά URLs, δημιουργήστε ένα Excel sheet.

### Ανακεφαλαίωση

1. Εξάγετε τα δεδομένα σας
2. Εισάγετε στο FastComments
3. Αντικαταστήστε το απόσπασμα JavaScript του Muut με <a href="https://fastcomments.com/auth/my-account/get-acct-code" target="_blank">του FastComments</a>

## Γιατί το Muut δεν θα μου στείλει το αρχείο μου;
Για μεγαλύτερους ιστότοπους, το Muut μπορεί να μην είναι σε θέση να δημιουργήσει το εξαγόμενο αρχείο λόγω τεχνικών περιορισμών από την πλευρά τους. Έχουμε τη δυνατότητα να αντλήσουμε τα υπάρχοντα δεδομένα του ιστότοπού σας και να τραβήξουμε τα σχόλια από αυτό, ωστόσο αυτό μπορεί να είναι
αρκετά ειδικό και εξαιτίας του χρόνου που απαιτείται θα χρειαστεί να έχετε το Πακέτο Υποστήριξης. <a href="https://fastcomments.com/auth/my-account/help" target="_blank">Μπορείτε να ζητήσετε βοήθεια με αυτό εδώ</a>.
