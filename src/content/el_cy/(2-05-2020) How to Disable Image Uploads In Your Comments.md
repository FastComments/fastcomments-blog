# [postlink]Πώς να απενεργοποιήσετε τις αναρτήσεις εικόνας στα σχόλιά σας[/postlink]

Με το FastComments μπορείτε να απενεργοποιήσετε τις αναρτήσεις εικόνας στην ενότητα σχόλιων της ιστοσελίδας σας είτε για ολόκληρη την ιστοσελίδα είτε μόνο για μια συγκεκριμένη σελίδα.

{{#isPost}}
Για να το κάνετε αυτό, μεταβείτε στο <a href="https://fastcomments.com/auth/my-account/customize-widget" target="_blank">Προσαρμογή</a> στο ταμπλό και προσθέστε έναν κανόνα προσαρμογής.

Η ενεργοποίηση της ρύθμισης φαίνεται έτσι:

<img 
    src="images/fc-disable-image-uploads.png"
    alt="Απενεργοποίηση Αναρτήσεων Εικόνας"
    title="Απενεργοποίηση Αναρτήσεων Εικόνας"
    class='lozad' />

... και πατήστε αποθήκευση.
Σε αυτή τη σελίδα πρέπει να ορίσετε το domain σας αν δεν είναι ήδη προ-συμπληρωμένο. Αφήστε το πεδίο URL ID κενό για να εφαρμόσετε τον κανόνα σε ολόκληρη την ιστοσελίδα σας.

Αυτή η σελίδα έχει απενεργοποιημένες τις αναρτήσεις εικόνας ως παράδειγμα.

{{/isPost}}
