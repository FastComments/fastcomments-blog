# [postlink]Νέες Ζωντανές Αναλύσεις FastComments Δημοσιεύθηκαν[/postlink]

{{#unless isPost}}
Η FastComments έχει παρέχει διάφορες Αναλύσεις από την έναρξή της, αλλά τώρα παρέχει μια σειρά από **ζωντανές** μετρήσεις που μπορούν να αναφερθούν.
{{/unless}}

{{#isPost}}

### Τι Νέο

Η [FastComments Analytics](https://fastcomments.com/auth/my-account/analytics) μπορεί τώρα να χρησιμοποιηθεί για να παρέχει μια **ζωντανή** αναφορά για τους χρήστες που χρησιμοποιούν
την πλατφόρμα ή το προϊόν σας.

Δίνεται ο συνολικός αριθμός των χρηστών που είναι συνδεδεμένοι, καθώς και η δυνατότητα **ταξινόμησης σελίδων με βάση τον αριθμό των χρηστών που είναι online**. Μπορείτε επίσης να ταξινομήσετε τις σελίδες
με βάση τον αριθμό των σχολίων σε κάθε σελίδα.

Θα διαπιστώσετε ότι αυτές οι αναφορές είναι ακριβείς και γρήγορες, καθώς έχουμε καταβάλει σημαντική προσπάθεια για να διασφαλίσουμε ότι
λειτουργούν έτσι ώστε η FastComments να μπορέσει να συνεχίσει να δικαιώνει το όνομά της. :)

### Πώς να το Αποκτήσετε

Όλοι οι πελάτες της FastComments σε όλες τις βαθμίδες έχουν τώρα πρόσβαση στους νέους αριθμούς ζωντανών αναλύσεων μέσω της [σελίδας FastComments Analytics](https://fastcomments.com/auth/my-account/analytics).

### Ακρίβεια Μετρήσεων

Οι ζωντανές μετρήσεις (χρήστες online, χρήστες online ανά σελίδα) αναφέρονται χωρίς καμία καθυστέρηση. Απαιτούν ανανέωση της σελίδας για να δείτε τον τελευταίο αριθμό, αλλά
αυτός ο αριθμός παρέχεται χωρίς καμία αποθήκευση ή καθυστέρηση. Οι άλλες μετρήσεις, όπως ο αριθμός των φορτώσεων σελίδας ή τα νέα σχόλια ανά ημέρα, μπορεί να έχουν καθυστέρηση έως ενός λεπτού.

### Βελτιώσεις Εικονιδίων Δραστηριότητας Χρηστών

Με αυτή την κυκλοφορία, έχουμε βελτιώσει την ακρίβεια των εικονιδίων δραστηριότητας χρηστών που εμφανίζονται σε κάθε σχόλιο που δείχνει αν ένας χρήστης
είναι online και βλέπει αυτή την αλληλουχία σχολίων. Πριν, μπορεί να υπήρχε κάποια καθυστέρηση ή το εικονίδιο να μην ήταν πάντα ακριβές. Αυτό έχει
βελτιωθεί με μια πλήρη αναδιάρθρωση του συστήματος παρακολούθησης δραστηριότητας χρηστών.

### Τεκμηρίωση

Η ενημερωμένη τεκμηρίωση αναλύσεων μπορεί να βρεθεί στο [docs.fastcomments.com](https://docs.fastcomments.com/guide-analytics.html).

### Στο Συμπέρασμα

Όπως κάθε κύρια κυκλοφορία, είμαστε ευτυχείς που μπορέσαμε να πάρουμε τον χρόνο για να βελτιστοποιήσουμε, να δοκιμάσουμε και να κυκλοφορήσουμε σωστά αυτή τη δυνατότητα. Ενημερώστε μας
παρακάτω αν ανακαλύψετε τυχόν προβλήματα.

Ευχαριστώ!
