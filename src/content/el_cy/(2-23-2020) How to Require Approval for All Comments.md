# [postlink]Πώς να Απαιτήσετε Έγκριση για Όλα τα Σχόλια[/postlink]

Κατά προεπιλογή, δεν απαιτείται διαδικασία έγκρισης με το FastComments. Οι χρήστες σας σχολιάζουν και τα σχόλιά τους είναι άμεσα διαθέσιμα σε όλο το διαδίκτυο (αν και με την ένδειξη "μη επιβεβαιωμένο" αν δεν είναι συνδεδεμένοι). Τι γίνεται όμως αν θέλετε να ελέγχετε και να εγκρίνετε χειροκίνητα κάθε σχόλιο;

{{#isPost}}

Η απενεργοποίηση της αυτόματης έγκρισης σχολίων σημαίνει ότι εσείς ή οι συντονιστές σας θα πρέπει να εγκρίνετε κάθε σχόλιο πριν εμφανιστεί δημόσια.

Αρχικά, μεταβείτε στη σελίδα Ρυθμίσεων Συντονισμού, μέσω Συντονισμός Σχολίων -> Επεξεργασία Ρυθμίσεων Συντονισμού.

Σε αυτή τη σελίδα υπάρχει μια επιλογή που ονομάζεται "Αυτόματη Έγκριση Σχολίων". Αφαιρέστε την επιλογή αυτή και πατήστε αποθήκευση.

Αυτό είναι. Όλα τα σχόλιά σας θα απαιτούν πλέον έγκριση.

<img 
    src="images/fc-auto-approval.png"
    alt="Ρύθμιση Αυτόματης Έγκρισης"
    title="Ρύθμιση Αυτόματης Έγκρισης"
    class='lozad' />

Για να εγκρίνετε τα σχόλια μπορείτε να φιλτράρετε όπως επιθυμείτε και να κάνετε κλικ στην επιλογή "Σημείωση ως Εγκεκριμένο". Αν η επιλογή αλλάξει σε "Σημείωση ως Μη Εγκεκριμένο", αυτό σημαίνει ότι αποθηκεύτηκε.

<img 
    src="images/fc-mark-approved.png"
    alt="Πώς να Σημειώσετε ένα Σχόλιο Εγκεκριμένο"
    title="Πώς να Σημειώσετε ένα Σχόλιο Εγκεκριμένο"
    class='lozad' />

Μπορείτε να δείτε πόσα σχόλια εγκρίνουν οι συντονιστές σας μέσω της σελίδας Συντονιστών:
