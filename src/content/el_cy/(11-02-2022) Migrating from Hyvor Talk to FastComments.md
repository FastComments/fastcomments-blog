# [postlink]Μεταφορά από το Hyvor Talk στο FastComments[/postlink]

{{#unless isPost}}
Διαβάστε αυτή την ανάρτηση για να δείτε πώς να μεταφέρετε πλήρως από το Hyvor Talk στο FastComments.

Αν χρησιμοποιείτε την εγγρ. κώδικα με το Hyvor Talk, τότε είναι απλά θέμα αφαίρεσης του μικρού αποσπάσματος που σας δίνουν και αντικατάστασής του με <a href="https://fastcomments.com/install-wizard" target="_blank">το δικό μας</a>.
Μπορείτε ακόμη να τρέξετε το Hyvor και το FastComments στον ίδιο ιστότοπο προσωρινά για να δοκιμάσετε και να προσαρμόσετε την εμφάνιση. <a href="https://fastcomments.com/auth/my-account/help" target="_blank">Μπορούμε να σας βοηθήσουμε με αυτό</a>.

Αν δεν χρησιμοποιείτε την εγκατάσταση με το απόσπασμα κώδικα, τότε οι οδηγίες θα διαφέρουν ανάλογα με την πλατφόρμα σας - ξανά <a href="https://fastcomments.com/auth/my-account/help" target="_blank">επικοινωνήστε μαζί μας</a>.

## Αν είστε σε WordPress

Το FastComments έχει ένα ειδικό plugin για το WordPress: <a href="https://wordpress.org/plugins/fastcomments/" target="_blank">https://wordpress.org/plugins/fastcomments/</a>.

Το plugin καθιστά την εγκατάσταση, τη ρύθμιση και τη συγχρονισμό πολύ εύκολα. Αν ακολουθήσετε αυτήν τη διαδρομή, μπορείτε να αγνοήσετε τις υπόλοιπες οδηγίες εδώ. Απλώς βεβαιωθείτε ότι έχετε συγχρονίσει τα σχόλια του Hyvor Talk πίσω στην εγκατάσταση WordPress σας.

Αν αυτό δεν λειτουργεί για εσάς όπως αναφέρουν μερικοί πελάτες μας, ίσως θελήσετε να χρησιμοποιήσετε το plugin του WordPress, αλλά να εκτελέσετε μια εισαγωγή δεδομένων εξάγοντας χειροκίνητα από το Hyvor Talk. Μπορείτε να βρείτε τις εισαγωγές δεδομένων στο [Manage Data -> Import Comments](https://fastcomments.com/auth/my-account/manage-data/import).

## Χειροκίνητη, Γενική Εγκατάσταση

### Εξαγάγετε τα Υπαρχόντα Σχόλια σας

Ο εισαγωγέας του FastComments θα μεταφέρει τις νηματικές σας σχόλια, τα ονόματα χρηστών, τις εικόνες προφίλ, τα emoji και τις ενσωματωμένες εικόνες. Θα μεταφέρουμε τις εικόνες στους διακομιστές μας χωρίς διακοπή.

Από το 2022, το Hyvor Talk δεν εξάγει emails ή ψήφους (up/down). Αν μπορείτε να αποκτήσετε μια εξαγωγή από αυτούς με αυτές τις πληροφορίες, μπορούμε να την εισάγουμε.

### Εισαγωγή του αρχείου .JSON

Αφού έχετε κατεβάσει την εξαγωγή του Hyvor, συνδεθείτε στον πίνακα ελέγχου του FastComments και πηγαίνετε <a href="https://fastcomments.com/auth/my-account/manage-data/import" target="_blank">εδώ για να εισάγετε το αρχείο</a>. Επιλέξτε το Hyvor Talk από το αναπτυσσόμενο μενού και ανεβάστε το αρχείο σας.

### Περιμένετε λίγα λεπτά

Η εισαγωγή του FastComments είναι "ασύγχρονη". Αυτό σημαίνει ότι η αποστολή του αρχείου και η επεξεργασία του είναι ξεχωριστά βήματα.

Έτσι, ανάλογα με το μέγεθος του αρχείου σας, μπορεί να λάβετε το μήνυμα επιτυχίας αμέσως. Στη σελίδα εισαγωγής στο κάτω μέρος υπάρχει ένας πίνακας - κάθε γραμμή αντιπροσωπεύει μια προσπάθεια εισαγωγής.
Μπορείτε να ανανεώσετε αυτή τη σελίδα με ασφάλεια για να δείτε την κατάσταση της εισαγωγής σας και πόσες γραμμές έχουν εισαχθεί μέχρι στιγμής.

### Όταν ολοκληρωθεί

Θα λάβετε ένα email όταν ολοκληρωθεί η εισαγωγή - είτε είναι επιτυχής είτε όχι. Μπορείτε να κλείσετε τη σελίδα αφού η αποστολή του αρχείου είναι επιτυχής και δείτε την εισαγωγή σας στη σελίδα Εισαγωγών. Θα έχει μια κατάσταση όπως "Ζητήθηκε" και όταν ξεκινά, η κατάσταση θα είναι "Εκτέλεση".

Είναι ασφαλές να επανεισάγετε όσες φορές χρειάζεται - ωστόσο αν η εισαγωγή σας αποτύχει τότε <a href="https://fastcomments.com/auth/my-account/help" target="_blank">επικοινωνήστε μαζί μας</a> ώστε να μπορέσουμε να σας βοηθήσουμε.

### Αντικατάσταση του Κώδικα Hyvor Talk με το FastComments

Αν χρησιμοποιείτε την εγκατάσταση με το απόσπασμα κώδικα με το Hyvor, τότε είναι απλά θέμα αφαίρεσης του μικρού αποσπάσματος που σας δίνουν και αντικατάστασής του με <a href="https://fastcomments.com/install-wizard" target="_blank">το δικό μας</a>.
Υποστηρίζουμε επίσης πολλές διεπαφές front end που [μπορείτε να βρείτε εδώ](https://fastcomments.com/install-wizard). Μπορείτε ακόμη να τρέξετε το Hyvor και το FastComments στον ίδιο ιστότοπο προσωρινά για να δοκιμάσετε και να προσαρμόσετε την εμφάνιση. <a href="https://fastcomments.com/auth/my-account/help" target="_blank">Μπορούμε να σας βοηθήσουμε με αυτό</a>.

Αν δεν χρησιμοποιείτε την εγκατάσταση με το απόσπασμα κώδικα, τότε οι οδηγίες θα διαφέρουν ανάλογα με την πλατφόρμα σας - ξανά <a href="https://fastcomments.com/auth/my-account/help" target="_blank">επικοινωνήστε μαζί μας</a>.

Προσφέρουμε έναν οδηγό εγκατάστασης αυτοεξυπηρέτησης [εδώ](https://fastcomments.com/install-wizard).

### Μεταφορά URL ταυτόχρονα

Τα σχόλια αυτά καθαυτά συνδέονται με τα πεδία id σελίδας στην εξαγωγή, οπότε εφόσον τα URL σας δεν αλλάξουν, η αλλαγή είναι εύκολη. Αν θέλετε να μεταφέρετε URL και να κρατήσετε τα σχόλια σας τότε <a href="https://fastcomments.com/auth/my-account/help" target="_blank">ενημερώστε μας</a> στέλνοντάς μας τα παλιά και νέα URL. Η σελίδα βοήθειας επιτρέπει την αποστολή μικρών αρχείων κειμένου, οπότε αν μεταφέρετε περισσότερα από μερικά URLs, δημιουργήστε ένα φύλλο Excel.

### ΕΕ

Αν είστε στην ΕΕ, πιθανόν να θελήσετε να δημιουργήσετε τον λογαριασμό σας στο [eu.fastcomments.com](https://eu.fastcomments.com) έτσι ώστε τα δεδομένα σας να παραμείνουν στην ΕΕ.

### Ανακεφαλαίωση

1. Εξάγετε τα δεδομένα σας
2. Εισάγετε στο FastComments
3. Αντικαθιστάτε το απόσπασμα JavaScript του Hyvor με το <a href="https://fastcomments.com/install-wizard" target="_blank">του FastComments</a>

## Γιατί το Hyvor Δεν Μου Στέλνει το Αρχείο μου;
Για μεγαλύτερους ιστότοπους, το Hyvor μπορεί να μην είναι σε θέση να δημιουργήσει το εξαγόμενο αρχείο λόγω τεχνικών περιορισμών από την πλευρά τους. Έχουμε τη δυνατότητα να ανακτήσουμε τον υπάρχοντα ιστότοπό σας και να εξάγουμε τα σχόλια από αυτόν, ωστόσο αυτό μπορεί να είναι
αρκετά προσαρμοσμένο και λόγω του χρόνου που απαιτεί θα χρειαστεί να έχετε το Πακέτο Υποστήριξης. <a href="https://fastcomments.com/auth/my-account/help" target="_blank">Μπορείτε να ζητήσετε βοήθεια με αυτό εδώ</a>.
