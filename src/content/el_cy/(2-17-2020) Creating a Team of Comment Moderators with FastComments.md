### Το Πρόβλημα

Έχετε πολλή περιεχόμενο και πολλά σχόλια. Αυτό είναι καλό - η βάση χρηστών σας είναι αρκετά ενεργή για να αλληλεπιδράσει μαζί σας. Αλλά τώρα είστε υπερφορτωμένοι
και χρειάζεται να κλιμακώσετε τη λειτουργία σας.

### Moderation FastComments

Τα εργαλεία Moderation του FastComments παρέχουν ένα αποδοτικό περιβάλλον για την εξέταση των σχολίων, την επισήμανσή τους ως spam ή (αν πρέπει) τη διαγραφή τους.

Η ταμπλέτα είναι επίσης αντιδραστική, έτσι μπορείτε να εξετάσετε και να συντονίσετε τα σχόλια εν κινήσει!

Αλλά η πρόσβαση στη συντονισμό σχολίων για τον ιστότοπό σας δεν περιορίζεται μόνο στον λογαριασμό σας. Κάποιος μπορεί να δημιουργήσει τον δικό του λογαριασμό FastComments και να προσκληθεί
στον ιστότοπό σας μέσω του πίνακα διαχείρισης, [ή σε μαζική κλίμακα μέσω του API](https://docs.fastcomments.com/guide-api.html#moderator-structure). Δεν χρειάζεται καν να γνωρίζει για το FastComments ή να δημιουργήσει λογαριασμό εκ των προτέρων. Ο σύνδεσμος πρόσκλησης που θα σας δείξουμε
πώς να στείλετε θα τους καθοδηγήσει στη δημιουργία ενός λογαριασμού.

Απλά συνδεθείτε στο λογαριασμό σας και περάστε το ποντίκι πάνω από Συντονισμός Σχολίων. Θα δείτε μια επιλογή για Διαχείριση Συντονιστών.

Εάν δεν έχετε προσθέσει ακόμα συντονιστές, αυτό το κουμπί θα σας οδηγήσει σε έναν οδηγό που θα σας καθοδηγήσει για να προσθέσετε έναν συντονιστή.

Απλά θα χρειαστεί να παράσχετε ένα όνομα και ένα email. Αυτόματα θα τους σταλεί ένας σύνδεσμος πρόσκλησης που θα τους καθοδηγήσει επίσης για τη δημιουργία λογαριασμού.

Προαιρετικά, μπορείτε επίσης να τοποθετήσετε τους συντονιστές και τα σχόλια σε ξεχωριστές ομάδες. [Μάθετε περισσότερα για τις Ομάδες Συντονισμού και περαιτέρω λειτουργίες συντονισμού εδώ.](https://docs.fastcomments.com/guide-moderation.html)

Εάν έχετε μια ομάδα συντονιστών, μπορείτε να δείτε την πρόοδό τους από το ίδιο σημείο - η λίστα Συντονιστών Σχολίων παρέχει στατιστικά για κάθε συντονιστή. Αυτό περιλαμβάνει
τον αριθμό των σχολίων που έχουν επισήμανει ως εξετασμένα, spam και διαγραμμένα.

Αυτό είναι. Αφού στείλετε αυτόν τον σύνδεσμο πρόσκλησης, μπορούν να δημιουργήσουν τον λογαριασμό τους και θα έχουν πλήρη πρόσβαση για να συντονίσουν τα σχόλιά σας. Θα βλέπουν μόνο μια απλοποιημένη
έκδοση της σελίδας Συντονισμός Σχολίων - δεν παρέχεται δυνατότητα αλλαγής ρυθμίσεων στους συντονιστές.

Καλές σχολιαστικές αναρτήσεις!
