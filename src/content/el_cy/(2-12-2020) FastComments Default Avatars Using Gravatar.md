# [postlink]FastComments Προεπιλεγμένα Αβατάρ Χρησιμοποιώντας Gravatar[/postlink]

### Τι είναι ένα Αβατάρ;

Ένα αβατάρ είναι το εικονίδιο προφίλ που συνδέεται με κάθε λογαριασμό και εμφανίζεται δίπλα σε κάθε σχόλιο.

### Τι είναι το Gravatar;

<a href="https://en.gravatar.com/" target="_blank">Gravatar</a> τοποθετεί τον εαυτό του ως "Ένα Παγκοσμίως Αναγνωρισμένο Αβατάρ".
Είναι μια υπηρεσία όπου μπορείς να καθορίσεις ένα αβατάρ για μια συγκεκριμένη διεύθυνση email. Όλα τα προϊόντα που εγγράφεσαι χρησιμοποιώντας αυτήν την email μπορούν στη συνέχεια
να χρησιμοποιούν αυτό το αβατάρ. Αυτό καταργεί ένα βήμα για τους ανθρώπους που εγγράφονται στο FastComments καθώς το αβατάρ τους ρυθμίζεται αυτόματα.

### Οπότε ποια είναι η αλλαγή;

Το FastComments θα αντλεί πλέον προεπιλεγμένα αβατάρ από το Gravatar. Εσύ ή οι χρήστες σου δεν χρειάζεται να κάνετε τίποτα για να αποκτήσετε αυτήν τη δυνατότητα.

Αν οι υπάρχοντες χρήστες σου έχουν ήδη καθορίσει το δικό τους αβατάρ - μην ανησυχείς - δεν θα αντικαταστήσουμε τα προσαρμοσμένα αβατάρ τους.
Επιπλέον, οι χρήστες μπορούν ακόμα να αντικαταστήσουν το αβατάρ Gravatar στο FastComments.
