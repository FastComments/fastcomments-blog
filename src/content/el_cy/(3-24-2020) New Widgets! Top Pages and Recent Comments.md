# [postlink]Νέα Widgets! Κορυφαίες Σελίδες και Πρόσφατα Σχόλια[/postlink]

Σήμερα εισάγουμε δύο νέους τύπους widgets, φέρνοντας το συνολικό αριθμό των υποστηριζόμενων widgets σε τρία: Μετρητής Σχολίων, Πρόσφατα Σχόλια και Κορυφαίες Σελίδες.

{{#isPost}}

Εάν είστε συνδεδεμένοι, μπορείτε να δείτε τη λίστα <a href="https://fastcomments.com/auth/my-account/widgets" target="_blank">εδώ</a>, αλλιώς μεταβείτε στο Ο Λογαριασμός Μου -> Λάβετε Κωδικό Widget -> Άλλα Widgets.

Έχετε κατά νου ότι όλα μας τα widgets έχουν πολύ μινιμαλιστικό στυλ - είναι σχεδιασμένα να είναι γρήγορα και εύκολα να προσαρμοστούν στις ανάγκες σας.

Το widget Πρόσφατα Σχόλια θα δείξει τα δέκα κορυφαία εγκεκριμένα σχόλια, ταξινομημένα κατά ημερομηνία.

Το widget Κορυφαίες Σελίδες θα δείξει τις 30 κορυφαίες σελίδες ταξινομημένες κατά αριθμό σχολίων.

Οι τιμές για αυτά τα widgets είναι προκαθορισμένες πριν φορτώσετε τη σελίδα, **και** αποθηκεύονται προσωρινά για να διασφαλίσουν την καλύτερη εμπειρία για τους χρήστες σας. Για τεκμηρίωση γύρω από τα widgets, πηγαίνετε στη λίστα των widgets και επιλέξτε ένα. Από εκεί θα δείτε τον κωδικό που μπορείτε να προσθέσετε στον ιστότοπό σας καθώς και οποιαδήποτε ρύθμιση αποδέχεται το widget.

Εάν διαπιστώσετε ότι αυτά τα widgets δεν παρέχουν τις πληροφορίες που χρειάζεστε ή θα θέλατε βοήθεια στο στυλ τους, μη διστάσετε να <a href="https://fastcomments.com/auth/my-account/help" target="_blank">επικοινωνήσετε</a>.

{{/isPost}}
