# [postlink]Ενημέρωση Αναζήτησης Σχολίων Μετριασμού[/postlink]

{{#unless isPost}}
Η FastComments σας επιτρέπει πλέον να αναζητάτε ανά σελίδα και χρήστη στη σελίδα μετριασμού σχολίων.
{{/unless}}

{{#isPost}}

### Τι είναι Νέο

Προηγουμένως, η FastComments σας επέτρεπε να αναζητάτε με φράσεις ή με ακριβή αντιστοιχία, στη σελίδα μετριασμού σχολίων.

Τώρα μπορείτε να αναζητήσετε σχόλια με τους παρακάτω τρόπους:

- Αναζήτηση λέξης με ελαστικότητα: **cats love**
- Ακριβής αντιστοιχία φράσης: **"I love cats."**
- **_Νέο!_** Με τίτλο Σελίδας: **page:"Τίτλος Σελίδας"**
  - Υποστηρίζει αυτόματη συμπλήρωση.
  - Δείχνει τον αριθμό των σχολίων για κάθε σελίδα στη λίστα προτάσεων αυτόματης συμπλήρωσης.
- **_Νέο!_** Με URL Σελίδας: **page:"https://example.com/some-page"**
  - Υποστηρίζει αυτόματη συμπλήρωση.
  - Δείχνει τον αριθμό των σχολίων για κάθε σελίδα στη λίστα προτάσεων αυτόματης συμπλήρωσης.
- **_Νέο!_** Με Χρήστη: **user:"Bob"**
  - Υποστηρίζει αυτόματη συμπλήρωση.

Μπορείτε να μοιραστείτε τα αποτελέσματα αναζήτησης με άλλους μετριαστές ή διαχειριστές μοιράζοντας το URL της σελίδας από τη σελίδα μετριασμού. Η τιμή του πεδίου αναζήτησης θα περιληφθεί στο URL στον περιηγητή σας αφού πατήσετε "Go".

### Πώς να το αποκτήσετε

Αυτή η δυνατότητα έχει διατεθεί σε όλους τους πελάτες!

### Τεκμηρίωση

Η ιστοσελίδα τεκμηρίωσης έχει ενημερωθεί, και η αφιερωμένη τεκμηρίωση για αυτήν τη δυνατότητα [θα διατηρείται εδώ](https://docs.fastcomments.com/guide-moderation.html#search).

### Στο Συμπέρασμα

Όπως με όλες τις κύριες εκδόσεις, είμαστε χαρούμενοι που μπορέσαμε να πάρουμε τον χρόνο να βελτιστοποιήσουμε, να δοκιμάσουμε και να κυκλοφορήσουμε σωστά αυτή τη δυνατότητα. Ενημερώστε μας παρακάτω αν ανακαλύψετε τυχόν προβλήματα.

Ευχαριστούμε!

{{/isPost}}
