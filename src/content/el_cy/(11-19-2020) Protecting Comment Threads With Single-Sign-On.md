# [postlink]Προστασία Θηκών Σχολίων Με Σ Single-Sign-On[/postlink]

{{#unless isPost}}
Δεν θεωρείτε τις θηκές σχολίων σας δημόσιες; Έχετε ρυθμίσει SSO για αυθεντικοποίηση; Αυτό μπορεί να είναι για εσάς.
{{/unless}}

{{#isPost}}

### <i class="circle">!</i> Αυτό το Άρθρο Περιέχει Τεχνική Ορολογία

#### Εισαγωγή

Το FastComments SSO <a href="/(4-13-2020)-setting-up-sso-with-fastcomments.html" target="_blank">(λεπτομέρειες εδώ)</a> παρέχει στους χρήστες σας έναν τρόπο
να σχολιάζουν χωρίς να χρειάζεται να συνδεθούν σε άλλη πλατφόρμα.

Ωστόσο, αυτό μόνο του δεν προστατεύει τις θηκές σχολίων σας, αφού από προεπιλογή τα δεδομένα των σχολίων είναι δημόσια διαθέσιμες πληροφορίες - οποιοσδήποτε μπορεί να δει
τη σελίδα μπορεί να δει τα σχόλια. Ωστόσο, αυτή η έκδοση αλλάζει αυτό.

#### Ρύθμιση Χωρίς Κώδικα

Μπορούμε να αποτρέψουμε την προβολή και την αλληλεπίδραση με τις θηκές σχολίων μας, όταν ρυθμιστεί το SSO, δημιουργώντας έναν <a href="https://fastcomments.com/auth/my-account/customize-widget" target="_blank">κανόνα προσαρμογής</a>.

Όταν το κάνετε αυτό, αναζητήστε SSO, και θα βρείτε αυτή την επιλογή:

<div class="text-center">
    <img src="/images/fc-sso-protection.png" title="Επιλογή Ρύθμισης για Κλείδωμα Σχολίων μέσω SSO" alt="Επιλογή Ρύθμισης για Κλείδωμα Σχολίων μέσω SSO" class="lozad" />
</div>

Ενεργοποιήστε το και αποθηκεύστε τον κανόνα προσαρμογής.

#### Προστασία Μόνο Σε Ορισμένο Τομέα ή Σελίδα

Για να προστατεύσετε μόνο έναν συγκεκριμένο Τομέα ή Σελίδα, θα ρυθμίσουμε απλά τον κανόνα προσαρμογής.

Στην κορυφή της διεπαφής προσαρμογής, θα βρούμε δύο πεδία, Τομέας και ID URL.

<div class="text-center">
    <img src="/images/fc-sso-protection-by-page.png" title="Επιλογή Ρύθμισης κατά Σελίδα ή Τομέα" alt="Επιλογή Ρύθμισης κατά Σελίδα ή Τομέα" class="lozad" />
</div>

Για να προστατεύσετε μόνο έναν συγκεκριμένο τομέα, εισάγετε τον τομέα σε ερώτημα στο πεδίο "domain".

Για να προστατεύσετε μια συγκεκριμένη σελίδα, εισάγετε μια διεύθυνση URL σελίδας στο πεδίο "URL ID". Εάν έχετε μια προσαρμοσμένη ενσωμάτωση με το FastComments, μπορείτε να εισάγετε έναν τύπο ID εδώ
αντί για μια διεύθυνση URL.

#### Προστασία Πέρα Από την Ανάγνωση

Ενεργοποιώντας αυτή την επιλογή, θα προστατεύσετε τη σελίδα ή τον τομέα από το να σχολιαστεί εκτός εάν ο χρήστης είναι συνδεδεμένος μέσω SSO. 

#### Προβλήματα

Οποιεσδήποτε χρήστες που δημιούργησαν σχόλια πριν από την ενσωμάτωσή σας SSO δεν θα μπορούν να τα δουν, εκτός αν συνδεθούν μέσω της ενσωμάτωσής σας SSO.

#### Στην Κατακλείδα

Ελπίζουμε να θεωρήσατε αυτόν τον οδηγό χρήσιμο και εύκολο στην ανάγνωση. Μην διστάσετε να σχολιάσετε παρακάτω με οποιαδήποτε ερώτηση.

Στην Υγειά σας!
