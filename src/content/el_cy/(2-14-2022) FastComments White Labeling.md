## Τι είναι νέο

Το FastComments μπορεί τώρα να ενσωματωθεί με άλλες εφαρμογές και τρίτους μεταπωλητές μέσω της Υπογραφής Παράλληλης Σημείωσης. Η Υπογραφή Παράλληλης Σημείωσης σας επιτρέπει να δημιουργείτε λογαριασμούς FastComments (που ονομάζονται ενοικιαστές), χρήστες, συντονιστές και άλλα.

Τώρα προσφέρουμε ένα [εκτενή API](https://docs.fastcomments.com/guide-white-labeling.html#white-labeling-using-the-api) για να αυτοματοποιήσετε την ενσωμάτωσή σας
με τρίτους.

Μπορείτε τώρα, μέσω του API:

- Να δημιουργήσετε πελάτες (παιδικούς ενοικιαστές) που διαχειρίζονται από τον ενοικιαστή σας.
- Να διαχειριστείτε τα πακέτα που διατίθενται στους πελάτες σας και πώς θα χειριστείτε την χρέωση.
- Να προσθέσετε και να διαχειριστείτε χρήστες στους διαχειριζόμενους ενοικιαστές σας.
- Να προσθέσετε και να διαχειριστείτε συντονιστές στους διαχειριζόμενους ενοικιαστές σας.
- Να προσκαλέσετε συντονιστές και να στείλετε συνδέσμους σύνδεσης στους χρήστες του ενοικιαστή.

## Χρέωση

Με τη χρέωση, υπάρχουν δύο επιλογές:

1. Μέσω του FastComments Flex, η χρήση των παιδικών ενοικιαστών σας συνδυάζεται αυτόματα και χρεώνεται στον γονικό ενοικιαστή. Αυτό μπορεί να παρακολουθείται μέσω της [Σελίδας Αναλύσεων Χρέωσης](https://fastcomments.com/auth/my-account/analytics/billing).
2. Μέσω του FastComments Pro, αποκτάτε πρόσβαση για να δημιουργήσετε έναν σταθερό αριθμό παιδικών ενοικιαστών για μια σταθερή μηνιαία τιμή.

## Αφαίρεση Μάρκας

Οι διαχειριζόμενοι ενοικιαστές μπορούν να έχουν ενεργοποιημένη την αφαίρεση μάρκας, που αφαιρεί τα λογότυπα μας από το widget σχολίων.

## Πώς να αποκτήσετε Υπογραφή Παράλληλης Σημείωσης

Αυτή τη στιγμή, τα σχέδια FastComments Flex και Pro προσφέρουν πρόσβαση στο API Υπογραφής Παράλληλης Σημείωσης.

## Σενάρια και Παραδείγματα

Ένα παράδειγμα script που χρησιμοποιεί το API για να ρυθμίσει έναν ενοικιαστή με χρήστες και συντονιστές [διαθέσιμο εδώ](https://github.com/FastComments/fastcomments-code-examples/tree/master/white-labeling).

## Άλλες Περιοχές Χρήσης

Τα APIs [χρηστών ενοικιαστών](https://docs.fastcomments.com/guide-api.html#tenant-user-structure) και [συντονιστών](https://docs.fastcomments.com/guide-api.html#moderator-structure) μπορούν να χρησιμοποιηθούν επίσης για
να προσθέσετε και να διαχειριστείτε αυτούς τους πόρους στον δικό σας ενοικιαστή, ανεξαρτήτως της υπογραφής μάρκας.
