# [postlink]Προσθήκη Υπονοούμενων[/postlink]

{{#unless isPost}}
Οι χρήστες μπορούν τώρα να προσθέτουν υπονοούμενα στα σχόλιά τους τα οποία είναι κρυμμένα μέχρι να περνάει το ποντίκι από πάνω τους.
{{/unless}}

{{#isPost}}

### Τι Νέο
Οι χρήστες του FastComments μπορούν τώρα να κάνουν κλικ σε ένα κουμπί "SPOILER" για να προσθέσουν υπονοούμενα στα σχόλιά τους. Μπορούν επίσης να γράψουν ετικέτες `<spoiler>some text</spoiler>` άμεσα αν το επιθυμούν.
Σημειώστε ότι η επιλογή εργαλείου για υπονοούμενα είναι απενεργοποιημένη από προεπιλογή.

### Ενεργοποίηση Υπονοούμενων

Στην προσαρμογή Widget μπορείτε απλά να ενεργοποιήσετε την επιλογή `Spoilers` και να το αποθηκεύσετε. Τα έγγραφα είναι [εδώ](https://docs.fastcomments.com/guide-customizations-and-configuration.html#enable-spoilers).
Τα έχουμε ενεργοποιήσει για αυτήν την ανάρτηση για να δείξουμε τη δυνατότητα.

### Για Προγραμματιστές & Προβλήματα

Η προεπιλεγμένη στυλιστική υπονοούμενου υποστηρίζει φωτεινές και σκοτεινές λειτουργίες. Αν γράψετε προσαρμοσμένο CSS για καλυμμένο κείμενο, θυμηθείτε να καλύψετε
και τη σκοτεινή λειτουργία αν η ιστοσελίδα σας την υποστηρίζει.

### Συμπέρασμα

Όπως σε όλες τις μεγάλες εκδόσεις, είμαστε χαρούμενοι που μπορέσαμε να αφιερώσουμε χρόνο για να βελτιστοποιήσουμε, να δοκιμάσουμε και να απελευθερώσουμε σωστά αυτή τη δυνατότητα. Ενημερώστε μας
κάτω αν ανακαλύψετε οποιαδήποτε προβλήματα.

Χαιρετισμούς!

{{/isPost}}
