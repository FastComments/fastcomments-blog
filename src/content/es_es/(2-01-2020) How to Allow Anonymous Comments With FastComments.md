# [postlink]Cómo Permitir Comentarios Anónimos Con FastComments[/postlink]

{{#unless isPost}}
Por defecto, FastComments requiere un correo electrónico para comentar si no has iniciado sesión. Utilizamos este correo electrónico para crear automáticamente una cuenta para el usuario y permitirle verificar su comentario. 
Para permitir comentarios anónimos, solo necesitas crear una regla de personalización. Define el dominio y, opcionalmente, la página en la que deseas 
permitir comentarios anónimos.
{{/unless}}

{{#isPost}}
Por defecto, FastComments requiere un correo electrónico para comentar si no has iniciado sesión. Utilizamos este correo electrónico para crear automáticamente una cuenta para el usuario y permitirle verificar su comentario. 
Para permitir comentarios anónimos, solo necesitas crear una regla de personalización. Define el dominio y, opcionalmente, la página en la que deseas 
permitir comentarios anónimos.

Esto significa que puedes controlar en una base de página por página dónde permites comentarios anónimos.
