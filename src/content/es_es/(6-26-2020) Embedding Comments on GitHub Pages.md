#### Para Desarrolladores
Instalar FastComments en una página web solo requiere insertar un pequeño fragmento de código. Esto significa que funciona en páginas estáticas sin complicaciones, incluido el alojamiento gratuito de Pages de GitHub.

#### Una Demostración

Hemos configurado un ejemplo de GitHub Page <a href="https://fastcomments.github.io/fastcomments-demo.github.io/" target="_blank">aquí</a>.

<a href="https://github.com/fastcomments/fastcomments-demo.github.io/blob/master/index.html#L29" target="_blank">Aquí</a> está el enlace a donde hemos agregado el fragmento de código para nuestra
cuenta de prueba.
