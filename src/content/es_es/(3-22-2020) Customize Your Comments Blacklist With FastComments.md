# [postlink]Personaliza tu Lista Negra de Comentarios con FastComments[/postlink]

Para algunos, la lista negra de palabras predeterminada es demasiado restrictiva o demasiado abierta.

{{#isPost}}

La lista negra de palabras predeterminada ahora se puede editar o cambiar completamente. Simplemente ve a Moderar Comentarios -> Editar Configuraciones de Moderación -> Editar Lista Negra.

<img 
    src="images/fc-customize-blacklist.png"
    alt="Personalizar Lista Negra"
    title="Personalizar Lista Negra"
    class='lozad' />

Las palabras en esta lista deben estar separadas por comas, pero puedes usar espacios o saltos de línea si te resulta más fácil de leer.

Cualquier cambio puede tardar como máximo cinco minutos en actualizarse.

{{/isPost}}
