# [postlink]FastComments en sitios con fondos oscuros[/postlink]

FastComments ahora se ajusta al tema de tu sitio, ya sea claro u oscuro.

{{#isPost}}
#### ¿Qué, Cómo?
Nuestro widget de comentarios verificará el área de la página en la que se encuentra, y todos los elementos envolventes, hasta la ventana del navegador para encontrar cuán "oscura" es el área detrás del área de comentarios.

#### Para desarrolladores
Instalar FastComments en una página web solo requiere insertar un pequeño fragmento de código. FastComments intentará calcular automáticamente la "luminancia" del área de la página en la que está instalado, para determinar si debe renderizar un tema oscuro o claro.

#### Una demostración
¡Sorpresa, esta página es la demostración!

#### Si todo lo demás falla...
El navegador es un entorno loco, y como todas las características de FastComments, tenemos que equilibrar velocidad y corrección. Hay algunos casos en los que la detección puede equivocarse. En estos casos, <a href="https://fastcomments.com/auth/my-account/help" target="_blank">háznoslo saber</a> y te ayudaremos a ajustar manualmente el widget para que coincida con tu sitio.

#### Configuración
Nada especial, solo instala FastComments como con cualquier otro sitio o plataforma.

#### En conclusión
No utilizamos ningún estilo o configuración elegantes para hacer que el widget a continuación funcione en este sitio con un fondo negro.

Esperamos que encuentres FastComments rápido y fácil de usar - ¡salud!
