# [postlink]Mejoras en la Carga de Imágenes[/postlink]

{{#unless isPost}}
Varias mejoras en la carga de imágenes y el editor WYSIWYG en esta actualización.
{{/unless}}

{{#isPost}}

### Novedades

Hoy se han lanzado una serie de correcciones de errores para la carga de imágenes y el modo de edición WYSIWYG (lo que ves es lo que obtienes).

1. Ahora se muestra un indicador de progreso al cargar imágenes.
2. Ahora se admite el pegado de imágenes, como capturas de pantalla (solo editor WYSIWYG).
3. Al editar comentarios, las cajas de comentarios ahora tienen el tamaño adecuado para mostrar todas las imágenes (solo editor WYSIWYG).
4. Al eliminar imágenes, la caja de comentarios se reduce como se esperaba (solo editor WYSIWYG).

### En Conclusión

Gracias a nuestros pacientes clientes que señalaron algunos de estos problemas. Esperamos que continúen disfrutando trabajar con FastComments.
