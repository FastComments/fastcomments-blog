### <i class="circle">!</i> Este Artículo Contiene Jerga Técnica

### Novedades

FastComments ahora tiene componentes de React Native que están a la par en características con nuestro widget VanillaJS.

Decidimos llamar a la biblioteca `fastcomments-react-native`. Puedes encontrar el repositorio de GitHub con el código fuente <a href="https://github.com/FastComments/fastcomments-react-native" target="_blank">aquí</a>.

El repositorio también contiene ejemplos de uso del widget.

Está en NPM, que puedes encontrar <a href="https://www.npmjs.com/package/fastcomments-react-native" target="_blank">aquí</a>.

### Cómo Funciona

La forma en que funciona este nuevo componente es que el componente de React es un envoltorio alrededor del widget existente de FastComments.
 
Esto significa que si añadimos características a nuestros componentes existentes o corregimos errores, ¡tú aún te beneficiarás de inmediato!

### Tiempo de Desarrollo

Los widgets de React Native pueden añadirse a tu aplicación en solo minutos con poco tiempo de desarrollo requerido.

### SSO

Todas las bibliotecas de React Native soportan SSO directamente, y el proceso de integración de SSO es el mismo que el de los componentes de Vanilla JS. El código de backend existente también se aplica.

### ¿Estilo Personalizado?

Los componentes de React Native pueden estilizarse utilizando CSS a través de la interfaz de personalización del widget, igual que los widgets basados en la web.
