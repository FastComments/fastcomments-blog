# [postlink]Avatares Predeterminados de FastComments Usando Gravatar[/postlink]

### ¿Qué es un Avatar?

Un avatar es el icono de perfil asociado con cada cuenta y que se muestra junto a cada comentario.

### ¿Qué es Gravatar?

<a href="https://en.gravatar.com/" target="_blank">Gravatar</a> se presenta como "Un Avatar Reconocido Globalmente".
Es un servicio donde puedes definir un avatar para un correo electrónico determinado. Cualquier producto al que te inscribas usando ese correo electrónico puede entonces
usar ese avatar. Esto elimina un paso para las personas que se inscriben en FastComments, ya que su avatar se configura automáticamente.

### ¿Entonces, cuál es el cambio?

FastComments ahora obtendrá avatares predeterminados de Gravatar. Tú o tus usuarios no tienen que hacer nada para obtener esta función.

Si tus usuarios existentes ya han definido su propio avatar, no te preocupes, no vamos a sobrescribir sus avatares personalizados.
Además, los usuarios todavía pueden sobrescribir el avatar de Gravatar en FastComments.
