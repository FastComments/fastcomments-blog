# [postlink]FastComments 进入 React Native[/postlink]

{{#unless isPost}}
需要将实时评论嵌入到使用 React Native 开发的应用中吗？我们可以帮您。
{{/unless}}

{{#isPost}}

### <i class="circle">!</i> 本文包含技术术语

### 新特性

FastComments 现在拥有与我们的 VanillaJS 小部件功能相当的 React Native 组件。

我们决定将该库命名为 `fastcomments-react-native`。您可以在 <a href="https://github.com/FastComments/fastcomments-react-native" target="_blank">这里</a> 找到包含源代码的 GitHub 仓库。

该仓库还包含小部件的示例用法。

它已经在 NPM 上发布，您可以在 <a href="https://www.npmjs.com/package/fastcomments-react-native" target="_blank">这里</a> 找到。

### 工作原理

这个新组件的工作方式是，React 组件是现有 FastComments 小部件的包装器。

这意味着如果我们为现有组件添加新功能或修复错误，您将立即受益！

### 开发时间

可以在几分钟内将 React Native 小部件添加到您的应用中，所需开发时间很少。

### 单点登录（SSO）

所有 React Native 库都开箱即用地支持 SSO，SSO 集成过程与 Vanilla JS 组件相同。现有的后端代码示例同样适用。

### 自定义样式？

React Native 组件可以通过小部件自定义 UI 使用 CSS 进行样式设置，与基于 Web 的小部件相同。

### 替代方案

您可以围绕构建自己UI

### 总结

通过发布 `fastcomments-react-native` 库及未来的库，我们希望能够使开发人员在使用现代开发方法时更容易采用 FastComments。

干杯！
