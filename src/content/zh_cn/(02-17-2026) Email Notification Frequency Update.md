# [postlink]邮箱通知频率更新[/postlink]

{{#unless isPost}}
用户现在可以控制他们每分钟、每小时摘要或每日摘要接收新评论和回复的电子邮件通知频率，所有这些通过一个共享设置，还可以针对每个订阅进行覆盖。
{{/unless}}

{{#isPost}}

### 新功能

我们添加了一个**新评论通知频率**设置，控制您接收回复通知和租户（新评论）通知的电子邮件频率。三个选项如下：

- **每分钟** - 新评论到达时立即收到电子邮件（每分钟检查）。
- **每小时摘要** - 每小时接收一次新评论的汇总。
- **每日摘要** - 每天接收一次新评论的汇总。

此设置适用于租户管理员和评论者，并适用于所有评论通知电子邮件。请注意，无论此设置如何，@提及通知始终会立即发送。

您还可以在订阅表中按每个订阅覆盖频率，以便对单独页面进行细致控制。

### 如何配置

1. 前往您的[通知设置](https://fastcomments.com/auth/my-account/edit-notifications)。
2. 使用**新评论通知频率**下拉菜单设置您的首选频率。
3. 可选地，在订阅表中为单独的订阅覆盖频率。
4. 点击**保存更改**。

默认设置为**每分钟**，与之前的行为一致。

### API 支持

通知频率也可以通过 API 获取。用户级别的设置为`notificationFrequency`字段，且可以通过订阅的`notificationFrequency`字段设置每个订阅的覆盖。有关详细信息，请参见[API 文档](https://docs.fastcomments.com/guide-api.html)。

### 总结

这让用户可以控制他们的收件箱，而无需完全退订页面。一个设置现在涵盖了回复和租户通知，管理起来更简单。
