# [postlink]电子邮件通知频率更新[/postlink]

{{#unless isPost}}
用户现在可以控制他们收到新评论和回复的电子邮件通知频率 — 每分钟、每小时摘要或每天摘要 — 只需一个共享设置，同时可以对每个订阅进行覆盖。
{{/unless}}

{{#isPost}}

### 新动态

我们添加了一个**新评论通知频率**设置，用于控制您收到回复通知和租户（新评论）通知的电子邮件通知频率。三个选项为：

- **每分钟** - 在新评论到达时立即收到邮件（每分钟检查一次）。
- **每小时摘要** - 每小时收到新评论的批量总结。
- **每天摘要** - 每天收到新评论的批量总结。

该设置对租户管理员和评论者均可用，并适用于所有评论通知电子邮件。请注意，@mention通知始终会立即发送，而不受此设置的影响。

您还可以在订阅表中按每个订阅覆盖频率，以便对单个页面进行更精细的控制。

### 如何配置

1. 前往您的[通知设置](https://fastcomments.com/auth/my-account/edit-notifications)。
2. 使用**新评论通知频率**下拉菜单设置您首选的频率。
3. 可选地，在订阅表中覆盖单个订阅的频率。
4. 点击**保存更改**。

默认设置为**每分钟**，这与之前的行为一致。

### API 支持

通知频率也可以通过 API 获取。用户级别的设置是<div class="code">notificationFrequency</div>字段，每个订阅的覆盖可以通过订阅的<div class="code">notificationFrequency</div>字段设置。有关详细信息，请参见[API文档](https://docs.fastcomments.com/guide-api.html)。

### 小结

这使用户可以控制他们的收件箱，而无需完全取消订阅页面。现在一个设置涵盖了回复和租户通知，使管理变得简单。
