# [postlink]评论审核的快捷键[/postlink]

{{#unless isPost}}
高级用户现在可以完全通过键盘审核评论，配备了全面的导航、操作等快捷键。
{{/unless}}

{{#isPost}}

### 新特性

评论审核页面现在支持全面的键盘快捷键。可以在评论之间导航，批准、删除、封禁，标记为垃圾邮件，切换上下文，查看日志等 - 所有操作都无需使用鼠标。

对于审核大量评论的团队来说，这是一件大事。每一个之前需要移动鼠标并点击按钮的操作，现在只需按一个键就能完成。

### 示例流程

假设你有一队评论需要审核。按 **j** 聚焦第一条评论。按 **c** 展开其上下文，查看周围的讨论。看起来不错，所以再次按 **c** 以折叠上下文，然后按 **a** 进行批准。
按 **j** 移动到下一条评论并重复。如果发现什么不对，按 **s** 标记为垃圾邮件或按 **d** 删除它。你可以在整个评论页面上工作，而无需伸手去摸鼠标。

### 导航

按 **j** 移动焦点到下一条评论，按 **k** 移动到上一条评论。聚焦的评论用蓝色轮廓突出显示，以便与选中的（已勾选）评论区分。

按 **x** 或 **Enter** 切换聚焦评论的选择状态。按住 **Shift** 并按 **j** 或 **k** 可同时选择当前评论并移动焦点 - 对于快速选择一系列评论非常有用。

### 操作

聚焦评论时，单键快捷方式触发与工具栏按钮相同的操作：

- **a** - 切换批准 / 取消批准
- **s** - 切换垃圾邮件 / 不是垃圾邮件
- **d** - 删除
- **u** - 取消删除
- **r** - 切换已审核 / 未审核
- **e** - 编辑
- **b** - 封禁 / 管理封禁
- **p** - 切换钉住 / 取消钉住
- **f** - 取消标记
- **i** - 查看来自 IP 的所有内容
- **t** - 设置信任因素
- **m** - 管理徽章
- **l** - 查看日志
- **c** - 切换上下文

### 全局快捷键

一些快捷键在评论是否聚焦时均可使用：

- **Ctrl+a** - 选择所有评论
- **Ctrl+z** - 撤销最后一个操作
- **/** - 聚焦搜索输入
- **[** 和 **]** - 导航到上一页或下一页

### 帮助和引导

随时按 **?** 打开帮助对话框，列出所有可用的快捷键。

按 **g** 切换“引导者”模式，在聚焦评论的操作按钮上覆盖小徽章，显示相应的快捷键。这在学习快捷键时非常方便。

在搜索框或任何其他输入字段中输入时，所有快捷键都会被抑制，并且当对话框打开时也会如此。
按 **Escape** 取消聚焦输入，返回快捷键模式。
