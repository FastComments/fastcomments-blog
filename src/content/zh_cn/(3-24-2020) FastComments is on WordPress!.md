# [postlink]FastComments 现已支持 WordPress![/postlink]

考虑从其他 WordPress 评论插件切换过来吗？试试 FastComments！

{{#isPost}}

我们很自豪地宣布，经过大量工作，我们已推出我们的 WordPress 插件。您可以在这里找到它：
<a href="https://wordpress.org/plugins/fastcomments/" target="_blank">https://wordpress.org/plugins/fastcomments/</a>。

我们发现，FastComments 的性能明显优于默认的 WordPress 评论，特别是在较大的评论线程中。

它将通过减轻服务器负担来加速您的 WordPress 网站 - 因为我们将渲染评论线程，而不是 WordPress。

基准测试没有说谎 - 在一个没有负载的服务器上运行的文章中，100 条评论的纯 WordPress 安装每秒能处理 86 个请求，然后就崩溃了。

启用 FastComments 后，这个数字跃升至 **超过 400**。
