# [postlink]Как отключить загрузку изображений в ваших комментариях[/postlink]

С помощью FastComments вы можете отключить загрузку изображений в разделе комментариев вашего веб-сайта как для всего сайта, так и для конкретной страницы.

{{#isPost}}
Для этого перейдите в <a href="https://fastcomments.com/auth/my-account/customize-widget" target="_blank">Настроить</a> на панели управления и добавьте правило настройки.

Включение этой настройки выглядит следующим образом:

<img 
    src="images/fc-disable-image-uploads.png"
    alt="Отключить загрузку изображений"
    title="Отключить загрузку изображений"
    class='lozad' />

... и нажмите сохранить.
На этой странице вы должны указать свой домен, если он еще не заполнен. Оставьте поле URL ID пустым, чтобы применить правило ко всему сайту.

На этой странице загрузка изображений отключена в качестве примера.

{{/isPost}}
