# [postlink]Как автоматически утверждать только проверенные комментарии[/postlink]

У вас активирована автоматическая модерация, но вы хотите, чтобы на вашем сайте или блоге отображались только проверенные комментарии?

{{#isPost}}

Вот как это сделать: перейдите в <a href="https://fastcomments.com/auth/my-account/moderate-comments/settings" target="_blank">Модерация комментариев -> Изменить настройки модерации</a>.

Оттуда установите флажок **Автоматически утверждать комментарии**, если он еще не установлен - затем выберите **Только автоматически утверждать проверенные комментарии** и нажмите **сохранить**.

<img 
    src="images/fc-verification-approves-comment.png"
    alt="Опция автоматического утверждения для проверенных комментариев выделена"
    title="Опция автоматического утверждения для проверенных комментариев выделена"
    class='lozad' />

Теперь любой комментарий, который не подтвержден по электронной почте, не будет отображаться, пока комментатор не подтвердит его. Также им будет показано дружелюбное сообщение
при оставлении комментария с просьбой проверить свою электронную почту.

Изменение этой настройки происходит мгновенно, однако оно не будет применимо к прошлым комментариям.

Удачных комментариев!

{{/isPost}}
