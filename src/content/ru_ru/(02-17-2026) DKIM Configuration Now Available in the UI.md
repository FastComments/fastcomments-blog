# [postlink]Конфигурация DKIM теперь доступна в интерфейсе[/postlink]

{{#unless isPost}}
FastComments теперь предоставляет интерфейс для настройки DKIM, упрощая процесс повышения показателей доставки электронной почты без использования API.
{{/unless}}

{{#isPost}}

### Что нового

Настройка DKIM для вашего пользовательского домена электронной почты стала проще. Ранее для настройки DKIM требовалось использовать
<a href="https://docs.fastcomments.com/guide-api.html#domain-config-structure" target="_blank">API конфигураций доменов</a> напрямую. Теперь вы можете
настраивать DKIM прямо из админ-интерфейса FastComments.

### Почему DKIM важен

Когда вы настраиваете From Domain для писем, которые FastComments отправляет от вашего имени, поставщики электронной почты должны подтвердить, что
эти письма являются легитимными. DKIM (DomainKeys Identified Mail) является одним из ключевых механизмов аутентификации, который помогает в этом.
Без правильной настройки DKIM письма, отправляемые от имени вашего домена, с большей вероятностью окажутся в папках спама.

### Как это настроить

Перейдите на страницу <a href="https://fastcomments.com/auth/my-account/configure-domains" target="_blank">Мои домены</a> и нажмите
"Показать дополнительные" для домена, который вы хотите настроить. Вы увидите настройки DKIM, где можно ввести имя вашего домена, селектор ключа
и приватный ключ.

После сохранения конфигурация DKIM будет применена ко всем письмам, отправляемым с этого домена.

### API по-прежнему работает

Если вы предпочитаете управлять конфигурацией DKIM программно, то
<a href="https://docs.fastcomments.com/guide-api.html#domain-config-structure" target="_blank">API конфигураций доменов</a> продолжает работать
как и раньше.

### В заключение

Мы надеемся, что это упростит процесс обеспечения того, чтобы ваши пользователи получали письма, которые FastComments отправляет от вашего имени. Дайте нам знать
ниже, если у вас есть какие-либо вопросы.

Всего хорошего!
