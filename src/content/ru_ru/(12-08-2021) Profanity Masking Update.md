# [postlink]Обновление маскировки нецензурной лексики[/postlink]

{{#unless isPost}}
Система обнаружения и маскировки нецензурной лексики FastComments получила обновление
{{/unless}}

{{#isPost}}

### Что нового

Ранее FastComments позволял определять только один глобальный черный список слов для всей вашей учетной записи.

Этот список слов для маскировки или для использования в случае обнаружения спама не мог быть определен для каждого языка. Он также мог содержать только список слов, а не фраз.

Теперь были сделаны следующие обновления:

1. Черный список можно определять по локали (язык + регион).
2. В дополнение к словам теперь можно маскировать фразы или использовать их для обнаружения спама.
3. Обновление конфигурации обнаружения нецензурной лексики теперь *живое* и *незамедлительное*. Ранее применение изменений могло занимать до пяти минут.

### Как это использовать

Ссылка на черный список слов не изменилась и остается по адресу [/auth/my-account/moderate-comments/blacklist](https://fastcomments.com/auth/my-account/moderate-comments/blacklist) и
по-прежнему доступна из `Moderate Comments -> Moderation Settings -> Edit Word Blacklist`.

На этой странице вы теперь увидите выпадающий список языков для настройки черного списка по локали.

После выбора локали вы можете определять слова или фразы для маскировки.

### Ограниуения

Можно определить до 5,000 слов или фраз для каждой локали. Если вам нужно увеличить этот лимит, свяжитесь с поддержкой.

### Это в действии!

Любые изменения, которые вы вносите в список слов или фраз для маскировки, теперь применяются к системе немедленно.

### Кто получает это

Все текущие и новые клиенты FastComments, на всех уровнях, теперь имеют доступ к новым функциям черного списка *слов* и *фраз*.

### В заключение

Как и все остальное в FastComments, мы надеемся, что вы найдете эту функцию быстрой и удобной в использовании.

Ура!
