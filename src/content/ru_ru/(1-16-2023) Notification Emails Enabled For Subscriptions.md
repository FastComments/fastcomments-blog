# [postlink]Уведомления по электронной почте для подписок включены[/postlink]

{{#unless isPost}}
FastComments теперь отправляет электронные письма пользователям для страниц, на которые они подписаны.
{{/unless}}

{{#isPost}}

### Что нового

С помощью FastComments пользователи могут нажать на значок колокольчика в виджете комментариев, чтобы подписаться на эту страницу. Мы будем уведомлять их
когда на этой странице будут оставлены новые комментарии или когда появится новый комментарий в цепочке ответов, в которой они участвуют.

Мы делали это много лет, однако мы не отправляли электронные письма для этих уведомлений - они были видны только в нашей панели управления.

Теперь мы будем отправлять электронные письма для этих уведомлений.

### Новые типы электронных писем

Мы теперь отправляем два новых типа электронных писем. Первый - когда у вас всего несколько уведомлений. Мы просто перечислим
новые комментарии в электронном письме.

Второй тип электронного письма, которое вы получите, содержит ссылки на каждую страницу, по которой вы получили уведомления. Это отправляется
когда вы получаете много уведомлений.

Эти уведомления по электронной почте предназначены только для новых комментариев на страницах, на которые вы подписаны.

### Расписание уведомлений по электронной почте

Уведомления по электронной почте отправляются каждый час, чтобы предотвратить избыточные электронные письма. Исключение составляют страницы, на которых более десяти тысяч
комментариев, в этом случае уведомления для этой страницы будут отправляться только один раз в день.

### Для владельцев сайтов - Настройка электронной почты

Скоро будет доступен новый шаблон электронного письма `Новые комментарии по подписке` для настройки.

### Для разработчиков и нюансы

Для пользователей SSO мы не будем отправлять уведомления по электронной почте по умолчанию. Вам нужно будет передать новый флаг: `optedInSubscriptionNotifications: true` в объекте пользователя или
в полезной нагрузке к виджету комментариев, чтобы активировать эти новые уведомления.

### В заключение

ниже, если вы обнаружите какие-либо проблемы.

Удачи!
