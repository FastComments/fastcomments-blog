# [postlink]Обновление поиска комментариев в модерации[/postlink]

{{#unless isPost}}
FastComments теперь позволяет вам искать по странице и пользователю на странице модерации комментариев.
{{/unless}}

{{#isPost}}

### Что нового

Ранее FastComments позволял вам искать по фразам или точным совпадениям строк на странице модерации комментариев.

Теперь вы можете искать комментарии следующими способами:

- Неясный поиск слов: **cats love**
- Точное совпадение фразы: **"I love cats."**
- **_Новинка!_** По заголовку страницы: **page:"Page Title"**
  - Поддерживает автозаполнение.
  - Показывает количество комментариев для каждой страницы в списке предложений автозаполнения.
- **_Новинка!_** По URL страницы: **page:"https://example.com/some-page"**
  - Поддерживает автозаполнение.
  - Показывает количество комментариев для каждой страницы в списке предложений автозаполнения.
- **_Новинка!_** По пользователю: **user:"Bob"**
  - Поддерживает автозаполнение.

Вы можете делиться результатами поиска с другими модераторами или администраторами, делясь URL страницы с страницы модерации. Значение поля поиска
будет включено в URL в вашем браузере после нажатия "Перейти".
