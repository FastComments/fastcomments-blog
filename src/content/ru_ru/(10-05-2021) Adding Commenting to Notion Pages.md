## Что нового

FastComments теперь является провайдером embed.ly и, как таковой, предоставляет API и сервисы, которые позволяют встраивать потоки комментариев
в приложения, поддерживающие [oEmbed](https://oembed.com).

Это означает, что теперь вы можете легко добавлять живые комментарии на сайт, созданный с помощью Notion, Google Sites или Nimbus Note.

## Как это использовать - Быстрая демонстрация

Когда вы вошли в свой аккаунт, вы можете просто перейти по адресу [fastcomments.com/create](https://fastcomments.com/create), чтобы создать ссылку на разговор, которую затем
можно добавить на вашу страницу Notion или в любое приложение, интегрированное с embed.ly.

Вы можете попробовать это сами, используя свою демонстрационную ссылку для встраивания, [fastcomments.com/c/demo](https://fastcomments.com/c/demo):

## Обновление 2023 - Поддержка Super.so

Теперь вы можете добавлять комментарии на сайты Super.so, которые построены на основе документов Notion, с использованием FastComments! [Вы можете найти документацию здесь.](https://docs.fastcomments.com/guide-installation-super-so.html)

## Google Sites

Google Sites также поддерживается, и процесс похож.

1. [Создайте разговор для встраивания.](https://fastcomments.com/create)
2. При редактировании вашего Google Site перейдите в `Вставка` -> `Встроить`. Выберите `По URL`.
3. Вставьте URL из Шага 1.
4. Нажмите `Вставить`. Готово!

## Nimbus Note

Nimbus Note также поддерживается, и процесс похож.

1. [Создайте разговор для встраивания.](https://fastcomments.com/create)
2. При редактировании вашей заметки добавьте блок для встраивания.
3. Используйте URL из Шага 1 в качестве URL для встраивания.
4. Нажмите `Вставить`. Готово!

### Кто это получит

Все текущие и новые клиенты FastComments на всех уровнях теперь имеют доступ к поддержке oEmbed.

### Встраивание существующего разговора

Чтобы встроить существующий разговор, просто введите значение <a href="https://docs.fastcomments.com/guide-customizations-and-configuration.html#url-id" target="_blank">URL ID</a>, переданное виджету комментариев,
на странице Создания разговора в качестве названия разговора.
