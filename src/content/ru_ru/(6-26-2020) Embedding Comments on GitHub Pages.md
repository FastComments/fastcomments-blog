#### Для разработчиков
Установка FastComments на веб-странице требует лишь вставить небольшой код. Это означает, что он работает на статических страницах без хлопот - включая бесплатный хостинг Pages от GitHub.

#### Демонстрация

Мы настроили пример страницы на GitHub <a href="https://fastcomments.github.io/fastcomments-demo.github.io/" target="_blank">здесь</a>.

<a href="https://github.com/fastcomments/fastcomments-demo.github.io/blob/master/index.html#L29" target="_blank">Вот</a> ссылка на то, где мы добавили код для нашей тестовой учетной записи.

#### Настройка

Вам нужно будет добавить ваш домен GitHub как допустимый домен в вашей учетной записи.
