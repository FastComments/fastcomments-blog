# [postlink]FastComments теперь доступен на WordPress![/postlink]

Рассматриваете возможность перехода с другого плагина для комментариев WordPress? Попробуйте FastComments!

{{#isPost}}

Мы рады сообщить, что после долгой работы мы запустили наш плагин для WordPress. Вы можете найти его здесь:
<a href="https://wordpress.org/plugins/fastcomments/" target="_blank">https://wordpress.org/plugins/fastcomments/</a>.

Мы обнаружили, что FastComments работает значительно лучше, чем стандартные комментарии WordPress, особенно для больших потоков комментариев.

Это ускорит ваш сайт на WordPress, снимая нагрузку с вашего сервера - поскольку мы будем обрабатывать потоки комментариев вместо WordPress.

Результаты тестов не лгут - при 100 комментариях к посту на сервере без нагрузки стандартная установка WordPress может обслуживать 86 запросов в секунду, прежде чем выйти из строя.

С включенным FastComments это число увеличивается до **более 400**.
