### Что нового

Теперь у FastComments есть компоненты для React Native, которые равны по функционалу нашему виджету для VanillaJS.

Мы решили назвать библиотеку `fastcomments-react-native`. Вы можете найти репозиторий на GitHub с исходным кодом <a href="https://github.com/FastComments/fastcomments-react-native" target="_blank">здесь</a>.

Репозиторий также содержит примеры использования виджета.

Он доступен на NPM, который вы можете найти <a href="https://www.npmjs.com/package/fastcomments-react-native" target="_blank">здесь</a>.

### Как это работает

Новый компонент работает так, что компонент React является оберткой вокруг существующего виджета FastComments.
 
Это означает, что если мы добавим функции к нашим существующим компонентам или исправим ошибки, вы сможете сразу же воспользоваться этим!

### Время разработки

Виджеты для React Native можно добавить в ваше приложение всего за несколько минут с минимальными затратами времени на разработку.

### SSO

Все библиотеки React Native поддерживают SSO из коробки, а процесс интеграции SSO такой же, как и для компонентов Vanilla JS. Существующие примеры кода на сервере также применимы.

### Пользовательский стиль?

Компоненты React Native могут быть стилизованы с помощью CSS через интерфейс настройки виджета, как и веб-виджеты.
