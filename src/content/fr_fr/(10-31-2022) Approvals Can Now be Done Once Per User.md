### Quoi de Neuf

Auparavant, si l'approbation manuelle était activée, chaque commentaire devait être modéré. Maintenant, si ***Seule l'approbation automatique pour les utilisateurs approuvés*** est activée,
un commentaire d'utilisateur n'a pas besoin d'approbation s'il a été approuvé auparavant.

Cela sert de sorte de porte d'entrée dans une communauté, ce qui peut être souhaitable pour les communautés qui veulent avoir des approbations mais ne veulent pas
approuver chaque commentaire.

### Comment l'Obtenir

Vous pouvez activer cette fonctionnalité via [Paramètres de Modération des Commentaires](https://fastcomments.com/auth/my-account/moderate-comments/settings), accessibles via
`Modérer les Commentaires -> Paramètres`.

Sous `Approbation des Commentaires`, sélectionnez `Approuver les Commentaires Automatiquement` puis `Seule l'approbation automatique pour les utilisateurs approuvés`.

### Maintenir la Fonctionnalité Existante

Si vous approuvez manuellement chaque commentaire aujourd'hui, et que vous souhaitez continuer à le faire, il vous suffit de ne rien faire. Nous n'activerons pas cette fonctionnalité automatiquement
ou par défaut.

### En Conclusion

Nous savons que cette fonctionnalité a été longtemps attendue par certains. Comme pour toutes les principales versions, nous sommes heureux d'avoir pu prendre le temps d'optimiser, de tester et de publier correctement cette fonctionnalité.

Santé !
