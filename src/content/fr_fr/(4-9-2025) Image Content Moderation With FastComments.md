# [postlink]Modération du Contenu d'Image Avec FastComments[/postlink]

{{#unless isPost}}
Gardez votre section de commentaires sécurisée avec cette mise à jour.
{{/unless}}

{{#isPost}}
FastComments prend maintenant en charge le blocage des images en fonction de leur contenu.

Les images téléchargées dans les sections de commentaires peuvent maintenant être automatiquement filtrées en fonction de différents niveaux de contenu inapproprié. Cela vous donne plus de contrôle sur ce qui apparaît dans les discussions de votre communauté.

Vous pouvez choisir parmi quatre niveaux de modération différents :

- **Désactivé** : Pas de modération (par défaut)
- **Bas** : Bloque le contenu sexuel explicite et la violence
- **Moyen** : Bloque également le contenu de haine, de harcèlement et d'automutilation  
- **Élevé** : Bloque en outre le contenu suggérant légèrement

Cette fonctionnalité fonctionne pour toutes les images nouvellement téléchargées dans toutes les sections de commentaires de votre site, offrant une modération cohérente sans nécessiter de révision manuelle.

#### Documentation

Vous pouvez trouver la documentation ici : [https://docs.fastcomments.com/guide-customizations-and-configuration.html#image-content-moderation](https://docs.fastcomments.com/guide-customizations-and-configuration.html#image-content-moderation).

Bonne Modération des Commentaires !
{{/isPost}}
