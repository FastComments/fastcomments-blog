# [postlink]Nouvelle documentation lancée[/postlink]

{{#unless isPost}}
Nous avons lancé un site juste pour notre documentation. 
{{/unless}}

{{#isPost}}

## Quoi de neuf

Jusqu'à présent, notre blog servait d'une double fonction en tant que documentation et mises à jour des actualités.

Fin de cette époque !

## Le site de documentation

Vous pouvez trouver notre nouveau site de documentation à [docs.fastcomments.com](https://docs.fastcomments.com).

## L'avenir de ce blog

Nous continuerons à publier des mises à jour et de la documentation sur ce blog, cependant il n'aura pas la même profondeur que la documentation hébergée sur [docs.fastcomments.com](https://docs.fastcomments.com).

## Retours

Si vous découvrez des lacunes dans notre documentation, faites-le nous savoir en commentant cette page.

## Contribuer

Le dépôt de documentation est disponible sur GitHub, consultez-le ici : https://github.com/FastComments/fastcomments-docs

Bien que nous accueillions les contributions, nous apprécierions que vous puissiez d'abord soumettre un problème pour éviter que quiconque ne duplique votre travail.

## En conclusion

Nous avons mis des mois d'efforts dans le nouveau site de documentation - le cadre lui-même et le contenu - et nous espérons que 
la communauté le trouvera utile.
