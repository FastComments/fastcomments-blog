#### Pour les développeurs
Installer FastComments sur une page web nécessite seulement d'insérer un petit extrait de code. Cela signifie que cela fonctionne sur des pages statiques sans tracas - y compris l'hébergement gratuit de Pages de GitHub.

compte de test.
