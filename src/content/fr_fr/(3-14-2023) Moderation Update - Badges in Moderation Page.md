# [postlink]Mise à jour de Modération - Badges dans la page de Modération[/postlink]

{{#unless isPost}}
Les modérateurs peuvent désormais attribuer ou retirer des badges aux utilisateurs directement depuis la page de modération des commentaires, sans avoir à se rendre dans le visualiseur de fils de commentaires.
{{/unless}}

{{#isPost}}

### Quoi de Neuf

Auparavant, lorsque les modérateurs souhaitaient attribuer des badges, ils devaient se rendre dans le Visualiseur de Fils de Commentaires, ou attribuer les badges directement depuis le widget de commentaire installé sur leur site ou application.

Cependant, de nombreux modérateurs préfèrent simplement passer leur temps sur la page Modérer les Commentaires.

Pour faciliter les choses pour tout le monde, les badges des utilisateurs sont désormais affichés sur chaque commentaire dans la page Modérer les Commentaires.

De plus, il existe désormais des options dans le menu pour attribuer et retirer des badges pour un utilisateur, directement dans le menu de chaque commentaire.

Ces actions sont en direct, tout comme dans le widget de commentaire ou le visualiseur de fils, et tous les commentaires de l'utilisateur dans la liste de modération seront mis à jour lorsque des badges sont ajoutés ou retirés.

### Nombre Maximum de Badges Affichés

Jusqu'à `cinq` badges sont affichés. Après cela, un indicateur indique que l'utilisateur a plus de cinq badges.

### Comment l'Obtenir

Cette fonctionnalité a été déployée pour tous les clients !

### En Conclusion

Comme pour toutes les majeures versions, nous sommes ravis d'avoir pu prendre le temps d'optimiser, tester et bien publier cette fonctionnalité. Faites-nous savoir ci-dessous si vous découvrez des problèmes.

Santé !

{{/isPost}}
