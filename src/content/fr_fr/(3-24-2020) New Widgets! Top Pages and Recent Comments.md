# [postlink]Nouveaux Widgets ! Pages Principales et Commentaires Récents[/postlink]

Aujourd'hui, nous introduisons deux nouveaux types de widgets, portant le total de widgets pris en charge à trois : Compte de Commentaires, Commentaires Récents et Pages Principales.

{{#isPost}}

Si vous êtes connecté, vous pouvez voir la liste <a href="https://fastcomments.com/auth/my-account/widgets" target="_blank">ici</a>, sinon allez dans Mon Compte -> Obtenir le Code du Widget -> Autres Widgets.

Gardez à l'esprit que tous nos widgets ont un style très minimaliste - ils sont conçus pour être rapides et faciles à adapter à vos besoins.

Le widget Commentaires Récents affichera les dix meilleurs commentaires approuvés, triés par date.

Le widget Pages Principales affichera les 30 meilleures pages triées par nombre de commentaires.

Les valeurs pour ces widgets sont prédéterminées avant que vous ne chargiez la page, **et** mises en cache pour garantir la meilleure expérience pour vos utilisateurs. Pour la documentation concernant les widgets, allez dans la liste des widgets et sélectionnez-en un. À partir de là, vous verrez le code que vous pouvez ajouter à votre site ainsi que toute configuration que le widget accepte.

Si vous trouvez que ces widgets ne fournissent pas les informations dont vous avez besoin, ou si vous souhaitez de l'aide pour les styliser, n'hésitez pas à <a href="https://fastcomments.com/auth/my-account/help" target="_blank">nous contacter</a>.
