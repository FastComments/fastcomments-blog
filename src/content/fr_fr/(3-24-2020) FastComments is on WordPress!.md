# [postlink]FastComments est sur WordPress![/postlink]

Vous envisagez de passer d'un autre plugin de commentaires WordPress ? Essayez FastComments !

{{#isPost}}

Nous sommes fiers d'annoncer qu'après beaucoup de travail, nous avons lancé notre plugin WordPress. Vous pouvez le trouver ici :
<a href="https://wordpress.org/plugins/fastcomments/" target="_blank">https://wordpress.org/plugins/fastcomments/</a>.

Nous avons constaté que FastComments fonctionne beaucoup mieux que les commentaires WordPress par défaut, surtout pour les fils de commentaires plus longs.

Cela accélérera votre site WordPress en soulageant votre serveur - puisque nous renderons les fils de commentaires au lieu de WordPress.

Les benchmarks ne mentent pas - avec 100 commentaires sur un post fonctionnant sur un serveur sans charge, une installation WordPress classique peut traiter 86 demandes par seconde avant de s'effondrer.

Avec FastComments activé, ce nombre passe à **plus de 400**.

    src="images/fc-wordpress-stats.png"

Nous espérons que les utilisateurs trouveront le plugin facile à utiliser, à personnaliser et qu'il fonctionne tout simplement. Il devrait également synchroniser vos commentaires WordPress sans effort dans les deux sens sans créer de charge supplémentaire sur vos serveurs.
