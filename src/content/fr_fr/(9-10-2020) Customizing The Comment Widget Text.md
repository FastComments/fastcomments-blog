# [postlink]Personnaliser le texte du widget[/postlink]

{{#unless isPost}}
La personnalisation du texte du widget est une question fréquente de support. Maintenant, c'est en libre-service.
{{/unless}}

{{#isPost}}

#### Quoi de neuf

Auparavant, la seule façon de personnaliser le texte dans le widget de commentaire était soit d'apporter des modifications au code lors de l'intégration du widget pour passer les traductions, soit de soumettre un ticket de support client.

#### Comment ça fonctionne

Comme pour d'autres personnalisations de widget, cela est maintenant géré sur la page des <a href="https://fastcomments.com/auth/my-account/customize-widget" target="_blank">Personnalisations</a>.

Entrez le domaine pour lequel vous souhaitez personnaliser le texte, et soit une URL spécifique, soit * pour toutes les pages.

Vers le milieu de la page, cliquez sur "Afficher le texte personnalisé", puis sur "Sélectionner la substitution de traduction".

Dans le menu déroulant, vous pouvez rechercher des traductions par le texte anglais par défaut dans le widget de commentaire pour trouver le texte que vous souhaitez remplacer.

Au fur et à mesure que vous apportez des modifications, elles apparaîtront dans l'aperçu en bas de la page (comme mis en surbrillance).

#### Quelles sont les modifications pour les personnalisations de traduction existantes

Si vous avez déjà personnalisé votre texte dans le widget de commentaire avec l'une des méthodes précédentes, rien ne devrait changer - vos substitutions devraient rester telles quelles.

#### En conclusion

Nous espérons qu'en fournissant plus d'outils en libre-service comme celui-ci, la configuration de FastComments sera plus facile pour tous les types de clients que nous avons. Faites-nous savoir dans la section des commentaires ci-dessous comment cela a fonctionné pour vous.

Cheers!
