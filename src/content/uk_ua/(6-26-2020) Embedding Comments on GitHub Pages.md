# [postlink]Вбудовування коментарів на GitHub Pages[/postlink]

{{#unless isPost}}
Хотіли б ви додати коментарі на статичний сайт, наприклад, на GitHub Pages?
{{/unless}}

{{#isPost}}
#### Для Розробників
Встановлення FastComments на веб-сторінці вимагає лише вставлення невеликої частини коду. Це означає, що він працює на статичних сторінках без жодних ускладнень - включаючи безкоштовний хостинг GitHub Pages.

#### Демонстрація

Ми налаштували приклад сторінки GitHub <a href="https://fastcomments.github.io/fastcomments-demo.github.io/" target="_blank">тут</a>.

<a href="https://github.com/fastcomments/fastcomments-demo.github.io/blob/master/index.html#L29" target="_blank">Ось</a> посилання на місце, де ми додали код для нашого
тестового облікового запису.

#### Налаштування

Вам потрібно буде додати ваш домен GitHub як дійсний домен у вашому обліковому записі.
