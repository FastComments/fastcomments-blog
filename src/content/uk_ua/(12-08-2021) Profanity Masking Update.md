# [postlink]Оновлення маскування ненормативної лексики[/postlink]

{{#unless isPost}}
FastComments виявлення та маскування ненормативної лексики отримало оновлення
{{/unless}}

{{#isPost}}

### Що нового

Раніше FastComments дозволяв визначити лише один глобальний чорний список слів для вашого акаунту.

Цей список слів для маскування або для використання виявлення спаму за бажанням не можна було визначити для кожної мови. Він також міг містити лише
список слів, а не фраз.

Тепер було внесено такі оновлення:

1. Чорний список можна визначити для кожної локалі (мова + регіон).
2. Крім слів, тепер можна маскувати й фрази або використовувати їх для виявлення спаму.
3. Оновлення конфігурації виявлення ненормативної лексики тепер *в реальному часі* та *негайно*. Раніше змінам могло знадобитися до п'яти хвилин, щоб вступити в силу.

### Як це використовувати

Посилання на чорний список слів не змінилося й залишилося на [/auth/my-account/moderate-comments/blacklist](https://fastcomments.com/auth/my-account/moderate-comments/blacklist) і
все ще доступне з `Moderate Comments -> Moderation Settings -> Edit Word Blacklist`.

На цій сторінці ви тепер побачите випадаючий список мов для налаштування чорного списку за локалями.

Після вибору локалі ви можете визначити слова або фрази для маскування.

### Обмеження

Можна визначити до 5,000 слів або фраз для кожної локалі. Якщо вам потрібно підвищити цей ліміт, зверніться до служби підтримки.

### Це в реальному часі!

Будь-які зміни, які ви вносите до списку слів або фраз для маскування, тепер негайно застосовуються до системи.

### Хто це отримує

Усі поточні та нові клієнти FastComments, на всіх тарифах, тепер мають доступ до нових функцій чорного списку *слів* і *фраз*.

### На завершення

Як і в решті FastComments, ми сподіваємось, що ви знайдете цю функцію швидкою та легкою у використанні.

Успіхів!
