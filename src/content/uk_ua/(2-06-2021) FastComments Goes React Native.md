### <i class="circle">!</i> Ця стаття містить технічну термінологію

### Що нового

FastComments тепер має компоненти React Native, які мають рівень функціональності на паралелі з нашим віджетом VanillaJS.

Ми вирішили назвати бібліотеку `fastcomments-react-native`. Ви можете знайти репозиторій GitHub з вихідним кодом <a href="https://github.com/FastComments/fastcomments-react-native" target="_blank">тут</a>.

Репозиторій також містить приклади використання віджету.

Він доступний на NPM, який ви можете знайти <a href="https://www.npmjs.com/package/fastcomments-react-native" target="_blank">тут</a>.

### Як це працює

Новий компонент працює таким чином: компонент React є обгорткою навколо існуючого віджету FastComments.

Це означає, що якщо ми додамо функції до наших існуючих компонентів або виправимо помилки, ви все ще отримаєте переваги відразу!

### Час розробки

Віджети React Native можна додати до вашого додатку всього за кілька хвилин з мінімальними витратами часу на розробку.

### SSO

Усі бібліотеки React Native підтримують SSO з коробки, і процес інтеграції SSO такий же, як і для компонентів Vanilla JS. Існуючі
приклади коду для бекенду також застосовуються.

### Кастомізація стилю?

Компоненти React Native можна стилізувати за допомогою CSS через UI налаштування віджетів, так само як і веб-віджети.
