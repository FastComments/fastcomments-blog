# [postlink]FastComments тепер має MCP сервер[/postlink]

{{#unless isPost}}
Тепер ви можете націлити Claude Code, Claude Desktop, Cursor або будь-який інший клієнт Model Context Protocol на FastComments і безпосередньо викликати наше API.
{{/unless}}

{{#isPost}}

### Що нового

FastComments тепер постачає хостингований [Model Context Protocol](https://modelcontextprotocol.io/) (MCP) сервер. MCP - це відкритий стандарт, який використовують AI-асистенти для виклику інструментів на стороні сервера. Націліть свій клієнт на один URL і він може перерахувати ваші коментарі, модерувати їх, керувати користувачами SSO, налаштовувати віджети та виконувати все, що робить [API](https://docs.fastcomments.com/guide-api.html).

### Де це знаходиться

Відкрийте **Інтеграція, потім MCP Сервер** у вашій панелі керування FastComments або відвідайте його безпосередньо за адресою
[fastcomments.com/auth/my-account/mcp-setup](https://fastcomments.com/auth/my-account/mcp-setup).
Виберіть, який API ключ ви хочете, щоб клієнт використовував, і сторінка згенерує заповнений URL плюс фрагменти для копіювання для файлів конфігурації Claude Code і Claude Desktop / Cursor.

### Claude Code

Однією командою для реєстрації:

    claude mcp add --transport http fastcomments 'https://fastcomments.com/mcp?tenantId=YOUR_TENANT_ID&API_KEY=YOUR_API_KEY'

Потім `/mcp` всередині сесії показує з’єднання і перераховує кожен інструмент, який сервер надає. Питайте речі на кшталт "покажи останні 20 позначених коментарів на /blog/launch", "заблокувати цього коментатора на всіх моїх сайтах", або "який наш показник спаму за останній тиждень". Claude просуває через правильний виклик інструменту.

### Claude Desktop та Cursor

Додайте цей блок до конфігурації MCP вашого клієнта (`claude_desktop_config.json` для Claude Desktop, `mcp.json` для Cursor):

```json
{
  "mcpServers": {
    "fastcomments": {
      "type": "http",
      "url": "https://fastcomments.com/mcp?tenantId=YOUR_TENANT_ID&API_KEY=YOUR_API_KEY"
    }
  }
}
```

Перезавантажте клієнт, і сервер з’явиться поряд із будь-якими іншими MCP серверами, які ви налаштували.

### Безпека

API ключ вбудований у URL, томуTreat URL як секрет. Не вставляйте його в публічні чати, скріншоти або коміти. Якщо ключ потрапить у відкритий доступ, змініть його на сторінці [API Keys](https://fastcomments.com/auth/my-account/api-secret) і знову скопіюйте фрагмент зі сторінки налаштування. Щоб обмежити те, що агент може робити, створіть окремий API ключ для кожного асистента та заблокуйте його для відповідного домену. Випадаюче меню на сторінці налаштування дозволяє вам змінити, який ключ використовують фрагменти за один клік.

### Чому ми це створили

Люди все більше використовують агентні інструменти для повсякденних завдань, взаємодіючи з SaaS-продуктами. Будь-хто, хто підключає сесію Claude або Cursor, щоб переглядати свою чергу модерування, або створює спеціального агента для виведення тем, гідних закріплення, тепер може повністю пропустити обгортковий шар.

### На завершення

Відкрийте [Інтеграція, потім MCP Сервер](https://fastcomments.com/auth/my-account/mcp-setup) у вашій панелі, скопіюйте фрагмент для вашого улюбленого MCP клієнта, і ви в ефірі. Повна документація знаходиться в [посібнику LLM Kit](https://docs.fastcomments.com/guide-llm-kit.html).

Як завжди, повідомте нам нижче, якщо у вас виникнуть будь-які питання.

З найкращими побажаннями!
