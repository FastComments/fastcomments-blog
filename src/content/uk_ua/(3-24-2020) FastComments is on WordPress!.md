# [postlink]FastComments тепер на WordPress![/postlink]

Розглядаєте можливість переходу з іншого плагіна коментарів для WordPress? Спробуйте FastComments!

{{#isPost}}

Ми раді оголосити, що після численних зусиль ми запустили наш плагін для WordPress. Ви можете знайти його тут:
<a href="https://wordpress.org/plugins/fastcomments/" target="_blank">https://wordpress.org/plugins/fastcomments/</a>.

Ми виявили, що FastComments працює значно краще, ніж стандартні коментарі WordPress, особливо для великих тем коментарів.

Це прискорить ваш сайт WordPress, зменшивши навантаження на ваш сервер - оскільки ми будемо виводити теми коментарів замість WordPress.

Тести не брешуть - при 100 коментарях на повідомленні, що працює на сервері без навантаження, стандартна установка WordPress може обробити 86 запитів на секунду, перш ніж впаде.

З увімкненим FastComments це число зростає до **понад 400**.
