### Što je novo

FastComments sada ima React Native komponente koje su u skladu s našim VanillaJS widgetom.

Odlučili smo nazvati biblioteku `fastcomments-react-native`. GitHub repozitorij sa izvorni kodom možete pronaći <a href="https://github.com/FastComments/fastcomments-react-native" target="_blank">ovdje</a>.

Repozitorij također sadrži primjer korištenja widgeta.

Dostupan je na NPM-u, što možete pronaći <a href="https://www.npmjs.com/package/fastcomments-react-native" target="_blank">ovdje</a>.

### Kako to funkcionira

Način na koji ova nova komponenta funkcionira jest da je React komponenta omotač oko postojećeg FastComments widgeta.
 
To znači da ako dodamo funkcije našim postojećim komponentama ili ispravimo greške, vi ćete odmah imati koristi!

### Vrijeme razvoja

React Native widgeti mogu se dodati u vašu aplikaciju u samo nekoliko minuta uz malo potrebnog vremena razvoja.

### SSO

Sve React Native biblioteke podržavaju SSO odmah, a proces integracije SSO je isti kao i za Vanilla JS komponente. Postojeći
primjeri backend koda vrijede i ovdje.

### Prilagođeno oblikovanje?

React Native komponente mogu se oblikovati pomoću CSS-a putem Widget Customization UI, isto kao i web-based widgeti.
