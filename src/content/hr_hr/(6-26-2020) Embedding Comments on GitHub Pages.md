#### Za programere
Instalacija FastComments na web stranicu zahtijeva samo umetanje malog koda. To znači da radi na statičnim stranicama bez problema - uključujući GitHubovo besplatno Pages hostanje.

#### Demo

Postavili smo primjer GitHub stranice <a href="https://fastcomments.github.io/fastcomments-demo.github.io/" target="_blank">ovdje</a>.

<a href="https://github.com/fastcomments/fastcomments-demo.github.io/blob/master/index.html#L29" target="_blank">Ovdje</a> je link na kojem smo dodali kod za naš
testni račun.
