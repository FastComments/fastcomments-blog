# [postlink]Uključivanje komentara na GitHub Pages[/postlink]

{{#unless isPost}}
Želite da dodate komentare na statički sajt, poput GitHub Pages?
{{/unless}}

{{#isPost}}
#### Za developere
Instalacija FastComments na veb stranicu zahteva samo umetanje malog dela koda. To znači da funkcioniše na statičkim stranicama bez ikakvih problema - uključujući GitHub-ovo besplatno hosting na Pages.

#### Demostracija

Postavili smo primer GitHub stranice <a href="https://fastcomments.github.io/fastcomments-demo.github.io/" target="_blank">ovde</a>.

<a href="https://github.com/fastcomments/fastcomments-demo.github.io/blob/master/index.html#L29" target="_blank">Ovde</a> je link do mesta gde smo dodali deo koda za naš
testni nalog.
