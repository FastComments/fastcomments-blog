### Šta je novo

FastComments sada ima React Native komponente koje su na nivou funkcionalnosti sa našim VanillaJS vidgetom.

Odlučili smo da biblioteku nazovemo `fastcomments-react-native`. Izvorni kod možete pronaći u GitHub repozitorijumu <a href="https://github.com/FastComments/fastcomments-react-native" target="_blank">ovde</a>.

Repozitorijum takođe sadrži primer korišćenja vidgeta.

Dostupan je na NPM-u, koji možete pronaći <a href="https://www.npmjs.com/package/fastcomments-react-native" target="_blank">ovde</a>.

### Kako radi

Način na koji ova nova komponenta radi je da je React komponenta omotac oko postojećeg FastComments vidgeta.

To znači da ako dodamo funkcije našim postojećim komponentama ili ispravimo greške, vi ćete odmah imati koristi!

### Vreme razvoja

React Native vidgeti mogu biti dodati vašoj aplikaciji za samo nekoliko minuta uz malo potrebnog vremena za razvoj.

### SSO

Sve React Native biblioteke podržavaju SSO odmah, a proces integracije SSO je isti kao i za Vanilla JS komponente. Već postojeći
primeri koda na backendu se takođe primenjuju.

### Prilagođeni stil?

React Native komponente mogu biti stilizovane pomoću CSS-a preko Widget Customization UI, kao i web-based vidgeti.
