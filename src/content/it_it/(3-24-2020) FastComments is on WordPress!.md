# [postlink]FastComments è su WordPress![/postlink]

Stai pensando di passare da un altro plugin per commenti WordPress? Prova FastComments!

{{#isPost}}

Siamo orgogliosi di annunciare, dopo molti sforzi, che abbiamo lanciato il nostro plugin per WordPress. Puoi trovarlo qui:
<a href="https://wordpress.org/plugins/fastcomments/" target="_blank">https://wordpress.org/plugins/fastcomments/</a>.

Abbiamo scoperto che FastComments offre prestazioni significativamente migliori rispetto ai commenti predefiniti di WordPress, specialmente per thread di commenti più ampi.

Accelererà il tuo sito WordPress alleggerendo il carico del tuo server - poiché renderizzeremo i thread di commento invece di WordPress.

I benchmark non mentono - con 100 commenti su un post eseguito su un server senza carico, un'installazione vanilla di WordPress può gestire 86 richieste al secondo prima di crollare.

Con FastComments abilitato, quel numero sale a **oltre 400**.

Speriamo che le persone trovino il plugin facile da usare, personalizzare e, in generale, "funzioni semplicemente". Dovrebbe sincronizzare senza soluzione di continuità i tuoi commenti di WordPress avanti e indietro senza mettere sotto stress i tuoi server.
