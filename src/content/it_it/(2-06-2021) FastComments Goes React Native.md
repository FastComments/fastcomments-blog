### <i class="circle">!</i> Questo Articolo Contiene Gergo Tecnico

### Novità

FastComments ora ha componenti React Native che sono alla pari in termini di funzionalità con il nostro widget VanillaJS.

Abbiamo deciso di chiamare la libreria `fastcomments-react-native`. Puoi trovare il repository GitHub con il codice sorgente <a href="https://github.com/FastComments/fastcomments-react-native" target="_blank">qui</a>.

Il repository contiene anche esempi di utilizzo del widget.

È disponibile su NPM, che puoi trovare <a href="https://www.npmjs.com/package/fastcomments-react-native" target="_blank">qui</a>.

### Come Funziona

Il funzionamento di questo nuovo componente è che il componente React è un involucro attorno al widget FastComments esistente.

Ciò significa che se aggiungiamo funzionalità ai nostri componenti esistenti o correggiamo bug, beneficerai immediatamente!

### Tempo di Sviluppo

I widget React Native possono essere aggiunti alla tua applicazione in pochi minuti con un tempo di sviluppo ridotto.

### SSO

Tutte le librerie React Native supportano SSO "out of the box", e il processo di integrazione SSO è lo stesso dei componenti Vanilla JS. Gli esempi di codice backend esistenti si applicano, così come.

### Stile Personalizzato?

I componenti React Native possono essere stilizzati utilizzando CSS tramite l'interfaccia di personalizzazione del widget, allo stesso modo dei widget basati sul web.

### Alternative

Potresti costruire la tua interfaccia

### In Conclusione

Rilasciando la libreria `fastcomments-react-native` e le librerie future, speriamo di rendere più facile per gli sviluppatori adottare FastComments mentre utilizzano metodologie di sviluppo moderne.

Salute!
