# [postlink]FastComments 現已上線於 WordPress！[/postlink]

考慮從其他 WordPress 評論插件轉換過來嗎？試試 FastComments 吧！

{{#isPost}}

我們自豪地宣布，經過大量工作，我們已經推出了我們的 WordPress 插件。您可以在這裡找到它：
<a href="https://wordpress.org/plugins/fastcomments/" target="_blank">https://wordpress.org/plugins/fastcomments/</a>。

我們發現 FastComments 的性能優於預設的 WordPress 評論，特別是在大型評論串中。

它將通過減輕伺服器的負擔，加快您的 WordPress 網站速度——因為我們將運行評論串，而不是 WordPress。

基準測試不會說謊——在一個沒有負載的伺服器上運行的帖子上，當評論數達到 100 條時，一個純粹的 WordPress 安裝每秒可以處理 86 個請求，然後就會崩潰。

啟用 FastComments 後，這個數字提升到了 **超過 400**。

我們希望人們發現這款插件易於使用，易於自定義，總的來說“運行良好”。它應該能無縫地將您的 WordPress 評論雙向同步，而不會對您的伺服器造成壓力。
