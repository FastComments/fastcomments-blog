# [postlink]在 GitHub Pages 上嵌入評論[/postlink]

{{#unless isPost}}
想要在靜態網站上添加評論，比如 GitHub Pages 嗎？
{{/unless}}

{{#isPost}}
#### 對開發者的建議
在網頁上安裝 FastComments 只需插入一小段代碼。這意味著它可以在靜態頁面上運作，無需任何麻煩 - 包括 GitHub 的免費 Pages 主機。

#### 示範

我們在這裡設置了一個示例 GitHub 頁面 <a href="https://fastcomments.github.io/fastcomments-demo.github.io/" target="_blank">這裡</a>。

<a href="https://github.com/fastcomments/fastcomments-demo.github.io/blob/master/index.html#L29" target="_blank">這裡</a> 是我們為測試帳戶添加代碼片段的位置的鏈接。

#### 設置

您需要將您的 GitHub 域名添加為您帳戶中的有效域。
