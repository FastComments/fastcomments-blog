# [postlink]FastComments 預設頭像使用 Gravatar[/postlink]

### 什麼是頭像？

頭像是與每個帳戶相關聯的個人圖標，顯示在每個評論旁邊。

### 什麼是 Gravatar？

<a href="https://en.gravatar.com/" target="_blank">Gravatar</a> 自我宣傳為「全球認可的頭像」。
這是一項服務，您可以為給定的電子郵件定義一個頭像。使用該電子郵件註冊的任何產品都可以使用該頭像。這樣可以為註冊 FastComments 的人省去一個步驟，因為他們的頭像會自動設置。

### 那麼，這有什麼變化？

FastComments 現在將從 Gravatar 中提取預設頭像。您或用戶不必做任何事情就可以獲得此功能。

如果您現有的用戶已經定義了自己的頭像 - 別擔心 - 我們不會覆蓋他們的自定義頭像。
此外，用戶仍然可以在 FastComments 中覆蓋 Gravatar 頭像。
