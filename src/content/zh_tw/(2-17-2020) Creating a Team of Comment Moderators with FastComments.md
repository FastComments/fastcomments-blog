# [postlink]使用 FastComments 創建評論版主團隊[/postlink]

### 問題

您擁有大量內容和評論。這是好事 - 您的用戶群如此活躍以至於與您互動。但現在您感到不堪重負，需要擴展您的運營。

### FastComments 版主管理

FastComments 的版主管理工具提供了一個高效的環境來審查評論，標記為垃圾郵件，或者（如果必要的話）刪除它們。

儀表板也是響應式的，因此您可以隨時隨地審查和管理評論！

但對於您網站的評論管理訪問並不僅限於您的帳戶。其他人可以創建自己的 FastComments 帳戶，並通過管理儀表板受邀加入您的網站，[或通過 API 批量邀請](https://docs.fastcomments.com/guide-api.html#moderator-structure)。他們甚至不需要事先了解 FastComments 或創建帳戶。我們將告訴您如何發送的邀請鏈接將指導他們創建帳戶。

只需登錄您的帳戶並將鼠標懸停在“管理評論”上。您將看到“管理版主”的選項。

<img 
    src="images/fc-add-moderator-menu.png"

如果您尚未添加任何版主，這個按鈕將帶您進入向導，指導您添加版主。

您只需提供名字和電子郵件。他們將自動收到一個邀請鏈接，該鏈接還將指導他們完成帳戶創建。

選擇性地，您還可以將版主和評論放入不同的組。 [在這裡了解更多關於版主組和其他管理功能的資訊。](https://docs.fastcomments.com/guide-moderation.html)

如果您擁有一支版主團隊，您可以在同一地方查看他們的進度 - 評論版主列表提供每位版主的統計信息，包括他們標記為已審查、垃圾郵件和已刪除的評論數量。

就這樣。在您發送那個邀請鏈接後，他們便可以創建他們的帳戶，並將擁有完全的評論管理權限。他們只會看到簡化版本的管理評論頁面 - 版主無法更改設置。

祝評論愉快！
