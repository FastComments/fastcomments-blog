# [postlink]評論管理的鍵盤快捷鍵[/postlink]

{{#unless isPost}}
進階用戶現在可以完全透過鍵盤管理評論，配備了一整套的導航、操作等快捷鍵。
{{/unless}}

{{#isPost}}

### 新功能

評論管理頁面現在支持全面的鍵盤快捷鍵。可以在評論之間導航，批准、刪除、禁言、標記為垃圾郵件、切換上下文、查看日誌等等——所有這些操作都無需觸碰滑鼠。

這對於需要管理大量評論的團隊來說非常重要。每一個之前需要移動滑鼠並點擊按鈕的操作現在只需按一下鍵。

### 示例流程

假設你有一個需要審核的評論隊列。按下 **j** 以聚焦第一條評論。按 **c** 以擴展其上下文並查看周圍的線索。看起來不錯，所以再按 **c** 把上下文折疊起來，然後按 **a** 來批准它。按 **j** 移動到下一條評論，再重複此操作。如果有什麼看起來不對，按 **s** 標記為垃圾郵件或 **d** 刪除它。你可以在整個評論頁面上工作而無需摸到滑鼠。

### 導航

按 **j** 將焦點移至下一條評論，按 **k** 移至前一條。聚焦的評論將用藍色輪廓突出顯示，以區分選定（已勾選）的評論。

按 **x** 或 **Enter** 來切換聚焦評論的選擇狀態。按住 **Shift** 後再按 **j** 或 **k** 可以在一個動作中選擇當前評論並移動焦點——對於快速選擇一系列評論非常有用。

### 操作

當評論已聚焦時，單鍵快捷鍵觸發與工具欄按鈕相同的操作：

- **a** - 切換批准 / 取消批准
- **s** - 切換垃圾郵件 / 非垃圾郵件
- **d** - 刪除
- **u** - 取消刪除
- **r** - 切換已審核 / 未審核
- **e** - 編輯
- **b** - 禁言 / 管理禁言
- **p** - 切換釘選 / 取消釘選
- **f** - 取消標記
- **i** - 查看所有來自 IP 的內容
- **t** - 設置信任因子
- **m** - 管理徽章
- **l** - 查看日誌
- **c** - 切換上下文

### 全局快捷鍵

幾個快捷鍵在評論是否聚焦的情況下都有效：

- **Ctrl+a** - 全選所有評論
- **Ctrl+z** - 撤銷最後一個操作
- **/** - 專注於搜索輸入
- **[** 和 **]** - 導航到上一頁或下一頁

### 幫助與指導

隨時按 **?** 來打開幫助對話框，列出所有可用的快捷鍵。

按 **g** 切換“指導者”模式，它會在聚焦評論的操作按鈕上覆蓋小徽章，顯示相應的快捷鍵。這在學習快捷鍵時非常方便。

在搜索框或任何其他輸入框中輸入文本時，所有快捷鍵都會被抑制，並且在對話框打開時也是如此。按 **Escape** 可以使聚焦輸入失去焦點並返回快捷鍵模式。
