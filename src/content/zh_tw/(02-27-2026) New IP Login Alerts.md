# [postlink]新的 IP 登入警報[/postlink]

{{#unless isPost}}
當從新的 IP 位址檢測到登入時，FastComments 現在會發送安全警報電子郵件，包括大致位置，以幫助用戶檢測未經授權的訪問。
{{/unless}}

{{#isPost}}

### 新內容

當我們檢測到從未見過的 IP 位址登入到您的帳戶時，FastComments 現在會自動發送安全警報電子郵件。該郵件包括：

- **登入的大致位置**（城市和國家）。
- **使用的 IP 位址**。
- **登入時間**。
- 如果登入不是您本人，則提供直接鏈接以**更改您的密碼**。

這適用於所有登入方式：基於密碼的登入、魔法鏈接以及雙重驗證流程。

### 隱私

我們不會儲存您的原始 IP 位址。我們僅儲存經過混淆的形式作為安全比較之用。大致位置是在登入時確定的，並不會被存儲。

**您的 IP 不會與第三方共享**以確定位置。我們擁有自己的服務，從一個（每週更新）版本的 MaxMind 數據庫中提取大致位置。

### 何時發送警報

當 **所有** 以下條件成立時會發送警報：

- 登入成功。
- 這不是用戶的第一次登入。
- 此帳戶的先前成功登入未見過該 IP 位址。

這意味著您不會在第一次登入時收到警報，並且不會對相同 IP 重複收到警報。

### 總結

這是保護您的帳戶安全邁出的一步。如果您收到不明的警報，我們建議您立即更改您的密碼。

如果您有任何反饋，請在下面告訴我們！

乾杯！
