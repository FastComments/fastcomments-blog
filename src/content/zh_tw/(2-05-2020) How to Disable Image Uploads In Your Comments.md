# [postlink]如何在您的評論中禁用圖片上傳[/postlink]

使用 FastComments，您可以禁用網站評論區的圖片上傳，無論是針對整個網站還是僅針對特定頁面。

{{#isPost}}
要做到這一點，請轉到儀表板中的 <a href="https://fastcomments.com/auth/my-account/customize-widget" target="_blank">自訂</a> 並添加自訂規則。

啟用此設置的樣子如下：

<img 
    src="images/fc-disable-image-uploads.png"
    alt="禁用圖片上傳"
    title="禁用圖片上傳"
    class='lozad' />

... 然後點擊保存。
在此頁面上，如果您的域名尚未自動填充，則必須定義您的域名。將 URL ID 保持為空，以便將該規則應用於整個網站。

此頁面已禁用圖片上傳，以作為範例。

{{/isPost}}
