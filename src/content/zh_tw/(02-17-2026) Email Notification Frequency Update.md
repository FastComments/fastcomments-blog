# [postlink]電子郵件通知頻率更新[/postlink]

{{#unless isPost}}
使用者現在可以控制他們接收新留言和回覆的電子郵件通知頻率，可以選擇每分鐘、每小時匯總或每日匯總，並且針對回覆和管理員通知有單獨的設定，此外還可以對每個訂閱進行覆蓋設置。
{{/unless}}

{{#isPost}}

### 新增內容

我們新增了 **回覆通知頻率** 和 **管理員通知頻率** 設定，用以控制您分別接收回覆和新留言的電子郵件通知頻率。每種設定有三個選項：

- **每分鐘** - 新留言到達後立即收到電子郵件（每分鐘檢查一次）。
- **每小時匯總** - 每小時收到一次新留言的匯總摘要。
- **每日匯總** - 每日收到一次新留言的匯總摘要。

回覆通知頻率對所有使用者開放，預設為 **每分鐘**。管理員通知頻率對網站管理員開放，預設為 **每小時匯總**。請注意，@提及通知始終會立即發送，與這些設定無關。

您也可以在訂閱表中針對每個訂閱覆蓋通知頻率，以便細緻控制個別頁面。

### 如何配置

1. 前往您的 [通知設定](https://fastcomments.com/auth/my-account/edit-notifications)。
2. 使用 **回覆通知頻率** 和 **管理員通知頻率** 下拉選單設置您的偏好頻率。
3. 可選擇性地在訂閱表中覆蓋單個訂閱的頻率。
4. 點擊 **儲存變更**。

### API 支援

使用者物件上的 `notificationFrequency` 欄位控制回覆通知頻率，而 `adminNotificationFrequency` 欄位控制管理員通知頻率。每個訂閱的覆蓋可以通過訂閱的 `notificationFrequency` 欄位設置。詳情請參見 [API 文檔](https://docs.fastcomments.com/guide-api.html)。

### 總結

這使得使用者能夠控制他們的收件箱，而無需完全取消訂閱頁面。
