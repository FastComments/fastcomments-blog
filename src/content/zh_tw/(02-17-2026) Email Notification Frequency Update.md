# [postlink]電子郵件通知頻率更新[/postlink]

{{#unless isPost}}
用戶現在可以控制他們收到新評論和回覆的電子郵件通知頻率 — 每分鐘、每小時摘要或每日摘要 — 透過單一的共用設置，並可以對每個訂閱進行覆寫。
{{/unless}}

{{#isPost}}

### 什麼是新的

我們添加了一個 **新評論通知頻率** 設置，控制您收到回覆通知和租戶（新評論）通知的電子郵件頻率。三個選項為：

- **每分鐘** - 在新的評論到達時立即收到電子郵件（每分鐘檢查一次）。
- **每小時摘要** - 每小時收到一次新評論的批量摘要。
- **每日摘要** - 每天收到一次新評論的批量摘要。

此設置對於租戶管理員和評論者均適用，並適用於所有評論通知電子郵件。請注意，@提及通知始終會立即發送，而不受此設置的影響。

您還可以在訂閱表中按每個訂閱覆寫頻率，以實現對個別頁面的精細控制。

### 如何配置

1. 前往您的 [通知設置](https://fastcomments.com/auth/my-account/edit-notifications)。
2. 使用 **新評論通知頻率** 下拉菜單設置您首選的頻率。
3. 可選地，在訂閱表中覆寫個別訂閱的頻率。
4. 點擊 **保存更改**。

默認值為 **每分鐘**，這與之前的行為相符。

### API 支援

通知頻率也可以透過 API 獲得。用戶級設置是 <div class="code">notificationFrequency</div> 欄位，並且可以通過訂閱的 <div class="code">notificationFrequency</div> 欄位設置每個訂閱的覆寫。詳情請參見 [API 文檔](https://docs.fastcomments.com/guide-api.html)。

### 總結

這使得用戶可以控制他們的收件箱，而不必完全取消訂閱頁面。現在一個設置涵蓋了回覆和租戶通知，使管理變得更簡單。
