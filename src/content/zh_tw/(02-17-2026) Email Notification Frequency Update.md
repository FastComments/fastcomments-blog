# [postlink]電子郵件通知頻率更新[/postlink]

{{#unless isPost}}
用戶現在可以控制他們每分鐘、每小時摘要或每日摘要接收新評論和回覆的電子郵件通知頻率，並可透過單一共享設置進行設置，還可以針對每個訂閱進行覆蓋。
{{/unless}}

{{#isPost}}

### 新功能

我們新增了一個 **新評論通知頻率** 設置，控制您收到回覆通知和租戶（新評論）通知的電子郵件頻率。三個選項為：

- **每分鐘** - 當新評論到達時，立即收到電子郵件（每分鐘檢查一次）。
- **每小時摘要** - 每小時接收一次新評論的批次摘要。
- **每日摘要** - 每日接收一次新評論的批次摘要。

此設置對於租戶管理員和評論者均可用，並適用於所有評論通知電子郵件。請注意，@提及通知始終會立即發送，無論此設置如何。

您還可以在訂閱表格中按訂閱覆蓋頻率，以便對個別頁面進行更精細的控制。

### 如何配置

1. 前往您的 [通知設置](https://fastcomments.com/auth/my-account/edit-notifications)。
2. 使用 **新評論通知頻率** 下拉選單設置您喜好的頻率。
3. 可選地，對訂閱表格中的個別訂閱覆蓋頻率。
4. 點擊 **保存變更**。

默認設置為 **每分鐘**，這與之前的行為相符。

### API 支持

通知頻率也可以通過 API 獲取。用戶級別設置為 `notificationFrequency` 欄位，且可以通過訂閱的 `notificationFrequency` 欄位設置每個訂閱的覆蓋。詳情請見 [API 文檔](https://docs.fastcomments.com/guide-api.html)。

### 總結

這讓用戶能夠控制他們的收件箱，而不必完全退訂頁面。現在一個設置涵蓋了回覆和租戶通知，使管理變得更加簡單。
