# [postlink]使用 FastComments 關閉評論串[/postlink]

有想要結束的討論嗎？這裡有選項！

{{#isPost}}

FastComments 的目標是促進討論並賦予用戶發聲的機會。然而，有時您可能會有不再相關的內容，或者想暫時禁用回覆，現在您可以做到這一點。

只需前往 <a href="https://fastcomments.com/auth/my-account/customize-widget" target="_blank">自定義頁面</a> 並勾選「防止新回覆」選項。

請注意，您應定義一個 URL，除非您希望在整個網站上禁用評論。
