### <i class="circle">!</i> 本文包含技術術語

### 新聞

FastComments 現在有與我們的 VanillaJS 小部件功能相當的 React Native 元件。

我們決定將這個庫命名為 `fastcomments-react-native`。你可以在 <a href="https://github.com/FastComments/fastcomments-react-native" target="_blank">這裡</a> 找到包含源代碼的 GitHub 倉庫。

該倉庫還包含小部件的示範用法。

它已在 NPM 上，可以在 <a href="https://www.npmjs.com/package/fastcomments-react-native" target="_blank">這裡</a> 找到。

### 工作原理

這個新元件的工作方式是，React 元件是現有 FastComments 小部件的包裝層。

這意味著如果我們對現有元件添加功能或修復錯誤，你會立即受益於這些變更！

### 開發時間

React Native 小部件可以在幾分鐘內輕鬆添加到你的應用程式中，所需的開發時間很少。

### SSO

所有的 React Native 庫都可以直接支援 SSO，並且 SSO 集成過程與 Vanilla JS 元件相同。現有的後端代碼示例也適用。

### 自定義樣式？

React Native 元件可以通過小部件自定義 UI 使用 CSS 進行樣式設計，與基於網頁的小部件相同。
